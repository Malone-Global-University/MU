06-Building Envelopes

This field manual provides practical, project-based instruction for constructing durable, weather-resistant residential building envelopes. Students learn how to install water-resistive barriers, flash openings, install windows and exterior doors, apply siding systems, and create a complete weatherproof exterior shell.

Overview

Building Envelopes develops the skills required to protect residential structures from water, air, and weather infiltration. Beginning with wall preparation and weather-resistive barriers, students progress through flashing systems, window and door installation, rainscreens, exterior cladding, sealing, repairs, and complete water management inspections before finishing with a full capstone weatherproofing project.

Learning Outcomes

  • Prepare framed walls for weather-resistant exterior construction.
  • Install house wrap, flashing, and water-resistive barrier systems correctly.
  • Install and weatherproof exterior windows and doors.
  • Construct rainscreen assemblies and install common residential siding systems.
  • Seal exterior joints, penetrations, vents, and utility openings against moisture intrusion.
  • Inspect, diagnose, and repair common building envelope failures.
  • Complete a fully weatherproof exterior shell on a residential practice structure.
