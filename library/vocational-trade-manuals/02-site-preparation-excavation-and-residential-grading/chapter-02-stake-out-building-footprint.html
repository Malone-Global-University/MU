Job Overview

This project qualifies you to transfer an approved building footprint from the site plan to the ground using control points, offset stakes, batter boards, and mason's string lines. The finished work product is a protected preliminary layout: corner references established from a controlling site reference, batter-board pairs set at a consistent working height at each corner, string lines run and tensioned between corresponding batter-board references, and a written record of preliminary side dimensions and control-point locations.

You will confirm utility-locate clearance, identify the controlling property line, baseline, benchmark, or survey control point, establish the first building reference corner and the primary baseline, mark approximate corner locations, install offset stakes and batter boards outside the future excavation area, run and tension mason's strings between batter-board references, verify string intersections with a plumb bob, mark required offset and footing lines, label every control point, and protect the finished layout.

This is the second project in this book. It begins only after the site-readiness decision issued in Project 01 authorizes proceeding, and it uses the existing-conditions record — property markers, hazards, access routes, and drainage notes — produced there. Every later chapter in this book, beginning with the precision squaring performed in Project 03, depends on the preliminary layout established here.

Skills You Will Demonstrate

  • SP-007 — Install offset stakes and corner stakes that establish the approximate building footprint from an approved site control.
  • SP-008 — Establish control points, including the controlling baseline, first reference corner, and primary building direction.
  • SP-009 — Install batter-board pairs at a consistent working height or documented elevation at each building corner.
  • SP-010 — Run and tension mason's string lines between corresponding batter-board references to represent the building lines.
  • SP-011 — Protect the layout by labeling control points and establishing physical protection around stakes, strings, and batter boards.
  • SP-012 — Verify the initial layout for visibility, stability, access clearance, and readiness for final squaring.

Where This Skill Fits

Project 01 produced the documented readiness decision and the existing-conditions record — property markers, hazards, access routes, and drainage notes — that controls where and how this project begins. This project applies that record for the first time to a physical layout: driving stakes, setting batter boards, and running strings that represent the actual building lines on the ground.

The preliminary layout produced here is deliberately approximate. Project 03 uses the 3-4-5 method and diagonal measurements to adjust these same string lines until they meet the project's square, dimension, and elevation tolerances. Do not treat this project's layout as final; it is the starting position that Project 03 refines.

Scope and Intended Use

This project covers:

  • reviewing the approved site plan, survey references, setbacks, orientation, dimensions, and benchmark information
  • confirming utility-locate clearance and permission before driving any stake
  • identifying the controlling property line, baseline, benchmark, or survey control point
  • establishing the first building reference corner and the primary baseline
  • installing offset stakes and batter-board pairs outside the future excavation and equipment-work area
  • running, tensioning, and verifying mason's string lines between batter-board references
  • labeling control points and protecting the finished preliminary layout

This project does not authorize the learner to:

  • drive any stake before utility-locate clearance and entry permission are confirmed for this project
  • disturb, relocate, or drive stakes over a property marker identified in the Project 01 boundary marker log
  • perform final squaring, diagonal verification, or benchmark transfer, which are addressed in Project 03
  • begin clearing, stripping, or excavation based on this project's preliminary layout alone; those operations require the verified layout from Project 03
  • proceed if the Project 01 site-readiness decision was stop-work or if a conditional-proceed item affecting this project's work area remains unresolved

Any control point, stake, or batter board that is disturbed, damaged, or found unreliable during this chapter must be corrected before layout work continues.

Trade Note

Every corner and offset in this chapter is measured from the controlling baseline and the first reference corner established in Step 4 — never from a previously placed stake that has not itself been checked against that baseline. Measuring stake-to-stake instead of baseline-to-stake is the most common source of an out-of-position footprint that Project 03 cannot fully correct.

Working Outward From One Controlling Reference

Establish the controlling reference and the first reference corner before measuring any other location. Every baseline, offset, and batter-board setback in this chapter is measured from that controlling reference and the established baseline, not from a previously placed stake that has not itself been checked against them. This preliminary layout is approximate; Project 03 adjusts it to final tolerance using the 3-4-5 method and diagonal measurements.

Measurement Cue — Measure From the Controlling Reference Every Time

Measure every baseline distance, corner offset, and batter-board setback from the controlling reference and the established first reference corner. Do not measure a new corner from a stake that was itself only approximately located. Compounding approximate measurements stake to stake can place the entire footprint outside its intended position before Project 03 squaring ever begins.

Critical Safety Cue — Utility-Locate Clearance Before Driving Stakes

Do not drive any stake, corner pin, or batter-board stake until current utility-locate clearance has been confirmed for the area. Striking a buried utility with a driven stake can cause serious injury, service interruption, or property damage. If a marked utility line lies within the planned stake or batter-board location, stop and obtain direction before proceeding at that location.

Field Safety Cue — Staking Area Awareness

Watch footing on uneven or previously disturbed ground while driving stakes and running string lines. Keep the hammer or stake driver's swing path clear of other workers. Where mason's string crosses a walking path, flag the line so it is visible and does not create a trip hazard.

Code Alert — Setback and Permit Verification

Confirm the approved setbacks, permitted building envelope, and any permit conditions affecting layout before establishing the baseline and first reference corner. Setback and permit requirements vary by jurisdiction; verify current local requirements rather than assuming those used on a prior site.

E. Preparation — Plan Review and Utility Clearance

  1. Review the approved site plan, survey references, setbacks, orientation, dimensions, and benchmark information. Confirm the approved footprint dimensions and any documented benchmark elevation before proceeding to Step 2.
  2. Confirm utility-locate clearance and permission before driving stakes. Obtain current utility-locate clearance for the staking area and confirm entry permission is still valid per the Project 01 site-readiness decision.

F. Controlling Reference and Corner Staking (Repeat for Each Corner)

  1. Identify the controlling property line, baseline, benchmark, or survey control point. Select the single reference that will control every measurement in this chapter and record it on the layout record sheet.
  2. Establish and mark the first building reference corner from the controlling site reference. Drive and label the corner stake for the first reference corner directly from the controlling reference.
  3. Establish the primary baseline in the required building direction. Mark the baseline direction from the first reference corner and record it on the layout record sheet.
  4. Measure and mark approximate locations for the remaining building corners. Measure each remaining corner from the baseline and the first reference corner, not from a previously placed stake that has not itself been checked against them, and drive and label each corner stake.
  5. Install offset stakes outside the future excavation and equipment-work area. For each corner, measure and record an offset distance and direction from the corner stake, and drive and label the offset stake at a location that will remain undisturbed during excavation.

G. Batter-Board Installation (Repeat for Each Corner)

  1. Install batter-board pairs at each corner far enough from excavation to remain undisturbed. Drive two batter-board stakes per corner, positioned so the rails will span behind the corner stake outside the excavation limit.
  2. Set batter-board rails at a consistent working height or documented elevation. Use the line level or the recorded benchmark elevation to set every rail at the same height across all corners, and fasten the rail to its stakes.
  3. Transfer baseline and corner references to durable marks on the batter boards. Mark or notch the top of each rail at the point directly above its corner stake, using the tape measure and plumb bob to confirm the transfer.

H. String Lines and Verification (Repeat for Each Building Line)

  1. Run mason's strings between corresponding batter-board references. Attach the string at the marked point on each batter board so it represents the intended building line.
  2. Adjust string tension so lines remain straight without pulling the batter boards out of position. Tension each string firmly and confirm the batter-board rails remain level and secure.
  3. Use plumb transfer to verify that string intersections correspond to the intended building corners. Hold the plumb bob at each string intersection and confirm it falls directly over the corner stake or offset reference; adjust the string attachment point on the batter board if it does not.
  4. Mark offsets, footing lines, wall lines, or excavation limits required by the approved plan. Record each required offset or limit on the layout record sheet as it is established.

I. Labeling, Protection, and Closeout

  1. Label each control point and batter board so the layout can be reconstructed. Use the same corner-naming convention on every stake, batter board, and the layout record sheet.
  2. Establish protection around stakes, strings, and batter boards. Flag or barricade the layout so it is visible to workers, delivery vehicles, and equipment operators.
  3. Record preliminary side dimensions and control-point locations. Measure and record each preliminary side length and every control-point location on the layout record sheet.
  4. Inspect the layout for visibility, stability, access clearance, and readiness for final squaring in Chapter 3. Confirm every stake and batter board is stable, every string is visible and tensioned, and access remains clear before ending the project.

Common Layout Inspection

Inspect the completed preliminary layout using the same checks regardless of footprint size or configuration.

  • Controlling-reference check: confirm the first reference corner and baseline were established directly from the identified controlling reference and are recorded on the layout record sheet.
  • Staking check: confirm every corner has a corner stake and at least one offset stake located outside the future excavation and equipment-work area.
  • Batter-board check: confirm every rail is set at the same consistent working height or documented elevation and is securely fastened to its stakes.
  • String check: confirm every string line is straight, tensioned without displacing its batter boards, and verified by plumb bob at every intersection.
  • Labeling and protection check: confirm every control point and batter board is labeled to match the layout record and that physical protection is established around the layout.

Complete the Layout Record Sheet in the Inspection Checklist section below before beginning Project 03.

Failure Points and Troubleshooting

A Corner Ends Up Noticeably Out of Position

Probable cause: the corner was measured from a previously placed stake instead of from the baseline and the first reference corner.
Confirmation: re-measure the affected corner directly from the first reference corner and the baseline and compare the result to the recorded preliminary dimension.
Correction: reposition the corner stake and its offset stake using the baseline and first reference corner, and update the layout record.
Reinspection: repeat the staking check for the corrected corner and confirm its batter boards and string lines are adjusted to match.

Batter-Board Rails Are Not at a Consistent Height

Probable cause: the line level was not used consistently across all corners, or a documented benchmark elevation was not transferred to every rail.
Confirmation: check each rail with the line level against the other rails, or re-measure each rail's elevation against the recorded benchmark value.
Correction: loosen and reset the fasteners on the affected rail and reposition it to match the consistent working height or documented elevation, then refasten.
Reinspection: repeat the batter-board check across all corners.

A String Line Pulled a Batter Board Out of Position

Probable cause: the string was over-tensioned, or the batter-board stakes were not driven deeply or firmly enough to resist the tension.
Confirmation: sight along the batter-board rail for movement or lean and check the stake depth and firmness in the ground.
Correction: reset and, if necessary, re-drive the batter-board stakes, then re-run the string at a tension that keeps it straight without displacing the batter board.
Reinspection: repeat the string check and the batter-board check for the affected corner.

A String Intersection Does Not Fall Over Its Corner Stake

Probable cause: the string was attached to the wrong mark on the batter board, or the corner stake itself was not established from the baseline and first reference corner.
Confirmation: re-check the corner stake's location against the baseline and first reference corner, then re-verify the string attachment points with the plumb bob.
Correction: if the corner stake is out of position, correct it first; then reattach the string at the correct mark on each batter board and re-verify with the plumb bob.
Reinspection: repeat the string-intersection verification for the corrected corner.

Control Points Cannot Be Identified After a Break in Work

Probable cause: stakes and batter boards were not labeled to match the layout record, or labels were placed where they were later obscured or worn away.
Confirmation: compare the physical stakes and batter boards against the layout record sheet and confirm each control point is still legibly labeled.
Correction: relabel every stake and batter board using the same corner-naming convention recorded on the layout record sheet.
Reinspection: repeat the labeling and protection check before continuing layout work.

A Stake or String Was Disturbed by Site Traffic

Probable cause: stakes, strings, or batter boards were not adequately flagged or protected in an area with worker, delivery, or equipment traffic.
Confirmation: inspect the disturbed control point against the layout record and determine whether its original position can be confirmed or must be re-established from the controlling reference.
Correction: re-establish the disturbed control point from the controlling reference and the first reference corner, and add or improve physical protection at that location.
Reinspection: repeat the relevant staking, batter-board, or string check for the corrected control point.

Cleanup

This project ends with a protected, labeled layout left in place for Project 03, not disturbed or removed.

  • Confirm every corner stake, offset stake, and batter board remains stable and correctly labeled
  • Confirm every string line remains tensioned, level, and undisturbed
  • Remove loose hardware, offcut lumber, and packaging from the layout area
  • Improve or add flagging or barricades around any control point exposed to site traffic
  • File the completed layout record sheet with the Project 01 existing-conditions record
