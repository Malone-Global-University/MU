Job Overview

This project qualifies you to assemble, support, and pressure-test one PEX water-supply branch using a single approved joining system from start to finish. The finished work product is one supported, protected, tested PEX branch — either a copper crimp-ring assembly or a stainless-steel clamp assembly — ready for the inspection required before concealment.

You will select one approved PEX system, review the branch drawing and pipe size, inspect the tubing and the crimp or clamp tool, route and support the tubing through the framing, cut tubing and install fittings with the selected ring or clamp method, install a test cap or manifold, apply the permitted water test pressure, inspect every connection, and correct and retest any failed connection before the branch is concealed.

This project builds directly on the pipe-cutting and deburring skills from Chapter 3 and the shutoff and drain-down discipline from Chapter 1. It establishes the supported, tested-branch discipline that Chapters 5 through 9 apply to copper, push-fit, compression, and DWV assemblies, and that the Chapter 20 capstone applies to a complete bathroom rough-in.

Skills You Will Demonstrate

  • PL-019 — Lay out a PEX branch run, marking penetrations, supports, bends, valves, and termination points from the approved branch drawing.
  • PL-020 — Cut PEX tubing square using the compatible tubing shear without kinks, flattening, or contamination.
  • PL-021 — Install PEX fittings using one approved joining system without blending crimp and clamp methods in the same assessed branch.
  • PL-022 — Crimp or clamp connections using the calibrated tool and confirm each connection with the required go/no-go gauge or visual method.
  • PL-023 — Support tubing at manufacturer- and code-controlled intervals while protecting it at penetrations and from damaging materials.
  • PL-024 — Pressure-check connections using the approved water test method and correct any failed connection before concealment.

Where This Skill Fits

PEX is one of three water-supply joining systems taught in this book, alongside soldered copper in Chapter 5 and push-fit or compression fittings in Chapter 6. Chapter 7 reuses this branch-building skill when replacing an angle stop, and Chapter 17 reuses it when roughing in a laundry box. The capstone in Chapter 20 requires the same lay-out, cut, join, support, and test sequence for the bathroom's hot and cold branches.

The method-separation discipline introduced here — completing one project using one joining system without blending methods — is the same discipline Chapter 6 applies to push-fit versus compression fittings, and the same discipline the book-level Method-Separation Control requires across every water-supply chapter.

Scope and Intended Use

This project covers:

  • selecting one approved PEX joining system — copper crimp rings or stainless-steel clamps — for the entire assessed branch
  • reviewing the branch drawing for pipe size, fitting sequence, valve locations, support points, bend limits, protection-plate locations, and test boundaries
  • inspecting tubing condition and calibrating or verifying the crimp or clamp tool before use
  • routing, cutting, and joining tubing using the selected method only
  • supporting and protecting tubing at the intervals and locations specified by the manufacturer and adopted code
  • filling, purging, and pressure-testing the branch at the permitted test pressure and duration
  • correcting and retesting any connection that fails inspection

This project does not authorize the learner to:

  • mix crimp-ring and clamp connections within one assessed branch unless the approved design specifically includes a transition
  • invent a support interval, bend radius, test pressure, or test duration not supplied by the manufacturer, adopted code, or the branch drawing
  • drill or notch a structural member, or route tubing through a prohibited fastener zone without an installed protection plate
  • conceal the branch before the required pressure test and inspection have passed
  • proceed with any tubing, tool, or connection condition identified as a stop-work condition in this chapter

Any tubing damage, tool-calibration failure, or connection that fails inspection must be corrected, replaced, or escalated to a qualified professional before the branch is concealed.

Trade Note

Crimp-ring and stainless-clamp systems use different tools, different gauge-verification procedures, and different failure signatures. A learner who has practiced one method has not automatically demonstrated the other. Complete and assess only the one method selected for this project unless the approved design explicitly calls for both.

Setup Standard

Do not crimp or clamp a single connection until one PEX joining system has been selected for the entire assessed branch and the crimp or clamp tool has been inspected and verified against the manufacturer's calibration or gauge procedure. A tool that fails calibration verification is a stop-work condition for every connection made with it.

Shop Tip

Dry-fit and mark the full tubing route — penetrations, supports, bends, and termination points — before making the first cut. Committing to a route after tubing is already joined leads to forced bends and unsupported spans that fail inspection.

Crimp-Ring and Stainless-Clamp Systems Compared

A crimp-ring connection uses a copper ring slid over the tubing and compressed with a crimp tool until it reaches the manufacturer-specified finished diameter, verified with a go/no-go gauge. A stainless-clamp connection uses a stainless-steel band tightened and cinched with a dedicated clamp tool until the band's ears are fully closed, verified visually and by the tool's release signal. The two methods require different tool setups, different gauge or visual verification steps, and different failure signatures, and this chapter's assessment applies to whichever single method is selected for the assigned branch.

Measurement Cue — Full Insertion Before Crimping or Clamping

Insert the fitting fully into the tubing before positioning the ring or clamp. Position the ring or clamp at the manufacturer-controlled distance from the tubing end, not by estimate. A connection crimped or clamped before full insertion is confirmed will not pass the go/no-go gauge or visual check.

Critical Safety Cue — Verified Isolation Before Testing

Confirm the branch is isolated from the building supply, per Chapter 1, before filling and pressurizing it for the water test. Never apply a permitted test pressure to a branch that remains connected to an unverified or uncontrolled source. Depressurize completely before disconnecting any test cap, valve, or manifold.

Field Safety Cue — Drilling and Framing Penetrations

Identify structural members, electrical wiring, and other utilities before drilling any framing penetration. Wear hearing and eye protection during drilling, and confirm the drill bit and route avoid the prohibited fastener zone or that a protection plate will be installed there.

Manufacturer Instruction Cue — Tool Calibration and Gauge Verification

Verify the crimp tool or clamp tool against the manufacturer's calibration or gauge procedure before making the first connection, and re-verify at the interval the manufacturer specifies. A tool that has not been verified, or that fails verification, is a stop-work condition for every connection made with it until it is corrected or replaced.

Code Alert — Support Spacing and Test Method

Support spacing for PEX tubing and the permitted water-test pressure and duration are set by the manufacturer's installation instructions and the adopted plumbing code, and both vary by jurisdiction. Confirm the exact support interval and test procedure required for this project before installation and before testing.

Method A — Copper Crimp Ring

G. Tubing and Tool Inspection (Both Methods)

  1. Inspect the tubing for UV damage, kinks, cuts, contamination, flattening, or incompatible markings. Reject and replace any specimen that fails this inspection.
  2. Inspect and verify the crimp or clamp tool using the manufacturer's calibration or gauge procedure. Do not proceed to the first connection until the tool passes verification.

H. Route Layout and Framing Preparation (Both Methods)

  1. Establish the tubing route and mark penetrations, supports, bends, valves, and termination points. Confirm the marked route matches the branch drawing before drilling.
  2. Drill or use approved framing openings without damaging structural members, electrical wiring, or other utilities. Verify each penetration location against the marked route.
  3. Install protection plates where the tubing passes within the prohibited fastener zone. Confirm plate placement covers the full width of the vulnerable zone.

I. Cutting and Connection Assembly (Both Methods)

  1. Measure and cut the first tubing section square. Confirm a clean, square cut before proceeding to the next step.
  2. Slide the specified ring or clamp onto the tubing in the required orientation. Confirm orientation against the manufacturer's instructions before inserting the fitting.
  3. Insert the compatible fitting fully and position the ring or clamp at the manufacturer-controlled distance from the tubing end. Confirm full insertion at the reference mark before crimping or clamping.
  4. Apply the crimp or clamp tool square to the connection and complete the tool cycle. Confirm the tool completes its full cycle without interruption.
  5. Inspect the completed connection using the required go/no-go gauge or visual confirmation method. Do not proceed to the next connection until this one passes inspection.
  6. Repeat the connection procedure for every fitting without reusing rings or clamps. Confirm a new ring or clamp is used at each connection.
Method B — Stainless Clamp

Where the project uses Method B, complete Sections G through I using the stainless-clamp tool, clamp orientation, and clamp engagement verification specified by the clamp manufacturer instead of the crimp-ring tool and gauge. Do not substitute crimp-ring components or verification steps into a clamp-method branch.

J. Routing, Support, and Protection (Both Methods)

  1. Route tubing with approved bends or bend supports and without kinks, abrasion, or excessive tension. Confirm each bend meets the manufacturer's minimum radius or uses a supported bend fitting.
  2. Install supports at manufacturer- and code-controlled intervals while allowing required movement. Confirm measured spacing against the branch drawing.
  3. Protect tubing at framing penetrations and isolate it from damaging metal edges or heat sources. Confirm every penetration and edge contact point is protected before concealment.

K. Pressure Testing and Correction

  1. Install approved test caps, valves, or manifolds. Confirm the test boundary is fully sealed before pressurizing.
  2. Fill and purge the branch using the approved water test method. Confirm air is fully purged before applying test pressure.
  3. Apply the permitted test pressure and duration under the required supervision. Confirm the exact pressure and duration values against the branch drawing or adopted code before applying them.
  4. Inspect every connection, tubing length, support, and termination for leakage or movement. Record the result of each inspected connection.
  5. Depressurize before correcting any failed connection. Do not attempt a correction under pressure.
  6. Remove and replace failed fittings or tubing using the system-approved correction method. Use a new ring or clamp; do not re-crimp or re-clamp a used one.
  7. Retest and document the final result before concealment. Confirm every connection passes before the branch is considered complete.

Common Readiness Inspection

Inspect every completed PEX branch using the same checks regardless of which method was selected.

  • Method check: confirm every connection in the branch uses the single selected joining method.
  • Tool check: confirm the crimp or clamp tool passed calibration or gauge verification before the first connection.
  • Connection check: confirm every connection passed the required go/no-go gauge or visual confirmation.
  • Routing check: confirm no kinks, abrasion, excessive tension, or under-radius bends are present.
  • Support and protection check: confirm supports and protection plates are installed at the specified locations and intervals.
  • Pressure-test check: confirm the branch held the permitted test pressure for the permitted duration with no leakage or movement.

Failure Points and Troubleshooting

A Crimp-Ring Connection Fails the Go/No-Go Gauge

Probable cause: the ring was positioned incorrectly, the tool did not complete a full cycle, or the tool is out of calibration.
Confirmation: re-verify the tool against the manufacturer's calibration procedure and inspect ring position on a test cutoff.
Correction: depressurize if already tested, cut out the failed connection, and remake it with a new ring following the full insertion, positioning, and crimp-cycle procedure.
Reinspection: apply the go/no-go gauge to the new connection before proceeding.

A Stainless Clamp Does Not Fully Close

Probable cause: the clamp tool jaws were misaligned, the clamp was the wrong size for the tubing, or the tool requires service.
Confirmation: inspect clamp ear closure visually and compare against the manufacturer's fully-closed reference.
Correction: remove the incomplete clamp, inspect the tubing and fitting for damage, and remake the connection with a new clamp.
Reinspection: visually confirm full ear closure on the new connection before proceeding.

The Branch Loses Pressure During the Water Test

Probable cause: an unconfirmed connection, a damaged tubing section, or an improperly sealed test cap or manifold.
Confirmation: inspect each connection, the tubing route, and the test-boundary seal in sequence while the branch is depressurized.
Correction: remake any connection identified as the leak source using the system-approved method, or reseat and reseal the test cap or manifold.
Reinspection: refill, purge, and repeat the full pressure test at the permitted pressure and duration.

Tubing Shows Kinks, Flattening, or Abrasion After Routing

Probable cause: the bend radius was tighter than the manufacturer's minimum, the tubing contacted a sharp framing edge, or it was pulled under excessive tension during routing.
Confirmation: inspect the tubing along its full length and compare suspect bends against the manufacturer's minimum radius specification.
Correction: cut out and replace the damaged section, or reroute using a supported bend fitting where the radius cannot be met.
Reinspection: re-inspect the corrected routing and repeat the pressure test before concealment.

Support Spacing Does Not Meet the Specified Interval

Probable cause: supports were installed by estimate rather than measured against the branch drawing or manufacturer instructions.
Confirmation: measure actual support spacing along the branch and compare against the specified interval.
Correction: install additional supports at the required interval without overtightening or restricting required tubing movement.
Reinspection: re-measure spacing after correction and confirm the branch is fully supported.

Crimp-Ring and Clamp Connections Were Mixed in One Branch

Probable cause: a different fitting or ring/clamp type was substituted mid-branch, often when the originally specified component was unavailable.
Confirmation: inspect every connection in the branch and compare the joining method used against the method recorded on the worksheet.
Correction: unless the approved branch drawing explicitly documents a transition, remake every connection using the single selected method.
Reinspection: re-verify method consistency across the full branch before retesting.

Cleanup

This project ends with the branch supported, protected, tested, and documented, ready for the inspection required before concealment.

  • Remove and dispose of cutoff tubing, spent rings or clamps, and packaging
  • Clean and store the crimp or clamp tool per the manufacturer's storage instructions
  • Confirm all test caps, valves, or manifolds remain in place until the required inspection is scheduled
  • Leave the branch isolated from the building supply per the Chapter 1 shutoff record until inspection authorizes restoration
  • Store the PEX Branch Test Worksheet with the project records for later reference
