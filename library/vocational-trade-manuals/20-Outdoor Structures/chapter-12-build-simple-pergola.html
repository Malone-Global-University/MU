Job Overview

This project qualifies you to build one small detached open-roof pergola from an approved design: footings and post bases already accepted, posts set plumb and braced, beams installed on approved bearing, rafters and purlins laid out and fastened, permanent bracing and uplift connectors installed, and the finished structure inspected for alignment and stability. The finished work product is a stable, plumb, level, permanently braced pergola frame with a documented load and use restriction.

You will lay out post and footing centers from the approved plan, confirm the footings and post bases pass inspection, set and brace posts, install beams on approved bearing, install rafters and purlins to the specified spacing, install permanent bracing and uplift connectors, and inspect the finished structure before removing temporary bracing.

This project assumes Project 04 (footings) and Project 05 (posts, where posts are installed ahead of beam work) are complete, or that this project's own footing and post steps are performed as part of the same sequence using the plans and procedures from those chapters. No beam is loaded, no rafter is fastened, and no temporary bracing is removed until the load path below it is confirmed complete.

Skills You Will Demonstrate

  • OS-067 — Lay out pergola post and footing centers from an approved plan.
  • OS-068 — Install beams on approved bearing, post caps, notches, or through-bolt details.
  • OS-069 — Install rafters to the approved spacing, overhang, and connection detail.
  • OS-070 — Install purlins or shade slats to the approved spacing and attachment.
  • OS-071 — Brace the structure permanently, including knee braces, diagonal bracing, and uplift connectors.
  • OS-072 — Inspect pergola alignment and stability before removing temporary bracing.

Where This Skill Fits

This project depends directly on the property and site information from Project 01, the work-area controls from Project 02, the layout method from Project 03, the footing work from Project 04, and the post-setting method from Project 05. It also shares its framing vocabulary — beam bearing, rafter spacing, temporary and permanent bracing, uplift connection — with the deck framing taught in Project 06 and the shed roof framing taught in Project 16.

The pergola is one of the two structures the Chapter 20 capstone project is built from, so the bracing habits, connector discipline, and load-restriction documentation established here carry forward without modification into the final integrated project.

Scope and Intended Use

This project covers:

  • laying out pergola post and footing centers from an approved plan
  • installing and inspecting footings and post bases, or confirming footings already completed in Project 04
  • setting, bracing, plumbing, and aligning posts
  • installing beams on approved bearing, post caps, notches, or through-bolt details
  • laying out and installing rafters and purlins to the specified spacing and overhang
  • installing permanent knee braces, diagonal bracing, tension devices, and uplift connectors
  • inspecting post plumb, beam level, rafter spacing, connectors, uplift path, and lateral stiffness

This project does not authorize the learner to:

  • invent post sizes, beam spans, rafter spans, footing dimensions, or connector schedules that are not supplied by the approved plan or manufacturer instructions
  • install roofing, solid roof sheathing, or any weatherproof covering over the rafters — this is an open-roof structure only
  • allow the structure to support snow accumulation, swings, hammocks, solar panels, planted vines beyond the design's stated plant-load allowance, or any other load not included in the approved design
  • remove temporary bracing before permanent bracing and uplift connectors are complete and inspected

Any post, beam, rafter, connector, or bracing condition that cannot be confirmed against the approved plan must be resolved before this project is declared complete.

Trade Note

A pergola looks like a simple structure because its roof is open, but its bracing and uplift path carry real wind load across a tall, exposed frame. The habit this chapter builds — installing permanent bracing before removing temporary bracing, and never treating an open lattice as load-free — is the same habit that governs the taller, heavier framing later in this book.

Setup Standard

Do not remove any temporary bracing on a post or beam until the permanent knee braces, diagonal bracing, and uplift connectors specified by the approved plan are installed and inspected. A pergola frame left standing on temporary bracing alone is not a stable structure.

Shop Tip

Cut and test-fit the first rafter as a pattern before cutting the rest of the set, the same way a stair stringer pattern is verified before it is repeated. A rafter pattern error caught on one board costs a few minutes; caught after every rafter is cut, it costs the whole set.

Common Field Fundamentals

Every framed structure in this book depends on the same sequence: accepted support below, plumb and braced posts, beams on approved bearing, spanning members at the approved spacing, and permanent bracing installed before temporary bracing is removed. A pergola is a useful place to learn this sequence because its frame is fully exposed — every connection is visible for inspection before any covering or infill conceals it.

Open-Roof Scope and Load Limits

This structure is designed as an open-roof pergola. It is not designed to carry roofing, sheathing, snow accumulation, or unapproved hanging loads such as swings, hammocks, or solar equipment. Any request to add one of these loads requires a return to the approved design and, where the load exceeds the pergola's stated capacity, an engineered revision before construction proceeds.

Code Alert — Permits, Setbacks, and Wind or Snow Criteria

Confirm whether the pergola requires a building permit, and confirm the approved wind and snow design criteria for the site before beam and rafter sizes are finalized. Height, footprint, and proximity to the property line may also trigger separate setback or zoning review.

Critical Safety Cue — Do Not Remove Temporary Bracing Early

Do not remove temporary bracing on any post or beam until permanent knee braces, diagonal bracing, and uplift connectors are installed and inspected. An unbraced frame can rack and fall under wind load, the weight of workers on a ladder, or accidental contact.

Field Safety Cue — Working at Beam and Rafter Height

Use a ladder or scaffold rated for the working height and footing conditions at this site. Keep the fall zone clear of other workers while raising beams or rafters, and never stand on unbraced framing.

Measurement Cue — Verify Post Spacing Against the Beam and Rafter Plan

Confirm post spacing matches the beam span and rafter span in the approved plan before any beam is set. A post grid that drifts from the approved spacing changes the span the beam and rafters must carry.

E. Preparation — Layout and Footing Confirmation

  1. Review the approved pergola dimensions, height, post grid, footing design, beam and rafter sizes, spans, overhangs, purlins, bracing, wind, uplift, and permit requirements. Confirm the finalized dimensions match the plan before any material is cut.
  2. Confirm the design is open-roof only. Verify it is not intended to support roofing, snow accumulation, swings, hammocks, vines beyond the stated allowance, solar panels, or other unapproved loads.
  3. Lay out post and footing centers from the approved plan. Transfer dimensions from the Project 03 layout references.
  4. Install and inspect footings and post bases, or confirm the Project 04 footings already installed have passed inspection. Do not proceed until bearing, cure, and anchor position are accepted.
  5. Set, brace, plumb, and align posts. Follow the Project 05 procedure, holding plumb on two adjacent faces with temporary bracing in two directions.

F. Beam Installation Procedure

  1. Mark beam bearing points and top elevations on each post. Transfer elevations using a string line, laser, or level referenced from a single fixed point.
  2. Cut and treat posts and beams to the approved lengths. Apply field-cut preservative to every cut end and drilled hole in treated material.
  3. Install beams on approved bearing, post caps, notches, or through-bolt details. Do not substitute a bearing method not shown on the approved plan.
  4. Install required beam splices only over approved supports. A splice between supports is not an approved bearing condition unless the plan specifically details one.
  5. Install temporary lateral bracing on every beam and post before proceeding to rafter work. Confirm the frame holds plumb and level under light hand pressure before continuing.

G. Rafter and Purlin Installation Procedure

  1. Lay out rafter positions on the beams from the approved spacing and reference point. Mark every rafter location before cutting begins.
  2. Cut the first rafter using an approved template or pattern. Test-fit it in place before cutting the remaining rafters.
  3. Cut remaining rafters to the approved pattern. Treat every field cut in pressure-treated material.
  4. Install rafters with listed ties, screws, bolts, or joinery as specified by the plan. Maintain the approved spacing, overhang, and alignment across the full frame.
  5. Install purlins or shade slats at the approved spacing and attachment. Maintain consistent spacing and alignment across the full roof plane.

H. Permanent Bracing, Uplift, and Closeout Procedure

  1. Install knee braces, diagonal bracing, tension devices, and uplift connectors per the approved plan. Follow manufacturer instructions for every listed connector.
  2. Do not remove temporary bracing until permanent bracing is complete and inspected. Confirm every permanent connection is installed before releasing any temporary support.
  3. Treat field cuts and install caps where specified. Seal every exposed cut in treated lumber before final inspection.
  4. Inspect post plumb, beam level, rafter spacing, purlins, connectors, uplift path, lateral stiffness, clearance, and sharp edges. Correct any deficiency before proceeding to the stability check.
  5. Perform the approved controlled stability check. Apply the specified lateral load, deflection check, or firm hand-check to the completed frame and confirm no unexpected movement.
  6. Document the structure and its load restrictions. Record beam and rafter sizes, connector schedule, bracing installed, and the stated open-roof, no-snow, no-hanging-load restriction for the completed structure.

Common Inspection Review

Inspect the completed pergola frame using the same checks regardless of the exact post grid or span selected.

  • Footing and post-base check: confirm bearing, cure, and anchor position were accepted before any post carried load.
  • Post check: confirm plumb on two adjacent faces, correct spacing, and secure connection at the base.
  • Beam check: confirm approved bearing, level or approved slope, and correct connector installation.
  • Rafter and purlin check: confirm the approved spacing, overhang, and pattern were followed across the full frame.
  • Bracing and uplift check: confirm every permanent knee brace, diagonal, and uplift connector is installed and that temporary bracing was not removed early.
  • Documentation check: confirm the load and use restriction is recorded for future reference.

Complete the Pergola Framing Worksheet in the Inspection Checklist section below before considering the structure finished.

Failure Points and Troubleshooting

Post Spacing Does Not Match the Beam or Rafter Span

Probable cause: posts were set from field measurements rather than the approved plan dimensions, or the layout drifted during footing work.
Confirmation: re-measure post-to-post spacing and compare it against the beam and rafter span called out in the approved plan.
Correction: do not force an oversized beam or rafter span to fit; correct the post position where possible, or return to the design for an approved revision.
Reinspection: repeat the layout check on the corrected post positions before beam installation continues.

Beam Was Loaded Before Footings Reached the Approved Cure Condition

Probable cause: beam installation proceeded on a schedule rather than on the concrete's actual condition.
Confirmation: check footing placement records against the elapsed cure time and the approved condition for loading.
Correction: remove load from the affected post and footing, confirm the footing has not been displaced or cracked, and wait for the approved cure condition before reloading.
Reinspection: repeat the footing and post-base check before continuing beam work.

Temporary Bracing Was Removed Before Permanent Bracing Was Installed

Probable cause: temporary bracing was mistaken for a finished condition, or removed to make room for later work before permanent connectors arrived.
Confirmation: compare the installed connector schedule against the approved plan's permanent bracing and uplift requirements.
Correction: re-install temporary bracing immediately, then complete every permanent knee brace, diagonal, and uplift connector before removing it again.
Reinspection: repeat the bracing and uplift check on the corrected frame.

Rafter Spacing or Pattern Varies Across the Frame

Probable cause: the first rafter pattern was not tested before the remaining rafters were cut, or spacing was marked without a consistent reference point.
Confirmation: measure spacing between every rafter and compare it to the approved plan.
Correction: remove and re-cut any rafter that does not match the approved pattern, re-marking spacing from a single consistent reference.
Reinspection: repeat the rafter and purlin check across the corrected frame.

An Unapproved Load Was Added to the Open-Roof Frame

Probable cause: roofing, solid sheathing, a hanging swing, or another load not included in the approved design was added after the frame was accepted.
Confirmation: compare the added load against the documented load and use restriction recorded at closeout.
Correction: remove the unapproved load and return to the approved design for an engineered revision before any such load is added.
Reinspection: repeat the full stability check once the frame is returned to its approved condition.

Cleanup

This project ends with the frame permanently braced, inspected, and documented, not with tools simply set down.

  • Remove and store all temporary bracing only after permanent bracing and uplift connectors are confirmed complete and inspected
  • Sweep and clear all cutoffs, fasteners, and debris from beneath and around the frame
  • Store unused connectors, fasteners, and lumber for reuse in later chapters
  • Photograph the completed frame from multiple angles for the project record
  • File the connector schedule, load restriction, and inspection results with the project records

Field Terms

Open-Roof Structure
A pergola frame with rafters and purlins but no solid roof deck or weatherproof covering.
Beam Bearing
The approved point of support carrying the beam's load into the post.
Rafter
The framing member spanning between beams, carrying purlins and roof-plane loads.
Purlin
A secondary member spanning across rafters at the approved spacing.
Knee Brace
A diagonal member connecting a post to a beam, resisting lateral movement.
Uplift Connector
A listed hardware connection resisting wind forces that would lift or separate framing members.
Temporary Bracing
Bracing installed during erection to hold members plumb until permanent bracing is complete.
Permanent Bracing
The specified knee braces, diagonals, and connectors that provide the finished structure's lateral stiffness.
Load and Use Restriction
The documented statement of what loads the completed structure is, and is not, designed to carry.
Stability Check
The approved controlled load, deflection, or firm-hand check performed on the completed frame.
