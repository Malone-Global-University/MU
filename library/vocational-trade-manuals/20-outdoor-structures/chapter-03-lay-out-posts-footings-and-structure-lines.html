Job Overview

This project qualifies you to transfer an approved set of structure dimensions from the site plan and structural drawings onto the actual ground, and to prove that transfer is square, dimensionally accurate, and protected before any excavation begins. The finished work product is a completed and documented Structure Layout: a controlling baseline, squared corner points verified by diagonal measurement, batter boards or offset stakes set outside the excavation limits, and marked post, footing, beam, and wall reference lines recorded on the layout worksheet.

You will establish a controlling baseline from the accepted readiness package, set and square the structure corners, verify both diagonals against the approved tolerance, transfer the verified lines to protected offsets, mark post and footing centers against the framing plan, and record every dimension before requesting layout approval.

This project assumes Project 01's site inspection and readiness package and Project 02's work-area setup are both complete and accepted. No hole is dug, no auger is used, and no footing is excavated until this project's layout has been recorded and approved. Excavation begins in Project 04, working from the post and footing centers this project establishes.

Skills You Will Demonstrate

  • OS-013 — Establish reference lines from the approved site plan, structural drawings, and accepted Project 01 site controls.
  • OS-014 — Square the layout using an approved squaring method and verify it by measuring both diagonals.
  • OS-015 — Mark post and footing center locations against the approved structural drawings.
  • OS-016 — Verify post, beam, and framing spacing against the approved framing plan before excavation is requested.
  • OS-017 — Check and adjust diagonals until the layout meets the approved tolerance, then recheck side lengths.
  • OS-018 — Document the final layout, including dimensions, diagonals, offsets, and elevations, and protect all reference controls.

Where This Skill Fits

Every footing excavated in Project 04, every post set in Project 05, and every beam and joist installed in Project 06 depends on the post and footing centers this project establishes. A layout that is out of square or out of tolerance carries that error into every later structural component, and correcting it after concrete is placed is far more disruptive than correcting it here, before a single hole is dug.

The habit built in this chapter — establishing a controlling baseline, squaring by the diagonal method rather than by eye, and protecting reference controls once they are verified — is the same habit that governs layout for a deck frame, a fence line, a pergola grid, or a shed footprint anywhere in this book.

Scope and Intended Use

This project covers:

  • reviewing approved dimensions, post grid, footing centers, beam lines, wall lines, and elevations from the site plan and structural drawings
  • confirming survey controls, utility markings, drainage, and permit conditions remain valid before layout begins
  • establishing a controlling baseline and setting structure corners
  • squaring the layout by an approved method and verifying both diagonals
  • setting batter boards or offset stakes outside the excavation limits and transferring structure lines to them
  • marking beam centerlines, post centerlines, wall faces, and footing centers
  • verifying the layout remains within property, setback, and easement limits
  • recording and protecting the completed layout for handoff to Project 04

This project does not authorize the learner to:

  • excavate any footing hole — layout approval must be recorded first, and physical excavation begins in Project 04
  • disturb, remove, or relocate a confirmed utility marking, property corner flag, or setback marking established in Project 01
  • finalize a layout that has not been squared and verified by diagonal measurement within the approved tolerance
  • proceed to footing excavation with an unrecorded or unapproved layout

Any survey control, utility marking, or property reference that cannot be confirmed as still valid must be resolved before this project's layout is finalized.

Trade Note

Every dimension in this chapter is transferred from the approved site plan and structural drawings — never estimated by eye or adjusted informally in the field to make the ground "look right." A layout that has not been verified by diagonal measurement is not square, no matter how it appears from a standing position. This is the single most common source of accumulated framing error in later chapters.

Setup Standard

Do not remove or relocate a corner stake once its diagonal has been verified. Transfer every verified corner to a protected batter board or offset stake before any stake within the excavation footprint is disturbed by later steps.

Shop Tip

Record diagonals in a simple table as you measure them — corner pair, target dimension, and measured dimension. A running record makes it far easier to isolate which single corner needs adjustment when the two diagonals do not yet match.

Squaring by the Diagonal Method

A rectangular layout is square only when both corner-to-corner diagonals are equal within the approved tolerance. A layout that "looks square" or that was set using a single framing-square reading at one corner has not been verified. Every corner is checked, and any adjustment to one corner is followed by rechecking every affected side length and diagonal.

Code Alert — Layout Within Verified Property Limits

Confirm the squared layout remains entirely within the property limits, setbacks, and easements confirmed in Project 01 before setting batter boards. A layout that was square on paper but drifted outside a setback during field transfer still requires correction before excavation is requested.

Critical Safety Cue — Utility Markings Remain the Controlling Reference

Do not drive a stake, batter-board support, or offset stake through or beyond a confirmed utility marking from Project 01. If a proposed post or footing center falls on or near a marked utility line, stop and resolve the conflict with the structure-type or footing plan before continuing layout.

Field Safety Cue — Driving Stakes and Pulling String Line

Wear eye protection while driving stakes with a sledgehammer or mallet, and keep hands clear of the stake head during each strike. Watch for taut string line at eye and ankle height when walking the layout area, and mark or flag string lines that cross a walking path.

Measurement Cue — Verify Every Diagonal, Not Just One

On a layout with more than four corners, or with an internal post grid, check the diagonal of every rectangular section, not only the overall perimeter. A perimeter that reads square can still contain an internal post that is out of position relative to the beam or joist plan.

E. Preparation — Plan Review and Baseline Selection

  1. Review the approved overall dimensions, post grid, footing centers, beam lines, wall lines, elevations, setbacks, and structure orientation. Confirm these match the structure type selected in Project 01.
  2. Confirm survey controls, utility markings, drainage findings, and permit conditions from Project 01 and Project 02 are still valid and undisturbed. Resolve any discrepancy before proceeding.
  3. Establish the controlling baseline. Select the single reference line the site plan calls for — a property line, the face of an existing structure, or a fixed survey point — and confirm it against the recorded Project 01 corner flags.

F. Corner Setting and Squaring Procedure

  1. Set the first corner or control point on the baseline. Mark it clearly and record its position relative to the baseline origin.
  2. Measure and mark the first structure line from the first corner. Record the measured length against the approved dimension.
  3. Establish the perpendicular line at the first corner using an approved squaring method. Use a framing square for an initial reference, then confirm with a measured method rather than the square alone.
  4. Set the remaining corners along the established lines. Record each corner's measured distance from the baseline and from the adjacent corner.
  5. Measure both diagonals of the rectangular layout. Record each diagonal on the worksheet diagonal-measurement table.
  6. Adjust control points until the measured diagonals meet the approved tolerance. Move only the corner or corners needed to bring the diagonals into agreement.
  7. Recheck side lengths after any diagonal adjustment. Confirm every side still matches its approved dimension before continuing.

G. Offset Transfer and Reference Marking (Repeat at Each Corner)

  1. Set batter boards or offset stakes outside the excavation limits at each verified corner. Position them far enough outside the footing area that they will not be disturbed by excavation.
  2. Transfer the verified structure lines to the offsets using string line and the recorded corner positions. Confirm the transferred lines still intersect at the approved corner points before removing any original corner stake.
  3. Mark beam centerlines, post centerlines, wall faces, and footing centers on the ground and on the offset string lines. Use the marking-paint or flag color reserved for structure-outline marking from Project 01.
  4. Verify post spacing against the beam, joist, fence-rail, pergola, or wall framing plan for the selected structure type. Confirm no post or footing center conflicts with a marked utility, a setback, or an easement.
  5. Mark doors, gates, stairs, landings, utilities, drains, and other fixed obstacles on the layout. Record their position relative to the structure lines.
  6. Establish the finished elevation and post-top or beam reference point. Record it using a hand level, string level, or laser level referenced to a fixed benchmark.

H. Verification and Approval

  1. Verify the completed layout remains within the property lines, setbacks, and easements confirmed in Project 01. Adjust and re-measure if any conflict is found.
  2. Record all dimensions, diagonals, post centers, offsets, and elevations on the layout worksheet. Confirm every entry corresponds to an actual field measurement, not an assumed or transcribed plan value.
  3. Obtain layout approval before excavation begins. Route the completed worksheet through the required review before Project 04 starts.
  4. Protect all reference controls. Flag or barricade batter boards, offset stakes, and string lines so they are not disturbed before excavation begins.

I. Recheck and Closeout Procedure

  1. Recheck every incomplete or unclear item. Classify each gap as a missing dimension, an unconfirmed diagonal, an unresolved spacing conflict, or an unprotected reference control before closing it out.
  2. Label and retain the completed layout worksheet. Store it with the Project 01 readiness package for reference throughout excavation and framing.

Common Inspection Review

Inspect the completed layout using the same checks regardless of which structure type was selected.

  • Baseline check: confirm the controlling baseline matches the approved site plan and Project 01 property controls.
  • Squaring check: confirm every rectangular section has measured, equal diagonals within the approved tolerance.
  • Offset check: confirm batter boards or offset stakes are set outside excavation limits and transferred lines match the verified corners.
  • Spacing check: confirm post, footing, beam, and wall centers match the approved framing plan.
  • Property-limits check: confirm the finished layout remains within verified property lines, setbacks, and easements.
  • Record check: confirm every dimension, diagonal, offset, and elevation is recorded and the worksheet is approved.

Complete the Layout Verification Worksheet in the Inspection Checklist section below before beginning Project 04.

Failure Points and Troubleshooting

Diagonals Do Not Match Within the Approved Tolerance

Probable cause: one or more corners were set using a framing square alone, without a follow-up diagonal measurement, or a corner stake shifted after it was set.
Confirmation: re-measure both diagonals and compare the difference to the approved tolerance.
Correction: adjust the corner or corners contributing the greatest error, then recheck all affected side lengths and both diagonals again.
Reinspection: repeat the squaring check on the corrected layout.

Post or Footing Center Falls on or Near a Marked Utility

Probable cause: the framing-plan spacing was transferred to the ground without cross-checking it against the utility markings recorded in Project 01.
Confirmation: compare the marked post center against the utility-locate sketch and the required clearance.
Correction: adjust the post or footing center, and the associated framing spacing, to clear the marked utility, and document the adjustment.
Reinspection: repeat the spacing check on the corrected centers.

Batter Board or Offset Stake Was Disturbed Before Excavation

Probable cause: the offset was set too close to the excavation limits, or was not flagged or barricaded after the layout was verified.
Confirmation: compare the current offset position and string-line intersection against the recorded worksheet values.
Correction: re-establish the offset from the recorded corner dimensions, reset the string line, and re-verify the diagonal before proceeding.
Reinspection: repeat the offset check on the corrected reference.

Layout Drifted Outside a Property Limit or Setback

Probable cause: the layout was squared correctly relative to itself but was not re-checked against the confirmed property corners after the baseline was transferred to the field.
Confirmation: measure the distance from each affected structure line to the nearest confirmed property corner or setback flag.
Correction: shift the entire squared layout, maintaining its internal dimensions and diagonals, until it clears the property limit or setback.
Reinspection: repeat the property-limits check on the corrected layout.

Post Spacing Does Not Match the Framing Plan

Probable cause: post centers were marked from a rough field estimate rather than measured from the structural drawings' beam and joist spacing.
Confirmation: compare each marked post center distance against the approved framing plan.
Correction: re-measure and re-mark post centers from the baseline and corrected corner positions to match the approved spacing.
Reinspection: repeat the spacing check on the corrected centers.

Layout Worksheet Is Missing Dimensions or Diagonal Entries

Probable cause: measurements were taken in the field but not recorded before stakes were adjusted or moved to the next corner.
Confirmation: review the worksheet against the Assessed Layout Schedule for missing components.
Correction: return to the site to re-measure any missing dimension while reference controls are still intact, or note on the worksheet any item that must be reconfirmed before approval.
Reinspection: repeat the record check on the completed worksheet.

Cleanup

This project ends with the layout protected and documented, not discarded.

  • Leave batter boards, offset stakes, and structure-line string in place; do not remove any reference control needed for Project 04 excavation
  • Photograph the final squared layout, including diagonal measurements in progress, for the layout record
  • Store unused stakes, batter-board stock, and string line for reuse in later chapters
  • Label and file the completed layout worksheet, diagonal-measurement table, and any updated sketch together
  • Return the approved layout record to the project records for reference throughout footing, post, and framing work
