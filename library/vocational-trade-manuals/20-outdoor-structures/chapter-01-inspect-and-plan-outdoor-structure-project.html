Job Overview

This project qualifies you to inspect a yard or building site, gather the information needed to plan an outdoor wood structure, and document that the site is ready for layout. The finished work product is a completed Site Inspection and Project Readiness Package: a marked sketch of the site, a completed utility-locate confirmation, a drainage and slope assessment, a structure-type recommendation with rationale, and a signed readiness worksheet.

You will walk the site, identify and record existing conditions, locate and flag property lines and setbacks, request and record a utility locate, evaluate drainage and slope across the build area, weigh site conditions against candidate structure types, and select and justify a structure type before any digging, layout, or material ordering begins.

This is the first project in the book. It assumes the exterior sequence covered in earlier books — site prep, concrete flatwork, masonry, hardscaping, and landscaping — is either complete or already understood, and it establishes the site information every later chapter in this book depends on. No footing is dug, no post is set, and no lumber is cut until this project's readiness package is complete.

Skills You Will Demonstrate

  • OS-001 — Inspect site conditions relevant to an outdoor wood structure, including existing grade, vegetation, structures, and access.
  • OS-002 — Identify property limits, setbacks, easements, and any HOA or deed restrictions that affect the build area.
  • OS-003 — Identify buried and overhead utilities and confirm a utility locate before any digging is planned.
  • OS-004 — Evaluate drainage and slope across the build area and note conditions that will affect footing, post, or floor decisions.
  • OS-005 — Select a structure type appropriate to the site conditions, intended use, and site constraints.
  • OS-006 — Document project readiness in a form usable by every later chapter in this book.

Where This Skill Fits

This project is the entry point for the entire book. Every layout in Project 03, every footing in Project 04, every post in Project 05, and the Chapter 20 capstone all reference the property limits, utility locate, drainage assessment, and structure-type decision produced here. A site that has not been inspected and documented under this project cannot proceed to layout in Project 03.

The habit built in this chapter — walking a site, recording what is actually there rather than what is assumed to be there, and confirming utilities before any tool touches the ground — is the same habit that governs every outdoor structure project in this book, from a small garden trellis to a full shed.

Scope and Intended Use

This project covers:

  • walking and recording existing site conditions relevant to an outdoor wood structure
  • locating and flagging property lines, setbacks, easements, and any recorded restrictions
  • requesting a utility locate and recording the results before planning any digging
  • evaluating drainage patterns and ground slope across the proposed build area
  • comparing site conditions against the requirements of common structure types covered later in this book
  • selecting and documenting a structure type with a stated rationale
  • completing a Site Inspection and Project Readiness Package

This project does not authorize the learner to:

  • dig any hole, footing, or post pit — utility locate confirmation and readiness sign-off must be complete first, and physical digging begins in Project 04
  • set stakes or string in a way that disturbs a confirmed utility marking
  • finalize a structure type, size, or location that conflicts with a documented property limit, setback, easement, or HOA restriction
  • proceed to layout in Project 02 or Project 03 with an incomplete or unsigned readiness package

Any site condition that cannot be confirmed, or any utility locate that has not been completed and marked, must be resolved before this project is declared complete.

Trade Note

Every conclusion in this chapter is recorded from direct observation of the actual site — never assumed from memory, a prior owner's description, or an old survey without verifying it against current ground conditions. Substituting assumption for inspection, particularly around property lines and buried utilities, is the single most common source of the costly errors this chapter is designed to prevent.

Setup Standard

Do not flag or stake a proposed structure location until the utility locate request (Step 6) has been submitted and the marked results have been recorded on the site sketch. A structure location flagged before the utility locate is confirmed must be re-flagged after confirmation.

Shop Tip

Use one flag color for property corners and setbacks, a second color for utility markings, and a third color for the proposed structure outline. Mixing colors between categories is how a setback flag gets mistaken for a structure corner during layout in Project 03.

Reference Conventions

Establish these references before any layout begins, and do not change them mid-project.

TermDefinition
Property lineThe recorded legal boundary of the lot; the reference every setback distance is measured from
SetbackThe minimum required distance a structure must be located from a property line, easement, or other reference, as set by local jurisdiction or HOA rule
EasementA recorded right for a utility, neighbor, or agency to use a defined portion of the property; structures generally cannot be placed within an active easement
Utility locateA confirmed marking of buried utility lines within the proposed build area, obtained before any digging is planned
Build areaThe proposed footprint of the structure plus the working clearance needed to construct it
Drainage pathThe route water follows across the site during and after rainfall; recorded so the structure does not block or redirect water onto a neighboring property or the house foundation
Grade / slopeThe rise or fall of the ground surface across the build area, recorded because it affects footing depth, post height, and floor-framing decisions in later chapters

Prerequisites

  • Access to the site during daylight hours, with permission to walk, flag, and photograph the full build area and its surroundings
  • A copy of the property survey, plat, or deed description, if one exists, for cross-checking observed property corners
  • Knowledge of, or the ability to look up, local setback, permit, and HOA requirements for the jurisdiction where the site is located
  • Contact information for the regional utility locate service, requested well in advance of any planned digging
  • A blank site-inspection worksheet and enough marking flags and stakes to complete the inspection without running out mid-task

Property Limits, Setbacks, and Restrictions

Every proposed structure location is checked against the recorded property line, not against a fence, hedge, or existing marker that may not sit on the actual boundary. A structure sited from an assumed line rather than a confirmed one risks an encroachment that can require the structure to be moved or removed after it is built.

Code Alert — Setbacks, Easements, and HOA Restrictions

Local setback distances, easement restrictions, and HOA rules vary by jurisdiction and by neighborhood. Verify the current requirements for the specific site before finalizing a structure location, and do not rely on a neighboring property's structure as evidence of what is allowed.

Critical Safety Cue — Utility Locate Before Any Digging

Contact the regional utility locate service and obtain a confirmed, marked locate before any digging is planned or performed anywhere in this book. Never dig, drive a stake deeper than flag depth, or probe the ground in an unmarked area. Contact with a buried utility line can cause serious injury, service outages, or death.

Field Safety Cue — Walking the Site

Watch footing on uneven ground, overgrown vegetation, and hidden holes or debris while walking and flagging the site. Wear closed-toe footwear at all times and use appropriate insect or sun protection for the conditions.

Measurement Cue — Record Drainage and Slope Across the Whole Build Area

Check slope and drainage at multiple points across the build area, not just at one corner. A site that appears level at the front corner can still slope significantly toward the back, which affects footing depth and post height decisions in later chapters.

E. Preparation — Document Review and Site Walk Planning

  1. Gather available reference documents. Locate the property survey, plat, or deed description if one exists, and note the recorded lot dimensions and any recorded easements.
  2. Look up current local requirements. Confirm setback distances, permit thresholds, and any HOA restrictions that apply to the jurisdiction where the site is located.
  3. Walk the full site once before flagging anything. Note existing grade changes, vegetation, structures, fences, and access routes without yet marking the ground.
  4. Identify the general area under consideration for the structure. Confirm it appears to fall within the buildable area based on the reference documents.
  5. Assemble marking materials by category. Confirm one flag or stake color is reserved for property corners and setbacks, a second for utility markings, and a third for the proposed structure outline.

F. Site Inspection and Layout Verification Procedure

  1. Locate and flag the property corners nearest the build area. Cross-check the flagged corners against the survey or plat where available.
  2. Measure and mark the required setback distance from each relevant property line, easement, or restriction. Record each measured distance on the worksheet.
  3. Record existing structures, fences, trees, and grade changes within and adjacent to the build area. Photograph each condition that may affect the structure's location or design.
  4. Confirm the build area falls entirely within the property limits and outside any easement or setback. Adjust the proposed area and re-measure if any conflict is found.

G. Utility Locate and Drainage Assessment (Repeat Drainage Checks at Each Point)

  1. Submit a utility locate request for the confirmed build area. Provide the locate service with accurate boundaries for the area to be marked.
  2. Wait for the locate to be completed and marked on site before proceeding. Do not place any structure-outline flag before the locate is confirmed.
  3. Record every marked utility line on the site sketch and photograph each marking. Note the utility type for each marked line where indicated.
  4. Check drainage and slope at each corner and at the center of the build area. Use the string and line level to record relative rise or fall between points.
  5. Note the direction water currently flows across the site during or after rainfall. Record whether the proposed structure location could block or redirect that flow onto a neighboring property or the house foundation.
  6. Record the drainage and slope findings on the worksheet. Flag any condition that will need to be addressed during footing, post, or floor planning in later chapters.

H. Structure-Type Selection and Readiness Sign-Off

  1. Compare at least two candidate structure types against the recorded site conditions and intended use. Weigh grade, drainage, access, size limits, and property restrictions for each candidate.
  2. Select and document the structure type with the strongest fit, including a stated rationale. Record the rationale on the worksheet for reference in later chapters.
  3. Flag the proposed structure outline using the reserved third flag color. Confirm the outline stays clear of all marked utilities, setbacks, and easements.
  4. Compile the completed sketch, photographs, worksheet, and rationale into the readiness package. Review every section for completeness before sign-off.
  5. Sign and date the readiness worksheet. Confirm the package is ready to hand off to Project 02.

I. Recheck and Closeout Procedure

  1. Recheck every incomplete or unclear item. Classify each gap as missing documentation, an unconfirmed measurement, an unresolved utility marking, or an unresolved drainage condition before closing it out.
  2. Label and retain the completed readiness package. Store it for reference and reuse throughout the rest of this book.

Common Inspection Review

Inspect the completed readiness package using the same checks regardless of which structure type was selected.

  • Property-limits check: confirm every flagged corner, setback, and easement is measured, recorded, and shown on the sketch.
  • Utility check: confirm the locate request was submitted and completed, and every marked line is recorded on the sketch before the structure outline was flagged.
  • Drainage and slope check: confirm slope and drainage were recorded at multiple points across the full build area.
  • Structure-type check: confirm at least two candidate types were compared and the final selection has a documented rationale.
  • Readiness check: confirm every section of the worksheet is complete and the package is signed and dated.

Complete the Site Inspection Worksheet in the Inspection Checklist section below before beginning Project 02.

Failure Points and Troubleshooting

Proposed Structure Location Encroaches on a Setback or Easement

Probable cause: the structure outline was flagged from an assumed property line, such as an existing fence, rather than from a confirmed corner and measured setback.
Confirmation: re-measure the setback distance from the confirmed property corner and compare it to the current jurisdictional requirement.
Correction: re-flag the structure outline from the confirmed corner and setback distance, adjusting the structure size or location as needed.
Reinspection: repeat the property-limits check on the corrected outline.

Structure Outline Was Flagged Before the Utility Locate Was Confirmed

Probable cause: the locate request was submitted late, or the structure outline was flagged while waiting for the locate results.
Confirmation: check the locate service's confirmation date and time against the date the structure outline was flagged.
Correction: remove the structure-outline flags, wait for the confirmed and marked locate, and re-flag the outline only after the marked utilities are recorded on the sketch.
Reinspection: repeat the utility check on the corrected sequence.

Drainage or Slope Was Recorded at Only One Point

Probable cause: the slope check was performed at a single, convenient corner instead of across the full build area.
Confirmation: compare the recorded worksheet entries against the number of points required by Section G, Step 4.
Correction: return to the site and record slope and drainage at each corner and the center of the build area.
Reinspection: repeat the drainage and slope check on the completed set of points.

Structure Type Was Selected Without a Documented Comparison

Probable cause: a structure type was chosen based on preference alone, without recording how it was weighed against site conditions or a second candidate type.
Confirmation: review the worksheet for a written rationale and at least one alternative structure type considered.
Correction: document at least two candidate types, compare them against the recorded site conditions, and record the rationale for the final selection.
Reinspection: repeat the structure-type check on the completed rationale.

Flag Colors Were Mixed Between Categories

Probable cause: property, utility, and structure-outline flags were not assigned distinct colors before the site walk began.
Confirmation: inspect the site sketch and physical flags for consistent color use by category.
Correction: re-flag the site using three distinct colors, updating the sketch legend to match.
Reinspection: repeat the property-limits and utility checks on the corrected flagging.

Readiness Package Is Missing Photographs or Worksheet Entries

Probable cause: photographs or worksheet entries were deferred until after the site walk and some conditions were not recorded before flags were removed or conditions changed.
Confirmation: review the readiness package against the Assessed Inspection Schedule for missing components.
Correction: return to the site to record any missing condition still visible, or note on the worksheet any condition that must be reconfirmed before Project 02 begins.
Reinspection: repeat the readiness check on the completed package.
