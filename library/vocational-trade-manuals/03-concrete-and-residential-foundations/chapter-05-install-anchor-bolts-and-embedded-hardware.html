Job Overview

This project qualifies you to lay out, place, stabilize, and verify anchor bolts and embedded hardware designated by the approved structural plan for the current concrete placement. The finished work product is a completed Embedded-Hardware Placement Record, a set of correctly located and aligned anchors and hardware verified against the structural plan and template, and photographs of the final placement taken before concealment or curing protection begins.

You will review the embed schedule and product instructions, confirm which hardware belongs in the current concrete element, inspect and stage the hardware, establish reference lines and mark every location, assemble and use placement templates, secure the hardware so it survives placement and consolidation, monitor and adjust wet-set items while the concrete remains workable, and verify final projection and spacing before the concrete hardens.

This project applies to footing, pier, grade-beam, and stem-wall placements that include designated embedded hardware. It follows directly from the reinforcement work in Project 03 and 04 and precedes the stem-wall form construction in Project 06. Embedded hardware installed incorrectly or omitted at this stage cannot be corrected later without cutting, drilling, or epoxy-anchoring into hardened concrete, so this project's verification steps must be completed before the concrete loses workability.

Skills You Will Demonstrate

  • FN-025 — Locate anchor bolts and embedded hardware from the verified reference controls.
  • FN-026 — Set anchor-bolt and hardware spacing according to the approved structural plan or product template.
  • FN-027 — Install bolts and hardware in wet concrete without displacing reinforcement or losing alignment.
  • FN-028 — Align embedded hardware for plumb, orientation, and elevation.
  • FN-029 — Verify projection height against the defined reference surface.
  • FN-030 — Inspect and document hardware placement before the concrete cures.

Where This Skill Fits

Project 01 gave you a verified layout with protected horizontal and vertical controls. Projects 02 through 04 used those controls to build footing forms, install reinforcement, and place footing concrete. This project uses the same protected references to locate embedded hardware in whichever concrete element is currently being placed — footing, pier, grade beam, or stem wall — and ties that hardware placement directly to the structural connections framing will depend on in Book 4.

The wet-set discipline taught here — installing hardware, monitoring it through consolidation, and rechecking it while the concrete remains workable — is the same discipline used again in Project 07 for stem-wall anchor bolts and hold-downs, and in Project 19 when sill plates are drilled and set over the anchors installed here.

Scope Control

This project applies only to anchor bolts, dowels, post bases, hold-down anchors, straps, sleeves, plates, or other hardware specifically detailed for installation in the current footing, pier, grade-beam, or wall placement.

Do not place sill-plate anchor bolts in strip footings unless the approved structural documents explicitly require that condition. Anchor hardware intended for the top of a stem wall must be installed during the applicable stem-wall placement in Project 07, not during this footing-stage project, unless the structural plan specifically designates footing-stage installation.

This project does not authorize the learner to:

  • substitute a different anchor size, type, spacing, or embedment depth than the approved structural plan specifies
  • heat, bend, relocate, cut, or field-modify an embedded anchor without written authorization from the structural designer or engineer
  • install hardware intended for a later concrete element into the current placement, or defer hardware required in the current element to a later placement
  • proceed with concrete placement when required hardware has not passed the pre-placement inspection in Step 10

Trade Note

Embedded hardware is the one component in this book that becomes essentially unrecoverable once the concrete hardens around it in the wrong location. A misaligned footing dimension can sometimes be corrected before concrete is placed; a missing anchor bolt in a hardened footing typically requires an engineer-approved retrofit anchor. Treat the pre-placement check in Step 10 and the wet-concrete monitoring in Steps 11–14 as the two points in this project where a mistake is still cheap to fix.

Setup Standard

Do not permit concrete discharge into the current form until every hardware location required by the structural plan has been marked, every template has been assembled and dry-fit, and the pre-placement hardware inspection in Step 10 has passed. Hardware added after concrete placement begins is a wet-set exception, not the planned procedure, and must still meet every spacing, projection, and alignment standard in this project.

Shop Tip

Secure every template to the form or reinforcement before concrete arrives rather than holding it by hand during discharge. A template that shifts during placement can move every bolt it holds out of tolerance at once, while a template fastened to the form or a stable reinforcement point holds its spacing through consolidation.

Common Field Fundamentals

Every embedded-hardware placement in this book, whether it is a footing dowel installed here or a stem-wall anchor bolt installed in Project 07, follows the same sequence: confirm which hardware belongs in the current element, mark every location from the verified references, secure the hardware so it survives placement, monitor and adjust it while the concrete remains workable, and verify final position before the concrete hardens. Learning that sequence at the footing stage means every later embedded-hardware placement in this book starts from a method you already trust.

Scope Discipline: Right Hardware, Right Placement

Before marking a single location, separate the embed schedule into hardware required in the current concrete element and hardware designated for a later placement. Sill-plate anchor bolts belong at the top of the stem wall in Project 07 unless the structural plan explicitly requires footing installation. Installing the wrong hardware in the wrong element, or omitting hardware that belongs in the current element, cannot be corrected without a structural retrofit.

Measurement Cue — Mark Every Location From the Verified References

Establish centerlines, edges, reference corners, wall faces, post locations, and elevation controls from the same protected controls verified in Project 01 before marking a single hardware location. Verify spacing from corners, joints, openings, reinforcement, form faces, and other embedded items, not from a previously placed piece of hardware.

Critical Safety Cue — Working Around Wet Concrete and Placement Equipment

Do not reach into an area where a pump hose, vibrator, or discharge chute is actively operating to adjust hardware. Signal the placing crew and wait for a controlled pause before entering the placement area to check or adjust wet-set items. Wear wet-concrete-rated boots and gloves before contacting fresh concrete, and avoid prolonged skin contact with wet concrete.

Field Safety Cue — Sharp Hardware and Protruding Anchors

Cap or cover exposed anchor threads and protruding straps before foot traffic resumes in the placement area. Treat every uncapped anchor projection as a puncture hazard until protective caps or covers are installed.

Code Alert — Embedment and Field-Modification Authority

Required embedment depth, edge distance, and spacing for anchors and hold-downs are set by the approved structural plan and vary by jurisdiction and by design. Never estimate an embedment depth or substitute a different anchor diameter. Do not heat, bend, relocate, cut, or field-modify an embedded anchor without written authorization from the engineer or designer of record.

Manufacturer Instruction Cue — Proprietary Hardware and Templates

Post bases, hold-down anchors, and proprietary strap systems are governed by the manufacturer's placement instructions for embedment, edge distance, torque, and orientation. Assemble and use only the placement template supplied or specified for the installed product, and do not substitute a shop-made jig for a manufacturer-required template.

E. Preparation — Document Review and Scope Confirmation

  1. Review the approved structural plan, embed schedule, product instructions, templates, edge distances, spacing, projection, orientation, and inspection requirements. Confirm the complete hardware list before staging any material.
  2. Confirm which hardware belongs in the current concrete element and which hardware must be deferred to later placements. Mark deferred items clearly on the embed schedule and set them aside from the current staging area.
  3. Inspect all bolts, anchors, plates, straps, sleeves, templates, nuts, washers, coatings, and accessories for correct identification and condition. Reject any item that does not match the embed schedule or shows damage, corrosion, or incorrect coating.

F. Layout and Location Marking

  1. Establish centerlines, edges, reference corners, wall faces, post locations, and elevation controls. Use the same protected controls verified in Project 01.
  2. Mark every embedded-item location on forms, templates, or protected controls. Label each mark with the hardware item it corresponds to on the embed schedule.
  3. Verify spacing from corners, joints, openings, reinforcement, form faces, and other embedded items. Compare every marked location to the approved plan before assembling templates.

G. Template Assembly and Hardware Preparation

  1. Assemble manufacturer-provided or project-specific placement templates where required. Dry-fit each template against its marked location before concrete arrives.
  2. Install nuts, washers, caps, or protective coverings needed to control projection and preserve threads. Confirm each covering matches the manufacturer or structural-plan requirement.

H. Securing Hardware Before Placement

  1. Secure hardware to forms, reinforcement, templates, or independent supports using an approved method that does not violate structural details. Confirm the securing method does not relocate, bend, or damage adjacent reinforcement.
  2. Check each item for plumb, orientation, elevation, embedment, projection, and clearance before concrete placement. Complete and sign the pre-placement hardware inspection before authorizing concrete discharge.

I. Monitoring and Adjustment During Concrete Placement

  1. During concrete placement, monitor hardware for movement, rotation, uplift, or loss of alignment. Assign a spotter to watch embedded items continuously while discharge and consolidation proceed nearby.
  2. Consolidate concrete around embedded items without striking or displacing them. Direct vibration away from direct contact with hardware and templates.
  3. Recheck and adjust wet-set items only while the concrete remains workable and without creating voids or unauthorized disturbance. Confirm the concrete has not begun to stiffen before making any adjustment.

J. Final Verification and Documentation

  1. Verify final projection using the defined reference surface. Measure every item's projection and record it on the Embedded-Hardware Placement Record.
  2. Verify center-to-center spacing and alignment against the approved plan or template. Record any deviation and its magnitude.
  3. Clean exposed threads and hardware surfaces before concrete hardens. Remove cement paste from threads, plates, and connector surfaces while it remains workable.
  4. Mark any item that moved beyond tolerance and obtain the required correction direction. Contact the named engineer or designer before attempting an independent correction.
  5. Do not heat, bend, relocate, cut, or field-modify embedded anchors without written authorization. Document any authorized field modification with the source of the authorization.
  6. Photograph the final placement before concealment or curing protection obscures the work. Photograph each hardware group from a position that shows its location relative to a fixed reference.
  7. Complete the embedded-hardware inspection record and maintain protection throughout curing. Sign and date the release section before curing protection is installed per Project 09.

Common Hardware Inspection

Inspect every embedded item using the same checks regardless of hardware type.

  • Scope check: confirm the item belongs in the current concrete element per the embed schedule, not a later placement.
  • Location check: confirm centerline, edge distance, and spacing match the approved plan, measured from the verified reference controls.
  • Orientation check: confirm the item faces the direction shown in the structural plan or manufacturer instructions.
  • Plumb and alignment check: confirm vertical hardware remains plumb and aligned within tolerance.
  • Projection check: confirm final projection matches the approved plan, measured from the defined reference surface.

Complete the Embedded-Hardware Placement Record in the Inspection Checklist section below before authorizing curing protection to begin.

Failure Points and Troubleshooting

Anchor Bolt Spacing Does Not Match the Embed Schedule

Probable cause: the bolt was marked from a previously placed bolt instead of the reference corner or wall face, or the template was not dry-fit before use.
Confirmation: re-measure the spacing from the reference corner or wall face identified in the embed schedule and compare it to the template.
Correction: relocate the bolt to the position required by the approved plan while the concrete remains workable, or contact the engineer if the concrete has begun to stiffen.
Reinspection: repeat the spacing check for every bolt in the affected group.

Hardware Moved Out of Plumb During Consolidation

Probable cause: the vibrator contacted the hardware directly, or the securing method did not hold the item rigidly during placement.
Confirmation: check plumb with a level immediately and compare to the pre-placement inspection record.
Correction: straighten the item while the concrete remains workable, without creating a void around the embedment; if the concrete has begun to stiffen, contact the engineer before disturbing it further.
Reinspection: repeat the plumb and alignment check after correction and again after final set.

Projection Height Does Not Match the Structural Plan

Probable cause: the reference surface was misidentified, the concrete surface finished at a different elevation than planned, or the hardware settled during consolidation.
Confirmation: re-verify the defined reference surface and re-measure projection from that surface.
Correction: raise or reset the item while the concrete remains workable; if correction is not possible before the concrete hardens, contact the engineer for a retrofit or shim solution.
Reinspection: repeat the projection check for every item on the same template or placement group.

Wrong Hardware Installed in the Current Placement

Probable cause: sill-plate anchor bolts or other later-stage hardware were staged and installed in the current footing or wall placement without confirming the embed schedule's scope designation.
Confirmation: compare every installed item against the embed schedule's designated placement stage.
Correction: stop work and obtain engineer or designer direction; do not remove hardware from hardened concrete without authorization.
Reinspection: repeat the scope check for the full hardware group before authorizing curing protection.

Hardware Displaced or Rotated by Reinforcement Contact

Probable cause: reinforcement placed or adjusted after hardware installation contacted and shifted the embedded item.
Confirmation: compare current hardware position and orientation to the pre-placement inspection record.
Correction: reposition the hardware while the concrete remains workable and confirm reinforcement clearance; if the concrete has stiffened, contact the engineer.
Reinspection: repeat the location and orientation check for the affected item and any adjacent reinforcement.

Cleanup

This project ends with verified, protected hardware and a completed placement record, not a torn-down worksite.

  • Confirm every exposed thread, protruding strap, or sharp hardware edge is capped or covered before foot traffic resumes
  • Remove unused templates, packaging, and hardware offcuts from the placement area
  • Store unused hardware and templates for the next applicable placement
  • Protect installed hardware from impact, contamination, and disturbance through curing per Project 09
  • Return the Embedded-Hardware Placement Record and photographs to the project record

Performance Check

With every hardware item located, spaced, secured, aligned, and verified for projection, review the finished Embedded-Hardware Placement Record against the Acceptance Standards table and confirm every item belongs in the current concrete element per the embed schedule, every location and spacing matches the approved plan within tolerance, every item is plumb and correctly oriented, every projection matches the approved plan as measured from the defined reference surface, and every out-of-tolerance item has been corrected or escalated to the engineer or designer. Complete the Embedded-Hardware Placement Record and resolve any unresolved deviation before authorizing curing protection to begin. If an item fails, identify whether the error occurred in document review, scope confirmation, layout, template use, securing, or wet-concrete monitoring before recording the correction and re-running the affected check.
