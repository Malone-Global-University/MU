03 – Concrete and Residential Foundations

This field manual teaches the complete construction of residential foundation systems, from layout verification through footing construction, stem walls, waterproofing, slab installation, and final sill plate installation. Students complete a sequence of practical projects that mirror professional residential foundation construction.

Overview

Residential foundations support every structure built above them. This manual develops the practical skills required to build durable, code-compliant concrete foundation systems. Students learn layout verification, form construction, reinforcement placement, concrete placement and finishing, waterproofing, drainage, slab construction, and foundation repair before completing an integrated capstone project that brings every skill together.

  2. Project 2: Build Residential Footing Forms

  6. Project 6: Build Stem Wall Forms
  7. Project 7: Reinforce and Pour Stem Walls
  8. Project 8: Strip Forms and Repair Concrete Defects
  9. Project 9: Cure and Protect New Concrete
  10. Project 10: Waterproof and Dampproof Foundation Walls
  11. Project 11: Install Foundation Drain Tile
  12. Project 12: Install Foundation Insulation and Protection Board
  13. Project 13: Backfill Around the Foundation
  14. Project 14: Build Slab Forms and Control Lines
  15. Project 15: Install Slab Base, Vapor Barrier, and Reinforcement

  17. Project 17: Finish a Concrete Slab

  19. Project 19: Install Sill Sealer and Sill Plates
  20. Project 20: Capstone — Build a Complete Residential Foundation System

Learning Outcomes

  • Verify foundation layouts, elevations, and dimensions before construction.
  • Construct footing, stem wall, and slab formwork to professional standards.
  • Install reinforcing steel, anchor bolts, and embedded hardware correctly.
  • Place, consolidate, screed, finish, cure, and protect structural concrete.
  • Install waterproofing, drainage systems, insulation, and foundation protection materials.
  • Complete slab-on-grade construction using proper reinforcement and finishing techniques.
  • Perform foundation repairs, backfilling, and final preparation for framing.
  • Construct a complete residential concrete foundation system that meets professional building practices.
