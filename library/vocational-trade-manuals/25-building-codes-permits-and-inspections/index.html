25-Building Codes, Permits and Inspections

This field manual provides practical instruction in residential building codes, permitting procedures, inspection preparation, and compliance documentation. Students learn how to navigate code requirements, prepare permit materials, coordinate inspections, and verify that construction work meets applicable standards.

Overview

Building Codes, Permits and Inspections develops the knowledge needed to manage residential construction compliance from planning through final approval. Beginning with code reference organization and permit requirements, students progress through zoning checks, foundation and framing inspections, building envelope requirements, mechanical system inspections, energy compliance, life-safety requirements, and final inspection preparation.

Learning Outcomes

  • Organize building code references and permit documentation.
  • Determine when residential construction projects require permits.
  • Prepare basic residential permit application materials.
  • Interpret code notes and compliance requirements on plans.
  • Evaluate site setbacks and zoning considerations.
  • Prepare construction work for foundation and framing inspections.
  • Coordinate roof, envelope, plumbing, electrical, and HVAC inspections.
  • Apply energy code and life-safety compliance requirements.
  • Prepare inspection checklists for residential construction phases.
  • Support successful project completion through code compliance.
