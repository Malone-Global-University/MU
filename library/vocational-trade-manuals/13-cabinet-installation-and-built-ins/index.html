13-Cabinet Installation and Built-Ins

This field manual provides practical, project-based instruction for residential cabinet installation, built-in storage systems, and finish carpentry applications. Students learn to plan layouts, install cabinets, fit components, build storage solutions, and complete professional cabinet and built-in projects.

Overview

Cabinet Installation and Built-Ins emphasizes hands-on skill development through complete residential storage and cabinetry projects. Beginning with layout, inspection, and preparation, students progress through base and wall cabinet installation, countertop installation, cabinet accessories, closet systems, built-in furniture, repairs, refinishing, and a complete cabinet and storage system capstone project.

Learning Outcomes

  • Inspect job conditions and plan cabinet installation layouts.
  • Establish accurate cabinet reference lines and measurements.
  • Install base, wall, tall, and pantry cabinets securely.
  • Adapt cabinet installations for uneven floors and walls.
  • Cut cabinet openings for plumbing and electrical services.
  • Install fillers, end panels, toe kicks, and countertop systems.
  • Install cabinet doors, drawers, hardware, pulls, and knobs.
  • Build and install shelving, closet organizers, and built-in storage.
  • Repair, adjust, and refinish existing cabinet systems.
  • Complete a full cabinet installation and built-in storage project.
