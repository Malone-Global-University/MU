Job Overview

This project qualifies you to calculate stair rise and run from the approved stair dimensions, lay out and cut a matched pattern stringer, trace and cut the remaining stringers from that pattern, and install a level, supported, aligned set of stringers in the floor opening and landing framed in Project 17. The finished work product is a matched set of installed stair stringers, temporarily guarded, with a completed stair calculation record and inspection results.

You will measure the actual floor-to-floor or floor-to-landing elevation difference, determine the approved number of risers and uniform unit rise, determine the approved unit run and total run, account for tread thickness and top and bottom bearing conditions, lay out and cut one pattern stringer, verify it at the opening, trace and cut the remaining stringers from that verified pattern, install top and bottom bearing supports, set and secure all stringers at the approved spacing, and verify rise uniformity, run uniformity, headroom, and support before the stair is used.

This project follows directly from Project 17, which framed and guarded the stair opening and landing. It uses the opening dimensions, landing elevation, and structural supports established there, and it produces the stringer set that carries treads, risers, and guards in later finish work.

Skills You Will Demonstrate

  • FR-103 — Calculate total rise, number of risers, unit rise, unit run, and total run from the approved stair dimensions and field measurements.
  • FR-104 — Lay out a pattern stringer using a framing square and stair gauges, marking plumb cuts, tread and riser notches, and the bottom cut from defined references.
  • FR-105 — Cut the pattern stringer and the matching stringers without overcutting inside corners, and finish inside corners using the approved hand-tool method.
  • FR-106 — Install top and bottom bearing supports and set stringers at the approved spacing, fully supported and aligned.
  • FR-107 — Verify every tread position for level and every riser position for uniformity across the installed stringer set.
  • FR-108 — Verify total stair width, rise uniformity, run uniformity, headroom, landing clearance, and connector installation, and document the results.

Where This Skill Fits

The stair calculation performed in this project controls every tread and riser a user will step on for the life of the structure. An error of a fraction of an inch in unit rise, repeated across a full flight, becomes a trip hazard that is difficult to detect by eye and expensive to correct once treads and finish stringers are installed. This project treats the calculation, the pattern stringer, and the verification of that pattern before it is traced onto the remaining stock as three separate, sequential checkpoints, because an error caught at the pattern stage costs one board; the same error caught after all stringers are cut costs the full stringer set.

The framing square and stair-gauge layout method taught here is the same method referenced anywhere else in this book that a site-built stair or stair-like component must be laid out from a calculated rise and run.

Scope and Intended Use

This project covers:

  • calculating total rise, riser count, unit rise, unit run, and total run for a basic straight residential stair from the approved stair dimensions and field-measured elevations
  • laying out, cutting, and verifying one pattern stringer
  • tracing and cutting matching stringers from the verified pattern
  • installing top and bottom bearing supports and setting the stringer set at the approved spacing
  • verifying tread level, riser uniformity, headroom, and landing clearance across the installed set

This project does not authorize the learner to:

  • select total rise, riser count, unit rise, unit run, stringer size, species, or spacing without the approved stair dimensions and project specification; these values come from the plans, adopted code provisions applicable to the project, or a qualified designer, not from this project's general description
  • trace and cut matching stringers from a pattern stringer that has not been test-fitted and verified at the opening
  • permit use of the stair by anyone before required temporary guards are installed and the stair has passed inspection
  • alter the stair opening, landing, or headers framed in Project 17 to accommodate a stringer layout error

Any discrepancy between the field-measured elevation, the approved stair dimensions, and the calculated rise and run must be resolved before cutting the pattern stringer.

Trade Note

Every riser in a flight of stairs must be the same height within the tolerance stated on the approved stair specification. This is verified by dividing the exact field-measured total rise by a whole number of risers so that no remainder is carried forward or absorbed into a single riser at the top or bottom of the flight.

Setup Standard

Do not trace and cut the matching stringers until the pattern stringer has been test-fitted at the opening and its top connection, bottom bearing, total rise, run, headroom, and landing relationship are verified (Step 15). A pattern error carried into every matching stringer requires the full set to be recut.

Shop Tip

Set the stair gauges on the framing square to the exact unit rise and unit run before marking the first tread and riser line, and do not adjust them again until the pattern stringer is fully marked. Resetting the gauges partway through layout is a common source of a single mismatched step in an otherwise uniform stringer.

Reference Conventions

Establish these references before any layout begins, and do not change them once the pattern stringer has been marked.

TermDefinition
Total riseThe exact field-measured vertical distance from the lower finished surface to the upper finished surface the stair connects, measured from the same reference surfaces used by the approved stair design
Unit riseThe height of one riser, calculated by dividing the total rise by the approved number of risers
Unit runThe horizontal depth of one tread, taken from the approved stair specification
Total runThe unit run multiplied by the number of treads, measured horizontally from the first riser to the last riser

Throat depthThe remaining solid width of the stringer measured from the inside corner of a tread/riser notch to the stringer's opposite edge

Common Field Fundamentals

Every stringer layout in this project follows the same sequence: measure the actual elevation, calculate rise and run, lay out and verify one pattern, then trace and cut matching stringers from that verified pattern. Skipping the verification step and cutting all stringers directly from a calculated but untested pattern is the most common cause of a stair that must be substantially rebuilt.

Rise, Run, and the Pattern-Verification Sequence

Total rise is measured, not assumed, at the exact location the stair will be built, because floor and landing elevations vary from the plan by ordinary construction tolerance. The number of risers is fixed by the approved stair specification; dividing the field-measured total rise by that fixed riser count produces the unit rise actually used for layout. A pattern stringer that checks out on paper but has not been set in the opening can still fail on headroom, top connection, or landing clearance — conditions the calculation alone does not confirm.

Measurement Cue — Uniform Rise Across the Full Flight

Every riser in the completed flight, including the top riser formed by the floor or landing surface and the bottom riser formed by the finished lower surface, must fall within the tolerance stated on the approved stair specification. Include finish flooring and tread-thickness allowances in the calculation before marking the pattern stringer, not after.

Critical Safety Cue — Verify the Pattern Before Tracing

Do not trace and cut matching stringers until the pattern stringer has been test-fitted at the opening and its top connection, bottom bearing, total rise, run, headroom, and landing relationship are confirmed against the approved specification. An unverified pattern error is repeated in every matching stringer traced from it.

Field Safety Cue — Open Opening Hazards

Maintain temporary guards at the stair opening throughout layout, cutting, and installation. Do not remove required guards to test-fit a stringer without installing an equivalent temporary barrier at the affected edge for the duration of the fit check.

Code Alert — Rise, Run, and Headroom Requirements Vary

Maximum and minimum unit rise, minimum unit run, riser uniformity tolerance, and minimum headroom vary by jurisdiction and stair use. Verify the applicable requirements and the approved project specification before finalizing the stair calculation.

E. Calculation (Steps 1–6)

  1. Review the approved stair dimensions, total rise, available run, stair width, tread and riser requirements, landing elevations, headroom, stringer material, support, and connection details. Confirm the document revision against the project's document-control log.
  2. Measure the actual finished-floor or framing elevation difference from the same reference surfaces used by the design. Record the field-measured total rise on the stair calculation record.
  3. Determine the approved number of risers and uniform unit rise. Divide the field-measured total rise by the approved number of risers and record the result.
  4. Determine the approved unit run and total stair run. Multiply the approved unit run by the number of treads and record the calculated total run.
  5. Account for tread thickness, finished flooring, top connection, bottom bearing, and landing conditions. Adjust the top and bottom riser layout on the calculation record to reflect these conditions before marking any stock.
  6. Record the complete stair calculation before marking material. Confirm every entry on the stair calculation record is complete and consistent with the approved specification.

F. Pattern Stringer Layout and Cutting (Steps 7–16)

  1. Select straight, sound stringer stock of the specified size, grade, species, and treatment. Sight each piece for crown, bow, and twist using the straightedge before selection.
  2. Reject stock with defects in the cut zones or conditions prohibited by the project specification. Set aside and label rejected stock; do not use it for the pattern or matching stringers.
  3. Mark one pattern stringer using a framing square and approved stair gauges or layout method. Set the stair gauges to the calculated unit rise and unit run before marking the first line.
  4. Mark the top plumb cut, tread and riser notches, bottom cut, and required landing adjustments. Mark every tread and riser line from the same gauge setting without readjusting the gauges mid-layout.
  5. Verify remaining throat depth and all layout dimensions before cutting. Measure the throat depth at the narrowest notch and compare it with the minimum stated in the approved specification.
  6. Cut the pattern stringer without overcutting inside corners. Stop each circular-saw cut at the marked inside-corner point.
  7. Finish inside corners using the approved hand-tool method. Complete each inside corner with a handsaw or jigsaw so no overcut extends past the marked lines.
  8. Test-fit the pattern stringer at the opening. Set the pattern stringer in position against the header and lower bearing surface without fastening it.
  9. Verify top connection, bottom bearing, total rise, run, headroom, and landing relationship. Measure headroom from the nose line of each tread position to the framing above, and compare against the minimum in the approved specification.
  10. Correct the pattern before tracing additional stringers. If a discrepancy is found, correct the pattern stringer or obtain design direction before proceeding to Section G.

G. Matching Stringers and Installation (Steps 17–23)

  1. Trace and cut the required matching stringers from the approved pattern. Clamp the verified pattern to each piece of matching stock and trace every line before cutting.
  2. Compare all stringers for matched rise, run, top, and bottom geometry. Lay all stringers together on a flat surface and check tread and riser line alignment across the set.
  3. Install top hangers, ledgers, bearing blocks, posts, or other specified supports. Install these supports at the header and lower landing before setting any stringer.
  4. Set and temporarily secure the outside stringers. Position the outside stringers at the approved stair-width dimension and temporarily fasten each end.
  5. Install intermediate stringers at the approved spacing. Use the chalk line and tape measure to confirm spacing before permanently fastening any stringer.
  6. Verify stringers remain parallel, aligned, fully supported, and consistently spaced. Check spacing at the top, middle, and bottom of the stringer set.
  7. Install temporary or permanent treads only as permitted by the project sequence. Confirm the approved sequence before placing any tread material on the installed stringers.

H. Verification and Documentation (Steps 24–27)

  1. Check every tread position for level and every riser position for uniformity. Record each measurement on the stair calculation record.
  2. Verify total stair width, rise uniformity, run uniformity, headroom, landing clearance, support, and connector installation. Compare every result against the Acceptance Standards table in Section D.
  3. Install required temporary guards and prohibit use until the stair has passed inspection. Confirm guards are in place at every open edge before leaving the installed stringer set unattended.
  4. Record calculations, pattern dimensions, final measurements, connections, and inspection results. Complete every entry on the stair calculation record and attach it to the project's permanent documentation.

Common Verification Inspection

Inspect the completed stringer set using the same checks regardless of stair width or stringer count.

  • Calculation check: confirm the stair calculation record shows a field-measured total rise, approved riser count, calculated unit rise, approved unit run, and calculated total run consistent with the approved specification.
  • Pattern check: confirm the pattern stringer was test-fitted and verified before any matching stringer was traced or cut.
  • Matching check: confirm every matching stringer agrees with the verified pattern within the approved tolerance.
  • Support check: confirm top and bottom bearing supports match the approved connection detail and provide full bearing contact.
  • Alignment check: confirm stringer spacing, level, and parallelism at the top, middle, and bottom of the installed set.
  • Guard check: confirm required temporary guards are installed and use of the stair is prohibited until inspection is complete.

Complete the Inspection Checklist below before treads, risers, or finish stringers are installed.

Failure Points and Troubleshooting

Riser Heights Are Not Uniform Across the Flight

Probable cause: the total rise was divided by an incorrect riser count, the stair gauges shifted during pattern layout, or tread thickness and finish-flooring allowances were not included before marking the pattern.
Confirmation: re-measure the field total rise, recalculate unit rise using the approved riser count, and compare every marked riser line on the pattern stringer against that value.
Correction: if the pattern is in error, correct it and re-verify before tracing further stringers; if stringers were already cut from an incorrect pattern, do not adjust individual risers in the field — recut the affected stringers from a corrected, re-verified pattern.
Reinspection: repeat the riser-uniformity check across the full installed set.

Throat Depth Is Below the Minimum

Probable cause: the stringer stock width is undersized for the calculated rise and run, or the tread/riser notch layout was marked too deep.
Confirmation: measure the remaining throat depth at the narrowest notch on the pattern stringer and compare it with the minimum stated in the approved specification.
Correction: obtain wider stringer stock or a revised stringer detail from the design professional; do not reduce the notch depth in a way that changes the calculated unit rise or unit run without an approved revision.
Reinspection: repeat the throat-depth check on the corrected pattern before tracing matching stringers.

Matching Stringers Do Not Agree With the Pattern

Probable cause: the pattern shifted during tracing, the tracing tool was not held tight to the pattern's reference edge, or a different stair-gauge setting was used to mark a "matching" stringer independently.
Confirmation: lay the suspect stringer against the verified pattern on a flat surface and measure the difference at each tread and riser line.
Correction: discard the mismatched stringer and re-trace a replacement directly from the verified pattern, clamped securely for the full trace.
Reinspection: repeat the stringer-matching check across the full set before installation.

Headroom Is Below the Approved Minimum

Probable cause: the framed opening from Project 17 does not match the assumption used in the stair calculation, or the total run was shortened without adjusting the opening.
Confirmation: measure headroom from the nose line of each tread position to the framing above and compare it with the minimum stated in the approved specification.
Correction: obtain a revised stair or opening detail from the design professional; do not reduce a riser count or unit rise in the field to gain headroom without an approved revision.
Reinspection: repeat the headroom check at every tread position once the revised detail is incorporated.

Stringer Spacing Is Inconsistent Top to Bottom

Probable cause: intermediate stringers were fastened before spacing was confirmed at all three check points, or the top or bottom bearing support was not installed square to the opening.
Confirmation: measure spacing between each pair of stringers at the top, middle, and bottom of the installed set and compare against the approved specification.
Correction: loosen the affected stringer's temporary fasteners, realign it using the chalk line reference, and re-fasten; correct the bearing support first if it is not square to the opening.
Reinspection: repeat the spacing check at all three points after correction.

Top or Bottom Bearing Does Not Provide Full Contact

Probable cause: the bearing support was installed at the wrong elevation, or the pattern stringer's top or bottom cut does not match the as-built header or landing condition.
Confirmation: inspect the bearing surface for gaps or partial contact at each stringer end and compare the as-built condition with the approved connection detail.
Correction: correct the bearing support elevation or obtain a revised connection detail from the design professional; do not shim a structural bearing connection without an approved method.
Reinspection: repeat the bearing check at every stringer end after correction.

Cleanup

This project ends with an installed, guarded stringer set and a completed calculation record, not with finish treads, risers, or guards installed.

  • Remove offcuts, rejected stock, and sawdust from the work area and the stair opening
  • Store any unused stringer stock protected from weather and moisture
  • Confirm temporary guards remain installed and effective at every open edge
  • Store the completed stair calculation record with the project's permanent documentation
  • Communicate any open discrepancy and its required decision to the responsible design professional or general contractor before treads, risers, or finish guards are installed

Field Terms

Total Rise
The field-measured vertical distance the stair connects, from the same reference surfaces used by the approved design.
Unit Rise
The height of one riser, calculated by dividing total rise by the approved riser count.
Unit Run
The horizontal depth of one tread, taken from the approved stair specification.
Total Run
The unit run multiplied by the number of treads.
Pattern Stringer
The single stringer laid out, cut, and verified first, used as the tracing reference for the remaining stringers.
Throat Depth
The remaining solid width of a stringer at a tread/riser notch.
Stair Gauges
Adjustable fittings clamped to a framing square to hold a consistent unit-rise and unit-run setting during layout.
Plumb Cut
The vertical cut at the top of a stringer that meets the header or upper bearing surface.
Seat Cut
The horizontal cut at the bottom of a stringer that bears on the lower floor or landing surface.
Headroom
The clear vertical distance from a tread's nose line to the framing or finish surface above it.
