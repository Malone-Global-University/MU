Job Overview

This project qualifies you to frame one approved simple gable roof using site-built common rafters and a ridge board. The finished work product is a complete, plumb, aligned rafter system — ridge board set at the calculated elevation, matched pattern-cut common rafters installed at the approved spacing, and all specified ties, blocking, outlookers, and gable framing in place — ready for roof sheathing in Project 16.

You will obtain the approved roof plan and its span, run, pitch, ridge height, overhang, rafter size, spacing, and connector data; lay out the roof centerline and eave lines; calculate or obtain the common-rafter length using the approved method and supplied project data; lay out, cut, and test-fit one pattern rafter; correct the pattern before duplicating it; set the ridge board and the first rafter pairs; install remaining rafter pairs at the approved spacing; and complete every specified tie, brace, and connector before signing off the roof-geometry inspection.

This project follows Project 13, Frame Ceiling Joists, which must be complete, inspected, and released before roof framing begins. It uses the completed exterior wall system from Chapters 7 through 12 as its bearing surface, and it produces the roof frame that Project 16, Install Roof Sheathing, will deck.

Skills You Will Demonstrate

  • FR-079 — Calculate rafter layout, including common-rafter run and length, from the approved roof plan and supplied project data.
  • FR-080 — Mark rafter cuts, including the ridge plumb cut, birdsmouth seat cut, heel location, and tail cut, from defined references on a pattern rafter.
  • FR-081 — Cut common rafters matching the verified pattern, with consistent orientation and legible labeling.
  • FR-082 — Install the ridge board at the calculated elevation and support it through the rafter-setting sequence.
  • FR-083 — Set rafter pairs at the approved spacing, seat them fully at the ridge and wall bearing, and install specified ties, straps, blocking, and gable framing.
  • FR-084 — Inspect roof geometry, including rafter spacing, seat bearing, ridge contact, roof plane, overhang, pitch, and connection completeness.

Where This Skill Fits

The rafter system framed here carries every later roof component. Roof sheathing in Project 16 is fastened directly to these rafters, and any error in ridge height, rafter spacing, or seat-cut depth here becomes a sheathing, ventilation, or fascia problem later. The pattern-rafter method taught in this project — lay out one rafter, verify it at the actual roof before cutting the rest — is the controlling method for every site-built rafter application in this book and protects the entire roof from a single layout mistake being repeated across every rafter.

This is one of two roof-framing methods presented in this book. Project 15 frames the same category of roof using prefabricated trusses. The two methods are not combined on the same roof unless an engineered transition detail specifically permits it, and this project must not be treated as interchangeable with truss installation.

Scope and Intended Use

This project covers:

  • reviewing the approved roof plan for span, run, pitch, ridge height, overhang, rafter size, spacing, ties, and connector requirements
  • confirming that the selected structure is designed for site-built rafters, not the prefabricated truss structure used in Project 15
  • calculating or obtaining common-rafter run and length using the approved method and supplied project data
  • laying out, cutting, and verifying one pattern rafter before duplicating it
  • installing the ridge board and setting rafter pairs at the approved spacing
  • installing ties, clips, straps, blocking, outlookers, and gable framing specified by the plan
  • inspecting completed roof geometry against the approved plan

This project does not authorize the learner to:

  • select rafter size, span, spacing, or pitch without an approved roof plan, prescriptive code table, or qualified designer's calculation
  • combine site-built rafters and prefabricated trusses on the same roof without an engineered transition detail
  • remove ceiling-joist or rafter-tie thrust resistance before the permanent connection is complete
  • work on the roof frame without the fall-protection, controlled-access, and material-restraint measures identified in Section G and the Field Fundamentals cues below

Any wall-top condition, span, or plan discrepancy discovered during layout must stop rafter cutting in the affected area until direction is obtained from the responsible design professional.

Trade Note

Every common rafter in this project is cut to match one verified pattern rafter, not to an independently remeasured length. Cutting each rafter from a fresh measurement, rather than tracing the corrected pattern, is the most common source of an uneven roof plane in site-built rafter framing.

Setup Standard

Do not cut the first common rafter until the pattern rafter has been test-fit at both roof sides and verified for ridge height, wall bearing, overhang, plumb cuts, and symmetry (Steps 10–12). A pattern rafter that has not been corrected and verified must not be used to mark additional rafters.

Shop Tip

Label the verified pattern rafter clearly and set it aside as the tracing master. Do not return it to general stock or use it as an installed rafter — if it is damaged or lost partway through cutting the remaining rafters, the layout and verification steps must be repeated in full.

Common Field Fundamentals

Every site-built rafter framed in this project follows the same sequence: establish the roof centerline and eave lines, calculate the rafter run and length from the approved plan data, lay out and verify one pattern rafter, duplicate the pattern, set the ridge and first rafter pairs, install remaining pairs and specified restraint, and inspect the finished roof geometry. Verifying the pattern before duplicating it protects the entire roof from a single layout error being repeated across every rafter.

Common-Rafter Length and the Pattern Method

The common-rafter length is derived from the run and the pitch stated on the approved roof plan, using the framing-square or rafter-calculator method supplied by the project specification. Do not select a rafter length by measuring an assumed roof plane or by matching an existing unverified rafter. The pattern rafter is the only rafter cut before test-fitting; every other rafter is traced from the pattern only after the pattern has been corrected and verified at both roof sides.

Measurement Cue — Verify the Birdsmouth Depth Before Cutting Further

Confirm that the birdsmouth seat cut does not exceed the approved depth limit stated on the roof plan or supplied project data before cutting the heel and seat cuts on the pattern rafter. An oversized birdsmouth reduces the rafter's effective depth at its most heavily loaded point and is a stop-work condition until the pattern is corrected.

Critical Safety Cue — Ridge and Rafter-Pair Support

Support the ridge board with temporary posts or bracing at the calculated elevation before releasing workers to set rafter pairs. Do not remove ridge support or release the first rafter pair from temporary restraint until the pair is fully seated at the wall bearing and fastened at the ridge. An unsupported ridge or an unrestrained rafter pair can shift or fall under its own weight or wind load.

Field Safety Cue — Working at Wall-Top and Roof Height

Use the fall-protection method identified in the project's fall-protection plan for all layout, cutting, and setting work performed at wall-top or roof height. Maintain a controlled exclusion zone below the work area during rafter setting and material handling, and clear cut-offs and loose fasteners from wall tops and roof surfaces before moving to the next rafter bay.

Code Alert — Prescriptive Rafter Tables and Thrust Resistance

Rafter size, span, spacing, and the required ceiling-joist or rafter-tie thrust-resistance connection vary by jurisdiction, pitch, and loading condition. Confirm that the rafter size and tie requirement on the approved roof plan match the applicable prescriptive table or engineered design before framing, and do not substitute an unapproved rafter size or omit a specified tie or strap.

G. Reference Layout and Rafter Calculation (Steps 1–7)

  1. Review the approved roof plan, span, run, pitch, ridge height, overhang, rafter size, spacing, ties, ceiling framing, connectors, and temporary-bracing plan. Record every value on the Roof Framing Data Record before layout begins.
  2. Confirm that the selected structure is designed for site-built rafters and is not the truss structure used in Chapter 15. Verify this against the approved plan's structure identification before proceeding.
  3. Verify wall dimensions, wall-top alignment, ceiling-joist or rafter-tie installation, and bearing readiness. Confirm Project 13 has passed its required inspection.
  4. Establish the roof centerline, ridge position, rafter-layout reference, and eave lines. Strike the roof centerline on the ceiling framing and mark the eave line on both wall plates from the approved overhang dimension.
  5. Obtain or calculate the common-rafter run and length using the approved method and supplied project data. Use the run, pitch, and rafter-length method specified by the project; do not estimate the length from an assumed roof plane.
  6. Determine plumb-cut, birdsmouth, tail, and ridge-shortening requirements from the controlled design. Record the ridge-shortening allowance for the ridge-board thickness before laying out the pattern rafter.
  7. Lay out one pattern rafter on straight stock using a framing square or approved calculator method. Select stock free of defects in the birdsmouth and plumb-cut zones.

H. Pattern Rafter Layout, Cutting, and Verification (Steps 8–12)

  1. Mark the ridge plumb cut, birdsmouth seat cut, heel location, and tail cut from defined references. Measure each cut line from the calculated rafter length and the approved overhang dimension.
  2. Verify that the birdsmouth does not exceed the approved depth. Compare the marked seat- and heel-cut depth against the plan's stated limit before cutting.
  3. Cut the pattern rafter and test-fit it at both roof sides. Support the pattern rafter at its ridge and wall-bearing locations without permanent fastening.
  4. Check ridge height, wall bearing, overhang, plumb cuts, and symmetry. Confirm the pattern matches the approved plan at both test locations before proceeding.
  5. Correct the pattern before using it to mark additional rafters. Re-cut or replace the pattern rafter if any check in Step 11 fails, and re-verify before duplication.

I. Duplicating Rafters and Setting the Ridge (Steps 13–16)

  1. Mark and cut matching rafters with consistent orientation and labels. Trace each rafter from the verified pattern using the same reference edge, and label each rafter with its plan location.
  2. Prepare the ridge board to the required length and support it at the calculated elevation. Install temporary ridge posts or bracing sized for the ridge board's weight and the roof span.
  3. Install the first opposing rafter pair and secure it temporarily. Seat each rafter fully at its birdsmouth and against the ridge board before installing temporary restraint.
  4. Check ridge plumb, pair alignment, bearing, and roof pitch. Verify the first pair against the approved plan values before continuing.

J. Setting Remaining Rafters and Installing Restraint (Steps 17–20)

  1. Install an opposing end pair and brace the ridge line. Add bracing sufficient to hold the ridge line straight through the remainder of the setting sequence.
  2. Set intermediate rafter pairs at the approved spacing. Confirm each pair's layout location on the wall plate and ridge before fastening.
  3. Install ties, clips, straps, blocking, outlookers, gable framing, and other specified restraint. Complete each connection per the Fastener and Hardware Schedule as each rafter pair is set.
  4. Maintain fall protection, controlled access, and material restraint throughout roof work. Verify the exclusion zone and fall-protection measures remain in place for every worker at wall-top or roof height.

K. Inspection and Correction (Steps 21–24)

  1. Sight and string the ridge and eave lines. Confirm both lines are straight within the approved plan tolerance.
  2. Verify rafter spacing, seat bearing, ridge contact, roof plane, overhang, pitch, and connection completeness. Complete the Roof Geometry Inspection Checklist below before covering any rafter with sheathing.
  3. Correct authorized defects without overcutting or unapproved sistering. Obtain direction from the design professional for any defect outside this project's authorized corrections.
  4. Record pattern-rafter dimensions, roof geometry, connections, and inspection results. Complete and sign the roof-framing record before releasing the structure for roof sheathing.

Common Verification Inspection

Inspect the completed roof frame using the same checks regardless of which bay or rafter pair it covers.

  • Documentation check: confirm every required plan, prescriptive table or engineered design, and connector schedule is current and referenced on the roof-framing record.
  • Geometry check: confirm ridge height, rafter length, spacing, birdsmouth bearing, overhang, and roof plane match the approved plan within its stated tolerance.
  • Connection check: confirm every rafter-to-ridge, rafter-to-wall, tie, strap, and gable-framing connection matches the approved connector schedule.
  • Restraint check: confirm required temporary bracing remains in place until permanent connections are complete, and permanent ties are installed at the specified spacing.
  • Discrepancy check: confirm every logged variance has an assigned decision, responsible authority, and, where authorized, a completed correction.

Complete the Roof Geometry Inspection Checklist in the Inspection Checklist section below before beginning Project 16.

Failure Points and Troubleshooting

Pattern Rafter Does Not Fit the Ridge or Wall Bearing

Probable cause: the rafter length was calculated from an incorrect run, the ridge-shortening allowance was omitted, or the plumb cuts were laid out from the wrong reference edge.
Confirmation: re-measure the actual run at the roof, re-check the calculated rafter length and ridge-shortening allowance against the approved method, and re-verify the reference edge used for layout.
Correction: re-lay out and re-cut the pattern rafter from corrected values; do not force an incorrect pattern into position or shim a permanent gap at the ridge or seat cut.
Reinspection: repeat the pattern test-fit at both roof sides before duplicating the rafter.

Birdsmouth Exceeds the Approved Depth

Probable cause: the seat cut or heel cut was marked without checking the plan's stated depth limit, or the rafter stock's actual depth differs from the assumed nominal size.
Confirmation: measure the actual birdsmouth depth on the pattern rafter and compare it against the approved plan limit.
Correction: discard the pattern rafter and re-lay out the seat cut within the approved depth using the actual rafter stock dimensions; do not use a rafter with an oversized birdsmouth.
Reinspection: repeat the birdsmouth depth check on the corrected pattern before cutting.

Ridge Line Is Not Straight

Probable cause: temporary ridge bracing was insufficient or was adjusted unevenly while rafter pairs were being set.
Confirmation: sight and string the ridge line along its full length and identify the location and magnitude of any deviation.
Correction: adjust temporary bracing to straighten the ridge before installing additional rafter pairs, and re-fasten any rafter connection disturbed by the adjustment.
Reinspection: repeat the string-line check after adjustment and before installing permanent ties.

Rafter Spacing Does Not Match the Layout

Probable cause: the layout marks on the wall plate or ridge were not transferred consistently, or a rafter shifted during fastening.
Confirmation: re-measure spacing between adjacent rafters at the wall plate and at the ridge and compare against the approved plan spacing.
Correction: loosen and reposition the affected rafter to the marked layout location before completing permanent fastening; do not leave a rafter fastened off its layout mark.
Reinspection: repeat the spacing check across the full roof length once the correction is complete.

Roof Planes Are Not Symmetrical

Probable cause: the roof centerline was struck incorrectly, or one side's rafters were cut or set from a different reference than the other side.
Confirmation: re-verify the roof centerline against the approved plan and compare rafter length, seat-cut location, and overhang on both roof sides.
Correction: correct the centerline or the affected side's rafters using the verified pattern before installing sheathing; obtain direction from the design professional if the discrepancy affects framing already permanently fastened.
Reinspection: repeat the symmetry check at multiple points along the roof length.

Required Tie or Strap Connection Is Missing or Incomplete

Probable cause: the connector schedule was not checked before proceeding to the next rafter pair, or a connection was left for later and not tracked.
Confirmation: compare every installed rafter, ridge, tie, and gable connection against the approved connector schedule.
Correction: install the missing or incomplete connection using the specified fastener, strap, or clip before sheathing covers the connection.
Reinspection: repeat the connection check for the affected bay and confirm no other bay has the same omission.

Cleanup
