Job Overview

This project qualifies you to prepare and install a plan-specified structural header over a selected load-bearing opening, along with the jack studs, bearing supports, and connectors that transfer the header's load down through the wall to the foundation, beam, or approved structure below. The finished work product is one or more installed headers, seated on verified bearing, fastened to the approved schedule, with a completed load-path inspection and installation record.

You will confirm that the header size and material come from the approved plans, prescriptive code table, engineered span table, or a qualified designer; inspect the header stock and any built-up components; verify the actual rough-opening width and required bearing; assemble multi-ply headers where specified; install the supporting jack studs, posts, or bearing blocks; lift and seat the header; fasten it to the approved schedule; and verify the continuous load path above and below the header before closing the work.

This project follows the rough-opening framing taught in Project 09. Project 09 established king studs, jack studs, header limits, sill members, and cripple-stud locations for both structural and non-structural openings. This project focuses specifically on the structural header itself: selecting the correct material and ply count from the approved schedule, assembling it correctly, and installing it with a verified bearing and load path. It feeds directly into Project 11, where the completed wall — including its installed headers — is raised, plumbed, and braced.

Skills You Will Demonstrate

  • FR-055 — Select header material from the approved header schedule, prescriptive code table, engineered table, or qualified designer, and identify its handling restrictions.
  • FR-056 — Assemble multi-ply, solid-sawn, or engineered headers according to the approved fastener pattern, spacer requirements, and orientation.
  • FR-057 — Install jack studs, posts, bearing blocks, or column assemblies that provide the required bearing beneath each header end.
  • FR-058 — Fasten the header to king studs, jack studs, straps, hangers, or other connectors according to the approved connection schedule.
  • FR-059 — Verify that the header's load path transfers continuously to the foundation, beam, or approved structure below, without missing or offset support.
  • FR-060 — Inspect the completed header installation for elevation, level, bearing length, ply contact, fastener pattern, connector seating, and wall-face alignment, and document the results.

Where This Skill Fits

A header carries the wall and roof or floor loads that would otherwise be carried by the wall studs removed to create a door or window opening. Every pound of load above a structural opening passes through the header, into its jack studs or posts, and down through the wall to the foundation or beam below. A header that is undersized, unsupported, or disconnected from that load path does not fail gradually — it fails as a structural event.

This project sits between Project 09, which frames the rough opening geometry, and Project 11, which raises and braces the completed wall. The load-path verification taught here — confirming that bearing continues uninterrupted from the header, through the jack studs, through the wall, and down to the structure below — is the same verification repeated in Project 20, where the capstone shed's door and window headers are checked against the same standard before the roof is framed.

Scope and Intended Use

This project covers:

  • confirming header size, material, ply count, and bearing requirements from the approved header schedule, prescriptive code provisions, engineered tables, or a qualified designer
  • inspecting solid-sawn, built-up, laminated veneer lumber (LVL), glulam, steel, or other specified header material for identification, grade, dimensions, damage, and moisture condition
  • assembling multi-ply headers with the approved spacers, fasteners, and orientation
  • installing jack studs, posts, bearing blocks, or column assemblies beneath each header end
  • lifting, seating, and fastening the header according to the approved schedule
  • verifying and documenting the continuous load path above and below the header

This project does not authorize the learner to:

  • select header size, species, grade, or ply count without an approved header schedule, prescriptive code table, engineered table, or qualified designer's direction
  • field-cut, drill, notch, or otherwise alter an engineered header (LVL, glulam, I-joist, or similar product) beyond what the manufacturer's instructions explicitly permit
  • substitute a different bearing condition, connector, or fastener for the one specified in the approved connection schedule
  • install or load a header with damaged plies, inadequate bearing, or an unresolved discrepancy from the approved plan

Header sizing for any load-bearing opening is structural work. If the approved plans, prescriptive table, or engineered table do not clearly cover the opening's span, load condition, or ply requirement, stop work and obtain direction from a qualified structural professional before proceeding.

Trade Note

A header schedule entry is not complete unless it specifies material, ply count, depth, bearing length, and fastening pattern together. A depth and species alone (for example, "4x8 header") do not tell you how many jack studs are required, how the plies are fastened together, or what bearing length each end needs. If any one of these values is missing from the approved documentation for the opening you are framing, obtain it before cutting header stock.

Setup Standard

Do not cut or assemble header material until the approved header schedule, prescriptive table, engineered table, or qualified designer's direction has been confirmed for the specific opening (Step 2). Header size, material, ply count, and bearing requirements are never inferred from opening width alone.

Shop Tip

Label each header immediately after cutting or assembly with its plan-schedule reference and its wall/opening location. Headers of similar depth are easy to mix up once several are staged together, and installing the wrong header at an opening creates a bearing or span discrepancy that is far more expensive to correct after the wall is sheathed.

Common Field Fundamentals

Every header installed in this project follows the same load-path logic: the roof or floor load above bears on the header, the header bears on the jack studs or posts at each end, and those jack studs or posts bear on the wall's bottom plate, which in turn bears on the floor framing or foundation established in Projects 01 through 03. A header is not a finished component until every link in that chain is verified, fastened, and free of gaps.

Header Sizing Authority

Header size, species, grade, ply count, and bearing length are never selected in the field from judgment or from the rough-opening width alone. These values must come from the approved plans, the header schedule supplied with the project specification, a prescriptive code table applicable to the exact span and load condition, an engineered span table for the specific product, or a qualified structural designer. If none of these sources clearly covers the opening being framed, stop work and obtain direction before cutting any header material.

Critical Safety Cue — Do Not Guess Header Size

Never select a header size by comparing it to a similar-looking opening elsewhere in the structure. Span, load condition, tributary width, and existing wall configuration all affect the correct header size, and two openings of the same rough width can require different headers. Confirm the exact schedule entry, prescriptive table row, or engineered value for each opening individually.

Manufacturer Cue — Engineered Header Restrictions

Laminated veneer lumber, glulam, and other engineered headers may not be field-cut, drilled, notched, or otherwise altered except where the manufacturer's published instructions explicitly permit it. Confirm allowable end cuts, hole locations, and handling and storage requirements from the manufacturer's current literature before cutting or drilling any engineered header.

Measurement Cue — Verify the Opening Before Cutting

Measure the actual rough-opening width in the field before cutting the header to length. Framed openings can vary slightly from the plan dimension even when built to the same layout. Cut the header to fit the verified field dimension and required bearing length, not to the nominal plan dimension alone, unless the approved schedule specifies an exact header length.

Field Safety Cue — Lifting and Temporary Support

Plan the header lift before removing any temporary support already in place at the opening. Assign additional workers or mechanical lifting assistance for headers exceeding the single-worker safe lifting limit established for this project. Install temporary support posts or T-braces under the header immediately after it is lifted into position, before releasing it for fastening.

C. Fabrication and Installation Materials

  • Header stock: solid-sawn, built-up, LVL, glulam, steel, or other material exactly as specified in the header schedule for each scheduled opening.
  • Spacers and filler panels: only the type and thickness specified for multi-ply built-up headers; do not substitute an unspecified filler thickness.
  • Jack studs, king studs, and posts: matching the species, grade, and size specified for the wall system and scheduled bearing condition.

  • Documentation: one header/installation record per header, listing schedule reference, material, dimensions, connectors, and inspection results.

E. Documentation Review and Material Verification (Steps 1–5)

  1. Review the structural header schedule, opening location, span, load condition, material type, ply count, depth, bearing, fastening, and connector details. Confirm the schedule entry that applies to the specific opening being framed before proceeding.
  2. Confirm that header sizing comes from approved plans, prescriptive code provisions, engineered tables, or a qualified designer. Record the source and reference used for the sizing decision on the header/installation record.
  3. Identify solid-sawn, built-up, laminated veneer lumber, glulam, steel, or other header material and its handling restrictions. Confirm any manufacturer-stated storage, handling, or moisture-exposure limits before staging the material.
  4. Inspect each header component for correct identification, grade, dimensions, damage, moisture condition, and orientation. Quarantine and log any component that fails inspection before it is used.
  5. Verify the actual rough-opening width and required bearing before cutting. Measure the field-built opening from Project 09 and compare it with the header schedule's length and bearing requirements.

F. Header Preparation and Assembly (Steps 6–8)

  1. Mark solid-sawn members from a defined reference end; do not field-cut engineered members outside manufacturer allowances. Use a framing square to mark the cut line square to the reference edge before cutting.
  2. Prepare approved spacers, insulation, filler panels, or built-up header plies. Cut spacer material to the exact thickness and pattern specified for the header assembly.
  3. Assemble multi-ply headers on a flat surface using the approved fastener pattern and orientation. Clamp plies together before fastening and verify full face contact along the entire length before driving fasteners.

G. Bearing Support and Header Lift (Steps 9–13)

  1. Install required jack studs, posts, bearing blocks, or column assemblies before lifting the header. Confirm each support is plumb, at the correct height, and fastened to the bottom plate before the header is set on it.
  2. Verify that supporting members transfer load to the foundation, beam, or approved structure below. Trace the load path from each jack stud or post down through the wall to its bearing point and confirm no gap or missing support exists.
  3. Plan the header lift and temporary support. Confirm worker count or mechanical assistance needed, and stage temporary support posts or T-braces before lifting begins.
  4. Lift and seat the header fully on each bearing support. Confirm full contact and the specified bearing length at each end before proceeding.
  5. Restrain the header against rotation and displacement. Install temporary support posts or T-braces immediately and confirm the header cannot shift before permanent fastening begins.

H. Fastening and Connection (Steps 14–16)

  1. Fasten the header to king studs, jack studs, straps, hangers, or other connectors according to the approved schedule. Install every fastener in the pattern, spacing, and orientation specified; do not substitute a different fastener type or reduce the specified quantity.
  2. Install upper cripples, blocking, rim, or framing required above the header. Confirm these members bear fully and are fastened per the wall schedule before removing temporary support.
  3. Complete hold-down, strap, portal-frame, or narrow-wall details where specified. Install each listed connector using its specified fasteners and confirm the connector engages both the header assembly and its supporting member.

I. Verification, Correction, and Documentation (Steps 17–21)

  1. Verify header elevation, level, bearing length, ply contact, fastener pattern, connector seating, and wall-face alignment. Compare each measurement against the Acceptance Standards table above.
  2. Inspect the load path above and below the header for missing or offset support. Confirm every bearing point in the path from roof or floor load to foundation or beam is complete and aligned.
  3. Stop work for damaged engineered members, inadequate bearing, field-altered headers, or unsupported concentrated loads. Log the condition and obtain direction from the design professional or manufacturer before continuing at that opening.
  4. Correct only deficiencies covered by the approved detail. Do not improvise a repair, substitute member, or added connector that is not shown in the approved documentation.
  5. Record header identification, material, span, bearing, fasteners, connectors, and inspection approval. Complete the header/installation record for each installed header before concealment or wall sheathing begins.

Common Verification Inspection

Inspect every completed header using the same checks regardless of material type or opening location.

  • Material check: confirm header material, grade, and ply count match the approved header schedule entry for the specific opening.
  • Bearing check: confirm full bearing contact and the specified bearing length at each header end.
  • Assembly check: confirm all plies of a built-up header are in full face contact and fastened in the approved pattern with no gaps.
  • Elevation and level check: confirm the header sits at the plan-specified elevation and reads level within the approved tolerance.
  • Connector check: confirm every strap, hanger, hold-down, and connector is installed with listed fasteners in the specified pattern and orientation.
  • Load-path check: confirm continuous, unobstructed bearing from the header through its supports to the foundation, beam, or approved structure below.

Complete the Header/Installation Record in the Inspection Checklist section below before wall sheathing or concealment begins.

Failure Points and Troubleshooting

Header Does Not Reach Full Bearing at One or Both Ends

Probable cause: the jack stud or post was cut short, the header was cut long, or the opening was framed out of square, leaving a gap at one end.
Confirmation: measure the actual bearing length at each end and compare it with the schedule's required bearing length; check jack-stud height and plumb.
Correction: shim only if the approved detail permits shimming for the specific bearing condition; otherwise remove the header, correct the jack-stud height or opening framing, and reseat it. Do not force a header into a bearing condition that reduces contact area below the specified length.
Reinspection: repeat the bearing-length check at both ends after correction.

Built-Up Header Plies Show Gaps Instead of Full Face Contact

Probable cause: the plies were not clamped before fastening, the stock is bowed or twisted, or the spacer thickness does not match the specified value.
Confirmation: inspect the full length of the assembled header for daylight or gaps between plies, and verify spacer thickness against the schedule.
Correction: disassemble and reassemble with correct clamping and spacer thickness, or replace bowed or twisted stock; do not fasten plies together with a visible gap.
Reinspection: repeat the ply-contact check along the full header length before installation.

Header Reads Out of Level After Installation

Probable cause: the jack studs on each side of the opening are cut to different lengths, or one bearing support settled during the lift.
Confirmation: measure jack-stud height on both sides of the opening and compare; check the level reading at multiple points along the header.
Correction: correct the jack-stud height on the low side per the approved detail; do not compensate by tilting the header or adding an unspecified shim thickness.
Reinspection: repeat the level check across the full header length after correction.

Connector Fastener Pattern Does Not Match the Schedule

Probable cause: a generic fastener was substituted for a listed connector fastener, or the fastener quantity or spacing was reduced from the specified pattern.
Confirmation: compare the installed fasteners against the connector manufacturer's specified type, quantity, and pattern for the connector in use.
Correction: remove substituted fasteners and install the specified listed fasteners in the complete pattern; do not leave a partial fastener pattern in place.
Reinspection: repeat the connector check and confirm every required fastener location is filled.

Load Path Is Interrupted Below the Header

Probable cause: the jack stud or post does not align with adequate bearing below (such as a floor joist, beam, or foundation point), or an intermediate framing member was omitted.
Confirmation: trace the load path from the header end down through the jack stud or post to its bearing point on the floor or foundation, checking for alignment and continuous support at every level.
Correction: stop work at that opening and obtain direction from the design professional; do not add an unapproved post, blocking, or bearing detail to close the gap without a documented decision.
Reinspection: repeat the full load-path check after the approved correction is installed.

Header Material Does Not Match the Schedule Entry

Probable cause: the wrong header was pulled from staged material, or the schedule reference used was not the current approved revision.
Confirmation: confirm the header schedule revision date, then compare the installed header's material, depth, and ply count against the current schedule entry for that specific opening.
Correction: remove the incorrect header and install the schedule-specified header; log the substitution and the correction on the header/installation record.
Reinspection: repeat the material check and the full Acceptance Standards review for the corrected header.

Field Terms

Header Schedule
The approved document specifying header material, ply count, depth, length, bearing length, and fastening pattern for each scheduled opening.
Bearing Length
The horizontal length of header end resting directly on the jack stud, post, or bearing block below it.
Ply
One layer of a multi-ply built-up header, fastened face-to-face with the others to form the finished member.
King Stud
The full-height stud at each side of a rough opening, extending from the bottom plate to the top plate.
Jack Stud (Trimmer Stud)
The stud that extends from the bottom plate to the underside of the header, providing direct bearing for the header end.
Load Path
The continuous structural route through which a load travels from above the header down to the foundation or beam below.
Prescriptive Table
An adopted code table specifying allowable header sizes for defined spans and load conditions.
Engineered Span Table
A manufacturer-published table specifying allowable spans for a specific engineered header product.
Hold-Down
A connector that resists uplift or overturning forces at a specified location, installed per manufacturer instructions.
Header/Installation Record
The document recording header identification, material, dimensions, connectors, fasteners, and inspection results for each installed header.
