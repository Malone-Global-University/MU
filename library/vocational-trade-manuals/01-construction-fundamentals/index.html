01 – Construction Fundamentals

Construction Fundamentals is the entry point to the Malone Global University Vocational Trade Field Manual series. Through a sequence of practical projects, students learn the essential skills used on every residential construction site, including safety, measurement, layout, cutting, fastening, material estimation, and basic structural assembly.

Overview

Unlike a traditional textbook, this field manual teaches construction by having students complete real projects. Each chapter builds practical jobsite experience while introducing professional tools, techniques, quality standards, and safe work practices. By the end of the manual, students will have completed several foundational construction projects while developing the core skills required for every residential building trade.

Projects

  1. Project 1: Set Up a Safe Residential Jobsite
  2. Project 2: Master Layout with Squares and Levels
  3. Project 3: Measure, Mark, and Cut Lumber Accurately
  4. Project 4: Introduction to Construction Fasteners and Structural Hardware
  5. Project 5: Build a Pair of Heavy-Duty Sawhorses
  6. Project 6: Lay Out a Small Building Footprint
  7. Project 7: Calculate Materials for a Small Project
  8. Project 8: Build a Complete Garden Storage Shed Floor Platform

Learning Outcomes

  • Apply professional jobsite safety procedures and hazard awareness.
  • Use measuring, marking, layout, and leveling tools accurately.
  • Perform precise cuts using common hand and power tools.
  • Select and install appropriate construction fasteners and structural hardware.
  • Construct practical woodworking projects using professional techniques.
  • Lay out simple building footprints using accepted construction methods.
  • Estimate material quantities and prepare basic construction takeoffs.
  • Develop the foundational skills required for all residential construction trades.
