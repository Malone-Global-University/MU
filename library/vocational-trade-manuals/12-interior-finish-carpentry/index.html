12-Interior Finish Carpentry

This field manual provides practical, project-based instruction for residential interior finish carpentry. Students learn to measure, cut, install, fit, repair, and finish interior trim systems including doors, windows, baseboards, crown molding, stairs, and built-in details.

Overview

Interior Finish Carpentry emphasizes precision craftsmanship through real-world residential trim installation projects. Beginning with finish carpentry planning and workstation preparation, students progress through measuring and cutting techniques, door and window trim, baseboards, molding, accent features, stair trim, surface preparation, repairs, and a complete practice room finishing project.

Learning Outcomes

  • Plan and prepare residential finish carpentry projects.
  • Set up a safe and efficient finish carpentry workstation.
  • Measure, cut, fit, and install interior trim accurately.
  • Install interior door casing, prehung doors, and door hardware.
  • Install window trim, stools, aprons, and casing assemblies.
  • Install baseboards, shoe molding, and decorative molding systems.
  • Cut and install crown molding and complex corner transitions.
  • Build decorative wall features and install closet trim components.
  • Install and finish stair trim components.
  • Prepare, repair, and complete professional-quality interior finishes.
  • Complete a full interior trim installation project.
