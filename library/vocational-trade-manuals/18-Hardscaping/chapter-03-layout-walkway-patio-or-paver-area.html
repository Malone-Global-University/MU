Job Overview

This chapter qualifies you to lay out and document one straight or curved paver area using approved boundaries, offsets, finished elevations, and pattern controls. The finished work product is one dimensionally verified hardscape layout with controlled straight lines, curves, pattern-control references, excavation limits, drainage marks, and documented elevations and diagonals.

You will establish a controlling baseline, set primary corner or center points, measure and square the boundaries, lay out any curves using the approved method, set offset stakes or batter boards beyond the excavation limit, transfer the boundary to those offsets, mark the finished paver edge and excavation limit, mark drains and fixed obstacles, establish independent pattern-control lines, mark high and low edges for drainage, and obtain layout approval before excavation begins.

This chapter depends directly on Chapter 1's readiness plan and Chapter 2's staged, protected work area. It supplies the exact boundaries, elevations, and reference points that Chapter 4 uses to excavate to the correct depth and slope, and that Chapters 7 and 8 use to install pavers on pattern lines that remain square and true regardless of any out-of-square wall, fence, or existing slab nearby.

Skills You Will Demonstrate

  • HC-013 — Establish project boundaries from approved dimensions, finished elevation, and site references.
  • HC-014 — Set reference lines, including baselines, perpendiculars, and offset controls beyond the excavation limit.
  • HC-015 — Square the layout using diagonal measurement and correct deviations within tolerance.
  • HC-016 — Mark curves using the approved offset, radius, arc, hose, or template method and verify tangency.
  • HC-017 — Verify dimensions, diagonals, curve points, and elevations against the approved layout.
  • HC-018 — Document the final hardscape layout, including all reference points, for use by later excavation and installation chapters.

Where This Skill Fits

Layout is the point in the project where a paper plan becomes a set of physical reference points that every later chapter depends on. An excavation dug from an unsquared or unverified layout produces a base, bedding, and paver field that must fight the error at every subsequent step. Pattern lines struck from an out-of-square wall or fence carry that error across the entire paver field.

This chapter also establishes the discipline of working from offset controls set beyond the excavation limit, so that reference points survive the excavation and base work in Chapters 4 through 6. A layout that cannot be recovered after excavation begins forces the crew to re-establish boundaries by guesswork, which this book does not permit.

Scope and Intended Use

This chapter covers:

  • reviewing approved dimensions, finished elevation, slope, pattern, borders, drains, edges, transitions, and excavation limits before layout begins
  • confirming utility markings and property boundaries remain valid before staking
  • establishing a controlling baseline and primary corner or center points
  • measuring straight boundaries and establishing perpendicular lines using an approved squaring method
  • measuring diagonals and adjusting the layout until it meets the stated tolerance
  • establishing curves using offsets, radii, arcs, hoses, or templates as specified, and verifying tangency to straight sections
  • setting offset stakes or batter boards beyond the excavation limit and transferring boundaries to them
  • marking the finished paver edge, excavation limit, drains, structures, obstacles, and pattern-control lines
  • marking high and low edges for drainage direction
  • recording all layout data and obtaining layout approval
  • protecting reference points during excavation

This chapter does not authorize the learner to:

  • stake, drive pins, or mark the ground before utility markings and property boundaries are reconfirmed as current and valid
  • proceed with layout on a site where Chapter 1 identified an unresolved stop-work condition
  • begin excavation using this chapter's layout before layout approval is recorded on the layout worksheet
  • invent an excavation depth, slope value, or finished elevation not supplied by the approved site plan, drainage plan, or chapter project specification

Any layout that fails the squaring, diagonal, or curve-tangency check in this chapter must be corrected and reverified before Chapter 4 excavation begins.

Trade Note

Every reference point set in this chapter is a promise to every later chapter that the boundary, elevation, or pattern line has not moved. Set offset controls generously beyond the excavation limit, record their exact position relative to the baseline, and protect them physically once excavation equipment is on site.

Required PPE for this project: closed-toe footwear and high-visibility clothing appropriate to the site; gloves when driving stakes or handling batter-board lumber; eye protection when driving stakes or pins into hard or compacted ground.

Setup Standard

Do not stake, drive pins, or mark the ground until utility markings from Chapter 1 remain visible and valid, and property boundaries have been reconfirmed. If markings have faded, been disturbed, or the utility-locate window has expired, request remarking before layout work continues.

Shop Tip

Record the exact offset distance from every batter board or offset stake back to its corresponding boundary point before excavation starts. If a stake is disturbed, this recorded offset lets you recover the original boundary point without re-surveying the entire layout.

Reference Conventions

Establish these references before any layout work begins, and record them on the layout worksheet.

TermDefinition
BaselineThe single controlling straight reference line from which all other layout lines, offsets, and pattern-control lines are measured
Offset stake or batter boardA stake or board set beyond the excavation limit that preserves a boundary or elevation reference after excavation begins
Squaring toleranceThe maximum permitted difference between the two diagonals of a rectangular layout before the layout must be corrected, as stated in the approved site plan or, if not stated there, a maximum difference of 1/4 inch per 10 feet of diagonal length
TangencyThe condition where a curved section meets a straight section without a visible angle or kink at the transition point
Pattern-control lineA reference line established from the baseline and verified corners, used to keep the paver pattern square and true independent of any adjacent wall, fence, or slab that may be out of square
High edge / low edgeThe layout-marked edges corresponding to the highest and lowest finished elevations of the paver area, controlling the direction of surface drainage

Common Field Fundamentals

Every paver area in this book, whether a straight walkway or a curved patio, is laid out from the same controlling sequence: establish a baseline, set corners or center points, square the layout by measurement rather than by eye, add curves only from an approved method, transfer the boundary to protected offset controls, and mark the finished edge, excavation limit, and pattern lines before excavation equipment arrives.

Baseline and cornersControlling straight reference, primary corner or center points, perpendiculars
SquaringDiagonal measurement, tolerance check, correction before proceeding
CurvesOffset, radius, arc, hose, or template method; tangency to straight runs
Offsets and markingBatter boards beyond excavation limit, finished edge, drains, pattern lines, drainage direction

Stop-Work Conditions

This chapter requires the layout technician to halt work, rather than proceed to marking or excavation, whenever a condition falls outside what the approved plan and this chapter's method can resolve on their own.

Critical Safety Cue — Utility Markings Not Reconfirmed

Do not drive a stake, pin, or probe into the ground until utility markings from Chapter 1 are visually reconfirmed as present, legible, and unambiguous at every point layout work will disturb. If a marking has faded, been removed, or is contradicted by a visible meter, riser, or vent, stop and request remarking before continuing in that area.

Measurement Cue — Square by Diagonal, Not by Eye

Confirm every rectangular or right-angle layout section using diagonal measurement against the squaring tolerance recorded in the Layout Definition Record. Do not accept a layout as square because the sides appear even; a layout can measure correctly on all four sides and still be significantly out of square.

Field Safety Cue — Stakes, Lines, and Trip Hazards

Keep mason's lines, stakes, and batter boards clearly visible and away from active pedestrian or vehicle paths on the site. Cap or flag exposed stake tops and remove or flag low lines that could catch a foot or an equipment tire during Chapter 2 access and later excavation work.

Code Alert — Boundary and Setback Verification

Do not finalize a layout that places the finished paver edge closer to a property line, easement, or setback than the distance confirmed in Chapter 1. If the approved dimensions appear to conflict with a verified boundary, stop layout work and refer the conflict back through Chapter 1's boundary-verification process before proceeding.

Manufacturer Instruction Cue — Curve Method Limits

When a curve is laid out using a manufactured radius tool, string-and-stake compass method, or flexible template, follow the manufacturer's or specified method's stated minimum radius and transition requirements rather than forcing a tighter curve than the method supports.

E. Preparation — Review and Reconfirm

  1. Review approved dimensions, finished elevation, slope, pattern, borders, drains, edges, transitions, and excavation limits. Complete the Layout Definition Record above using the approved site plan, drainage plan, and pattern or border drawing.
  2. Confirm utility markings and property boundaries. Walk the project area and visually reconfirm every utility marking from Chapter 1 remains present and legible, and reconfirm property boundary points against the property survey. Request remarking before continuing if any marking is missing, faded, or ambiguous.

F. Baseline and Primary Points

  1. Establish the controlling baseline. Set the baseline from the fixed reference recorded in Chapter 1, using a permanent feature such as a foundation corner or survey marker, and mark its ends with stakes.
  2. Set primary corner or center points. Measure from the baseline using the approved dimensions and mark each primary corner, or each curve center point, with a stake or pin.
  3. Measure straight boundaries. Measure the length of every straight boundary segment from its established corner points and record each measurement.
  4. Establish perpendicular lines using approved squaring methods. Use a framing square, the 3-4-5 triangle method, or an equivalent approved method to establish each required perpendicular from the baseline.

G. Squaring and Curves

  1. Set remaining corners. Complete the layout of every remaining primary corner using the established perpendiculars and boundary measurements.
  2. Measure diagonals and adjust until the layout meets tolerance. Measure both diagonals of every rectangular or right-angle section, compare the difference to the recorded squaring tolerance, and adjust corner stakes and remeasure until the layout is within tolerance.
  3. Establish curves using offsets, radii, arcs, hoses, or templates as specified. Lay out every curved section using the exact method specified in the Layout Definition Record, marking curve points at regular intervals sufficient to define the curve.
  4. Verify curve tangency and transitions to straight sections. Sight along each transition point and confirm no visible angle or kink exists where a curve meets a straight section; adjust curve points until the transition is tangent.

H. Offset Controls and Ground Marking

  1. Set offset stakes or batter boards beyond excavation. Place offset stakes or batter boards at a distance beyond the marked excavation limit sufficient to survive excavation equipment and spoil placement, and record the exact offset distance from each to its corresponding boundary point.
  2. Transfer boundaries to offset controls. Use a mason's line, plumb bob, or equivalent method to transfer every primary corner and curve point to its corresponding offset stake or batter board, and record the transfer on the worksheet.
  3. Mark the finished paver edge. Apply marking paint or marking flour along the verified boundary and curve lines to show the exact finished paver edge.
  4. Mark excavation limits including base extension and edge restraint. Mark a second line outside the finished-edge line at the distance specified for base extension and edge-restraint installation.
  5. Mark drains, structures, columns, posts, trees, utilities, and fixed obstacles. Mark the exact location of every fixed feature that falls within or adjacent to the excavation limit.
  6. Establish pattern-control lines independent of irregular walls or fences. Strike pattern-control lines from the verified baseline and corners rather than from any adjacent wall, fence, or slab, and mark them clearly on the ground.
  7. Mark high edge, low edge, and drainage direction. Compare the marked layout to the approved drainage plan and mark the high edge, low edge, and an arrow or label indicating drainage direction.

I. Documentation, Approval, and Protection

  1. Record dimensions, diagonals, curve points, elevations, and reference controls. Complete every field on the Hardscape Layout Worksheet, cross-checking each recorded measurement against the marked layout.
  2. Obtain layout approval. Review the completed worksheet against the Acceptance Standards table, and sign and date the layout approval before excavation equipment is brought onto the marked area.
  3. Protect reference points during excavation. Flag, cap, or otherwise protect every offset stake and batter board, and brief the excavation crew on which marks must not be disturbed.

Common Site Inspection

Review every layout using the same checks regardless of project shape.

  • Utility and boundary check: confirm utility markings and property boundaries were reconfirmed before any ground disturbance began.
  • Baseline check: confirm the baseline and primary corner or center points are recorded relative to the Chapter 1 fixed reference.
  • Squaring check: confirm both diagonals of every rectangular or right-angle section are recorded and within the stated tolerance.
  • Curve check: confirm every curved section was laid out using the specified method and verified tangent at each transition.
  • Offset check: confirm every offset stake or batter board is set beyond the excavation limit with its offset distance recorded.
  • Marking check: confirm the finished edge, excavation limit, drains, obstacles, pattern-control lines, and drainage marks all appear on the ground and match the worksheet.

Complete the Hardscape Layout Worksheet in the Inspection Checklist section below before beginning Chapter 4.

Failure Points and Troubleshooting

Layout Passes Side Measurements but Fails Diagonal Check

Probable cause: the layout was squared by measuring only the four sides and adjusting them to match the approved dimensions, without measuring either diagonal.
Confirmation: measure both diagonals of the affected section and compare the difference to the recorded squaring tolerance.
Correction: hold one corner fixed, shift the adjacent corners along their boundary lines until both diagonals fall within tolerance, then remeasure all four sides to confirm the approved dimensions are still met.
Reinspection: repeat the squaring check for the corrected section and any adjoining section affected by the correction.

Curve Shows a Visible Angle at the Tangent Point

Probable cause: curve points were set too far apart, or the curve method's minimum radius was not followed near the transition to the straight section.
Confirmation: sight along the transition from a low angle and compare against the specified minimum radius for the curve method in use.
Correction: add additional curve points near the transition and reset the curve using the specified method and minimum radius until the transition reads as a smooth arc.
Reinspection: repeat the curve-tangency check at every transition point on the affected curve.

Offset Stake Disturbed Before Excavation Is Complete

Probable cause: the offset stake or batter board was not set far enough beyond the excavation limit, or was not protected from equipment and spoil placement.
Confirmation: compare the current stake position, if it can still be found, against its recorded offset distance from the corresponding boundary point.
Correction: use the recorded offset distance and the surviving reference points to reset the disturbed stake, or reestablish it from an undisturbed adjacent offset control and the recorded dimensions.
Reinspection: repeat the boundary-transfer check for the affected corner or curve point before excavation continues.

Pattern-Control Lines Struck From an Out-of-Square Wall or Fence

Probable cause: pattern-control lines were established using an existing wall, fence, or slab edge as a reference instead of the verified baseline and corners.
Confirmation: measure the adjacent wall, fence, or slab against the verified baseline and corners to check for a discrepancy.
Correction: re-strike the pattern-control lines from the verified baseline and corners, and record the corrected reference on the worksheet.
Reinspection: repeat the marking check for all pattern-control lines before Chapter 7 or Chapter 8 installation begins.

Drainage Marks Do Not Match the Approved Drainage Plan

Probable cause: the high edge and low edge were marked from visual impression rather than from the finished-elevation data in the drainage plan.
Confirmation: compare the marked high edge and low edge against the finished elevations stated in the drainage plan and the fixed reference established in Chapter 1.
Correction: remark the high edge, low edge, and drainage-direction label using the finished-elevation data, and update the worksheet.
Reinspection: repeat the drainage-marks check before layout approval is issued.

Layout Approved With an Incomplete Worksheet

Probable cause: a required record, such as an offset distance or a diagonal measurement, was not recorded before the layout was signed as approved.
Confirmation: cross-check every field in the Layout Record Schedule against the completed worksheet to confirm no field is blank.
Correction: return to the site, take the missing measurement, and record it before the approval is considered valid.
Reinspection: repeat the full worksheet review before excavation begins.

Cleanup

This chapter ends with a completed, signed layout worksheet and a physically marked, protected layout, not a verbal description of where the project will go.

  • Confirm all offset stakes and batter boards are capped, flagged, or otherwise protected from equipment traffic
  • Store the completed layout worksheet with the Chapter 1 readiness plan and the approved site plan and drawings
  • File layout photographs alongside their corresponding worksheet entries
  • Brief the excavation crew on the marked excavation limit, protected reference points, and marked obstacles before Chapter 4 begins
  • Retain the complete layout plan for reference throughout Chapters 4 through 20

Field Terms

Baseline
The single controlling straight reference line from which all other layout lines are measured.
Offset Stake / Batter Board
A stake or board set beyond the excavation limit that preserves a boundary or elevation reference during excavation.
Squaring Tolerance
The maximum permitted difference between the two diagonals of a rectangular layout before correction is required.
Tangency
The condition where a curve meets a straight section without a visible angle or kink at the transition.
Pattern-Control Line
A reference line struck from the baseline and verified corners, used to keep the paver pattern true independent of adjacent structures.
High Edge / Low Edge
The layout-marked edges corresponding to the highest and lowest finished elevations, controlling drainage direction.
Excavation Limit
The marked line outside the finished paver edge that accounts for base extension and edge-restraint installation.
3-4-5 Method
A squaring method using proportional side lengths of 3, 4, and 5 units to establish a true right angle.

Performance Check

With the site laid out and every record on the Hardscape Layout Worksheet completed, review the finished layout against the Acceptance Standards table and confirm that utility and boundary markings were reconfirmed before disturbance, the baseline and corners are tied to the Chapter 1 fixed reference, every diagonal falls within the stated squaring tolerance, every curve is tangent to its adjoining straight section, every offset control is set beyond the excavation limit with its offset distance recorded, and the finished edge, excavation limit, obstacles, pattern-control lines, and drainage marks are all present and correct. If any item is incomplete or out of tolerance, correct it and reinspect the affected record before issuing the layout approval and beginning Chapter 4.
