Job Overview

This project qualifies you to install one base-cabinet run over an uneven finished floor: positioning each cabinet against the Project 03 reference lines, checking it level and plumb, shimming it at approved load points, joining it to its neighbor, and securing the run to wall framing with structural cabinet-mounting screws. The finished work product is a level, plumb, aligned, joined, and structurally secured base-cabinet run — inspected against a long straightedge and accepted for fillers, panels, and countertop templating in Project 08 and Project 09.

You will verify the Project 03 reference lines and floor high point, confirm the cabinet sequence, inspect each cabinet box, position and level the first cabinet at the controlling corner or high point, shim it at approved load points, check the face-frame alignment, clamp and join each subsequent cabinet, locate studs behind hanging rails, and secure every cabinet to framing with structural cabinet screws, correcting twist, rocking, separated faces, and wall gaps before the run is accepted.

This project assumes Project 01 issued an approved or conditional cabinet-installation readiness decision, Project 02 established a protected and organized work area, and Project 03 established the floor high point and every controlling reference line for this run. It sits immediately before Project 05, which installs the wall-cabinet run above or adjacent to this base-cabinet run, and before Project 08, which fits fillers, end panels, and toe kicks once every base, wall, and tall cabinet in the room is permanently secured.

Skills You Will Demonstrate

  • CB-019 — Position base cabinets against the Project 03 reference lines in the approved sequence, beginning at the controlling corner, high point, or controlling cabinet.
  • CB-020 — Shim cabinets level and plumb using stable, noncompressible shims at approved load points.
  • CB-021 — Align cabinet faces and face frames flush across the run before joining.
  • CB-022 — Fasten cabinets together using approved cabinet-connection screws without damaging finished faces.
  • CB-023 — Secure cabinets to wall framing using approved structural cabinet screws at verified stud or backing locations.
  • CB-024 — Inspect base-cabinet alignment for level, plumb, face alignment, run length, and countertop-support readiness.

Where This Skill Fits

The floor high point and reference lines confirmed in Project 03 control where this run begins and how each cabinet is shimmed. Every later component in the room — the wall-cabinet run in Project 05, the fillers and toe kicks in Project 08, the countertop template in Project 09, the sink opening in Project 10, and the doors, drawers, and hardware in Projects 12 and 13 — depends on this base-cabinet run being level, plumb, aligned, and permanently secured before it is concealed by fillers or countertops.

The leveling, shimming, joining, and structural-fastening method used here — establishing a controlling starting point, checking level and plumb before shimming, shimming only at approved load points, joining adjacent faces before fastening to the wall, and locating studs before driving structural fasteners — is the same method used for wall cabinets in Project 05 and tall cabinets in Project 06.

Scope and Intended Use

This project covers:

  • verifying the Project 03 reference lines, floor high point, and cabinet sequence before positioning cabinets
  • inspecting each cabinet box for square, damage, loose joints, and correct designation before installation
  • positioning, leveling, and plumbing base cabinets beginning at the approved corner, high point, or controlling cabinet
  • shimming cabinets at approved load points using stable, noncompressible shims
  • clamping and joining adjacent cabinet faces or boxes using approved cabinet-connection screws
  • locating studs or backing behind cabinet hanging rails and securing cabinets to framing using approved structural cabinet screws
  • maintaining specified appliance, corner, filler, and wall clearances throughout the run
  • verifying the completed run for level, plumb, face alignment, and countertop-support readiness with a long straightedge

This project does not authorize the learner to:

  • select cabinet dimensions, cabinet quantities, or run sequence independently of the approved cabinet plan and cabinet schedule
  • fasten cabinets to drywall alone in place of verified stud or backing locations, or substitute drywall screws for structural cabinet screws
  • trim or remove shims, install fillers, or template countertops before the run has passed the Acceptance Standards inspection
  • proceed with installation over an unresolved floor, wall, or service deficiency identified in Project 01 and not yet corrected

Any cabinet box found damaged, out of square, or with a loose joint at the start of this project must be set aside and reported before it is installed into the run.

Trade Note

Every later fastening and finish project in this book assumes this base-cabinet run was leveled, plumbed, joined, and structurally secured before fillers or countertops concealed the connections. Trimming shims or installing fillers before the full run passes inspection is the most common cause of the twist, rocking, and wall-gap conditions that Chapter 19 must trace back to this project during repair.

Setup Standard

Do not position the first cabinet until the Project 03 reference lines and floor high point have been verified in place (Step 1) and the cabinet sequence, fillers, corners, appliance openings, and end panels have been confirmed against the approved cabinet plan and schedule (Step 2).

Shop Tip

Begin at the corner, high point, or controlling cabinet identified on the approved plan, not at whichever end of the run is most convenient to reach first. Every following cabinet is leveled and aligned from that starting cabinet, and starting at the wrong point compounds a small floor variation into a run-length or corner-fit problem by the far end.

Shimming at Approved Load Points Only

A base cabinet transfers its load through its sides, rails, or other manufacturer-identified bearing points, not through its thin bottom panel. Shimming an unsupported bottom panel to close a floor gap distorts the cabinet box and creates a hidden stress point that later shows up as a sticking drawer or a cracked toe-kick joint. Identify the cabinet's approved load points before shimming, and use stable, noncompressible shim material at each one.

Measurement Cue — Check Level Front-to-Back and Side-to-Side Before Shimming

Check each cabinet top for level in both directions, and check both sides for plumb, before inserting any shim. Shimming a cabinet that has not first been checked in both directions produces a box that reads level in one direction but is twisted out of plane in the other, a condition that will not be visible until the countertop or an adjacent cabinet is installed.

Critical Safety Cue — Do Not Substitute Drywall Screws for Structural Cabinet Screws

Do not fasten a base cabinet's hanging rail or back to the wall using drywall screws or any fastener not specified as a structural cabinet-mounting screw. A base cabinet supports countertop, appliance, and stored-item loads that a drywall screw is not designed to carry over time. Locate the stud or approved backing first, and drive only the specified structural cabinet screw.

Field Safety Cue — Support Cabinets Before Releasing Clamps

Do not remove a clamp or support from a positioned cabinet until it is either shimmed and self-supporting on the floor or fastened to its neighbor or the wall. An unsupported cabinet can tip or shift out of position, damaging the finish or creating a pinch hazard at the adjacent cabinet edge.

Manufacturer Cue — Cabinet-Connection and Structural Screw Specifications

Use the exact cabinet-connection screw specified for joining face frames or box sides, and the exact structural cabinet screw specified for fastening to studs or backing. Do not use a longer or larger-diameter screw than specified to compensate for a perceived weak connection; an oversized fastener can split a face frame or protrude through a finished surface.

Code Alert — Verify Backing Requirements Where Local Amendments Apply

Some jurisdictions apply specific backing or fastening requirements to base cabinets supporting heavy countertop materials or built-in appliances. Verify the current requirement with the authority having jurisdiction and the general contractor before finalizing the structural-fastening plan for this run.

E. Reference Verification and Cabinet Preparation (Steps 1–4)

  1. Verify the Chapter 3 reference lines and floor high point. Confirm the marked lines and benchmark remain intact and unchanged since Project 03.
  2. Confirm cabinet sequence, fillers, corners, appliance openings, and end panels. Compare the staged cabinets against the approved cabinet plan and schedule before positioning any cabinet.
  3. Remove doors, drawers, and shelves where permitted. Store removed components with matching labels as established in Project 02.
  4. Inspect each cabinet box for square, damage, loose joints, and correct designation. Set aside and report any cabinet that fails this inspection before it is installed.

F. Positioning and Leveling the First Cabinet (Steps 5–11)

  1. Begin at the approved corner, high point, or controlling cabinet. Confirm the starting location against the Project 03 layout sketch before positioning.
  2. Position the first cabinet against the layout. Align it with the marked base-cabinet reference line and run limit.
  3. Check cabinet top for level front-to-back and side-to-side. Use the 4-foot level at both orientations before shimming.
  4. Check cabinet sides for plumb. Confirm both sides independently with the 4-foot level.
  5. Insert stable, noncompressible shims under cabinet sides, rails, or approved load points. Place shims only at the manufacturer-identified bearing locations.
  6. Do not shim unsupported thin cabinet bottoms. Confirm the shim contacts a rated load point before proceeding.
  7. Check the face frame or front edge against the run reference. Confirm alignment with the marked cabinet-run line before clamping the next cabinet.

G. Joining Adjacent Cabinets (Steps 12–17)

  1. Clamp the adjacent cabinet faces or boxes flush. Confirm no visible step or gap remains at the seam before drilling.
  2. Pre-drill joining locations as required. Use the specified bit size and depth for the cabinet-connection screw.
  3. Join cabinets using approved cabinet-connection screws. Drive each fastener to the specified seating condition without overdriving.
  4. Protect finished face frames from clamp damage. Use padded clamp faces or cauls at every clamped location.
  5. Repeat leveling, plumbing, shimming, clamping, and joining for each cabinet. Confirm each new cabinet against both the floor reference and the previously installed cabinet before joining.
  6. Maintain specified appliance, corner, filler, and wall clearances. Confirm each clearance against the approved cabinet plan as the run progresses.

H. Structural Fastening to Wall Framing (Steps 18–22)

  1. Locate wall studs or backing behind cabinet rails. Use the Project 03 stud markings, confirmed with the stud finder.
  2. Pre-drill cabinet hanging rails or backs as required. Use the specified bit size and depth for the structural cabinet screw.
  3. Secure cabinets to framing using approved structural cabinet screws. Drive each fastener into a verified stud or approved backing to the specified seating condition.
  4. Do not use drywall screws as cabinet-mounting fasteners. Confirm every structural fastener matches the specified cabinet-mounting screw before proceeding.
  5. Trim or secure shims only after the run is accepted. Leave shims in place and untrimmed until the Acceptance Standards inspection is complete.

I. Run Inspection and Correction (Steps 23–26)

  1. Verify top plane with a long straightedge. Check the full run for continuity and flatness within the approved tolerance.
  2. Check face alignment, door and drawer openings, run length, and countertop support. Compare each measurement against the approved cabinet plan and schedule.
  3. Correct twist, rocking, separated faces, and wall gaps before fillers or countertops. Re-shim, re-clamp, or re-fasten any cabinet that fails this check.
  4. Document fastener and shim locations. Record each structural-screw, connection-screw, and shim location on the base-cabinet installation record.

Common Verification Inspection

Inspect the completed run using the same checks regardless of which room or cabinet run it covers.

  • Reference check: confirm the run was positioned from the correct controlling corner, high point, or cabinet and the confirmed floor high point.
  • Level and plumb check: confirm every cabinet is level front-to-back and side-to-side and plumb on both sides.
  • Joining check: confirm adjacent faces are flush with no visible step or gap at the seam.
  • Structural check: confirm every cabinet is fastened to a verified stud or approved backing with the specified structural cabinet screw, with no drywall screws used as cabinet-mounting fasteners.
  • Top-plane check: confirm the full run is continuous and flat across a long straightedge within the approved tolerance.
  • Clearance check: confirm appliance, corner, filler, and wall clearances match the approved cabinet plan and schedule.

Failure Points and Troubleshooting

A Cabinet Rocks After Shimming

Probable cause: a shim placed at an unsupported point rather than an approved load point, or a shim that has compressed or shifted after seating.
Confirmation: check the cabinet with the 4-foot level in both directions and inspect each shim location against the manufacturer's approved load points.
Correction: remove and reposition the shim at an approved load point, using stable, noncompressible shim material, and recheck level and plumb before rejoining or refastening.
Reinspection: repeat the level and plumb check and confirm the cabinet no longer rocks under a controlled hand check.

Adjacent Cabinet Faces Do Not Align Flush

Probable cause: one cabinet was leveled and shimmed independently of its neighbor, or the clamp was released before the joining screws were fully seated.
Confirmation: hold a straightedge across the seam and measure the offset at the widest point.
Correction: loosen the joining screws, re-clamp the faces flush, and reinstall the joining screws to the specified seating condition.
Reinspection: recheck the seam with the straightedge and confirm no visible step remains.

The Top Plane Is Not Continuous Across the Run

Probable cause: an intermediate cabinet was leveled to the floor rather than to the established top-plane reference from the controlling cabinet, or a shim settled after the initial check.
Confirmation: hold the 6-foot straightedge across the affected section and measure the gap at its widest point.
Correction: re-shim the affected cabinet to match the top-plane reference established from the controlling cabinet, not to a new independent floor reading.
Reinspection: repeat the straightedge check across the full run before proceeding to Project 05.

A Structural Cabinet Screw Does Not Engage a Stud or Backing

Probable cause: the stud location marked in Project 03 was not confirmed again at the cabinet's final position, or the cabinet shifted slightly during joining.
Confirmation: recheck the stud location with the stud finder directly behind the cabinet's hanging rail before redriving the fastener.
Correction: relocate the fastener to the confirmed stud or approved backing location; do not rely on a drywall screw or toggle-type anchor as a substitute for the specified structural cabinet screw.
Reinspection: confirm the fastener is fully seated in the verified stud or backing before proceeding to the next cabinet.

A Run-Length or Appliance-Opening Dimension Does Not Match the Plan

Probable cause: a cabinet was installed out of the approved sequence, or an accumulated small gap at each joined seam has compounded across the run.
Confirmation: measure the completed run length and each appliance opening and compare against the approved cabinet plan and schedule.
Correction: identify the affected seam or cabinet, loosen the joining screws, and reposition the cabinet to the correct sequence and clearance before refastening.
Reinspection: remeasure the run length and appliance openings after correction and before Project 05 begins.
