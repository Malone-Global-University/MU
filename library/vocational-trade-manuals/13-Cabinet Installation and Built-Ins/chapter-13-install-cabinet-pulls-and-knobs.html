Job Overview

This project qualifies you to lay out and install pulls and knobs on a controlled cabinet set using repeatable templates and clean, supported drilling. The finished work product is a completed set of cabinet doors and drawer fronts with hardware installed at consistent locations, orientation, and spacing across the entire run, inspected against the approved hardware schedule and elevation. This project depends on the aligned, fully adjusted doors and drawer fronts completed in Project 12 and produces the final cosmetic and functional hardware condition carried into the Chapter 20 capstone punch inspection.

You will review the approved hardware schedule and elevation, confirm every door and drawer front is fully aligned, build or select a rigid template, test the template on matching scrap, drill and install hardware working from consistent reference edges, check alignment across adjacent doors and drawers after each group, correct template drift before continuing, and inspect the finished spacing, orientation, finish condition, and fastener engagement of every installed piece.

This project assumes the cabinet doors, drawer fronts, hinges, and slides installed and adjusted in Project 12 are complete, aligned, and operating correctly before any hardware hole is drilled. It sits immediately after Project 12 and immediately before the Chapter 19 repair project, which corrects any hardware condition that develops after installation, and before the Chapter 20 capstone, which integrates this project's hardware installation into the complete cabinet system.

Skills You Will Demonstrate

  • CB-073 — Create or select a rigid hardware template matching the approved hardware style, size, center-to-center spacing, and orientation.
  • CB-074 — Mark hardware locations on each door and drawer front from the approved cabinet type, top, bottom, left, right, door, and drawer references.
  • CB-075 — Drill clean, perpendicular, supported hardware holes at the marked locations without breakout or finish damage.
  • CB-076 — Install pulls and knobs with the correct screws and washers, tightened securely without crushing the door or drawer face.
  • CB-077 — Maintain consistent spacing, orientation, and reference-edge use across every door and drawer front in the cabinet set.
  • CB-078 — Inspect hardware alignment, spacing, finish protection, screw engagement, and operation across the completed set.

Where This Skill Fits

This project depends directly on Project 12: a door or drawer front that is not fully aligned before hardware drilling will carry that misalignment into every hardware hole drilled from it, and a template built or tested before Project 12's adjustments are final will drift as soon as doors are re-hung. The reference-edge discipline used here — marking every hardware location from the same top, bottom, left, right, door, and drawer references rather than by eye — is the same discipline the Chapter 19 repair project relies on when tracing a misaligned pull back to an inconsistent reference edge or an untested template.

The finished hardware set produced here is the last cosmetic and functional installation step before the Chapter 20 capstone's final punch inspection, which checks appliance clearances, door swings, drawer extension, and every moving and non-moving cabinet component together.

Scope and Intended Use

This project covers:

  • reviewing the approved hardware style, size, center-to-center spacing, orientation, location, and accessibility requirements
  • confirming that doors and drawer fronts are fully aligned before drilling begins
  • creating or selecting a rigid hardware template and testing it on scrap matching the cabinet material
  • verifying screw length through the scrap assembly before installing any hardware in a finished face
  • clamping the template, marking hole centers, supporting the back face, and drilling perpendicular, supported holes
  • installing pulls and knobs with the correct screws and washers, tightened securely without crushing the face
  • repeating installation using the same reference edges and checking alignment across adjacent doors and drawers after each group
  • correcting template drift before continuing and inspecting spacing, orientation, finish protection, screw engagement, and operation

This project does not authorize the learner to:

  • select hardware style, size, center-to-center spacing, or location independently of the approved hardware schedule and elevation
  • drill a hardware hole in a finished door or drawer front before the template has been tested on matching scrap and the screw length has been verified
  • enlarge a misdrilled hole to hide a layout error unless the specific hardware system permits it
  • substitute a longer or different-diameter screw than the one specified for the hardware and substrate

Any hardware location that conflicts with the approved schedule, or a substrate condition that will not support the specified screw, must stop drilling in that location until direction is obtained.

Trade Note

A hardware hole is one of the few cabinet defects that cannot be fully hidden once drilled. Treating the template test on scrap as an optional step, rather than as the last opportunity to catch a spacing or screw-length error before it appears on a finished face, is the most common cause of hardware rework traced back to this project during the Chapter 19 repair project.

Setup Standard

Do not begin marking any hardware location until every door and drawer front in the cabinet set is confirmed fully aligned from the completed Project 12 adjustment (Step 2). Do not drill a finished face before the template has been tested on scrap matching the cabinet material and the screw length has been verified through that scrap assembly (Steps 5–6).

Shop Tip

Mark the template itself with the cabinet type, top, bottom, left, right, door, and drawer references before the first hole is drilled. A template without its own reference marks is the most common source of an inverted or mirrored hardware location on the next door or drawer front.

Establish these references before any marking or drilling begins, and do not change them once hardware installation has started on a given cabinet run.

Common Field Fundamentals

Every hardware installation in this book, whether a single knob or a full cabinet set, follows the same order: confirm the approved schedule, confirm the doors and drawers are ready, build and test a rigid template, mark and drill from consistent references, install and tighten the hardware, and check alignment across the group before continuing. Learning that sequence here protects every hardware installation performed later in this book and during the Chapter 19 repair project.

Separating Reference-Edge Errors From Substrate or Screw Errors

Every hardware defect found during this project traces to one of two causes: an inconsistent reference edge or template position, or an incompatible screw length or substrate condition. A pull that lands at a different height than its neighbor is a reference-edge or template-drift problem, correctable by rechecking the template against the marked reference edges before the next hole is drilled. A screw that bottoms out before drawing the hardware tight, or that pushes through a finished face, is a screw-length or substrate problem, correctable only by verifying the screw length against the actual door or drawer-front thickness on scrap before continuing.

Measurement Cue — Verify Center-to-Center Spacing Against the Hardware Schedule

Confirm the template's center-to-center spacing matches the approved hardware schedule before the first hole is drilled in any finished face. Do not assume a template is correct because it matches a similar hardware piece installed elsewhere; verify it against the current schedule for this cabinet set.

Critical Safety Cue — Do Not Substitute a Longer Screw

Do not install a hardware screw longer than the length verified on matching scrap. A screw that is too long for the door or drawer-front thickness will penetrate the finished face, producing a defect that cannot be corrected without replacing the component. Verify screw length on scrap before installing hardware in any finished face.

Field Safety Cue — Support Removed Doors and Drawer Fronts

Support every removed door or drawer front on a padded, stable surface during marking, drilling, and installation. Do not clamp a finished face directly against a hard or unpadded surface, and do not drill with the component unsupported or hand-held.

Manufacturer Cue — Bit Size and Drilling Depth

Use the drill bit diameter specified for the hardware screw, and drill only to the depth required to seat the screw and washer as specified. Do not use a bit sized for a different hardware system, and do not drill deeper than required for the specified fastener.

Code Alert — Accessible Hardware Requirements Vary by Jurisdiction

Required hardware type, operable force, and mounting height for accessible cabinet runs vary by adopted code and jurisdiction. Verify the current requirement with the authority having jurisdiction and the approved hardware schedule before confirming that installed hardware meets an intended accessibility requirement.

E. Schedule Review and Readiness Confirmation (Steps 1–2)

  1. Review the approved hardware style, size, center-to-center spacing, orientation, location, and accessibility requirements. Confirm the hardware schedule revision matches the current cabinet plan and elevation before proceeding.
  2. Confirm doors and drawer fronts are fully aligned before drilling. Verify the Project 12 alignment record shows completed reveal consistency and hardware operation for every door and drawer front in the cabinet set.

F. Template Creation and Verification (Steps 3–7)

  1. Create or select a rigid hardware template. Confirm the template's center-to-center spacing and hole diameter match the approved hardware schedule.
  2. Mark cabinet type, top, bottom, left, right, door, and drawer references on the template. Use the marking pencil to record every reference the template will be positioned against.
  3. Test the template on scrap matching the cabinet material. Mark, drill, and install a sample piece of hardware on the scrap before drilling any finished face.
  4. Verify screw length through the scrap assembly. Confirm the specified screw seats the hardware and washer securely without penetrating the opposite face.
  5. Clamp the template to the first door or drawer front. Align the template's marked references against the door or drawer front's actual top, bottom, left, and right edges.

G. Marking and Drilling (Steps 8–13)

  1. Confirm the marked position against the approved elevation. Check the template's position against the hardware schedule and elevation before marking hole centers.
  2. Mark hole centers with an awl or approved punch. Mark each hole center firmly enough to prevent bit wander during drilling.
  3. Support the back face with a sacrificial block to reduce breakout. Position the backer block directly behind each marked hole center before drilling.
  4. Drill perpendicular holes using the specified bit. Hold the drill square to the face and drill to the depth required for the specified fastener.
  5. Drill from both faces only when the approved material method requires it. Follow the specified drilling method for the door or drawer-front substrate; do not drill from both faces unless the method requires it.
  6. Clean chips without scratching the finish. Use the vacuum or brush to remove chips before installing hardware.

H. Installation, Repetition, and Inspection (Steps 14–21)

  1. Install the pull or knob with the correct screws and washers. Confirm the screw and washer match the hardware schedule before installing.
  2. Tighten securely without crushing the door or drawer face. Stop tightening once the hardware is snug and fully seated against the face.
  3. Repeat using the same reference edges. Position the template against the same top, bottom, left, and right references for each subsequent door and drawer front.
  4. Check alignment across adjacent doors and drawers after each group. Compare installed hardware height and spacing against its neighbors before continuing to the next group.
  5. Correct template drift before continuing. Recheck the template against the marked reference edges if any installed piece is out of alignment with its neighbors.
  6. Do not enlarge holes to hide major layout errors unless the hardware system permits it. Identify and correct the reference-edge or template error before drilling additional holes.
  7. Inspect spacing, orientation, finish protection, screw engagement, and operation. Confirm every installed piece against the Acceptance Standards table.
  8. Document template dimensions and hardware locations. Record the final template dimensions and any correction on the hardware installation record.

Common Verification Inspection

Inspect the completed hardware installation using the same checks regardless of which cabinet run or set it covers.

  • Schedule check: confirm installed hardware style, spacing, orientation, and location match the approved hardware schedule and elevation.
  • Reference check: confirm every hole was marked and drilled from the same tested template and reference edges.
  • Hole-quality check: confirm holes are clean, perpendicular, and free of breakout or finish damage on both faces.
  • Fastener check: confirm the verified screw length and specified washer are installed and fully seated without crushing the substrate.
  • Alignment check: confirm consistent height, side spacing, and orientation across every door and drawer front in the set.
  • Documentation check: confirm the template test, screw-length verification, and final inspection are recorded on the hardware installation record.

Complete the Hardware Installation Record in the Inspection Checklist section below before considering the cabinet run ready for the Chapter 19 repair project or the Chapter 20 capstone.

Failure Points and Troubleshooting

An Installed Pull Sits at a Different Height Than Its Neighbor

Probable cause: the template was not positioned against the same reference edge used on the neighboring door or drawer front, or the template shifted during clamping.
Confirmation: measure both installed pulls against the approved hardware schedule and against each other from the same reference edge.
Correction: remove the affected hardware, recheck the template against the marked reference edges, and redrill only if the hardware system's permitted trim method allows correcting the existing hole; otherwise replace the affected component.
Reinspection: repeat the alignment check across the corrected door or drawer front and its neighbors.

A Screw Bottoms Out Before Drawing the Hardware Tight

Probable cause: the installed screw is shorter than the length verified on scrap, or the actual door or drawer-front thickness differs from the tested scrap.
Confirmation: measure the actual door or drawer-front thickness at the hardware location and compare it against the scrap used for screw-length verification.
Correction: install the correct screw length specified for the actual substrate thickness; do not substitute a longer screw without reverifying it on matching scrap first.
Reinspection: confirm the hardware seats fully and tightens securely without penetrating the opposite face.

A Drilled Hole Shows Breakout on the Back Face

Probable cause: the backer block was not positioned directly behind the marked hole center, or the drill was not held perpendicular during drilling.
Confirmation: inspect the back face for chip-out and compare the hole's condition against the acceptance standard for finish protection.
Correction: if the hardware system's permitted trim method allows it, correct the breakout using the approved method; otherwise replace the affected door or drawer front.
Reinspection: confirm the corrected or replaced component meets the hole-quality acceptance standard before installing hardware.

The Template Position Drifts Across a Long Cabinet Run

Probable cause: an accumulated small error in reference-edge placement carried from one door or drawer front to the next without rechecking against the approved elevation.
Confirmation: measure the template position against the approved elevation at several points across the run, not only against the immediately preceding piece.
Correction: stop installation, recheck the template against the original reference edges and the approved elevation, and correct the position before continuing.
Reinspection: repeat the alignment check across the full run once the template position is corrected.

An Installed Knob or Pull Loosens Shortly After Installation

Probable cause: the screw was not tightened to a fully seated condition, or the substrate was crushed or stripped during installation.
Confirmation: inspect the screw engagement and substrate condition at the affected hole.
Correction: repair the damaged fastener hole using an approved method, or replace the affected component; do not overtighten a replacement screw into a stripped hole.
Reinspection: confirm the corrected hardware is secure and does not loosen under normal operation.

Cleanup

This project ends with installed, inspected cabinet hardware, not with templates or removed components left in place.

  • Remove and store the hardware template and any test scrap
  • Store the hardware installation record, screw-length verification note, and supporting photographs with the project's permanent documentation
  • Confirm every door and drawer front operates correctly with hardware installed before leaving the site
  • Communicate any open discrepancy and its required correction to the responsible designer or general contractor before the Chapter 19 or Chapter 20 project begins
