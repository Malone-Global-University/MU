05-Residential Roofing

This field manual provides practical, project-based instruction for installing, flashing, ventilating, repairing, and maintaining residential asphalt shingle roofing systems. Students learn professional roofing practices while completing progressively more complex field projects.

Overview

Residential Roofing develops the skills required to install a complete residential roofing system from roof deck preparation through final ridge ventilation and drainage. Students learn safe roofing practices, weatherproofing techniques, flashing installation, ventilation, gutter systems, leak repair, and roof restoration before completing a capstone roofing project.

Learning Outcomes

  • Prepare roof decks for safe and code-compliant roofing installation.
  • Install drip edge, underlayment, and ice-and-water protection systems.
  • Lay out and install asphalt shingles using professional techniques.
  • Install flashing systems around valleys, walls, and roof penetrations.
  • Complete ridge ventilation, soffits, fascia, gutters, and drainage systems.
  • Diagnose and repair common roofing failures and water leaks.
  • Complete a fully roofed residential practice structure from deck preparation through final inspection.
