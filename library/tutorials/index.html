Tutorials Overview

Tutorials are used to explain how to perform basic functions like installing software and other mundane processes. This keeps lesson pages focused on the main learning topic by reducing the repetition of basic steps.

Categories

  • Computer Basics

    Basic computer skills, file management, folders, downloads, screenshots, keyboard shortcuts, and everyday digital tasks.

  • Software Installation

    Step-by-step guides for installing, updating, configuring, and uninstalling common software used in MGU courses.

  • Web and Internet

    Tutorials for browsers, URLs, downloads, cloud tools, accounts, web navigation, bookmarks, and safe internet use.

  • Programming Tools

    Guides for setting up coding environments, editors, terminals, Python, Node.js, Git, GitHub, and related development tools.

  • Documents and Files

    Help with PDFs, DOCX files, spreadsheets, compression, file conversion, backups, and organizing course materials.

  • Website Maintenance

    Internal MGU tutorials for editing HTML, updating links, managing components, fixing paths, clearing cache, and maintaining pages.

  • Media and Images

    Tutorials for image formats, resizing, screenshots, icons, WebP conversion, basic editing, and organizing visual assets.

  • Security and Accounts

    Basic guidance for passwords, account recovery, two-factor authentication, safe downloads, phishing awareness, and privacy settings.

  • Research and Library Use

    Guides for using the MGU Library, finding dictionary entries, navigating the encyclopedia, using references, and organizing research.

  • Troubleshooting

    General problem-solving guides for common computer, browser, software, file, website, and course-access issues.
