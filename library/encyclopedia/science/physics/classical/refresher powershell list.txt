# PowerShell Refresher — Practical Fundamentals

## Lesson Goal

Rebuild working familiarity with everyday PowerShell so you can comfortably navigate folders, manipulate files, inspect directories, and run scripts without stopping to look up syntax constantly.

---

## Part 1: Understanding What PowerShell Is

PowerShell is both:

1. A command shell (like Command Prompt, Bash)
2. A scripting language built around **objects**, not plain text

That means commands return structured data you can pipe into other commands.

Example:

```powershell
Get-ChildItem | Sort-Object Name
```

This does not pass text lines.
It passes file objects.

---

## Part 2: Navigation Commands

### Show Current Directory

```powershell
Get-Location
```

Alias:

```powershell
pwd
```

---

### Change Directory

```powershell
Set-Location C:\MGU
```

Alias:

```powershell
cd C:\MGU
```

Move up one folder:

```powershell
cd ..
```

Move up multiple:

```powershell
cd ..\..
```

---

### List Files/Folders

```powershell
Get-ChildItem
```

Alias:

```powershell
ls
dir
```

List recursively:

```powershell
Get-ChildItem -Recurse
```

## Part 3: Reading / Writing Files

### Read File Contents


---

### Write New File

```powershell

```

Overwrites file.

### Append to File

```powershell

```

## Part 4: Creating / Moving / Deleting Files

### Create Folder

```powershell

```

---

```powershell

```

---

### Copy File

```powershell

```

### Move / Rename

```powershell

```

---

### Delete

```powershell

```

Delete folder recursively:

```powershell

```

### Reverse Sort

---

### Save Output to File

```powershell

```

* `<=` = only in first file
* `=>` = only in second file

## Part 11: Mental Model for Pipelines

Think of PowerShell pipelines as:

```text
Get Stuff
→ Filter Stuff
→ Transform Stuff
→ Output Stuff
```

Example:

```powershell
Get-ChildItem *.webp |
Sort-Object Name |
Select-Object Name |
Out-File images.txt
```

Translation:

> Find webp files
> Sort alphabetically
> Keep only names
> Save to file

# PowerShell Refresher Part II — Scripting, Automation, and Real Workflow Use

## Lesson Goal

Move from “running commands” to “thinking procedurally” in PowerShell:

* automate repetitive tasks
* write reusable scripts
* manipulate filenames/text
* batch process folders
* understand control flow

---

## Part 1: The Core Programming Mindset

PowerShell scripting is mostly:

```text
Get data
Loop through data
Check conditions
Do something to matching items
Output results
```

That pattern drives nearly every automation script.

---

## Part 2: Variables and Data Types

### Variables

```powershell
$name = "Joseph"
$count = 57
$path = "C:\PATH"
```

Use them:

```powershell
Write-Output $name
```

---

### Arrays

Store lists:

```powershell
$chapters = @(
    "chapter-1",
    "chapter-2",
    "chapter-3"
)
```

Access item:

```powershell
$chapters[0]
```

---

## Part 3: Loops

### ForEach Loop

Most common for file work:

```powershell
Get-ChildItem *.html | ForEach-Object {
    Write-Output $_.Name
}
```

`$_` means:

> Current object in pipeline

---

### Traditional Foreach

```powershell
foreach ($file in Get-ChildItem *.html) {
    Write-Output $file.Name
}
```

Often easier to read in scripts.

---

## Part 4: Conditional Logic

### If Statements

```powershell
if ($count -gt 50) {
    Write-Output "Large count"
}
```

---

### Comparison Operators

```text
-eq   equal
-ne   not equal
-gt   greater than
-lt   less than
-ge   greater/equal
-le   less/equal
```

---

### Example

```powershell
$fileCount = (Get-ChildItem).Count

if ($fileCount -eq 0) {
    Write-Output "Folder empty"
}
```

---

## Part 5: String Manipulation

### Replace Text

```powershell
$name = "chapter-01.html"
$name -replace ".html", ""
```

---

### Replace Typo in Filenames

```powershell
Get-ChildItem *.webp | Rename-Item -NewName {
    $_.Name -replace "potenial", "potential"
}
```

---

## Part 6: Filtering

### Where-Object

Filter pipeline objects:

```powershell
Get-ChildItem | Where-Object {
    $_.Extension -eq ".html"
}
```

---

### Real Example

Find large files:

```powershell
Get-ChildItem -Recurse | Where-Object {
    $_.Length -gt 5MB
}
```

---

## Part 7: Functions

Reusable mini-tools.

```powershell
function Say-Hello {
    param($Name)

    Write-Output "Hello, $Name"
}
```

Run:

```powershell
Say-Hello "Joseph"
```

---

## Part 8: Practical Batch Automation Examples

### Rename All Files to Lowercase

```powershell
Get-ChildItem | Rename-Item -NewName {
    $_.Name.ToLower()
}
```

---

### Add Prefix to Files

```powershell
Get-ChildItem *.html | Rename-Item -NewName {
    "physics-" + $_.Name
}
```

---

### Remove Spaces From Filenames

```powershell
Get-ChildItem | Rename-Item -NewName {
    $_.Name -replace " ", "-"
}
```

## Part 9: Reading and Modifying File Contents

### Replace Text in Many Files

```powershell
Get-ChildItem *.html | ForEach-Object {
    (Get-Content $_.FullName) `
```

### Bulk Change Dates

```powershell
Get-ChildItem *.html | ForEach-Object {
    (Get-Content $_.FullName) `
        -replace "4-29-2025", "4-29-2026" |
    Set-Content $_.FullName
}
```

## Part 1: Parsing Filenames and Paths

### Filename Components

```powershell id="6sm8j1"
$file = Get-Item "chapter-57.html"

$file.Name
$file.BaseName
$file.Extension
$file.FullName
$file.DirectoryName
```

Returns:

```text id="6bbx08"
chapter-57.html
chapter-57
.html
C:\MGU\chapter-57.html
C:\MGU
```

---

## Part 2: Advanced Regex Extraction

### Extract Chapter Number

```powershell id="s3h8ma"
if ($file.BaseName -match '\d+') {
    $chapterNumber = [int]$matches[0]
}
```

Useful for numeric sorting.

---

### Extract Multiple Groups

```powershell id="98z0vv"
$name = "lesson-24-3"

if ($name -match 'lesson-(\d+)-(\d+)') {
    $unit = $matches[1]
    $lesson = $matches[2]
}
```

---

## Part 3: Sorting Numerically Instead of Alphabetically

Alphabetical sort causes:

```text id="7s79be"
chapter-1
chapter-10
chapter-11
chapter-2
```

Fix:

```powershell id="k8z4rj"
Get-ChildItem *.html |
Sort-Object {
    [int]($_.BaseName -replace '\D','')
}
```

---

## Part 6: Generating Index Pages Automatically

### Basic Folder Index Builder

```powershell id="hvkk4m"
$items = Get-ChildItem -Directory

$links = foreach ($item in $items) {

</ul>
"@

$html | Set-Content "index.html"
```

## Part 7: Frontmatter / Metadata Parsing

If file starts like:

```yaml id="wtqfp5"
---
title: Physics Foundations
uploaded: 5-4-2026
tags:
  - science
---
```

Extract block:

```powershell id="fzjwjf"
$content = Get-Content file.md -Raw

if ($content -match '(?s)^---(.*?)---') {
    $frontmatter = $matches[1]
}
```

## Part 8: Validation Scripts

### Find Missing Required YAML Fields

```powershell id="puz8bm"
Get-ChildItem *.md | ForEach-Object {
    $content = Get-Content $_.FullName -Raw

    if ($content -notmatch 'title:') {
        Write-Output "$($_.Name) missing title"
    }
}
```

---

### Detect Broken Internal Links

```powershell id="1r1vdt"
Get-ChildItem *.html | ForEach-Object {
    $content = Get-Content $_.FullName -Raw

    [regex]::Matches($content,'href="([^"]+)"') |
    ForEach-Object {
        $href = $_.Groups[1].Value

        if (-not (Test-Path $href)) {
            Write-Output "Broken link: $href"
        }
    }
}
```

### Generate On This Page from H2 Headings

```powershell id="jlwmu5"
$content = Get-Content index.html -Raw

[regex]::Matches($content,'<h2 id="([^"]+)">(.*?)</h2>') |

    "<li><a href=""#$($_.Groups[1].Value)"">$($_.Groups[2].Value)</a></li>"
```

```powershell id="e1uxvn"

ForEach-Object { $_.Groups[1].Value }

$ids | Group-Object | Where-Object Count -gt 1
```
