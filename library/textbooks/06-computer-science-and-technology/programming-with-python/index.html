Programming with Python

Programming with Python introduces learners to the fundamentals of computer programming using one of the most readable and versatile languages in modern computing. Students build practical problem-solving skills through variables, control flow, functions, data structures, files, classes, testing, and complete applications.

Overview

This textbook develops programming fluency through progressive instruction in Python. Students begin with core syntax and simple programs, then move into decisions, repetition, functions, structured data, file handling, exception management, object-oriented programming, testing, and project development. The course emphasizes both logical thinking and practical software creation.

Learning Outcomes

  • Write and run Python programs using core language features.
  • Use variables, expressions, decisions, and loops to control program behavior.
  • Design functions and reusable code to simplify problem solving.
  • Work with strings, lists, dictionaries, files, and structured data.
  • Handle errors and exceptions responsibly.
  • Apply object-oriented programming concepts using classes and objects.
  • Test, debug, and improve program quality.
  • Build small, complete Python applications.
  • Read and write JSON data for modern software workflows.
  • Develop confidence in computational thinking and software development.
