Computer Science and Technology

The Computer Science and Technology shelf organizes complete course-style learning PDF formatted ebooks across computing, programming, systems, data, web development, artificial intelligence, and machine learning foundations. Each textbook is designed for sustained study, chapter-by-chapter instruction, and long-term technical reference.

Overview

This shelf gathers MGU technology and computing textbooks into a single library index. The collection includes foundational computer science, data and information systems, Python programming, web development, data science, artificial intelligence, and machine learning mathematics.

These textbooks are built for structured learning rather than quick reference. They organize technical topics into chapters, examples, exercises, and conceptual progressions so students can build durable skill across the full technology pathway.

Core Computing

These texts establish the conceptual base for computation, algorithms, software, hardware, networking, and security.

  • Computer Science Foundations

    A computer science textbook covering algorithms, data, abstraction, programming concepts, hardware, software, networks, security, computation, and problem solving.

Data and Information Systems

These texts focus on data structures at the organizational level, including databases, records, transactions, and information architecture.

  • Data Databases and Information Systems

    A data and information systems textbook covering data models, databases, SQL, information architecture, records, transactions, systems, and organizational data use.

Programming

These texts develop coding fluency, problem solving, and software construction skills.

  • Programming with Python

    A Python programming textbook covering syntax, variables, control flow, functions, data structures, files, errors, modules, projects, and computational thinking.

Web Development

These texts support front-end development and accessible site construction.

  • Web Development HTML CSS and JavaScript

    A web development textbook covering HTML structure, CSS styling, JavaScript behavior, accessibility, responsive design, site organization, and front-end development.

Data Science, AI, and Machine Learning

These texts connect statistics, modeling, computation, and intelligent systems into a modern analytical pathway.

  • Data Science Foundations

    A data science textbook covering data collection, cleaning, visualization, probability, statistics, modeling, databases, computation, and responsible data analysis.

  • Artificial Intelligence Foundations

    An artificial intelligence foundations textbook covering logic, search, representation, reasoning, machine learning, neural networks, ethics, and intelligent systems.

  • Machine Learning Mathematics

    A machine learning mathematics textbook covering linear algebra, calculus, probability, optimization, loss functions, gradients, regression, classification, and model evaluation.
