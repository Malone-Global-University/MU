Monophthongs and Vowel Systems

From individual vowel qualities to organized phonemic inventories

Chapter Objectives

By the end of this chapter, you will be able to:

  1. Identify monophthongs in controlled data and distinguish them from dynamic vowel qualities.
  2. Distinguish a phonemic vowel inventory from the phonetic realizations of its members.
  3. Analyze how a vowel system spaces its members within the vowel quadrilateral to maintain contrast.
  4. Compare vowel inventories of different sizes without ranking them as simpler or more complex.
  5. Identify a contextual allophonic pattern within a vowel system from controlled data.
  6. Create a small, verified vowel chart for a specific language variety.

An Observable Problem

Chapter 15 gave you a chart with roughly twenty-three monophthong positions, arranged as pairs across three columns and four main rows. No language uses all of them. Spanish is regularly described as having five vowel phonemes. Standard Southern British English is regularly described as having twelve or more monophthong phonemes, depending on the analysis. Hawaiian has five. Norwegian has as many as nine or ten, depending on the dialect and the analyst.

This raises a question that the full quadrilateral, by itself, cannot answer: given a set of possible vowel positions, how does a particular language decide which ones to use, and how many it needs? A five-vowel language does not simply pick five positions at random from the twenty-three available. Something more systematic is going on.

Before continuing, examine the following two observations:

  1. Suppose a language has exactly three vowel phonemes. Predict where in the quadrilateral those three vowels are most likely to be positioned, and explain your reasoning using what you know about auditory distinctiveness from Chapter 15.
  2. Now suppose a second language has twelve vowel phonemes occupying the same quadrilateral. Would you expect the average distance between neighboring vowels to be larger or smaller than in the three-vowel language? What does this predict about how precisely listeners of the twelve-vowel language must distinguish nearby vowel qualities?
Vowel Systems

Inventory Versus Realization

Purpose: apply the Chapter 5 phoneme-allophone distinction directly to vowels, establishing the difference between the abstract set of contrastive vowel categories a language uses and the concrete phonetic values that realize them.

A phonemic vowel inventory is the set of vowel categories that contrast meaning in a language or variety — the vowels that, if substituted for one another, could change one word into a different word or into a nonsense form. A phonetic realization is the actual articulatory and acoustic value produced for a given phoneme in a given context, speaker, and style.

These are not the same claim, and confusing them creates two common errors. The first error is assuming that a language's phonemic transcription tells you the exact phonetic target of each vowel; Chapter 15 already warned against this for individual vowel symbols, and the warning applies with equal force to entire systems. The second error is assuming that because a phonetic realization varies — by speaker, by position in the word, by speech rate — the underlying phonemic category is somehow unstable. It is not: the category can remain constant while its phonetic expression shifts in predictable, rule-governed ways.

Building a vowel inventory, the skill this chapter develops, means identifying the phonemic categories first, using contrast as the primary evidence, and only afterward describing the phonetic range each category covers. This order matters. Beginning with phonetic detail before establishing contrast risks treating predictable variation as if it were a separate phoneme, inflating the inventory with categories the language does not actually use contrastively.

Concept Check — Two Claims, One Symbol

When a vowel chart for a language places the symbol /i/ at the high front corner, it is making a phonemic claim: this language has a vowel category that contrasts with its other vowel categories and that occupies the high front region of the vowel space more closely than any other category in the system. It is not claiming that every token of that phoneme, in every word, spoken by every speaker, hits an identical acoustic target. The phonemic chart is a map of contrastive categories. A phonetic chart, plotting measured or observed realizations, is a different and complementary document — one you will build later in this chapter and again in Chapter 30.

Reconstructing an Inventory from Controlled Data

Purpose: give learners a repeatable procedure for identifying a language's vowel phonemes from a small, controlled dataset, using minimal and near-minimal pairs as the primary evidence.

Reconstructing a vowel inventory is an application of the contrast-and-distribution reasoning introduced in Chapter 5, now scaled up to a full system. The basic procedure has four steps:

  1. Collect comparable forms. Gather a set of words or forms that share as much phonological material as possible except for the vowel under comparison — ideally minimal pairs, where a single vowel substitution is the only difference between two forms with two different meanings.
  2. Test for contrast. If substituting one vowel quality for another changes meaning, the two qualities represent separate phonemes. If substitution does not change meaning — and the vowels instead appear in complementary, non-overlapping contexts — the qualities may be allophones of a single phoneme rather than separate phonemes.
  3. Tally the distinct phonemic categories. Once contrast has been tested across enough data, list the categories that have been shown to contrast. This list is the provisional phonemic inventory.
  4. Plot the inventory. Place each confirmed phoneme in the vowel quadrilateral using its most typical, careful-speech realization as the representative point.

This procedure depends on the data being genuinely comparable and the meanings genuinely distinct — an unverified or invented pair is not evidence. It also depends on testing enough forms: a single minimal pair establishes that two vowel qualities can contrast in that specific environment, but a well-supported inventory claim normally draws on multiple pairs across different phonological contexts.

Evidence Note — Why Multiple Pairs Matter

A single minimal pair is suggestive but not conclusive. If a source provides only one pair distinguishing two vowel qualities, and that pair happens to occur in an unusual phonological environment, the contrast might be an isolated exception rather than a general pattern in the system. Responsible inventory-building looks for the same contrast confirmed across several word pairs in several environments before treating it as a stable phonemic distinction, and it treats a single unconfirmed pair as a hypothesis to test further rather than as a settled conclusion.

System Spacing and Contrast

Purpose: explain why vowel systems tend to distribute their members to maximize auditory distinctiveness, connecting this pattern to the trapezoid shape introduced in Chapter 15.

Once a phonemic inventory has been reconstructed, a further question becomes available: how are the vowels arranged relative to one another within the vowel space? A well-documented cross-linguistic tendency, sometimes discussed under the heading of dispersion, is that vowel systems tend to space their phonemes so as to keep them auditorily distinct from one another, particularly in smaller systems where every vowel must do more work to remain distinguishable from its neighbors.

This tendency is a description of a common pattern, not a rule that every language must follow. It helps explain why very small vowel systems recur across unrelated languages in strikingly similar shapes — typically anchored near the periphery of the quadrilateral, where auditory distinctiveness is greatest — while larger systems increasingly fill in the interior and central regions, where more, closely spaced contrasts become necessary.

Dispersion is a useful lens for describing why a system looks the way it does, but it is descriptive rather than predictive in any strict sense: it does not tell you with certainty what any specific unfamiliar language's inventory will be, and languages are not obligated to maximize dispersion. Treat it as one piece of evidence to weigh alongside the actual contrastive data you collect.

Comparing Systems

Small Vowel Systems

Purpose: give learners a concrete, well-documented example of a small vowel system and show how its members occupy peripheral, well-dispersed positions.

A widely cited example of a small vowel system is the five-vowel inventory found in Spanish and in many other languages, including Modern Greek, Swahili, and Japanese (analyzed at the phonemic level). The five vowels are typically transcribed /i, e, a, o, u/ and occupy positions close to the peripheral high, mid, and low regions of the quadrilateral.

a

Notice that this system uses only unrounded front vowels and rounded back vowels, avoiding the front rounded and back unrounded positions discussed in Chapter 15. It also uses only peripheral positions, avoiding the central column entirely at the phonemic level. Five well-separated peripheral vowels give listeners a high degree of auditory contrast with relatively few categories to track — a pattern consistent with the dispersion tendency described above.

Because a five-vowel inventory uses so few categories, its members can tolerate a fairly wide range of phonetic variation without risking confusion with a neighboring phoneme. This does not mean the phonetic realizations are unstable — only that the system has more auditory space between neighboring categories than a larger system would have.

Larger Vowel Systems

Purpose: show how larger vowel systems fill in additional height, backness, and central positions, and connect this to the narrower auditory margins between neighboring phonemes.

English vowel systems, in most described varieties, are considerably larger than the five-vowel type. General American English is commonly analyzed with around eleven to twelve monophthong phonemes (analyses vary in how they treat certain vowels before /r/ and in unstressed syllables), and Standard Southern British English is commonly analyzed with a similar or slightly larger monophthong count, in addition to a substantial diphthong inventory that Chapter 17 will address directly.

A system of this size cannot rely only on the most peripheral chart positions. It typically uses additional height distinctions — for example, separating a higher lax vowel from a fully high tense vowel, as in the contrast between the vowels of beat and bit in most English varieties — and it makes active use of the central column, including schwa and, in many varieties, additional central vowels such as those discussed in Chapter 15.

Because more categories are packed into the same quadrilateral, the average auditory distance between neighboring vowels is smaller than in a five-vowel system. This has a direct pedagogical consequence: learners moving from a smaller-inventory first language into a larger-inventory target language, or the reverse, often need focused perceptual training on precisely the contrasts that their first language's system does not require them to distinguish.

Variation Note — Which English Vowel System?

There is no single "English vowel system." General American, Standard Southern British, Scottish Standard English, Australian English, and many other varieties differ in vowel count, vowel quality, and which historical contrasts remain distinct. Some varieties merge vowels that others keep separate — for instance, several North American varieties do not distinguish the vowels of cot and caught, while others do. Any specific vowel count or chart for "English" must name the variety it describes. This chapter's discussion of English vowel-system size is general background; a defensible inventory claim for a specific variety requires the same contrast-testing procedure described earlier in this chapter, applied to that variety's own data.

Size Is Not Complexity

Purpose: prevent the common misconception that inventory size correlates with linguistic sophistication, difficulty, or overall system complexity.

A five-vowel system is not simpler than a twelve-vowel system in any general sense, and a twelve-vowel system is not more advanced than a five-vowel system. Inventory size is one dimension of a language's sound system among many, and languages regularly compensate in other dimensions. A language with few vowel phonemes may have an elaborate consonant inventory, a complex tone system, or phonotactic patterns that place heavy functional demands elsewhere. A language with many vowel phonemes may have a comparatively simple consonant inventory or syllable structure. Complexity, to the extent the term is useful at all, must be evaluated across the whole system, not read off a single number.

This principle follows directly from the Master Instructions' broader policy against ranking languages by simplicity, difficulty, or sophistication, and it applies with particular force to vowel counts, which are an easy but misleading shorthand for "advanced" or "basic" language design.

Common Mistake

Error: Describing a five-vowel language as phonologically "simple" or a twelve-vowel language as phonologically "rich" or "advanced."

Correction: Vowel inventory size reflects how a language organizes one part of its sound system. It says nothing about the language's overall structural complexity, expressive capacity, or age. Describe systems by their actual properties — inventory size, spacing, use of length or nasalization, and so on — rather than by evaluative labels.

Allophony and Notation

Contextual Allophony in Vowel Systems

Purpose: extend the Chapter 5 allophone concept to a worked vowel example, showing how a single phoneme can have systematically different realizations in predictable contexts.

Once an inventory has been established, the next analytical task is often to identify how each phoneme's realization shifts in different phonological environments. This is the same distribution reasoning from Chapter 5: if two phonetically different vowel qualities never contrast — that is, they never distinguish two different words from one another — and instead occur in predictable, non-overlapping (complementary) environments, they are treated as allophones of the same phoneme rather than as separate phonemes.

A frequently cited English example involves the vowel commonly transcribed /æ/. In many varieties of North American English, this vowel is realized with a noticeably raised and often nasalized quality before nasal consonants, as in man or hand, and with a lower, unraised quality elsewhere, as in cat or bag. The raised and unraised qualities do not distinguish separate words from one another; the difference is predictable from the following consonant. This is a textbook case of a single phoneme with two conditioned allophones, describable with the phonemic and phonetic delimiters introduced in Chapter 5: the phoneme is /æ/, and its allophones might be represented as [ɛə] or a raised, nasalized variant before nasals and [æ] elsewhere, depending on the variety and the transcription tradition consulted.

Evidence Note — Confirming an Allophonic Pattern

Claiming an allophonic pattern responsibly requires more than noticing one example. A defensible claim specifies: the two (or more) phonetic realizations involved; the environments in which each occurs; confirmation that the environments are complementary, meaning the realizations do not both occur in the same environment; and confirmation that no minimal pair distinguishes the two realizations. Any specific allophonic claim about a named variety should be checked against a scholarly source for that variety, since the details of raising, nasalization, and conditioning environment vary across regions and are actively studied in the sociophonetic literature.

Centralization, Raising, and Lowering

Purpose: introduce, at a controlled and minimal level, the diacritics needed to represent the kind of vowel-quality shift illustrated in the allophony section above, without expanding into the full diacritic inventory reserved for Chapter 25.

Chapter 25 will consolidate the complete set of diacritics used across the course. At this stage, only three are introduced, and only because they are needed to represent the kind of allophonic shift just discussed:

New notation — vowel-quality shift diacritics
Diacritic Name Meaning Example
◌̈CentralizedThe vowel is produced closer to the central column than its base symbol would suggest.ë — a centralized realization of e
◌̝RaisedThe vowel is produced with a higher tongue position than its base symbol would suggest.e̝ — a raised realization of e
◌̞LoweredThe vowel is produced with a lower tongue position than its base symbol would suggest.e̞ — a lowered realization of e

These diacritics attach to a base vowel symbol to indicate a directional shift from that symbol's reference position, rather than requiring a completely different symbol for every minor phonetic adjustment. They are useful precisely because allophonic variation is often a small, directional shift rather than a jump to a different phonemic category, and the notation should reflect that relationship rather than obscuring it.

Note: these three diacritics are introduced productively for the limited purpose of representing vowel allophony in this chapter's exercises. The full diacritic set, including advancement, retraction, and other combining marks, is consolidated in Chapter 25.

Broad Chart Versus Phonetic Plot

Purpose: distinguish two different products a phonetician might build from the same data — a broad phonemic chart and a phonetic plot showing realizational detail — and connect this distinction forward to Chapter 24.

Two different, equally legitimate charts can be built from the same vowel data, and it is important to know which one you are building at any given moment.

A broad (phonemic) chart plots one point per phoneme, using slashes and a representative or citation-form realization. Its purpose is to show the contrastive system: how many vowel categories the language has and their approximate relative positions.

A phonetic plot may show multiple points per phoneme — one for each measured or observed allophonic variant — using square brackets. Its purpose is to show the actual range of realizations a single phonemic category covers across different contexts, speakers, or styles.

Neither chart is more accurate than the other; they answer different questions. A broad chart answers "how many contrastive vowel categories does this system have, and roughly where are they?" A phonetic plot answers "what does this specific category actually sound like across the contexts in which it occurs?" Chapter 24 will formalize this broad-versus-narrow distinction for full transcription; this chapter previews it specifically for vowel-system charts, since a vowel inventory is one of the first places learners are asked to decide which kind of chart they are building.

Across Languages

Across Languages — Japanese: A Compact, Symmetrical Five-Vowel System

Standard Japanese is commonly analyzed as having five short vowel phonemes, /i, e, a, o, u/, each of which also has a phonemically distinct long counterpart, transcribed with a following length mark, as in /iː, eː, aː, oː, uː/. The short-vowel qualities occupy peripheral positions broadly comparable to the Spanish system discussed earlier in this chapter, though the phonetic realization of Japanese /u/ is often described as less strongly rounded and somewhat more central than the cardinal reference value for [u] — a useful reminder of the Chapter 15 principle that a shared symbol does not guarantee identical realization. The length contrast itself is a system-level property that Chapter 18 will treat directly; for this chapter, the relevant point is that Japanese demonstrates how a small, well-dispersed quality inventory can be doubled in functional load by adding a second, independent dimension of contrast rather than by adding more quality distinctions.

Across Languages — German: A Larger System Built on Height and Length

Standard German is commonly described as having a substantially larger monophthong inventory than Spanish or Japanese, with numerous scholarly analyses counting fourteen or more phonemically distinct monophthong qualities once tense-lax and long-short pairs are included, along with the front rounded vowels /y/ and /ø/ introduced for chart literacy in Chapter 15. This inventory fills in positions that the Spanish and Japanese systems leave empty, including the front rounded column and several near-peripheral interior positions. The system illustrates that larger inventories do not simply repeat the same five anchor positions with added detail; they actively use additional regions of the vowel quadrilateral that smaller systems do not phonemicize at all.

Across Languages — Hawaiian: A Small System with Long Words

Hawaiian is commonly analyzed as having the same five peripheral vowel qualities found in Spanish and Japanese, /i, e, a, o, u/, each with a phonemically distinct long counterpart. A small vowel-quality inventory of this kind places no limit on word length or on the amount of information a language can express; Hawaiian words and phrases can be as long and as informationally rich as those of any large-inventory language. This example directly supports the chapter's central caution: do not infer anything about a language's overall expressive capacity, word length, or grammatical complexity from the size of its vowel inventory alone.

Common Mistakes

Common Mistake

Error: Counting every phonetically distinct vowel quality heard in a recording as a separate phoneme.

Correction: Phonetic variation is expected in any recording; not every audible difference is contrastive. Before adding a category to a phonemic inventory, test whether the difference distinguishes meaning or instead occurs in predictable, complementary environments. An inflated inventory built from unfiltered phonetic detail misrepresents the system.

Common Mistake

Error: Assuming that a language's vowel inventory must be symmetrical — for example, assuming that every front vowel height must have a matching back vowel height, or that every unrounded vowel must have a rounded counterpart.

Correction: Real vowel systems are frequently asymmetrical. A system may have more front vowel heights than back, may lack a rounded counterpart at a given position, or may use the central column unevenly. Build the inventory from the actual contrastive evidence rather than from an assumption that the system must mirror itself. The Scope and Sequence for this course explicitly cautions against assuming symmetry, and the cross-language examples above illustrate systems that are not symmetrical.

Common Mistake

Error: Treating vowel inventory size as a proxy for how "advanced," "expressive," or "difficult" a language is.

Correction: As discussed above, inventory size reflects one design choice among many that a sound system can make, and it says nothing about the language's expressive range, grammatical complexity, or historical development. Describe systems by their properties, not by comparative value judgments.

Common Mistake

Error: Building a phonemic vowel chart and a phonetic vowel plot as if they were the same document, mixing bracket types or plotting multiple realizations under a single phonemic symbol without labeling them.

Correction: Decide at the outset which chart you are building. A phonemic chart uses slashes and one representative point per category. A phonetic plot uses brackets and may show multiple points per category, each labeled with its conditioning environment. Mixing the two without clear labeling makes the resulting chart impossible for another reader to interpret correctly.

1. A phonemic inventory is the set of vowel categories that contrast meaning; a phonetic realization is the actual value produced in a specific context. Building an inventory requires testing for contrast because not every audible difference is meaningful — some differences are predictable, conditioned variation that belongs to a single phonemic category.   2. Steps: collect comparable forms, test for contrast, tally the confirmed contrastive categories, and plot the resulting inventory. Multiple confirming pairs are recommended because a single pair might reflect an isolated exception rather than a general, productive contrast in the system.   3. Dispersion is the cross-linguistic tendency for vowel systems to space their phonemes to maximize auditory distinctiveness, particularly in smaller systems. It helps explain why small systems recur in similar peripheral shapes, since spreading a few vowels across the most distinct positions maximizes contrast per category. It is a descriptive tendency, not a strict rule; languages are not required to maximize dispersion, and actual inventories must be established from contrastive evidence, not predicted from the tendency alone.

1. In a five-vowel system, phonemes are spread across a few widely separated peripheral positions, giving large auditory margins. In a larger system such as German, additional phonemes fill in the space between those peripheral positions and extend into the front-rounded and interior regions, reducing the average distance between neighbors.   2. A language may compensate with a larger consonant inventory, a tone system, or elaborate phonotactics, as noted in the chapter; conversely, a large-vowel-inventory language might have a comparatively simple consonant system.   3. The cross-language examples show, for instance, that Spanish-type systems have no phonemic front rounded vowels at all, illustrating an asymmetry between the front and back columns.

Accept any consistent, correctly applied example.   3. A broad chart plots one representative point per phoneme using slashes; a phonetic plot may show multiple labeled points per phoneme using brackets, each tied to a conditioning environment. To show two allophones conditioned by stress, a phonetic plot is required, since a broad chart by design collapses conditioned variants into a single phonemic point.

Cumulative Review

1. The phoneme-allophone distinction applies to vowels exactly as it does to consonants: the test is still contrast versus complementary, predictable distribution, and it does not depend on the segment type. Nothing about the underlying logic changes for vowels; what differs is only the kind of phonetic detail typically involved in vowel allophony — quality shifts such as raising, lowering, and centralizing, as opposed to consonantal properties such as aspiration or place assimilation.   2. The reference-vowel method and paired-column structure give a reconstructed inventory a shared coordinate system: once a phoneme's approximate position is established through contrast testing, it can be plotted on the same quadrilateral used for the primary reference vowels, allowing direct comparison with other languages' inventories and with the reference values themselves. Plotting a reconstructed inventory onto this shared chart, rather than a language-specific ad hoc diagram, keeps the analysis comparable across languages.   3. Adding a second contrastive dimension such as length allows a language to support more phonemic vowel categories without adding new vowel qualities, because each existing quality can now combine with each value of the new dimension (for example, short and long) to produce a distinct phoneme. This is consistent with the Japanese example in this chapter, where five qualities combine with a length contrast to yield ten vowel phonemes rather than requiring ten distinct qualities.
