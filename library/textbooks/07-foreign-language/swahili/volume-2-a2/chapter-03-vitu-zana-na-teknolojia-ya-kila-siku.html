Vitu, Zana, na Teknolojia ya Kila Siku

Objects, Tools, and Everyday Technology · Speak and Write Swahili, Volume II: Expanding Everyday and Community Communication

Chapter Objectives

By the end of this chapter, you will be able to:

  1. Name and describe common tools, devices, and household items using class 7/8 (ki-/vi-) and class 9/10 concord.
  2. Apply adjective and demonstrative agreement correctly to class 7/8 and class 9/10 nouns.
  3. Recognize object markers for things (-ki-/-vi-, -i-/-zi-) in context, without yet being required to produce them independently.
  4. Explain the basic function or purpose of a common object in simple connected sentences.
  5. Distinguish hiki/hivi and hii/hizi from the class 1/2 demonstratives huyu/hawa learned in Chapter 2.
  6. Interpret a dialogue and two short readings describing everyday objects and digital tools.
  7. Write a short explanatory paragraph (110–140 words) describing how to use a common object.

Communication Scenario

At the same Klabu ya Kiswahili meeting in Nairobi, the group has moved from "Watu Muhimu" to a lighter activity: "Vitu Vyangu Muhimu" — My Important Things. Each member brings one object from their bag — a phone, a key, a notebook, a pair of glasses — and explains what it is, what class it belongs to, and how they use it. The activity that follows, "Ni cha Nani?" (Whose is it?), asks members to identify ownership and describe objects without pointing.

  1. This chapter develops classes 7/8 and 9/10 — the two most common non-human noun classes for everyday objects — into a system for describing and identifying things.
  2. Object markers for things are introduced only receptively this chapter; you will recognize them in the dialogue and readings, but full productive control comes later in this volume.

Pronunciation and Spelling

Kutofautisha ki- na vi- — Distinguishing ki- and vi-

The class 7/8 prefixes ki- and vi- differ only in their vowel, but that vowel carries the entire singular/plural distinction. English speakers sometimes blur the two under fast speech; keep the vowel quality crisp and stable in both.

SingularPluralListening focus
kitabuvitabuki- (close front-ish) vs. vi- (close front, shorter glide)
kitivitiDo not let the second vowel of kiti collapse toward English "kitty."
kikombevikombeKeep both syllables of the prefix-plus-root sequence distinct.
Sound Focus — The nasal class 9/10 prefix

Class 9/10 nouns often begin with a nasal consonant or no audible prefix at all (simu, meza, gari). Where a nasal prefix is present before certain consonants, pronounce it as a light, connected nasal rather than a separate syllable — for example, in words like ndoo (bucket) the n blends directly into the following consonant. Listen for this pattern rather than trying to isolate the nasal as its own beat.

Grammar Instruction

This chapter develops classes 7/8 and 9/10 into a system for describing and identifying things, and introduces object markers for things as a receptive-only system to prepare for productive use later in the volume.

3.1 Adjective concord with class 7/8 nouns

Purpose: describe tools, devices, and household items accurately by matching adjectives to class 7/8 nouns.

Language Building Block
class 7/8 noun + ki-/ch- (singular) or vi-/vy- (plural) + adjective root
Kitabu hiki ni kikubwa.
This book is big. (ki- + -kubwa)
Vitabu hivi ni vikubwa.
These books are big. (vi- + -kubwa)
NounSingular adjectivePlural adjective
kiti / vitikizuri (nice)vizuri
kikombe / vikombekidogo (small)vidogo
kifaa / vifaachenye manufaa* (useful)vyenye manufaa

*Adjectives beginning with a vowel, such as -enye ("having"), take ch- in the singular and vy- in the plural rather than the full ki-/vi- form. This pattern will recur with several roots and is worth noticing now.

3.2 Adjective and quantity concord with class 9/10 nouns

Purpose: describe common everyday objects that do not change form between singular and plural.

NounAdjective agreement (same for sg. and pl.)
simusimu nzuri (nice phone/phones), simu mpya (new phone/phones)
garigari kubwa (big car/cars)
pichapicha nyingi (many pictures)
  • Simu yangu ina skrini kubwa. My phone has a big screen.
  • Vifaa hivi ni vipya. These devices are new.

3.3 Demonstrative concord: hiki/hivi and hii/hizi

Purpose: point out or single out a specific object among others, distinguishing class 7/8 from class 9/10 forms and from the class 1/2 forms learned in Chapter 2.

Noun classSingular ("this")Plural ("these")
Class 7/8hikihivi
Class 9/10hiihizi
Class 1/2 (review)huyuhawa
  • Hiki ni kitabu changu. This is my book.
  • Hii ni simu yangu mpya. This is my new phone.

3.4 Object markers for things: recognition only

Purpose: recognize how Swahili refers back to a previously mentioned thing, without yet being asked to produce this system independently.

Just as -m- refers back to a person (Chapter 2), Swahili uses class-specific markers inside the verb to refer back to a thing already mentioned. You will meet two of these markers receptively in this chapter's dialogue and readings:

Language Building Block — Recognition Only
subject marker + tense/aspect marker + object marker (class 7/8 or 9/10) + verb root + final vowel
Ninakichaji kila usiku.
I charge it every night. (referring to a class 7/8 noun like kifaa; ni- + -na- + -ki- + chaj- + -i)
Ninaichaji simu yangu, kisha ninaizima.
I charge my phone, then I turn it off. (referring to a class 9/10 noun like simu; -i- is the class 9 object marker)
Noun classObject markerStatus this chapter
Class 7 (singular)-ki-Receptive
Class 8 (plural)-vi-Receptive
Class 9 (singular)-i-Receptive
Class 10 (plural)-zi-Receptive
Language Note — Why this system stays receptive for now

You already have strong productive control of person object markers from Chapter 2. Object markers for things follow the same verb-slot position, but they require secure recognition of noun class first. In this chapter, focus on recognizing -ki-/-vi- and -i-/-zi- when you hear or read them, and on understanding which noun each marker refers back to. Independent production of these markers becomes an active learning target later in this volume, once class 7/8 and 9/10 concord is fully consolidated.

You are not expected to produce these object markers correctly in your own speaking or writing yet, and they will not appear as a required item in this chapter's assessment.

Model Dialogue

Muktadha: Ni jioni ya "Vitu Vyangu Muhimu" katika Klabu ya Kiswahili. James anamwonyesha Halima vitu vilivyo mfukoni mwake.

(Setting: It is "My Important Things" evening at the Swahili Club. James is showing Halima the items in his bag.)

HALIMAJames, una nini mfukoni mwako leo?James, what do you have in your bag today?
JAMESNina vitu vichache tu. Hiki ni kifaa changu cha kazi — kompyuta ndogo. Ninaitumia kila siku ofisini.I only have a few things. This is my work device — a small computer. I use it every day at the office.
HALIMANzuri sana. Na hii, ni nini? Skrini yake ni kubwa.Very nice. And this, what is it? Its screen is big.
JAMESHiyo ni simu yangu mpya. Ina programu nyingi na betri nzuri. Ninaichaji usiku, kisha ninaitumia mchana kutwa.That is my new phone. It has many apps and a good battery. I charge it at night, then I use it all day.
HALIMANa hivi vitabu vidogo? Ni vya nini?And these small books? What are they for?
JAMESVitabu hivi ni vya Kiswahili. Ninavisoma kila jioni ili niboreshe lugha yangu.These books are Swahili books. I read them every evening to improve my language.
HALIMAVizuri sana! Mimi pia nina kifaa muhimu — funguo zangu. Bila hizi, siwezi kuingia nyumbani!Very good! I also have an important item — my keys. Without these, I can't get into my house!
JAMESKweli kabisa. Kila mtu ana kitu kimoja asichoweza kuishi bila chake.Absolutely true. Everyone has one thing they can't live without.

Vocabulary note: mfukoni = in the bag/pocket (locative form; full locative agreement is developed later in this volume, as in Chapter 2's mtaani).

Writing Workshop

  1. Soma mfano. Reread Maandishi 1 (Vitu Vyangu Muhimu). Underline every class 7/8 or 9/10 adjective, every demonstrative, and every object marker for a thing that you find.
  2. Panga mawazo yako. Choose one common object you use often. Note its noun class, one adjective describing it, and two or three steps for how you use it.
  3. Andika rasimu yako ya kwanza. Write a connected 110–140-word paragraph in Swahili explaining what the object is and how you use it. Use at least one class 7/8 noun, one class 9/10 noun, and at least one demonstrative, with consistent agreement throughout.
  4. Kagua kwa orodha hapa chini. Check your draft against the editing checklist before writing your final version.
  5. Andika nakala safi. Produce a clean final version to share with a partner or keep for your records.

Orodha ya ukaguzi — Editing checklist:

  • Every class 7/8 noun has matching adjective agreement (ki-/ch- singular, vi-/vy- plural).
  • Every class 9/10 noun uses the correct unchanged adjective agreement pattern.
  • Demonstratives (hiki/hivi, hii/hizi) are used correctly for the intended noun class.
  • Person object markers from Chapter 2 remain accurate wherever people are mentioned.
  • Any object markers for things are used only if you feel confident; they are not required this chapter.
  • All verbs are written in normal unhyphenated Swahili spelling.
  • Capitalization, spacing, punctuation, and ng' are correct throughout.

Kazi ya ziada — Additional task: Exchange your draft with a partner. Find one class 7/8 or 9/10 noun in your partner's writing and confirm together that the adjective agreement is correct.

Culture and Communication

Culture in Context — Mobile phones and daily life

In many Swahili-speaking communities, mobile phones (simu) serve not only as communication tools but also as platforms for mobile banking, small business, and staying connected with family across long distances. The specific apps, carriers, and habits vary widely by country, generation, and setting, but the phone's central place in daily communication is broadly shared across much of the region.

Culture in Context — Sharing and caring for shared objects

In many households and workplaces across Swahili-speaking communities, tools and equipment are often shared among family members, colleagues, or neighbors rather than treated as strictly individual property. This is not universal, and expectations vary by setting, but it can shape how people talk about ownership and use of everyday objects — sometimes with less emphasis on exclusive possession than is common in some English-speaking contexts.

Regional Variation — Technology vocabulary

Some technology vocabulary varies between English-influenced loanwords and older or more established Swahili terms, depending on region, generation, and setting. For example, some speakers prefer simu for "phone" in all contexts, while others may use English-origin terms in casual speech. The terms in this chapter represent widely understood Standard Swahili forms suitable for general communication; more specialized or regionally marked technology vocabulary remains a recognition-only extension of this chapter's core system.

Common Mistakes

Common Mistake — Using class 1/2 demonstratives with objects

Error: Huyu ni kitabu changu. (using the person demonstrative huyu with a class 7/8 noun)

Correction: Hiki ni kitabu changu. — Huyu/hawa are reserved for people (class 1/2); objects use class-appropriate demonstratives such as hiki/hivi or hii/hizi.

Common Mistake — Forgetting that class 9/10 nouns do not change form

Error: Ninazo simus mbili. (attempting to add an English-style plural marker to a class 9/10 noun)

Correction: Nina simu mbili. — Class 9/10 nouns like simu use the same form for singular and plural; number is shown by context, agreement, or a numeral, not by changing the noun itself.

Common Mistake — Attaching an adjective without matching the ki-/vi- prefix

Error: Vitabu kubwa (using the singular adjective form kubwa without the plural agreement prefix)

Correction: Vitabu vikubwa. — The adjective must carry the class 8 plural prefix vi- to agree with the plural noun vitabu.

Common Mistake — Trying to produce object markers for things before they are secure

Error: Forcing an object marker for a thing into every sentence this chapter, and guessing incorrectly at the class-specific form.

Correction: This chapter only requires you to recognize -ki-/-vi- and -i-/-zi- when reading or listening. In your own speaking and writing, it is acceptable and expected to repeat the noun instead, as in Nina kifaa. Ninatumia kifaa hiki kila siku. Productive use becomes a learning target in a later chapter.
