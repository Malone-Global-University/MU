Volume II, Chapter 7: Object Concords and Definite Reference · Speak and Write Zulu: Practical Independence

Chapter Objectives

By the end of this chapter, you will be able to:

  1. Explain what an object concord is, and why isiZulu speakers use one instead of repeating a noun that has already been mentioned.
  2. Form and use object concords for people, using -m- for class 1 and -ba- for class 2.
  3. Form and use object concords for a small set of familiar things, using -wu- for class 3, -yi- for class 4, -li- for class 5, and -yi- for class 9.
  4. Place an object concord correctly inside the verb template, between the tense marker and the verb root.
  5. Use object concords with the present, the resultative past from Chapter 5, and the simple future from Chapter 6.
  6. Distinguish, in context, between a sentence that names a noun directly and a sentence that refers back to that noun with an object concord.
  7. Read and listen to a short passage in which speakers avoid repeating a noun by using an object concord instead.
  8. Write a short exchange in which you mention a specific person or thing once, then refer back to it using an object concord.
  9. Continue applying the concord audit habit, now checking that every object concord matches the class of the noun it replaces.

Communication Scenario

The following Monday, Zanele finds Nomvula at her desk, still holding the report Zanele mentioned the week before. Rather than naming the report again in every sentence, Zanele and Nomvula begin referring back to it, and to each other, using a short marker built into the verb itself. The same habit lets them talk about the office, a letter, and a colleague, without repeating any of these words more than once each.

  1. Before you begin, think, in English, about how often you say "it" or "them" instead of repeating a noun you already mentioned. Notice how naturally your English conversation avoids repetition.
  2. Think of one specific object, such as a report, a letter, or a message, that came up earlier in a conversation you had recently. You will practise referring back to a similar object in isiZulu by the end of this chapter.

Pronunciation and Click Review

Object concords are short, usually a single consonant and vowel, and they sit tucked between the tense marker and the verb root. Because they are short, they are easy to swallow in fast speech. Practise saying them clearly and separately before blending them into natural speed.

Pronunciation Focus — Hearing the object concord

Say ngiyawubona slowly, breaking it into its parts, ngi-ya-wu-bona, then say it again at a natural, connected speed. Compare it with ngiyambona, broken into ngi-ya-m-bona. Listen for the short object concord in the middle of each word, and notice how it changes the meaning from I see it, referring to the work, to I see him or her, referring to a person.

Grammar: Object Concords

7.1 What an object concord does

Purpose: refer back to a specific person or thing already mentioned, without repeating the noun.

Once a noun has been named in a conversation, isiZulu speakers usually stop repeating it. Instead, they place a short marker, called an object concord, inside the verb itself. The object concord agrees with the class of the noun it replaces, the same way a subject concord agrees with the class of a subject noun.

Language Focus

Object concords work like the concord audit you already practise for subjects, applied to objects instead. Before choosing an object concord, identify the noun class of the person or thing being referred to, then select the matching object concord from the table below. This is the same discipline you already use for subject concords, possessives, and demonstratives.

7.2 Object concords for classes 1, 2, 3, 4, 5, and 9

ClassExample nounObject concordEnglish
1umuntu-m-him, her
2abantu-ba-them (people)
3umsebenzi-wu-it (class 3)
4imibiko-yi-them (class 4)
5ihhovisi-li-it (class 5)
9incwadi-yi-it (class 9)

Note on scope: Only these six classes are active for productive object-concord use in this chapter. Other classes, and person object concords such as me, you, and us, remain outside this chapter's scope, and will be developed in later chapters.

Language Building Block — Object concord placement
subject concord + tense marker + object concord + root + final vowel
The object concord sits immediately before the verb root, after any tense marker. It never appears alone, and it always agrees with the class of the noun it refers back to.
Ngiyawubona umsebenzi. Ngiyawubona.
I see the work. I see it.
  • UNomvula umbonile umbiko. Uwubonile. Nomvula has seen the report. She has seen it. resultative past, class 3, object concord -wu-
  • Ngizoyifunda incwadi kusasa. Ngizoyifunda. I will read the letter tomorrow. I will read it. simple future, class 9, object concord -yi-
  • UZanele umbonile uNomvula ehhovisi. Zanele saw Nomvula at the office. resultative past, class 1, object concord -m-, referring to a person by name

7.3 Object concords across present, past, and future

Purpose: show that the object concord works the same way no matter which tense carries it.

An object concord does not change its form when the tense changes. Only its position, always directly before the root, stays fixed. Compare the same object concord across three tenses below.

  • Ngiyawubhala umbiko. I am writing the report. present, class 3
  • Ngiwubhalile umbiko. I have written the report. resultative past, class 3
  • Ngizowubhala umbiko. I will write the report. simple future, class 3

In each sentence above, the noun umbiko is still stated for clarity. In natural conversation, once a noun has been mentioned once, later sentences usually drop it and rely on the object concord alone, as in the dialogue below.

Model Dialogue

Isimo: NgoMsombuluko, uZanele uthola uNomvula esedeskini lakhe, esaphethe umbiko abakhuluma ngawo ngesonto eledlule.
Setting: On Monday, Zanele finds Nomvula at her desk, still holding the report they discussed the week before.

UZANELE Sawubona, Nomvula. Yilowo umbiko esikhulume ngawo? Usewubhalile? Hello, Nomvula. Is that the report we spoke about? Have you already written it?
UNOMVULA Yebo, ngiwubhalile izolo, kodwa manje ngiyawufunda futhi, ukuze ngiwuhlele kahle. Yes, I wrote it yesterday, but now I am reading it again, so that I can organize it well.
UZANELE Kuhle. Uzowunika bani, uma usuwuqedile? Good. Who will you give it to, once you have finished it?
UNOMVULA Ngizowunika uMandla, ngoba umsebenzi lo, ngokwakhe. Ngizombona ehhovisi maduze. I will give it to Mandla, because this work is his. I will see him at the office soon.
UZANELE Kulungile. Nazi incwadi kaMandla naye, angithi uzoyifunda ehhovisi? Alright. Here is Mandla's letter too, won't you read it at the office?
UNOMVULA Yebo, ngizoyifunda naye, bese ngimnika yona nomsebenzi wakhe ndawonye. Yes, I will read it too, and then I will give it to him together with his work.

This dialogue uses object concords for class 1, class 3, and class 9, across present, resultative past, and simple future. Every noun class and noun phrase pattern used was already active by the end of Chapter 6.

Imibuzo yokuqonda — Comprehension questions:

  1. UNomvula uwubhale nini umbiko? (When did Nomvula write the report?)
  2. Uzomnika bani umbiko? (Who will she give the report to?)
  3. Kungani umsebenzi ungokaMandla? (Why is the work Mandla's?)

Writing Workshop

In this chapter, you will write a short exchange in which you name a specific person or thing once, then refer back to it using an object concord, combining this with the present, past, and future forms already active in your writing.

Umsebenzi wokubhala — Writing task: Write six to eight sentences describing a report, a letter, or a colleague, or invented equivalents. Name the person or thing once, then use at least three different object concords to refer back to it, across at least two different tenses.

  1. Choose your topic. In English, decide on one specific report, letter, or person you will describe.
  2. Name it once. Introduce the noun in your first sentence, with its correct noun class.
  3. Refer back to it. In your following sentences, replace the noun with the matching object concord, rather than repeating it.
  4. Vary the tense. Use at least one present, one resultative past, and one simple future sentence, each carrying the same object concord.
  5. Add one sequenced pair. Use bese, from Chapter 5 and Chapter 6, to link two related actions involving the same object.
  6. Audit your draft. Check that every object concord matches the class of the noun it replaces, and that you have not repeated a noun unnecessarily, before writing your final version.

Isikhungo sokuhlola — Editing checklist:

  • The noun is named once, then replaced with an object concord in later sentences.
  • At least three object concords are used, matching the correct noun classes.
  • Object concords appear correctly in at least two different tenses.
  • Bese is used at least once to link two related actions.
  • Every subject concord and object concord matches its noun's class.
  • No unscheduled vocabulary or grammar appears beyond Volume I and Chapters 1 through 7.

Speaking Practice

Ukuzilolonga — Controlled practice: swapping the noun for the object concord

Your partner says a short sentence naming a person or thing. Restate the sentence out loud, replacing the noun with the matching object concord, then confirm together which class you chose.

Umsebenzi wokuxhumana — Communicative task: the shared report

Working with a partner, invent a short report, letter, or task that belongs to both of you. Take turns describing what has already happened to it, what is happening to it now, and what will happen to it soon, using an object concord each time instead of repeating the noun.

Useful checking expressions: Uwubonile umbiko? (Have you seen the report?)  |  Uzoyifunda nini incwadi? (When will you read the letter?)  |  Ngizombona nini uMandla? (When will I see Mandla?)

Ukuzilolonga wedwa — Solo alternative

Record yourself describing the exchange you wrote about in the Writing Workshop, as a spoken account. Listen back, and check that every object concord you used still matches its noun's class.

Culture and Communication

Culture in Context — Avoiding repetition politely

In many isiZulu speaking communities, repeating a noun too many times in a short conversation can sound stiff or overly formal, rather than natural. Speakers commonly move quickly to an object concord once a person or thing has been named, and learners who do the same will often sound more fluent and easier to follow.

Culture in Context — Referring to people respectfully

When an object concord refers to a person rather than a thing, speakers often continue to use the person's name occasionally throughout a longer conversation, even after the object concord has been introduced, as a way of maintaining warmth and respect. This is worth noticing as you listen to longer exchanges in later chapters.

Regional Note — Variation in everyday speed

In fast, everyday speech, some speakers shorten certain object concords further than the careful forms presented in this chapter. This textbook models a clear, standard pattern suitable for learners. Keep listening for small variations as your listening skills develop in later chapters and volumes.

Common Mistakes

Common Mistake — Placing the object concord in the wrong position

Error: Placing the object concord before the tense marker, rather than after it, for example saying ngiwuzobhala when a simple future meaning is intended.

Correction: Keep the tense marker before the object concord, directly followed by the root: ngizowubhala, not ngiwuzobhala, when describing something that will happen to a class 3 noun.

Common Mistake — Repeating the noun and the object concord together

Error: Using both the full noun and the object concord in the same sentence without a clear reason, for example saying Ngiyawubona umbiko wona when a single, natural reference is intended.

Correction: Once a noun has already been named, use the object concord alone in later sentences: Ngiyawubona, not Ngiyawubona umbiko wona, unless you are deliberately emphasizing the noun.

Common Mistake — Choosing an object concord that does not match the noun class

Error: Using a class 3 object concord for a class 9 noun, for example saying Ngiyawubona incwadi when referring to a letter, which belongs to class 9.

Correction: Match the object concord to the noun's class before speaking: Ngiyayibona incwadi, not Ngiyawubona incwadi, since incwadi takes the class 9 object concord -yi-.
