Appendix A

Precalculus Readiness Review

This appendix reviews the prerequisite knowledge most important for success in calculus. Students who find any section unfamiliar should review the corresponding chapters in Trigonometry and Precalculus in the MGU Mathematics Series before proceeding.

A.1 Functions and Function Notation

A function assigns exactly one output to each input. If f is a function and x is an input value, then f(x) is the corresponding output.

Key Ideas

Domain — the set of all valid input values.

Range — the set of all possible output values.

Vertical line test — a graph represents a function if and only if every vertical line crosses it at most once.

Composition — (f ∘ g)(x) = f(g(x)). The output of g becomes the input of f.

Inverse functions — f⁻¹ undoes f. If f(a) = b, then f⁻¹(b) = a.

Quick ReviewA.1 Problems
  1. Let f(x) = 3x² − 5. Find f(0), f(2), and f(−1).
  2. Let g(x) = √(x + 4). Find the domain of g.
  3. Let f(x) = 2x + 1 and g(x) = x². Find (f ∘ g)(3) and (g ∘ f)(3).
  4. Find the inverse of f(x) = (x − 3)/5.

A.4 Slope and Linear Functions

Slope Formula

m = (y₂ − y₁)/(x₂ − x₁) = Δy/Δx

Slope-intercept: y = mx + b

Point-slope: y − y₁ = m(x − x₁)

Parallel lines: equal slopes. Perpendicular lines: slopes are negative reciprocals.

A.5 Polynomial and Rational Functions

Polynomials are defined and continuous everywhere. Rational functions p(x)/q(x) are undefined where q(x) = 0, and can have vertical asymptotes, horizontal asymptotes, holes, and oblique asymptotes.

A.6 Exponential and Logarithmic Functions

Key Properties

f(x) = aˣ: growth if a > 1; decay if 0 < a < 1. Natural base: e ≈ 2.71828.

ln(ab) = ln a + ln b

ln(a/b) = ln a − ln b

ln(aʳ) = r ln a

Change of base: log_a(x) = ln x / ln a

A.8 Algebra Review for Calculus

Skills Used Constantly in Calculus

Factoring polynomials (common factor, difference of squares, trinomial). Simplifying rational expressions by canceling common factors. Rationalizing numerators and denominators using conjugates. Laws of exponents: xᵐ · xⁿ = xᵐ⁺ⁿ, xᵐ/xⁿ = xᵐ⁻ⁿ, (xᵐ)ⁿ = xᵐⁿ. Fractional exponents: x^(1/n) = ⁿ√x, x^(m/n) = (ⁿ√x)ᵐ. These appear constantly in evaluating limits, differentiating, and integrating.

Appendix B

Function Parent Graphs Reference

For each function, the key calculus-relevant features — derivative, integral, and graphical behavior — are noted alongside the shape description.

  • f(x) = cHorizontal line at height c. Derivative: f'(x) = 0 everywhere. Integral: ∫c dx = cx + C.
  • f(x) = mx + bStraight line, slope m, y-intercept b. Derivative: f'(x) = m. Integral: ∫(mx + b)dx = (m/2)x² + bx + C.
  • f(x) = x²Upward parabola, vertex at origin. Minimum at x = 0. Derivative: 2x. Concave up everywhere: f''(x) = 2 > 0.
  • f(x) = x³S-shaped curve through origin. Increasing everywhere. Inflection point at x = 0 where concavity changes. Derivative: 3x².
  • f(x) = √xIncreasing curve defined for x ≥ 0. Slope decreases (curve flattens). Derivative: 1/(2√x). Not differentiable at x = 0: vertical tangent.
  • f(x) = |x|V-shape, vertex at origin. Not differentiable at x = 0: corner. Derivative: 1 for x > 0, −1 for x < 0, undefined at x = 0.
  • f(x) = 1/xTwo hyperbola branches. Vertical asymptote at x = 0, horizontal at y = 0. Derivative: −1/x². Integral: ln|x| + C.
  • f(x) = eˣAlways increasing, always positive. Remarkable: d/dx[eˣ] = eˣ. The function equals its own derivative. Concave up everywhere.
  • f(x) = ln xDefined for x > 0, increasing, concave down. Vertical asymptote at x = 0. Passes through (1, 0). Derivative: 1/x.
  • sin x, cos xPeriodic waves, period 2π, amplitude 1. d/dx[sin x] = cos x. d/dx[cos x] = −sin x. Integrals: ∫sin x dx = −cos x + C, ∫cos x dx = sin x + C.

C.6 Trigonometric Limits Essential for Calculus

Foundational Trig Limits

lim(x→0) [sin x / x] = 1

lim(x→0) [(1 − cos x) / x] = 0

These limits are the foundation of all trigonometric derivatives. They are proved using the Squeeze Theorem.

D.3 One-Sided Limits

Two-Sided Limit Condition

The two-sided limit lim(x→a) f(x) = L exists if and only if both one-sided limits exist and are equal:

lim(x→a⁻) f(x) = L and lim(x→a⁺) f(x) = L

D.4 Limits at Infinity

lim(x→∞) 1/xⁿ = 0 for any n > 0

For rational functions, the limit at infinity is determined by the ratio of the leading terms. Compare degrees of numerator and denominator.

D.5 Squeeze Theorem

Squeeze Theorem

If g(x) ≤ f(x) ≤ h(x) near x = a, and lim(x→a) g(x) = lim(x→a) h(x) = L, then lim(x→a) f(x) = L.

Used to prove lim(x→0) [sin x / x] = 1 and other results.

Appendix E

Continuity Reference

E.1 Definition of Continuity at a Point

Three Conditions for Continuity at x = a

1. f(a) is defined.

2. lim(x→a) f(x) exists.

3. lim(x→a) f(x) = f(a).

If any condition fails, the function has a discontinuity at x = a.

E.2 Types of Discontinuity

TypeDescriptionGraph Appearance
RemovableLimit exists, but f(a) is undefined or f(a) ≠ limit.Hole in the graph
JumpOne-sided limits exist but are not equal.Graph jumps vertically
InfiniteAt least one one-sided limit is infinite.Vertical asymptote
OscillatingFunction oscillates without settling near x = a.Rapid oscillation (e.g., sin(1/x) near 0)

E.3 Continuity of Common Functions

Continuous Everywhere on Their Domains

Polynomial functions · Rational functions (where denominator ≠ 0) · Radical functions (on their domains) · Trigonometric functions (on their domains) · Exponential functions · Logarithmic functions · Compositions and sums of continuous functions

E.4 Intermediate Value Theorem

Intermediate Value Theorem (IVT)

If f is continuous on [a, b] and k is any value between f(a) and f(b), then there exists at least one value c ∈ (a, b) such that f(c) = k.

The IVT guarantees existence — it does not find the value. A continuous function cannot skip from one value to another without passing through every value in between.

Appendix H

Related Rates Strategy Guide

Related rates problems describe two or more quantities that are both changing with respect to time. The goal is to find the rate of change of one quantity given the rate of change of another.

H.1 The Seven-Step Strategy

  • Step 1Read carefully. Identify every quantity that changes with time, the given rate, and the unknown rate.
  • Step 2Draw and label. Sketch the situation. Label changing quantities with variable names; label constants as constants.
  • Step 3Write an equation relating the variables (geometric, physical, or algebraic).
  • Step 4Differentiate both sides with respect to t using implicit differentiation. Every changing quantity produces a rate term.
  • Step 5Substitute known values of quantities and given rates at the specific moment described.
  • Step 6Solve for the unknown rate.
  • Step 7Check and interpret. Confirm sign and units. State the answer in context.

H.2 Common Related Rate Equations

Right triangle (a² + b² = c²): 2a·(da/dt) + 2b·(db/dt) = 2c·(dc/dt) Circle area (A = πr²): dA/dt = 2πr·(dr/dt) Sphere volume (V = 4/3·πr³): dV/dt = 4πr²·(dr/dt) Cylinder (V = πr²h): dV/dt = πr²·(dh/dt) + 2πrh·(dr/dt) Cone (V = 1/3·πr²h): Differentiate using the product rule after relating r and h.

H.3 Common Mistakes in Related Rates

Critical Errors to Avoid

Substituting a specific value of r, h, or x before differentiating — differentiate first, then substitute. Forgetting to differentiate every changing term. Ignoring the sign of a rate (a decreasing quantity has a negative rate). Misidentifying which rate is given and which is unknown. Forgetting units on rates — always express as a quantity per unit of time.

Appendix I

Optimization Strategy Guide

I.1 The Eight-Step Strategy

  • Step 1Read carefully. Identify what is being maximized or minimized.
  • Step 2Assign variables. Name the quantity Q to be optimized and the constrained variable.
  • Step 3Write the objective function. Express Q as a function of one variable, using the constraint to eliminate extras.
  • Step 4Find the domain. Identify all physically or mathematically valid values.
  • Step 5Differentiate. Find Q'(x) and set it equal to zero to locate critical numbers.
  • Step 6Classify. Use the First Derivative Test, Second Derivative Test, or Closed Interval Method.
  • Step 7Check endpoints. If the domain is a closed interval, evaluate Q at all critical numbers and endpoints.
  • Step 8Interpret. State the optimal value and the conditions that produce it. Include units.

I.2 Closed Interval Method

Closed Interval Method

If Q is continuous on [a, b]:
1. Find all critical numbers of Q on (a, b).
2. Evaluate Q at all critical numbers and at both endpoints a and b.
3. The largest value is the absolute maximum. The smallest is the absolute minimum.

Appendix J

Antiderivative Rules Formula Sheet

An antiderivative of f(x) is any function F(x) such that F'(x) = f(x). The family of all antiderivatives is the indefinite integral: ∫f(x)dx = F(x) + C.

J.1 Basic Antiderivative Rules

RuleFormula
Constant∫k dx = kx + C
Power Rule (n ≠ −1)∫xⁿ dx = xⁿ⁺¹/(n+1) + C
Constant Multiple∫k·f(x) dx = k·∫f(x) dx
Sum∫[f + g] dx = ∫f dx + ∫g dx
Difference∫[f − g] dx = ∫f dx − ∫g dx
Power Rule Exception

The power rule ∫xⁿ dx = xⁿ⁺¹/(n+1) + C does not apply when n = −1. That case is handled by the natural log rule: ∫(1/x) dx = ln|x| + C.

J.2 Exponential and Logarithmic Antiderivatives

∫eˣ dx = eˣ + C ∫e^(kx) dx = (1/k)e^(kx) + C ∫aˣ dx = aˣ/ln a + C (a > 0, a ≠ 1) ∫(1/x) dx = ln|x| + C

J.3 Trigonometric Antiderivatives

∫sin x dx = −cos x + C ∫cos x dx = sin x + C ∫sec²x dx = tan x + C ∫csc²x dx = −cot x + C ∫sec x tan x dx = sec x + C ∫csc x cot x dx = −csc x + C

J.4 Inverse Trigonometric Antiderivatives

∫1/√(1 − x²) dx = arcsin x + C ∫1/(1 + x²) dx = arctan x + C

Appendix L

Riemann Sum Reference

L.1 Partition Setup

Dividing [a, b] into n Equal Subintervals

Width of each subinterval: Δx = (b − a) / n

Partition points: x₀ = a, x₁ = a + Δx, x₂ = a + 2Δx, …, xₙ = b

L.2 Types of Riemann Sums

TypeFormulaSample Point
Left-endpointLₙ = Σ f(xₖ₋₁)·Δx, k=1 to nLeft endpoint of each subinterval
Right-endpointRₙ = Σ f(xₖ)·Δx, k=1 to nRight endpoint of each subinterval
MidpointMₙ = Σ f((xₖ₋₁ + xₖ)/2)·ΔxMidpoint — generally most accurate

L.3 Convergence

As n → ∞

lim(n→∞) Rₙ = lim(n→∞) Lₙ = lim(n→∞) Mₙ = ∫ₐᵇ f(x) dx

All three Riemann sums converge to the definite integral provided f is integrable on [a, b].

L.4 Sigma Notation Formulas

Σᵢ₌₁ⁿ c = cn Σᵢ₌₁ⁿ k = n(n+1)/2 Σᵢ₌₁ⁿ k² = n(n+1)(2n+1)/6 Σᵢ₌₁ⁿ k³ = [n(n+1)/2]²

L.5 Signed Area vs. Total Area

Critical Distinction

When f(x) > 0: the definite integral gives positive area. When f(x) < 0: the definite integral gives negative area (below the x-axis). Net area = area above axis − area below axis. Total area = area above axis + area below axis. To find total area, split the integral at each zero and take the absolute value of each piece.

Appendix M

Fundamental Theorem of Calculus Reference

M.1 Accumulation Function

Definition

Let f be continuous on [a, b]. The accumulation function is: F(x) = ∫ₐˣ f(t) dt

F(x) represents the net signed area under f from a to x. As x increases, F tracks how area accumulates.

M.2 FTC Part 1

Fundamental Theorem of Calculus, Part 1

If f is continuous on [a, b] and F(x) = ∫ₐˣ f(t) dt, then F is differentiable and:

F'(x) = f(x)

The derivative of an accumulation function returns the original integrand. Rate and accumulation are inverse processes.

With chain rule (extended form): d/dx [∫ₐ^(g(x)) f(t) dt] = f(g(x)) · g'(x)

M.3 FTC Part 2

Fundamental Theorem of Calculus, Part 2

If f is continuous on [a, b] and F is any antiderivative of f, then:

∫ₐᵇ f(x) dx = F(b) − F(a) = [F(x)]ₐᵇ

To evaluate a definite integral: find any antiderivative of the integrand and evaluate the difference at the endpoints.

M.4 Net Change Theorem

∫ₐᵇ F'(x) dx = F(b) − F(a)

The integral of a rate of change gives total net change in the quantity. This is FTC Part 2 stated in terms of what the function and its derivative represent.

M.5 Average Value of a Function

f_avg = (1/(b − a)) · ∫ₐᵇ f(x) dx

M.6 Mean Value Theorem for Integrals

Mean Value Theorem for Integrals

If f is continuous on [a, b], there exists c ∈ (a, b) such that f(c) = f_avg. A continuous function must actually equal its average value somewhere on the interval.

Appendix N

Area and Volume by Integration Reference

N.1 Area Between Two Curves

Area Formula

If f(x) ≥ g(x) on [a, b]: A = ∫ₐᵇ [f(x) − g(x)] dx

Always integrate top minus bottom (or right minus left when integrating with respect to y). Find intersection points by setting f(x) = g(x) and solving for x. If the curves cross more than once, split the integral at each crossing.

N.2 Volume by Slicing

V = ∫ₐᵇ A(x) dx where A(x) is the cross-sectional area at position x

N.3 Disk Method

V = π ∫ₐᵇ [f(x)]² dx (region between y = f(x) and x-axis, rotated about x-axis)

N.4 Washer Method

V = π ∫ₐᵇ ([f(x)]² − [g(x)]²) dx (outer radius f, inner radius g, rotated about x-axis)

N.5 Shell Method

V = 2π ∫ₐᵇ x · f(x) dx (region bounded by y = f(x), rotated about y-axis)

N.6 Summary of Volume Methods

Appendix O

Differential Equations Preview Reference

O.1 What a Differential Equation Is

A differential equation is an equation that relates an unknown function to its derivatives. The unknown is a function, not a number. A solution is a function y = f(x) that satisfies the equation on some interval. A general solution contains an arbitrary constant C; an initial condition determines the specific value of C.

O.2 Separable Differential Equations

Separable Form and Solution Strategy

A differential equation is separable if it can be written as: dy/dx = g(x) · h(y)

Separate variables: dy/h(y) = g(x) dx

Then integrate both sides: ∫1/h(y) dy = ∫g(x) dx

O.3 Exponential Growth and Decay

dy/dt = ky → y = Ce^(kt) k > 0: exponential growth. k < 0: exponential decay. C = y(0) is the initial value.
ModelFormula
Population growthP(t) = P₀e^(kt)
Radioactive decayA(t) = A₀e^(−kt)
Continuous compound interestA(t) = Pe^(rt)
Newton's Law of CoolingT(t) = T_env + (T₀ − T_env)e^(−kt)

O.4 Slope Fields

What a Slope Field Shows

A slope field draws short line segments at each point (x, y) with slope equal to dy/dx = f(x, y). Together, these segments show the direction of all solution curves. A specific solution curve follows the flow of the slope field. Slope fields are used to visualize solutions when exact formulas are difficult or impossible to find.

Appendix P

Sequence and Series Preview Reference

P.1 Sequences

A sequence is an ordered list a₁, a₂, a₃, …, aₙ, … A sequence converges to L if lim(n→∞) aₙ = L. If no finite limit exists, the sequence diverges.

P.2 Infinite Series

An infinite series is the sum of all terms of a sequence: Σaₙ = a₁ + a₂ + a₃ + … A series converges if the sequence of partial sums Sₙ = Σₖ₌₁ⁿ aₖ converges as N → ∞.

P.3 Geometric Series

Geometric Series

Form: Σₙ₌₀^∞ arⁿ = a + ar + ar² + …

Converges if and only if |r| < 1, with sum: a / (1 − r)

If |r| ≥ 1, the series diverges.

P.4 Power Series (Introduction)

A power series centered at x = a is: Σₙ₌₀^∞ cₙ(x − a)ⁿ = c₀ + c₁(x−a) + c₂(x−a)² + … It converges for x values within its radius of convergence. Power series allow functions to be expressed as infinite polynomials.

P.5 Taylor Polynomials

Taylor Polynomial of Degree n Near x = a

Pₙ(x) = f(a) + f'(a)(x−a) + f''(a)/2!(x−a)² + … + f⁽ⁿ⁾(a)/n!(x−a)ⁿ

When a = 0: Maclaurin polynomial.

P.6 Key Maclaurin Series

eˣ = 1 + x + x²/2! + x³/3! + … sin x = x − x³/3! + x⁵/5! − … cos x = 1 − x²/2! + x⁴/4! − … 1/(1−x) = 1 + x + x² + x³ + … (|x| < 1)
Full Series Theory Is in Calculus II

Convergence tests, radius of convergence, Taylor series derivation, and error bounds are covered fully in Calculus II in the MGU Mathematics Series.

Calculator and Graphing Technology Skills

Technology should support understanding, not replace reasoning. When technology produces a number, graph, or approximation, students should still explain what it means in context.

  • Viewing WindowInclude all relevant intervals, zeros, and asymptotes. Zoom in on points of interest: zeros, extrema, inflection points, discontinuities.
  • Limit EstimationUse tables (evaluate f near x = a from both sides) or graphs (zoom in). Caution: technology can show a filled point at x = a even when f(a) is undefined. Always confirm algebraically.
  • Derivative EstimationUse the numerical derivative feature (nDeriv or d/dx). Graph f and f' together to verify sign, increasing/decreasing, and zero crossings. Use the tangent-line tool to visualize slope at a specific point.
  • f, f', f'' TogetherWhere f' > 0: f is increasing. Where f' < 0: decreasing. Where f' = 0: potential extremum. Where f'' > 0: concave up. Where f'' < 0: concave down. Where f'' = 0: potential inflection point.
  • Numerical IntegrationUse fnInt or equivalent. Compare numerical results to exact antiderivative results as a check. Desmos and GeoGebra can shade area under curves and display definite integral values.
  • Slope FieldsUse GeoGebra or Desmos to generate slope fields. Identify solution curve directions, sketch approximate solutions through initial conditions, and verify that exact solutions follow the slope field.

Glossary

Key Terms Defined

This glossary defines the major terms introduced in Calculus. Entries are listed alphabetically. For fuller explanations, return to the chapter where each term first appears.

Absolute extremum
The largest (absolute maximum) or smallest (absolute minimum) value a function attains on its entire domain or on a specified closed interval. Compare with local extremum.
Accumulation function
A function F(x) = ∫ₐˣ f(t) dt that tracks how total signed area under f grows as x increases. See Fundamental Theorem, Part 1.
Antiderivative
A function F(x) such that F'(x) = f(x). Every function has infinitely many antiderivatives, differing only by a constant C.
Average rate of change
The ratio [f(b) − f(a)]/(b − a), representing the slope of the secant line through (a, f(a)) and (b, f(b)).
Average value
The quantity (1/(b−a))∫ₐᵇ f(x) dx — the value f would need to be constant to produce the same total accumulation over [a, b].
Chain rule
d/dx[f(g(x))] = f'(g(x)) · g'(x). The differentiation rule for composite functions.
Closed Interval Method
A procedure for finding absolute extrema of a continuous function on [a, b] by evaluating at all critical numbers and both endpoints.
Concave down
A function is concave down on an interval where its graph bends downward; equivalently where f''(x) < 0.
Concave up
A function is concave up on an interval where its graph bends upward; equivalently where f''(x) > 0.
Constant of integration
The arbitrary constant C included in every indefinite integral, representing the family of all antiderivatives.
Continuity
A function f is continuous at x = a if f(a) is defined, the limit exists, and the limit equals f(a). All three conditions must hold.
Critical number
A value x = c in the domain of f where either f'(c) = 0 or f'(c) does not exist. Critical numbers are candidates for local extrema.
Definite integral
The limit of Riemann sums ∫ₐᵇ f(x) dx, representing net signed accumulated area under f from x = a to x = b.
Derivative
The instantaneous rate of change of a function: f'(x) = lim(h→0) [f(x+h) − f(x)]/h. Also the slope of the tangent line at x.
Differentiability
A function is differentiable at x = a if f'(a) exists. Differentiability requires continuity; continuity alone does not guarantee differentiability.
Differential equation
An equation involving an unknown function and one or more of its derivatives. A solution is a function satisfying the equation.
Disk method
Computes volume of revolution by integrating circular cross-sections: V = π∫[f(x)]² dx about the x-axis.
Exponential decay
A quantity decreasing according to y = Ce^(kt) where k < 0. The quantity shrinks by a fixed percentage per unit time.
Exponential growth
A quantity increasing according to y = Ce^(kt) where k > 0. The quantity grows by a fixed percentage per unit time.
First Derivative Test
Classifies a critical number as a local max (f' changes + to −), local min (f' changes − to +), or neither (no sign change).
FTC Part 1
If F(x) = ∫ₐˣ f(t) dt and f is continuous, then F'(x) = f(x). The derivative of an accumulation function returns the integrand.
FTC Part 2
If F is any antiderivative of continuous f, then ∫ₐᵇ f(x) dx = F(b) − F(a). Connects antiderivatives to definite integrals.
Geometric series
Σarⁿ converges to a/(1 − r) when |r| < 1. Diverges when |r| ≥ 1.
Horizontal asymptote
A horizontal line y = L that the graph of a function approaches as x → ∞ or x → −∞.
Implicit differentiation
A technique for differentiating equations where y is not solved explicitly in terms of x, using the chain rule to generate dy/dx terms.
Indefinite integral
The family of all antiderivatives of a function: ∫f(x) dx = F(x) + C. No bounds; represents a function family, not a number.
Inflection point
A point where the concavity of f changes. Occurs where f''(x) = 0 or f''(x) is undefined, confirmed by a sign change in f''.
Initial condition
A specific value of the solution at a particular input, used to determine the constant C in a differential equation.
Instantaneous rate of change
The derivative f'(a): the rate at which f is changing at a single instant x = a, as opposed to the average rate over an interval.
IVT
Intermediate Value Theorem: if f is continuous on [a, b] and k is between f(a) and f(b), then f(c) = k for some c ∈ (a, b).
Limit
The value that f(x) approaches as x approaches a: lim(x→a) f(x) = L. Describes approach behavior, not necessarily function value.
Local extremum
A local maximum or minimum — where f(c) is larger (or smaller) than all nearby function values. Compare with absolute extremum.
Net area
The value of a definite integral, counting area above the x-axis as positive and below as negative. May differ from total area.
Net Change Theorem
∫ₐᵇ F'(x) dx = F(b) − F(a). The integral of a rate of change equals total net change in the quantity.
Objective function
In an optimization problem, the function expressing the quantity to be maximized or minimized.
One-sided limit
The value f(x) approaches from only one direction: from the left (x → a⁻) or from the right (x → a⁺).
Optimization
Finding the maximum or minimum value of a function subject to a constraint. Uses derivatives to locate critical values.
Power rule (diff.)
d/dx[xⁿ] = nxⁿ⁻¹ for any real number n.
Power rule (int.)
∫xⁿ dx = xⁿ⁺¹/(n+1) + C for n ≠ −1.
Power series
An infinite series Σcₙ(x − a)ⁿ representing a function within its radius of convergence.
Product rule
d/dx[fg] = f'g + fg'.
Quotient rule
d/dx[f/g] = (f'g − fg')/g².
Related rates
Problems where two or more quantities change with time and are connected by an equation. Differentiate implicitly with respect to t.
Riemann sum
An approximation of a definite integral using a finite number of rectangles. As n → ∞, Riemann sums converge to the definite integral.
Second Derivative Test
If f'(c) = 0: f''(c) > 0 → local min; f''(c) < 0 → local max; f''(c) = 0 → inconclusive.
Secant line
A line through two points on a curve. Its slope equals the average rate of change between the two x-values.
Separable DE
A differential equation dy/dx = g(x)h(y) whose variables can be separated and integrated on opposite sides.
Sequence
An ordered list of numbers a₁, a₂, …, aₙ, … A sequence converges if its terms approach a finite limit as n → ∞.
Series
The sum of the terms of a sequence. Converges if the sequence of partial sums converges to a finite number.
Shell method
Computes volumes of revolution using thin cylindrical tubes: V = 2π∫ x·f(x) dx (rotation about y-axis).
Signed area
Area measured with sign: positive when f is above the x-axis, negative when below. The definite integral gives signed area.
Slope field
A graphical representation of dy/dx = f(x, y) drawn by placing short line segments of slope f(x, y) at a grid of points.
Squeeze Theorem
If g(x) ≤ f(x) ≤ h(x) near x = a and both g and h have limit L there, then f also has limit L at x = a.
Tangent line
The line touching a curve at one point with the same slope as the curve there. Slope at x = a is f'(a).
Taylor polynomial
A polynomial built from f's value and derivatives at x = a: Pₙ(x) = Σ f⁽ᵏ⁾(a)/k! · (x−a)ᵏ. Approximates f near a.
Total area
The sum of absolute areas of all regions between the curve and the x-axis, without sign cancellation.
Vertical asymptote
A vertical line x = a near which function values grow without bound. Typically occurs where a denominator equals zero.
Washer method
Computes volumes of revolution with holes: V = π∫([f]² − [g]²) dx, where f is outer radius and g is inner radius.
