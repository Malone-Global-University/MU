Chapter 9 — Front Matter

Purpose and Overview

This chapter introduces exponential functions as one of the most important and widely applicable function families in mathematics. Unlike linear functions, which grow by adding a constant amount, exponential functions grow by multiplying by a constant factor. This distinction is not merely technical — it changes everything about how a quantity behaves over time. A population that doubles every ten years, a bank account earning compound interest, a radioactive substance losing half its mass at a steady rate, and a viral video accumulating views at an accelerating pace all follow exponential patterns.

Students who have completed Algebra I have seen exponential expressions in the context of exponent rules and basic growth patterns. This chapter builds on that foundation and develops the full structure of exponential functions: their equations, graphs, transformations, parameters, and real-world interpretations. By the end of this chapter, students should be able to identify an exponential function, graph it with and without transformations, interpret its parameters in context, and apply exponential models to problems in finance, science, and everyday life.

This chapter also lays the groundwork for Chapter 10, which introduces logarithmic functions as the inverse of exponential functions. The language and structure learned here will make logarithms far more accessible.

Opening Question

A town had a population of 5,000 people in the year 2000. Each year, the population grows by 8 percent. A neighboring town started with the same population but adds exactly 400 people each year.

Which town will be larger in the year 2030? By how much? Keep this question in mind as you work through the chapter. By Section 9.5, you will have the tools to answer it precisely — and the answer may surprise you.

Section 9.1

What Makes a Function Exponential

In Algebra I, you studied several important function families: linear functions, quadratic functions, and basic exponential patterns. In this chapter, you will develop exponential functions fully — understanding their structure, their behavior, and the wide variety of situations they model.

Recall that a linear function has the form \(f(x) = mx + b\). What characterizes a linear function is that each time the input increases by 1, the output increases by adding the same constant \(m\). This is called constant additive change.

An exponential function works differently. Each time the input increases by 1, the output is multiplied by the same constant. This is called constant multiplicative change, and it leads to a very different shape of growth.

Definition — Exponential Function

An exponential function is a function of the form \[f(x) = a \cdot b^x\] where \(a\) is a nonzero real number called the initial value, \(b\) is a positive real number with \(b \neq 1\) called the base, and \(x\) is the input variable appearing in the exponent.

Student Warning — Variable in the Exponent

The defining feature of an exponential function is that the variable \(x\) appears in the exponent, not the base. The function \(f(x) = x^3\) is a power function — the variable is the base. The function \(f(x) = 3^x\) is an exponential function — the variable is the exponent. These are completely different families with completely different behaviors. Do not confuse them.

Why the Base Matters

The behavior of an exponential function depends entirely on the value of the base \(b\).

Base ValueFunction TypeBehavior
\(b > 1\)Exponential growthOutput increases as \(x\) increases; graph rises steeply to the right
\(0 < b < 1\)Exponential decayOutput decreases as \(x\) increases; graph falls toward zero to the right
\(b = 1\)Not exponential\(f(x) = a \cdot 1^x = a\), a constant function — excluded from the definition
\(b \leq 0\)Not defined for all \(x\)Excluded; would produce undefined or imaginary values
Example 9.1.1
Identifying an Exponential Pattern from a Table
  • ProblemDetermine whether each table represents a linear or an exponential function.

    Table A: \(x\) = 0, 1, 2, 3, 4 and \(f(x)\) = 3, 6, 12, 24, 48
    Table B: \(x\) = 0, 1, 2, 3, 4 and \(g(x)\) = 5, 8, 11, 14, 17
  • ThinkFor Table A, compute the ratio of consecutive outputs. For Table B, compute the difference between consecutive outputs. A constant ratio means exponential; a constant difference means linear.
  • Solve Table A ratios: \(6/3 = 2\), \(12/6 = 2\), \(24/12 = 2\), \(48/24 = 2\). Constant ratio of 2 → exponential with base 2.

    Table B differences: \(8-5 = 3\), \(11-8 = 3\), \(14-11 = 3\), \(17-14 = 3\). Constant difference of 3 → linear.
  • CheckTable A equation: \(f(x) = 3 \cdot 2^x\). Verify: \(f(3) = 3 \cdot 2^3 = 3 \cdot 8 = 24\). ✓
Answer: Table A is exponential (\(f(x) = 3 \cdot 2^x\)). Table B is linear (\(g(x) = 3x + 5\)).

Section 9.2

Exponential Growth

When a quantity grows exponentially, it does not increase by the same amount each time period. It increases by the same factor each time period. The result is that growth accelerates: the larger the quantity becomes, the faster it grows.

A classic example is a population of bacteria that doubles every hour. If you start with 100 bacteria at hour 0, you have 200 at hour 1, 400 at hour 2, 800 at hour 3, and so on. Each hour, the population is multiplied by 2. In ten hours, the population exceeds 100,000. This acceleration is the hallmark of exponential growth.

Exponential Growth Form

\[f(x) = a \cdot b^x \quad \text{where } a > 0 \text{ and } b > 1\]

Here, \(a\) is the starting value (the output when \(x = 0\)), and \(b\) is the growth factor — the number by which you multiply at each step. Each increase of 1 in the input multiplies the output by \(b\).

Example 9.2.1
Writing an Exponential Growth Equation
  • ProblemA colony of bacteria starts with 200 cells and doubles every hour. Write a function \(f(t)\) that gives the number of bacteria after \(t\) hours.
  • ThinkThe starting value is 200. The growth factor (doubling) is 2. Each hour, the count is multiplied by 2, so the exponent is \(t\).
  • Solvef(t) = 200 · 2^t
  • CheckAt \(t = 0\): \(f(0) = 200 \cdot 1 = 200\). ✓ At \(t = 3\): \(f(3) = 200 \cdot 8 = 1600\). Doubling three times: \(200 \to 400 \to 800 \to 1600\). ✓
Answer: \(f(t) = 200 \cdot 2^t\), where \(t\) is measured in hours.
Example 9.2.2
Evaluating for a Specific Time
  • ProblemUsing the function from Example 9.2.1, how many bacteria are in the colony after 8 hours?
  • ThinkSubstitute \(t = 8\) into \(f(t) = 200 \cdot 2^t\).
  • Solvef(8) = 200 · 2^8 = 200 · 256 = 51,200
Answer: After 8 hours, there are 51,200 bacteria.

Section 9.3

Exponential Decay

When a quantity decays exponentially, it decreases by the same multiplicative factor at each step. Instead of multiplying by something greater than 1, you multiply by something between 0 and 1. The result is that the quantity gets smaller and smaller, approaching zero but never quite reaching it.

A common example is radioactive decay. A radioactive isotope may lose half its atoms in a fixed amount of time, called its half-life. If you start with 80 grams, after one half-life you have 40, then 20, then 10, then 5, and so on. The substance never completely disappears, but it becomes negligibly small. This approach toward zero without ever touching it is one of the defining features of exponential decay.

Exponential Decay Form

\[f(x) = a \cdot b^x \quad \text{where } a > 0 \text{ and } 0 < b < 1\]

The only structural difference from exponential growth is that the base \(b\) is a fraction between 0 and 1. Each increase of 1 in the input multiplies the output by \(b\), making the quantity smaller.

Example 9.3.1
Writing an Exponential Decay Equation
  • ProblemA cup of coffee starts at 180°F. Each minute, the temperature is 90% of the previous minute's temperature. Write an exponential function for the temperature after \(t\) minutes.
  • ThinkEach minute you keep 90% of the previous temperature, so you multiply by 0.9. The starting temperature is 180. The decay factor is 0.9.
  • SolveT(t) = 180 · (0.9)^t
  • CheckAt \(t = 0\): \(T(0) = 180\). ✓ At \(t = 1\): \(T(1) = 180 \cdot 0.9 = 162\). That is a 10% reduction. ✓ At \(t = 2\): \(T(2) = 180 \cdot 0.81 = 145.8\). ✓
Answer: \(T(t) = 180 \cdot (0.9)^t\), where \(t\) is measured in minutes.
Example 9.3.2
Evaluating a Decay Function
  • ProblemUsing the cooling function from Example 9.3.1, estimate the temperature of the coffee after 10 minutes.
  • ThinkSubstitute \(t = 10\) into \(T(t) = 180 \cdot (0.9)^t\).
  • Solve T(10) = 180 · (0.9)^10 (0.9)^10 ≈ 0.3487 T(10) ≈ 180 · 0.3487 ≈ 62.8°F
Answer: After 10 minutes, the coffee is approximately 62.8°F.
Interpretation Note — The Asymptote

Notice that the coffee never cools to exactly 0°F in this model. The function \(T(t) = 180 \cdot (0.9)^t\) approaches zero but never reaches it. This is a characteristic feature of exponential decay: the function has a horizontal asymptote at \(y = 0\). The graph gets closer and closer to the x-axis, but never touches it.

Section 9.4

Initial Value and Growth Factor

Every basic exponential function \(f(x) = a \cdot b^x\) is completely determined by two parameters. The initial value \(a\) is the output when \(x = 0\), because \(b^0 = 1\) for any base. The base \(b\) is the growth or decay factor — the number by which the output is multiplied each time the input increases by 1. Finding both parameters is the key step in writing an exponential equation from data.

Example 9.4.1
Finding the Equation from a Table Starting at x = 0
  • ProblemFind the exponential equation for the table: \(x\) = 0, 1, 2, 3 and \(f(x)\) = 12, 36, 108, 324.
  • ThinkBecause the table starts at \(x = 0\), the initial value is simply \(f(0) = 12\). Find the base by computing the ratio of consecutive outputs.
  • Solve b = f(1)/f(0) = 36/12 = 3 Verify: 108/36 = 3 and 324/108 = 3 ✓ f(x) = 12 · 3^x
  • Check\(f(2) = 12 \cdot 9 = 108\). ✓
Answer: \(f(x) = 12 \cdot 3^x\)
Example 9.4.2
Finding the Equation from Two Points (no x = 0)
  • ProblemAn exponential function passes through \((2,\, 20)\) and \((4,\, 80)\). Find the equation.
  • ThinkUse \(f(x) = a \cdot b^x\) and substitute both points to create a system. Divide the two equations to eliminate \(a\) and solve for \(b\) first.
  • Solve f(2) = a · b^2 = 20 ...(1) f(4) = a · b^4 = 80 ...(2) Divide (2) by (1): b^2 = 80/20 = 4, so b = 2 Substitute into (1): a · 4 = 20, so a = 5 f(x) = 5 · 2^x
  • Check\(f(2) = 5 \cdot 4 = 20\). ✓ \(f(4) = 5 \cdot 16 = 80\). ✓
Answer: \(f(x) = 5 \cdot 2^x\)

Section 9.5

Percent Change and Exponential Models

In many real-world applications, growth or decay is described as a percent change per period. A savings account earns 5% interest per year. A town's population grows by 2% per year. A car loses 15% of its value each year. These descriptions connect naturally to the exponential form \(f(t) = a \cdot b^t\).

Percent Change and the Base

If a quantity grows by \(r\%\) per period (where \(r\) is written as a decimal), the growth factor is \(b = 1 + r\).

If a quantity decays by \(r\%\) per period, the decay factor is \(b = 1 - r\).

In either case: \[f(t) = a \cdot b^t\] where \(a\) is the initial value and \(t\) is the number of time periods.

Example 9.5.1
Population Growth at a Percent Rate
  • ProblemA city has a population of 48,000. Each year, the population grows by 2.5%. Write a function for the population after \(t\) years, and estimate the population after 10 years.
  • ThinkInitial value: 48,000. Growth rate: 2.5% = 0.025. Growth factor: \(1 + 0.025 = 1.025\).
  • Solve P(t) = 48,000 · (1.025)^t P(10) = 48,000 · (1.025)^10 ≈ 48,000 · 1.2801 ≈ 61,444
  • CheckThe population grew by about 13,444 people over 10 years. A 2.5% annual rate compounding is reasonable for modest steady growth. ✓
Answer: \(P(t) = 48{,}000 \cdot (1.025)^t\). After 10 years: approximately 61,444 people.
Example 9.5.2
Depreciation of a Car
  • ProblemA car is purchased for \$28,000. It loses 12% of its value each year. Write a function for the car's value after \(t\) years, and find the value after 5 years.
  • ThinkDecay rate: 12% = 0.12. Decay factor: \(1 - 0.12 = 0.88\). Initial value: 28,000.
  • Solve V(t) = 28,000 · (0.88)^t V(5) = 28,000 · (0.88)^5 ≈ 28,000 · 0.5277 ≈ $14,776
Answer: \(V(t) = 28{,}000 \cdot (0.88)^t\). After 5 years: approximately \$14,776.
Returning to the Opening Question

Town A: starts at 5,000, grows 8% per year → \(A(t) = 5{,}000 \cdot (1.08)^t\)

Town B: starts at 5,000, adds 400 per year → \(B(t) = 5{,}000 + 400t\)

After 30 years: \(A(30) \approx 5{,}000 \cdot 10.063 \approx 50{,}313\)   and   \(B(30) = 5{,}000 + 12{,}000 = 17{,}000\).

Town A ends up with more than three times the population of Town B. Even a modest 8% annual growth, applied repeatedly, vastly outpaces any constant additive rate. This is the power of multiplicative growth.

Graphing Exponential Functions

To understand the graphs of exponential functions, it helps to begin with two parent cases and identify their key features. All exponential graphs share a common structure, and these parent cases establish the pattern.

\(x\)\(f(x) = 2^x\) (Growth)\(g(x) = (1/2)^x\) (Decay)
\(-3\)\(1/8\)\(8\)
\(-2\)\(1/4\)\(4\)
\(-1\)\(1/2\)\(2\)
\(0\)\(1\)\(1\)
\(1\)\(2\)\(1/2\)
\(2\)\(4\)\(1/4\)

Key Features of \(f(x) = a \cdot b^x\) with \(a > 0\)

Domain: All real numbers \((-\infty,\, \infty)\)

Range: All positive real numbers \((0,\, \infty)\) when \(a > 0\)

y-intercept: \((0,\, a)\), since \(f(0) = a \cdot b^0 = a \cdot 1 = a\)

Horizontal asymptote: \(y = 0\) — the graph approaches but never touches the x-axis

No x-intercepts: The graph never crosses the x-axis

Growth (\(b > 1\)): Rises steeply right, flattens left. Always increasing.

Decay (\(0 < b < 1\)): Falls steeply right, rises left. Always decreasing.

Section 9.7

Transformations of Exponential Functions

Just as the transformation rules from Chapter 1 apply to all function families, they apply to exponential functions as well. The general transformed exponential function is:

General Transformed Exponential Function

\[f(x) = a \cdot b^{x - h} + k\]

a — vertical stretch or compression; if negative, reflection across x-axis

h — horizontal shift (right if \(h > 0\), left if \(h < 0\))

k — vertical shift; moves the horizontal asymptote from \(y = 0\) to \(y = k\)

b — growth or decay factor (unchanged by transformations)

Example 9.7.1
Vertical Shift — Moving the Asymptote
  • ProblemCompare \(f(x) = 2^x\) and \(g(x) = 2^x + 3\). Describe the transformation and identify the new asymptote and y-intercept of \(g\).
  • ThinkAdding 3 to the entire function shifts the graph 3 units upward. Every output increases by 3, including the asymptote.
  • Solve Transformation: shift up 3 units New asymptote: y = 3 (was y = 0) New y-intercept: g(0) = 2^0 + 3 = 1 + 3 = 4
Answer: Shift up 3. Asymptote: \(y = 3\). y-intercept: \((0, 4)\).
Example 9.7.2
Full Transformation Analysis
  • ProblemDescribe the transformation from \(y = 3^x\) to \(g(x) = -2 \cdot 3^{x+1} + 4\). Identify the y-intercept and horizontal asymptote of \(g\).
  • ThinkRead the parameters: \(a = -2\), \(h = -1\) (shift left 1), \(k = 4\) (shift up 4).
  • Solve a = -2: vertical stretch by 2, reflected across x-axis h = -1: shift left 1 unit k = 4: shift up 4 units Asymptote: y = 4 y-intercept: g(0) = -2·3^(0+1) + 4 = -2·3 + 4 = -6 + 4 = -2
  • CheckThe graph is reflected, stretched, shifted left 1, shifted up 4. The y-intercept is \((0, -2)\) and the asymptote is \(y = 4\). ✓
Answer: Reflect across x-axis, stretch vertically by 2, shift left 1, shift up 4. Asymptote: \(y = 4\). y-intercept: \((0, -2)\).

Draw a coordinate plane showing \(y = 3^x\) as a dashed curve and \(g(x) = -2 \cdot 3^{x+1} + 4\) as a solid curve. Mark the horizontal asymptote of \(g\) at \(y = 4\) with a labeled dashed line. Mark the y-intercept of \(g\) at \((0, -2)\). Show that the solid curve falls steeply to the right (because of the negative coefficient) and approaches \(y = 4\) from below on the left side of the plane.

Section 9.8

Compound Interest

When money is deposited in a savings account, it earns interest. With simple interest, the interest is calculated only on the original principal — you earn the same fixed dollar amount each period. With compound interest, interest is calculated on the principal plus any interest already earned, which means each period's interest is larger than the last. Compound interest produces exponential growth.

Compound Interest Formula

\[A = P \left(1 + \frac{r}{n}\right)^{nt}\]

A = final amount (balance after \(t\) years)

P = principal (initial deposit)

r = annual interest rate as a decimal (e.g., 6% → 0.06)

n = number of compounding periods per year

t = number of years

Compounding Frequency\(n\)
Annually1
Semiannually2
Quarterly4
Monthly12
Daily365
Example 9.8.1
Annual Compounding
  • ProblemYou deposit \$2,000 in an account that earns 6% annual interest, compounded annually. How much do you have after 5 years?
  • Think\(P = 2000\), \(r = 0.06\), \(n = 1\), \(t = 5\).
  • Solve A = 2000(1 + 0.06/1)^(1·5) = 2000 · (1.06)^5 (1.06)^5 ≈ 1.3382 A ≈ 2000 · 1.3382 ≈ $2,676.45
Answer: After 5 years, the balance is approximately \$2,676.45.
Example 9.8.2
Monthly Compounding — Same Rate, More Periods
  • ProblemYou deposit \$2,000 at 6% annual interest, compounded monthly. How much do you have after 5 years?
  • Think\(P = 2000\), \(r = 0.06\), \(n = 12\), \(t = 5\). Note that \(r/n = 0.06/12 = 0.005\) and \(nt = 60\).
  • Solve A = 2000(1 + 0.005)^60 = 2000 · (1.005)^60 (1.005)^60 ≈ 1.3489 A ≈ 2000 · 1.3489 ≈ $2,697.70
  • CheckMonthly compounding gives about \$21 more than annual compounding. More frequent compounding always yields a higher balance for the same rate. ✓
Answer: After 5 years with monthly compounding, the balance is approximately \$2,697.70.
Example 9.8.3
Finding the Principal (Working Backward)
  • ProblemHow much should you deposit now at 4% annual interest compounded quarterly to have \$10,000 in 8 years?
  • Think\(A = 10{,}000\), \(r = 0.04\), \(n = 4\), \(t = 8\). Solve for \(P\).
  • Solve 10,000 = P · (1.01)^32 (1.01)^32 ≈ 1.3749 P = 10,000 / 1.3749 ≈ $7,273
Answer: You should deposit approximately \$7,273 now.
Student Warning — Converting Percent to Decimal

Always convert the interest rate to a decimal before using the formula. A rate of 6% becomes \(r = 0.06\), not \(r = 6\). Using \(r = 6\) in the formula would give an absurd result: a growth factor of \((1 + 6/1) = 7\), meaning the balance would multiply by 7 each year — a 600% annual return, not 6%.

Section 9.9

Continuous Growth as an Introduction

In Section 9.8, you saw that more frequent compounding produces slightly more interest. What happens if you compound interest continuously — at every possible instant, not just once a month or once a day? Mathematically, this is expressed by taking the compound interest formula to the limit as \(n \to \infty\). The result introduces one of the most important constants in all of mathematics: Euler's number, denoted \(e\).

\[e \approx 2.71828182845\ldots\]

The number \(e\) is irrational: its decimal expansion never repeats and never ends. Like \(\pi\), it appears throughout mathematics and science because of its natural connection to growth, change, and calculus.

Continuous Growth and Decay Formula

\[A = Pe^{rt}\]

A = final amount   P = initial amount   e ≈ 2.718   r = rate as a decimal (positive for growth, negative for decay)   t = time in years

Example 9.9.1
Continuous Compounding
  • ProblemYou invest \$3,000 at 5% annual interest, compounded continuously. How much do you have after 4 years?
  • ThinkUse the continuous formula. \(P = 3000\), \(r = 0.05\), \(t = 4\).
  • Solve A = 3000 · e^(0.05 · 4) = 3000 · e^0.2 e^0.2 ≈ 1.2214 A ≈ 3000 · 1.2214 ≈ $3,664.21
Answer: After 4 years of continuous compounding at 5%, the balance is approximately \$3,664.21.
Scope Note

A full treatment of the number \(e\), its origin from the compounding limit, and its central role in calculus belongs in Precalculus and Calculus I. At this level, students should recognize the continuous growth formula, know that \(e \approx 2.718\), and be able to apply it in straightforward problems. The derivative of \(e^x\) is \(e^x\) itself — a remarkable property you will encounter in Calculus I.

Section 9.10

Half-Life

The half-life of a substance is the amount of time it takes for the quantity to reduce to half its original amount. Half-life is used most commonly in nuclear physics to describe radioactive decay, but the concept applies anywhere a quantity decreases exponentially at a constant rate. The exponent in the half-life model counts the number of half-life periods that have passed — not the raw time.

Half-Life Model

\[Q(t) = Q_0 \cdot \left(\frac{1}{2}\right)^{t/h}\]

Q(t) = amount remaining at time \(t\)   Q₀ = initial amount   h = half-life (same units as \(t\))   t/h = number of half-lives elapsed

Example 9.10.1
Radioactive Decay
  • ProblemA sample contains 400 milligrams of a radioactive isotope with a half-life of 5 years. Write a function for the amount remaining after \(t\) years, and find the amount remaining after 20 years.
  • Think\(Q_0 = 400\), \(h = 5\). After 20 years, exactly \(20/5 = 4\) half-lives have passed.
  • Solve Q(t) = 400 · (1/2)^(t/5) Q(20) = 400 · (1/2)^4 = 400 · (1/16) = 25 mg
  • CheckFour halvings from 400: \(400 \to 200 \to 100 \to 50 \to 25\). ✓
Answer: \(Q(t) = 400 \cdot \left(\dfrac{1}{2}\right)^{t/5}\). After 20 years, 25 mg remain.
Example 9.10.2
Carbon Dating — Setting Up the Equation
  • ProblemCarbon-14 has a half-life of approximately 5,730 years. A fossil contains 30% of its original Carbon-14. Set up the equation to find the age of the fossil. (Do not solve for \(t\) — that requires logarithms.)
  • ThinkWe want \(Q(t)/Q_0 = 0.30\). Substitute into the half-life model and cancel \(Q_0\).
  • Solve 0.30 · Q₀ = Q₀ · (1/2)^(t/5730) 0.30 = (1/2)^(t/5730)
Answer: The equation \(0.30 = \left(\dfrac{1}{2}\right)^{t/5730}\) cannot be solved with algebra alone. We will solve it in Chapter 10 using logarithms.
Student Warning — The Exponent Is t/h, Not t or h

The exponent in the half-life model is \(\dfrac{t}{h}\) — the number of half-lives that have elapsed. A common error is writing just \(h\) or just \(t\) as the exponent. If you do that, you are computing the amount after exactly one half-life, not after \(t\) time units. Always divide the elapsed time by the half-life to count the number of periods.

Section 9.11

Doubling Time

The doubling time of a growing quantity is the amount of time it takes for the quantity to double. Just as half-life measures the speed of decay, doubling time measures the speed of growth. The model is structured the same way: the exponent counts the number of doubling periods that have passed.

Doubling Time Model

\[Q(t) = Q_0 \cdot 2^{t/d}\]

Q₀ = initial amount   d = doubling time   t = elapsed time (same units as \(d\))   t/d = number of doublings elapsed

Example 9.11.1
Population Doubling
  • ProblemA city of 25,000 people has a population doubling time of 20 years. Write a function for the population after \(t\) years, and find the population after 60 years.
  • Think\(Q_0 = 25{,}000\), \(d = 20\). Sixty years is \(60/20 = 3\) doubling periods.
  • Solve P(t) = 25,000 · 2^(t/20) P(60) = 25,000 · 2^3 = 25,000 · 8 = 200,000
  • CheckThree doublings from 25,000: \(25{,}000 \to 50{,}000 \to 100{,}000 \to 200{,}000\). ✓
Answer: \(P(t) = 25{,}000 \cdot 2^{t/20}\). After 60 years: 200,000 people.
The Rule of 72

A famous approximation in finance and science: to estimate the doubling time of a quantity growing at \(r\%\) per period, divide 72 by the rate: \(\text{Doubling time} \approx \dfrac{72}{r}\). For example, at a 6% annual growth rate, the doubling time is approximately \(72/6 = 12\) years. This is a useful mental shortcut, widely used in personal finance. The exact answer requires logarithms (Chapter 10).

Example 9.11.2
Using the Rule of 72
  • ProblemAn investment grows at 9% per year. Estimate how long it takes to double.
  • SolveDoubling time ≈ 72/9 = 8 years
Answer: The investment approximately doubles every 8 years.

Example 9.12.1
Choosing and Applying a Model — Revenue Growth
  • ProblemA company's revenue grows at a constant rate of 15% per year. In its first year, the company earned \$500,000. Estimate the revenue in year 7 (interpreting year 1 as \(t = 0\)).
  • ThinkPercent growth model. Initial value \$500,000, growth rate 15% = 0.15, so growth factor is 1.15. Year 7 is \(t = 6\).
  • Solve R(t) = 500,000 · (1.15)^t R(6) = 500,000 · (1.15)^6 ≈ 500,000 · 2.313 ≈ $1,156,535
Answer: By year 7, the company's revenue is approximately \$1,156,535.
Example 9.12.2
Bacteria — Tripling Every 90 Minutes
  • ProblemA bacterial culture starts with 800 cells and triples every 90 minutes. How many bacteria are present after 6 hours?
  • ThinkFirst convert: 6 hours = 360 minutes. The tripling period is 90 minutes, so the number of tripling periods is \(360/90 = 4\). Use a doubling-time style model adapted for tripling.
  • Solve Q(360) = 800 · 3^(360/90) = 800 · 3^4 = 800 · 81 = 64,800
Answer: After 6 hours, there are 64,800 bacteria.
Example 9.12.3
Drug Concentration
  • ProblemA patient receives 500 mg of a drug. The drug is eliminated at 20% per hour. Write a function for the amount remaining, and find the amount after 8 hours.
  • ThinkDecay rate 20% → decay factor \(1 - 0.20 = 0.80\).
  • Solve D(t) = 500 · (0.80)^t D(8) = 500 · (0.80)^8 ≈ 500 · 0.1678 ≈ 83.9 mg
Answer: After 8 hours, approximately 83.9 mg of the drug remain in the body.

Section 9.13

Common Exponential Function Mistakes

Mistake 1 — Confusing Exponential and Linear Functions
Outputs 3, 6, 12, 24 → student writes f(x) = 3x (linear)

Correction: A constant ratio (each output multiplied by 2) means exponential, not linear. Check differences for linear; check ratios for exponential. The equation is \(f(x) = 3 \cdot 2^x\).

Mistake 2 — Multiplying the Coefficient into the Base
4 · 2³ = 8³ = 512

Correction: \(4 \cdot 2^3 = 4 \cdot 8 = 32\). The coefficient \(a\) does not merge into the base. Evaluate the power first, then multiply by the coefficient.

Mistake 3 — Forgetting to Convert Percent to Decimal
5% growth rate → f(t) = 100 · (1 + 5)^t = 100 · 6^t

Correction: 5% = 0.05 as a decimal. The growth factor is \(1 + 0.05 = 1.05\), and the function is \(f(t) = 100 \cdot (1.05)^t\).

Mistake 4 — Confusing Growth Factor and Growth Rate
"The growth factor is 1.08, so the growth rate is 108%."

Correction: If the growth factor is \(b = 1.08\), the growth rate is \(r = b - 1 = 0.08 = 8\%\). The growth factor is always 1 more than the rate (when \(r\) is expressed as a decimal).

Mistake 5 — Misidentifying the Initial Value
Table starts at x = 2 with f(2) = 20 → student writes a = 20

Correction: The initial value is \(f(0)\), not the first listed value. If the table starts at \(x = 2\), work backward or use two points to find \(a\) properly.

Mistake 6 — Believing the Function Eventually Equals Zero
"The decay function eventually reaches zero after enough time."

Correction: Exponential functions with positive initial values never actually equal zero. They approach zero as a horizontal asymptote but never reach it. The domain includes all real numbers; the range for positive \(a\) is always \((0, \infty)\).

Mistake 7 — Using Simple Interest Instead of Compound
Compound interest problem: A = P + P · r · t

Correction: Compound interest uses \(A = P\left(1 + \dfrac{r}{n}\right)^{nt}\). Simple interest grows linearly; compound interest grows exponentially. The problem will specify which type.

Mistake 8 — Wrong Exponent in the Half-Life Formula
Q(t) = 400 · (1/2)^h instead of Q(t) = 400 · (1/2)^(t/h)

Correction: The exponent must be \(t/h\) — the number of half-lives elapsed. Plugging in just \(h\) gives the amount after exactly one half-life, regardless of what \(t\) is. The denominator \(h\) converts the time into the number of periods.

Section 9.14

Preparing for Logarithms

You have now developed a thorough understanding of exponential functions. You can write them, graph them, transform them, identify their parameters, and apply them to finance, science, and real-world models.

But one class of problems has appeared repeatedly in this chapter that you could not finish. In Example 9.10.2 and related problems, equations like

\[0.30 = \left(\frac{1}{2}\right)^{t/5730} \qquad \text{or} \qquad 1000 \cdot (1.06)^t = 5000\]

arose naturally, and you could not solve for \(t\) because the variable is locked in the exponent. No algebraic tool studied so far can isolate \(t\) when it appears as an exponent.

This is precisely the problem that logarithms were invented to solve. A logarithm is the inverse operation of exponentiation — it answers the question: "What power must I raise this base to in order to get this number?" Just as subtraction undoes addition and division undoes multiplication, a logarithm undoes an exponent.

Connecting to Chapter 10

If \(2^t = 8\), then \(\log_2(8) = t = 3\). The logarithm base 2 of 8 is the exponent that makes \(2^t = 8\) true.

In Chapter 10, you will define logarithms, evaluate them, learn their properties, graph logarithmic functions, and use them to solve every exponential equation you could not finish here — including the carbon dating problem, the doubling time problem, and the compound interest problem in which you want to find the time, not the balance.

Chapter Summary

  • Sec 9.1An exponential function has the form \(f(x) = a \cdot b^x\) with the variable in the exponent, not the base. Constant multiplicative change distinguishes exponential from linear (constant additive change).
  • Sec 9.2Exponential growth occurs when \(b > 1\). Each unit increase in \(x\) multiplies the output by \(b\). Growth accelerates over time.
  • Sec 9.3Exponential decay occurs when \(0 < b < 1\). Each unit increase in \(x\) multiplies by \(b < 1\), reducing the quantity. The function approaches but never reaches zero.
  • Sec 9.4The initial value \(a = f(0)\) is read directly when the table includes \(x = 0\). The base \(b\) is the constant ratio of consecutive outputs. Two points determine a unique exponential equation.
  • Sec 9.5For percent growth rate \(r\), the growth factor is \(1 + r\). For percent decay rate \(r\), the decay factor is \(1 - r\). Always convert percent to decimal before computing.
  • Sec 9.6All basic exponential graphs have domain all reals, range \((0, \infty)\) for \(a > 0\), y-intercept \((0, a)\), and horizontal asymptote \(y = 0\). Growth curves rise right; decay curves fall right.
  • Sec 9.7The transformed form is \(a \cdot b^{x-h} + k\). A vertical shift by \(k\) moves the horizontal asymptote to \(y = k\). A negative \(a\) reflects the graph across the x-axis.
  • Sec 9.8Compound interest: \(A = P\left(1 + r/n\right)^{nt}\). More frequent compounding produces a higher balance. The formula extends the exponential model to finance.
  • Sec 9.9Continuous growth: \(A = Pe^{rt}\), where \(e \approx 2.718\). This is the limiting case as \(n \to \infty\) in the compound interest formula. Euler's number \(e\) is central to calculus and science.
  • Sec 9.10Half-life model: \(Q(t) = Q_0 \cdot (1/2)^{t/h}\). The exponent \(t/h\) counts the number of half-lives elapsed. The substance never reaches zero.
  • Sec 9.11Doubling time model: \(Q(t) = Q_0 \cdot 2^{t/d}\). Rule of 72: doubling time \(\approx 72/r\) (where \(r\) is the percent rate). Exact solutions require logarithms.
  • Sec 9.12Choose the exponential model based on the information given: basic form, percent change, compound interest, continuous, half-life, or doubling time. All are forms of the same underlying exponential structure.
  • Sec 9.13Eight common mistakes: confusing growth and linear; merging coefficient into base; forgetting percent-to-decimal conversion; confusing growth rate and growth factor; misidentifying initial value; thinking the function reaches zero; using simple instead of compound interest; wrong half-life exponent.
  • Sec 9.14When the variable appears in the exponent, ordinary algebra cannot isolate it. Logarithms are the inverse operation of exponentiation and will solve these equations in Chapter 10.

Key Terms Review

exponential function
A function of the form \(f(x) = a \cdot b^x\) where \(a \neq 0\), \(b > 0\), and \(b \neq 1\). The variable appears in the exponent.
initial value
The value of the function when \(x = 0\); the coefficient \(a\) in \(f(x) = a \cdot b^x\). Found by evaluating \(f(0) = a \cdot 1 = a\).
growth factor
The base \(b\) when \(b > 1\). Each unit increase in \(x\) multiplies the output by \(b\). Larger than 1 for growth.
decay factor
The base \(b\) when \(0 < b < 1\). Each unit increase in \(x\) multiplies the output by \(b\), reducing it. Between 0 and 1 for decay.
exponential growth
A pattern in which a quantity increases by a constant multiplicative factor over equal time intervals. Characterized by \(b > 1\).
exponential decay
A pattern in which a quantity decreases by a constant multiplicative factor over equal time intervals. Characterized by \(0 < b < 1\).
percent change
The rate of increase or decrease expressed as a percentage of the previous value. Growth rate \(r\) gives factor \(1+r\); decay rate gives factor \(1-r\).
compound interest
Interest calculated on both the principal and the accumulated interest from prior periods. Uses the formula \(A = P(1 + r/n)^{nt}\).
principal
The initial amount of money on which interest is calculated, denoted \(P\) in the compound interest formula.
natural base
Euler's number \(e \approx 2.71828\). Used as the base in continuous growth models and central to calculus.
continuous growth
An exponential model using \(A = Pe^{rt}\), representing growth that compounds at every instant rather than at discrete intervals.
half-life
The time required for a quantity undergoing exponential decay to reduce to half its original amount. Model: \(Q(t) = Q_0 \cdot (1/2)^{t/h}\).
doubling time
The time required for a quantity undergoing exponential growth to double. Model: \(Q(t) = Q_0 \cdot 2^{t/d}\). Approximated by the Rule of 72.
horizontal asymptote
A horizontal line that a graph approaches but never reaches. For basic exponential functions, the asymptote is \(y = 0\). Vertical shifts move it to \(y = k\).
Rule of 72
An approximation: doubling time \(\approx 72/r\), where \(r\) is the annual percent growth rate. Useful for quick mental calculations in finance.
transformation
A shift, reflection, stretch, or compression applied to a parent function. The general form is \(a \cdot b^{x-h} + k\).
compounding period
The frequency with which interest is calculated and added: annually (\(n=1\)), quarterly (\(n=4\)), monthly (\(n=12\)), daily (\(n=365\)).
logarithm
The inverse operation of exponentiation. Answers the question: "What exponent gives this value?" Introduced fully in Chapter 10.
