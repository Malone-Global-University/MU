Appendix A

Calculus Readiness Review

Appendix A · Sections A.1 – A.6
Limits · Differentiation · Integration · Double Integrals · Series
Required calculus background for Chapters 4–7 and all continuous probability theory

A.1 Limits and Continuity

A limit describes the value a function approaches as the input approaches a given point. For probability, limits arise in defining density functions, verifying that total probability equals one, and understanding convergence in the law of large numbers and central limit theorem.

Key Results

If f is continuous at a, then lim(x → a) f(x) = f(a).

Squeeze theorem: if g(x) ≤ f(x) ≤ h(x) near a and lim g(x) = lim h(x) = L, then lim f(x) = L.

A.2 Differentiation Review

The derivative f′(x) measures the instantaneous rate of change of f at x. In probability, derivatives appear when converting between cumulative distribution functions and probability density functions: if F(x) is the CDF of a continuous random variable, then f(x) = F′(x) is the PDF.

RuleFormula
Power ruled/dx [xⁿ] = nxⁿ⁻¹
Product ruled/dx [uv] = u′v + uv′
Chain ruled/dx [f(g(x))] = f′(g(x)) · g′(x)
Exponentiald/dx [eˣ] = eˣ; d/dx [eᵃˣ] = aeᵃˣ
Natural logarithmd/dx [ln x] = 1/x

A.3 Integration Review

The definite integral ∫ₐᵇ f(x) dx measures the signed area under f between a and b. In probability, probability is area under a density curve, expected value is a weighted integral, and variance is computed by integrating a squared deviation.

TechniqueFormula
Power rule∫ xⁿ dx = xⁿ⁺¹/(n+1) + C, n ≠ −1
Exponential∫ eˣ dx = eˣ + C; ∫ eᵃˣ dx = eᵃˣ/a + C
Logarithm∫ (1/x) dx = ln|x| + C
SubstitutionLet u = g(x), du = g′(x) dx
Integration by parts∫ u dv = uv − ∫ v du
Improper integrals∫ₐ^∞ f(x) dx = lim(b → ∞) ∫ₐᵇ f(x) dx

A.4 Double Integrals (Preview)

Joint distributions of two continuous random variables require integration over a two-dimensional region. The double integral ∬_R f(x, y) dA computes the total probability over a region R. Fubini's theorem permits evaluation as iterated integrals when f is continuous:

∬_R f(x, y) dA = ∫ₐᵇ [∫_c^d f(x, y) dy] dx

A.5 Series and Summation Review

Discrete probability uses infinite sums. A geometric series with ratio r, |r| < 1:

Σₙ₌₀^∞ rⁿ = 1/(1 − r)

The Taylor series for eˣ, which underlies the Poisson distribution and moment generating functions:

eˣ = Σₙ₌₀^∞ xⁿ/n!

Appendix B

Integration Techniques for Probability

B.1 Improper Integrals Over Infinite Domains

Many probability density functions are defined on [0, ∞) or (−∞, ∞). If the limit is finite, the integral converges. A density function must satisfy ∫ f(x) dx = 1 over its entire domain.

Example B.1
Verifying the exponential density integrates to 1
  • Givenf(x) = λe^(−λx) for x ≥ 0, λ > 0
  • Solve∫₀^∞ λe^(−λx) dx = λ · [−e^(−λx)/λ]₀^∞ = λ · (1/λ) = 1 ✓

B.2 Integration by Parts in Expected Value

Example B.2
E[X] for the exponential distribution
  • SetupE[X] = ∫₀^∞ x λe^(−λx) dx. Let u = x, dv = λe^(−λx) dx.

B.3 Gamma Function

The gamma function is defined for α > 0 as Γ(α) = ∫₀^∞ xᵅ⁻¹ e⁻ˣ dx. It appears in the gamma distribution, beta distribution, normal distribution normalization, and chi-squared distribution.

PropertyStatement
Base caseΓ(1) = 1
RecursionΓ(α) = (α − 1)Γ(α − 1) for α > 1
Integer valuesΓ(n) = (n − 1)! for positive integers n
Half-integerΓ(1/2) = √π

B.4 Beta Function

B(α, β) = ∫₀¹ xᵅ⁻¹ (1 − x)^(β−1) dx = Γ(α)Γ(β)/Γ(α + β)
Connection

The beta function normalizes the beta distribution PDF and appears in Bayesian inference when computing posteriors with Beta conjugate priors.

B.5 Gaussian Integral

The key result underlying the normal distribution normalization:

∫₋∞^∞ e^(−x²) dx = √π Therefore: ∫₋∞^∞ e^(−x²/2) dx = √(2π)

Appendix C

Linear Algebra Readiness Review

C.1 Vectors and Matrices

A matrix is a rectangular array of numbers. An m × n matrix has m rows and n columns. In regression and multivariate analysis, data and parameters are organized in matrices. Matrix multiplication: the (i, j) entry of AB is the dot product of row i of A and column j of B.

C.2 Transpose and Symmetric Matrices

The transpose Aᵀ swaps rows and columns. A matrix is symmetric if A = Aᵀ. Covariance matrices in multivariate probability are always symmetric and positive semidefinite.

C.3 Matrix Inverse

A is invertible if AA⁻¹ = A⁻¹A = I. In multiple regression, the ordinary least squares formula is:

β̂ = (XᵀX)⁻¹ Xᵀy, provided XᵀX is invertible

C.4 Determinants

det(A) ≠ 0 if and only if A is invertible. Determinants appear in multivariate change-of-variable formulas through the Jacobian.

C.5 Eigenvalues and Eigenvectors (Preview)

A scalar λ and nonzero vector v satisfy Av = λv if λ is an eigenvalue and v is the corresponding eigenvector. Eigenvalues of covariance matrices describe the spread in principal directions and are central to principal component analysis.

Three Key Concepts

Disjoint (mutually exclusive): A ∩ B = ∅. They cannot both occur.

Exhaustive: A₁ ∪ A₂ ∪ ··· ∪ Aₙ = Ω. Together they cover all outcomes.

Partition: A collection {A₁, …, Aₙ} that is both pairwise disjoint and exhaustive.

Appendix E: Probability Rules Reference

Kolmogorov Axioms: (1) P(A) ≥ 0 for every event A. (2) P(Ω) = 1. (3) If A₁, A₂, … are pairwise disjoint, then P(A₁ ∪ A₂ ∪ ···) = P(A₁) + P(A₂) + ···

Independence

Events A and B are independent if and only if P(A ∩ B) = P(A) · P(B). A collection of events A₁, …, Aₙ is mutually independent if the multiplication rule holds for every subcollection.

Appendix K: Limit Theorems Reference

Law of Large Numbers

Let X₁, X₂, … be i.i.d. with mean μ and finite variance. Then for every ε > 0:

lim(n → ∞) P(|X̄ₙ − μ| > ε) = 0

The sample mean converges in probability to the population mean.

Central Limit Theorem

Let X₁, X₂, … be i.i.d. with mean μ and finite variance σ². Then as n → ∞:

(X̄ₙ − μ) / (σ/√n) →^d N(0, 1)

X̄ₙ is approximately N(μ, σ²/n) for large n. Practical guidance: n ≥ 30 is often sufficient for approximate normality when the population is not severely skewed or heavy-tailed.

Normal Approximation to the Binomial

If X ~ Binomial(n, p) and np ≥ 10 and n(1−p) ≥ 10, then X ≈ N(np, np(1−p)).

Continuity correction: P(X ≤ k) ≈ Φ((k + 0.5 − np)/√(np(1−p)))

Denominator of Sample Variance

The sample variance S² = (1/(n−1)) Σ(Xᵢ − X̄)² uses n − 1, not n, to correct for using X̄ instead of the unknown μ. This makes S² an unbiased estimator: E[S²] = σ².

Interpreting Confidence Intervals

A 95% CI does not mean the true parameter is in this specific interval with probability 95%. It means the method produces intervals containing the true parameter in 95% of repeated applications under the stated assumptions.

Appendix N: Hypothesis Testing Reference

Structure of a Hypothesis Test

1. State hypotheses: H₀ (null) versus H₁ (alternative).

2. Choose significance level α (common values: 0.01, 0.05, 0.10).

3. Identify test statistic and its distribution under H₀.

4. Compute the observed test statistic from the data.

5. Compute the p-value: P(result at least as extreme as observed | H₀ true).

6. Decision: reject H₀ if p-value < α.

7. State conclusion in context.

p-Value Interpretation

The p-value is not the probability that H₀ is true. It is the probability, under H₀, of a test statistic at least as extreme as observed. A small p-value indicates that the data would be surprising if H₀ were true. Statistical significance does not imply practical importance.

Appendix O: Likelihood and Bayesian Reference

For observed data x = (x₁, …, xₙ) and parameter θ, the likelihood function is:

L(θ; x) = f(x₁; θ) · f(x₂; θ) · ··· · f(xₙ; θ)

The log-likelihood is ℓ(θ; x) = ln L(θ; x). The MLE is θ̂ = argmax L(θ; x).

DistributionMLE
Bernoulli(p)p̂ = x̄
Poisson(λ)λ̂ = x̄
Normal(μ, σ²)μ̂ = x̄, σ̂² = (1/n)Σ(xᵢ − x̄)²
Exponential(λ)λ̂ = 1/x̄
Bayesian Framework

Prior: π(θ) — beliefs about θ before seeing data.

Posterior: π(θ | x) ∝ L(θ; x) · π(θ)

In words: posterior ∝ likelihood × prior

Appendix T

Advanced Statistics and Data Science Readiness Check

Complete these problems before beginning advanced coursework in mathematical statistics, machine learning, econometrics, or quantitative research methods. If you can work through all six sections confidently, you have the probability and statistics foundation required for the next level.

Section 1Probability and Random Variables
  1. Let X have PDF f(x) = 3x² for x ∈ [0, 1]. Find E[X], Var(X), and the median.
  2. Let X ~ Poisson(4). Find P(X ≤ 2) and P(X = 5).
  3. Let X and Y be independent with X ~ Exponential(2) and Y ~ Exponential(3). Find P(X < Y).
  4. Find the MGF of X ~ Binomial(5, 0.4) and use it to find E[X] and E[X²].
Section 2Joint Distributions
  1. Let f(x, y) = 2 for 0 < x < y < 1. Find the marginal distributions of X and Y. Are X and Y independent?
  2. Let X and Y have a discrete joint distribution. Find Cov(X, Y) and ρ(X, Y).
  3. If X₁, …, Xₙ are i.i.d. Normal(μ, σ²), find the distribution of X̄ and of Σ(Xᵢ − X̄)²/σ².
Section 3Estimation and Inference
  1. Find the MLE of p for a Binomial(n, p) model given data x₁, …, xₙ (each 0 or 1).
  2. Construct a 95% confidence interval for μ given x̄ = 42, s = 6, n = 25. Interpret in words.
  3. A study finds a two-sided p-value of 0.03. State what this means and what decision is made at α = 0.05 and at α = 0.01.
Section 4Regression and Modeling
  1. In simple linear regression, β̂₁ = 2.4, SE(β̂₁) = 0.8, n = 20. Test H₀: β₁ = 0 at α = 0.05.
  2. R² = 0.72 and adjusted R² = 0.68 in a multiple regression with 3 predictors and n = 50. Interpret both values.
  3. One-way ANOVA yields F = 6.3 with df = (3, 36). The p-value is 0.001. State the conclusion.
Section 5Bayesian and Likelihood Reasoning
  1. A coin is flipped 20 times and shows 14 heads. Compute the log-likelihood for p ∈ {0.5, 0.6, 0.7, 0.8} and identify which is largest.
  2. Prior: p ~ Beta(2, 2). Data: 8 successes in 10 trials. Find the posterior distribution and compute the posterior mean.
Section 6Stochastic Processes
  1. A Markov chain has transition matrix P = [[0.7, 0.3], [0.4, 0.6]]. Find the stationary distribution.
  2. Events arrive as a Poisson process with rate λ = 3 per hour. Find the probability of exactly 5 arrivals in 2 hours and the expected waiting time until the 4th arrival.
Glossary

Key Terms

Definitions for all major terms introduced in this textbook. Terms are listed alphabetically. Section numbers indicate where each term is first introduced.

alternative hypothesis
The hypothesis contradicting H₀; represents what the researcher suspects may be true. Accepted only when there is sufficient evidence against H₀. (9.3)
Bayes' theorem
P(A|B) = P(B|A)P(A)/P(B); allows updating prior probabilities using observed evidence. (2.11)
Bayesian inference
Framework treating unknown parameters as random variables; updates beliefs through posterior ∝ likelihood × prior. (10.9–10.12)
Bernoulli random variable
Discrete random variable taking values 0 or 1 with probabilities 1−p and p, modeling a single binary trial. (3.8)
beta distribution
Continuous distribution on [0,1] with parameters α, β > 0. Used for proportions and as a conjugate prior for Binomial. (4.12)
bias (estimator)
Bias(T) = E[T] − θ. An unbiased estimator has Bias = 0. (8.4)
binomial distribution
Distribution of the number of successes in n independent Bernoulli(p) trials. (3.9)
central limit theorem
The standardized sample mean converges in distribution to N(0,1) as n → ∞, regardless of population shape (given finite μ and σ²). (7.8)
conditional probability
P(B|A) = P(A ∩ B)/P(A). The probability of B given that A has occurred. (2.7)
confidence interval
An interval [L, U] such that under repeated sampling, (1−α)×100% of such intervals contain the true parameter. The confidence level is a property of the method, not a single interval. (8.7)
conjugate prior
A prior that produces a posterior in the same distributional family when combined with a given likelihood. (10.12)
continuous random variable
A random variable whose probability is described by a PDF; probabilities are areas under the density curve. (4.1)
convergence in probability
Xₙ →^p X if P(|Xₙ−X| > ε) → 0 for every ε > 0. (7.6)
correlation
ρ(X,Y) = Cov(X,Y)/(SD(X)·SD(Y)); standardized covariance measuring linear association on [−1,1]. (5.10)
covariance
Cov(X,Y) = E[XY] − E[X]E[Y]; measures joint variability. Positive = tend to move together. (5.9)
cumulative distribution function
F(x) = P(X ≤ x) for all real x. Non-decreasing, right-continuous, with limits 0 and 1. (3.4, 4.4)
discrete random variable
A random variable whose possible values form a finite or countably infinite set; probability assigned via PMF. (3.1–3.2)
event
A subset of the sample space Ω; a collection of outcomes to which probability can be assigned. (1.4)
expected value
The probability-weighted average of all possible values; the long-run average under repeated sampling. (3.5, 4.6)
exponential distribution
Continuous distribution on [0,∞) modeling waiting times between Poisson process events; characterized by the memoryless property. (4.9)
gamma distribution
Continuous distribution on [0,∞) with shape α and scale β; generalizes the exponential and includes chi-squared as a special case. (4.11)
geometric distribution
Distribution of the number of trials until the first success in i.i.d. Bernoulli(p) trials; discrete memoryless property. (3.10)
hypothesis test
A structured procedure for evaluating evidence against a null hypothesis using a test statistic and p-value. (9.1)
independence (events)
A and B are independent if P(A ∩ B) = P(A)P(B); knowing A occurred gives no information about B. (2.8)
independence (random variables)
X and Y are independent if f(x,y) = fₓ(x)·fᵧ(y) for all (x,y). (5.7)
joint distribution
The probability distribution describing the simultaneous behavior of two or more random variables. (5.2)
law of large numbers
The sample mean X̄ₙ converges in probability (weak) or almost surely (strong) to μ as n → ∞. (7.7)
law of total probability
For a partition {A₁,…,Aₙ}: P(B) = Σᵢ P(B|Aᵢ)P(Aᵢ). (2.10)
likelihood function
L(θ; x) = f(x₁;θ)···f(xₙ;θ); measures how plausible each value of θ is given the observed data. (10.2)
marginal distribution
The distribution of one variable obtained by summing or integrating over the values of the other in a joint distribution. (5.5)
Markov chain
A sequence of random variables satisfying the Markov property: P(future|past, present) = P(future|present). (13.5)
maximum likelihood estimator
The parameter value that maximizes the likelihood function given the observed data. (10.6)
mean squared error
MSE(T) = Var(T) + [Bias(T)]²; combines variability and bias of an estimator. (8.6)
moment generating function
Mₓ(t) = E[eᵗˣ]; encodes all moments and uniquely determines the distribution when it exists in an interval around 0. (6.8)
normal distribution
The bell-shaped distribution N(μ, σ²); the most important in statistics due to the CLT. (4.10)
null hypothesis
The default hypothesis representing no effect, no difference, or the status quo. (9.3)
p-value
The probability, under H₀, of observing a test statistic at least as extreme as computed from the data. (9.5)
Poisson distribution
Distribution of event counts in a fixed interval under a Poisson process with rate λ; PMF = e^(−λ)λˣ/x!. (3.13)
Poisson process
A counting process with independent, stationary increments where counts in intervals of length t follow Poisson(λt). (13.9)
posterior distribution
The conditional distribution of a parameter given the data: π(θ|x) ∝ L(θ;x)·π(θ). (10.11)
power (test)
1 − β = P(Reject H₀ | H₁ true). Power increases with larger n, larger effect size, or larger α. (9.9)
prior distribution
In Bayesian inference, π(θ) expresses beliefs about the parameter before observing data. (10.10)
probability density function
f(x) ≥ 0 with ∫ f(x) dx = 1; P(a ≤ X ≤ b) = ∫ₐᵇ f(x) dx. (4.2)
probability mass function
p(x) assigning probability to each value of a discrete random variable; p(x) ≥ 0 and Σ p(x) = 1. (3.3)
random variable
A function from a sample space to the real numbers, assigning a numerical value to each outcome. (3.1)
random walk
A stochastic process formed by accumulating i.i.d. steps; a simple random walk takes steps of ±1. (13.4)
residual
The difference between an observed value and the fitted value from a regression model: eᵢ = yᵢ − ŷᵢ. (11.7)
sample space
Ω; the set of all possible outcomes of a random experiment. (1.5)
sampling distribution
The distribution of a statistic computed from repeated samples of the same size from the same population. (7.10)
significance level
Pre-specified threshold α below which H₀ is rejected; equal to the Type I error rate under H₀. (9.6)
standard error
The standard deviation of a sampling distribution; SE(X̄) = σ/√n for the sample mean. (7.5)
stationary distribution
π satisfying πP = π and Σπⱼ = 1; the long-run proportion of time a Markov chain spends in each state. (13.8)
stochastic process
A collection of random variables indexed by time or another parameter; models systems that evolve randomly. (13.1)
Type I error
Rejecting H₀ when H₀ is true; its probability equals α. (9.8)
Type II error
Failing to reject H₀ when H₀ is false; its probability is β = 1 − Power. (9.8)
unbiased estimator
An estimator T such that E[T] = θ for all θ; correct on average. (8.4)
uniform distribution
Continuous distribution on [a,b] with density 1/(b−a) everywhere on the interval. (4.8)
variance
Var(X) = E[(X−μ)²]; the expected squared deviation from the mean, measuring spread. (3.6, 4.7)

Appendix T — Readiness Check
Selected Answers: Sections 1–6
T.1 — PDF f(x) = 3x² on [0,1]
E[X] = ∫₀¹ x·3x² dx = 3∫₀¹ x³ dx = 3/4 E[X²] = ∫₀¹ x²·3x² dx = 3∫₀¹ x⁴ dx = 3/5 Var(X) = 3/5 − (3/4)² = 3/5 − 9/16 = 48/80 − 45/80 = 3/80 Median m: ∫₀^m 3x² dx = m³ = 1/2 → m = (1/2)^(1/3) = 2^(−1/3) ≈ 0.794
T.3 — P(X < Y), X ~ Exp(2), Y ~ Exp(3) independent
P(X < Y) = ∫₀^∞ P(X < y) · fᵧ(y) dy = ∫₀^∞ (1 − e^(−2y)) · 3e^(−3y) dy = 1 − 3∫₀^∞ e^(−5y) dy = 1 − 3·(1/5) = 2/5
T.9 — 95% CI for μ: x̄ = 42, s = 6, n = 25
t*(df = 24, 95%) ≈ 2.064 CI: 42 ± 2.064 · (6/5) = 42 ± 2.477 ≈ (39.52, 44.48) Interpretation: We are 95% confident the true population mean lies between 39.52 and 44.48. (The confidence level describes the method's long-run performance, not this specific interval.)
T.14 — Log-likelihoods for coin (14 heads in 20 flips)
ℓ(p) = 14 ln(p) + 6 ln(1−p) (up to additive constants) ℓ(0.5) ≈ 14 ln(0.5) + 6 ln(0.5) = 20 ln(0.5) ≈ −13.86 ℓ(0.6) ≈ 14 ln(0.6) + 6 ln(0.4) ≈ −12.47 ← largest among these four ℓ(0.7) ≈ 14 ln(0.7) + 6 ln(0.3) ≈ −12.79 ℓ(0.8) ≈ 14 ln(0.8) + 6 ln(0.2) ≈ −14.61 Note: The MLE is p̂ = 14/20 = 0.70, which is between 0.6 and 0.8.
T.15 — Posterior: Beta(2,2) prior, 8 successes in 10 trials
Posterior: Beta(2 + 8, 2 + 2) = Beta(10, 4) Posterior mean = 10/(10 + 4) = 10/14 ≈ 0.714
T.16 — Stationary distribution, P = [[0.7, 0.3], [0.4, 0.6]]
Solve πP = π and π₁ + π₂ = 1: 0.3π₁ = 0.4π₂ → π₁ = (4/3)π₂ (4/3)π₂ + π₂ = 1 → π₂ = 3/7, π₁ = 4/7 Stationary distribution: π = (4/7, 3/7)
