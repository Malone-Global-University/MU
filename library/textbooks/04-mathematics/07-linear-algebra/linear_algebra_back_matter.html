Appendix A

Algebra and Function Readiness Review

This appendix reviews the algebraic and function skills that linear algebra depends on most heavily. Students who are comfortable with this material will find the transition into vectors, matrices, systems, and transformations much smoother. If any section feels unfamiliar, review the corresponding material in the MGU Algebra II or Trigonometry and Precalculus textbooks before continuing.

A.1 Real Numbers, Variables, and Expressions

Linear algebra works with real numbers, symbolic variables, and structured expressions. Students should be fluent with arithmetic on signed numbers, fractions and rational expressions, integer exponents and square roots, distributing and factoring, and substitution. Signed arithmetic appears constantly in row operations, matrix multiplication, and determinant computation. Fractions appear in row reduction and inverse calculations. Magnitudes of vectors involve square roots, and eigenvalue equations involve polynomial roots.

A.2 Solving Linear Equations and Inequalities

A linear equation in one variable has the form ax + b = c. Solving means isolating the variable:

ax + b = c → ax = c − b → x = (c − b)/a, a ≠ 0

Linear algebra extends this skill to systems of many linear equations in many variables. Students should also recall that multiplying or dividing by a negative number reverses an inequality sign.

A.3 Solving Systems of Two Linear Equations

A system of two linear equations in two unknowns uses the form a₁x + b₁y = c₁ and a₂x + b₂y = c₂. The substitution method solves one equation for one variable and substitutes into the other. The elimination method multiplies equations by constants and adds or subtracts to eliminate one variable — this is the direct algebraic ancestor of row reduction. Geometrically, each equation is a line; the solution is the intersection point, if it exists.

Geometric Interpretation

Two lines may intersect at one point (unique solution), be parallel and never meet (no solution), or be the same line (infinitely many solutions). This three-way classification extends naturally to systems in three or more variables in Chapter 3.

A.5 Graphing in the Coordinate Plane

Students should be comfortable plotting points, identifying slope and y-intercept from y = mx + b, recognizing horizontal and vertical lines, interpreting intersections as shared solutions, and reading basic function graphs. The standard xy-plane becomes the two-dimensional vector space ℝ² in Chapter 2.

A.6 Polynomials and Polynomial Equations

Students should be able to add, subtract, and multiply polynomials; factor quadratics; use the quadratic formula; and find polynomial roots. Eigenvalues in Chapter 12 require solving characteristic equations. For 2 × 2 matrices these are quadratic equations, making factoring and the quadratic formula essential tools.

Quadratic formula: x = (−b ± √(b² − 4ac)) / (2a)

A.7 Absolute Value and Distance

The absolute value |a| measures distance from a to 0 on the number line. In two dimensions, the distance between (x₁, y₁) and (x₂, y₂) is the foundation for vector magnitude in Chapter 2 and distance in vector spaces in Chapter 11.

d = √((x₂ − x₁)² + (y₂ − y₁)²)

Appendix B

Coordinate Geometry and Trigonometry Review

B.1 Points, Lines, and Slopes

A point in the plane is an ordered pair (x, y). Useful line forms: slope-intercept y = mx + b, point-slope y − y₁ = m(x − x₁), and standard form ax + by = c (used in systems). Parallel lines have equal slopes; perpendicular lines have slopes that are negative reciprocals (−1/m). This perpendicularity condition is directly related to the dot product condition for orthogonal vectors in Chapter 11.

Appendix E

Row Reduction Reference

E.1 Elementary Row Operations

OperationNotationDescription
SwapRᵢ ↔ RⱼExchange row i and row j
ScaleRᵢ ← c·RᵢMultiply row i by nonzero scalar c
AddRᵢ ← Rᵢ + c·RⱼAdd c times row j to row i
Key Principle

Each elementary row operation preserves the solution set of the system. This is why row reduction finds equivalent systems that are easier to solve.

E.3 Row Echelon Form (REF)

A matrix is in REF when: (1) all zero rows are at the bottom, and (2) each nonzero row has a leading nonzero entry (pivot) strictly to the right of the pivot in the row above. Pivot positions form a staircase descending left to right.

E.4 Reduced Row Echelon Form (RREF)

RREF satisfies all REF conditions plus: every pivot equals 1, and every pivot is the only nonzero entry in its column. RREF is unique for each matrix and gives solutions most directly.

E.6 Row Reduction Procedure

  • 1Identify the leftmost nonzero column. This is a pivot column.
  • 2If the top entry of that column is zero, swap with a row below that has a nonzero entry there.
  • 3Use row addition to create zeros in all entries below the pivot.
  • 4Cover the pivot row and repeat on the remaining submatrix.
  • 5Scale each pivot row so its pivot equals 1.
  • 6Use row addition to create zeros above each pivot. (RREF complete.)

Appendix F

Systems of Linear Equations Strategy Guide

F.2 Strategy for Solving a Linear System

  • ①Write the augmented matrix [A | b].
  • ②Row reduce to RREF using elementary row operations.
  • ③Check for consistency — any row [0 ··· 0 | c] with c ≠ 0 means no solution.
  • ④Identify pivot variables (basic) and free variables (non-pivot columns).
  • ⑤Write the solution: unique if no free variables; parametric if free variables exist.

F.3 Solution Types and Geometric Meaning

OutcomeAlgebraGeometry in ℝ²
Unique solutionOne pivot per variable, no free variablesLines intersect at one point
No solutionContradiction row in RREFLines are parallel
Infinitely manyAt least one free variableLines are coincident

F.4 Parametric Solution Format

Let each free variable equal an independent parameter. Express each basic variable as a linear function of the parameters. Write the solution as a vector equation:

x = p + t₁v₁ + t₂v₂ + ··· Example: x₁ = 3 − 2x₃, x₂ = 1 + x₃, x₃ free (set x₃ = t): x = (3, 1, 0)ᵀ + t(−2, 1, 1)ᵀ

F.6 Common System-Solving Mistakes

Six Recurring Errors

1. Forgetting to write the augmented matrix — right-hand side gets lost during row operations.

2. Using an incorrect row operation — adding a row to itself or forgetting to scale consistently.

3. Misreading pivot positions — a zero is not a pivot; swap or move to the next row.

4. Stopping at REF instead of continuing to RREF — REF requires back-substitution; RREF gives the solution directly.

5. Forgetting free variables — a column without a pivot has a free variable; do not set it to zero by default.

6. Missing the inconsistency check — always look for rows of the form [0 ··· 0 | c] before writing a solution.

Appendices G – H

Matrix Operations & Determinant Reference

G.3 Matrix Multiplication

[AB]ᵢⱼ = Σₛ aᵢₛ bₛⱼ = (row i of A) · (column j of B)

Requirements: A must be m × k, B must be k × n; the result AB is m × n. The inner dimensions must match.

Key Properties of Matrix Multiplication

Associative: (AB)C = A(BC)

Distributive: A(B+C) = AB + AC

Identity: AIₙ = A and IₘA = A

NOT commutative in general: AB ≠ BA

Transpose reversal: (AB)ᵀ = BᵀAᵀ

Inverse reversal: (AB)⁻¹ = B⁻¹A⁻¹

G.4 Column Combination View of Ax

Ax = x₁(col 1 of A) + x₂(col 2 of A) + ··· + xₙ(col n of A)

This column combination interpretation is one of the most important perspectives in linear algebra. It directly connects matrix-vector multiplication to span and linear combinations.

J.3 Subspace Test

Three-Part Subspace Test

A nonempty subset H of a vector space V is a subspace if and only if:

1. The zero vector 0 is in H.

2. If u, v ∈ H, then u + v ∈ H. (closed under addition)

3. If v ∈ H and c ∈ ℝ, then cv ∈ H. (closed under scalar multiplication)

Equivalently: every linear combination of vectors in H is also in H.

K.3 Linear Independence Test

Solve: c₁v₁ + c₂v₂ + ··· + cₖvₖ = 0 Independent ↔ only solution is c₁ = c₂ = ··· = cₖ = 0 (trivial) Dependent ↔ nontrivial solution exists (some cᵢ ≠ 0)

Quick checks: Any set containing 0 is dependent. Two vectors are dependent iff one is a scalar multiple of the other. More than n vectors in ℝⁿ are always dependent.

Appendix L
Rank and Nullity Reference
Rank · Nullity · Rank-Nullity Theorem · Consistency · Inequalities
Rank-Nullity Theorem

For any m × n matrix A:

rank(A) + nullity(A) = n

Pivot columns contribute to rank. Free-variable columns contribute to nullity. Together they account for all n columns.

Consequence: For a square n×n matrix, rank(A) = n ↔ A is invertible ↔ nullity(A) = 0.

Appendix O
Diagonalization Reference
Diagonalizability · When possible · Matrix powers · Discrete dynamical systems · Similar matrices
Diagonalization Formula

A = PDP⁻¹

Columns of P: linearly independent eigenvectors of A.

Diagonal entries of D: corresponding eigenvalues (same order as columns of P).

Condition: A is diagonalizable ↔ A has n linearly independent eigenvectors.

Sufficient condition: n distinct eigenvalues guarantee diagonalizability.

Matrix powers: Aᵏ = PDᵏP⁻¹ Dᵏ = diag(λ₁ᵏ, λ₂ᵏ, …, λₙᵏ) (just raise each diagonal entry to the kth power) Discrete dynamical system: x_{k+1} = Ax_k → x_k = Aᵏx_0 If x_0 = c₁v₁ + ··· + cₙvₙ (in eigenvector basis): x_k = c₁λ₁ᵏv₁ + c₂λ₂ᵏv₂ + ··· + cₙλₙᵏvₙ

P.4 Linear Regression Setup

Given data points (t₁,y₁), …, (tₘ,yₘ), fitting the best-fit line y = β₀ + β₁t leads to the least squares system with design matrix:

A = [ 1 t₁ ] y = (y₁, y₂, …, yₘ)ᵀ [ 1 t₂ ] [ ··· ] [ 1 tₘ ] Solve normal equations AᵀA β̂ = Aᵀy for β̂ = (β₀, β₁)ᵀ

This framework extends to polynomial and any model that is linear in the parameters, even if nonlinear in t.

Appendix R

Calculus Readiness Bridge Guide

This appendix explicitly describes how linear algebra prepares students for calculus, multivariable calculus, differential equations, and advanced applied mathematics. Every major idea in this textbook has a direct forward connection.

R.1 From One Variable to Many Variables

In Calculus I, functions have one input and one output: y = f(x). In multivariable calculus, functions have many inputs and possibly many outputs: y = f(x) where x ∈ ℝⁿ. The derivative of a vector-valued function at a point is a matrix of partial derivatives called the Jacobian matrix. Understanding matrix multiplication, linear transformations, and the meaning of Ax makes the Jacobian natural rather than mysterious.

R.2 Linear Approximation and Tangent Planes

Calculus I (one variable): f(x) ≈ f(x₀) + f'(x₀)(x − x₀) Multivariable Calculus (n variables): f(x) ≈ f(x₀) + J(x − x₀) where J is the Jacobian matrix (matrix of partial derivatives)

The tangent line of Calculus I becomes a tangent plane (or tangent hyperplane) in higher dimensions. The derivative becomes a matrix. Linear algebra is the language of linear approximation in many dimensions.

R.4 Systems of Differential Equations

Many important differential equations problems involve systems where several quantities change simultaneously. The solution to dx/dt = Ax involves eigenvalues and eigenvectors of A:

x(t) = c₁e^(λ₁t)v₁ + c₂e^(λ₂t)v₂ + ··· + cₙe^(λₙt)vₙ

Students who have mastered eigenvalues, eigenvectors, and diagonalization will find that the only new ingredient in differential equations is the exponential function e^(λt). The structure is entirely linear algebraic.

R.6 Multivariable Optimization

The Hessian matrix (matrix of second partial derivatives) classifies critical points using eigenvalues: all positive eigenvalues → local minimum; all negative → local maximum; mixed signs → saddle point. This direct application of eigenvalue analysis makes multivariable optimization systematic.

Glossary

Key terms used throughout this textbook, listed alphabetically. Chapter references indicate where each term is introduced or most fully developed. All terms are also indexed at the back.

  • Algebraic multiplicityCh 12
    The number of times an eigenvalue appears as a root of the characteristic polynomial. The geometric multiplicity is always less than or equal to the algebraic multiplicity.
  • Augmented matrixCh 3
    A matrix formed by appending the right-hand side vector b to the coefficient matrix A, written [A | b], used to represent and solve a system of linear equations by row reduction.
  • BasisCh 9
    A set of vectors in a vector space that is linearly independent and spans the entire space. Every vector in the space has a unique representation as a linear combination of the basis vectors. A basis is a minimal spanning set and a maximal independent set.
  • Characteristic equationCh 12
    The equation det(A − λI) = 0 whose solutions are the eigenvalues of the matrix A.
  • Characteristic polynomialCh 12
    The polynomial det(A − λI) whose roots are the eigenvalues of A. Its degree equals the size of the matrix.
  • Column spaceCh 10
    The set of all linear combinations of the columns of a matrix A, written Col(A). A subspace of ℝᵐ for an m × n matrix. Equals the range of the linear transformation T(x) = Ax.
  • Consistent systemCh 3
    A system of linear equations that has at least one solution. Identified by the absence of a contradiction row in the RREF of the augmented matrix.
  • DeterminantCh 6
    A scalar value det(A) associated with a square matrix that measures area scaling (2×2), volume scaling (3×3), orientation, and invertibility. A matrix is invertible if and only if its determinant is nonzero.
  • Diagonal matrixCh 13
    A square matrix with zeros in all off-diagonal positions. Diagonal matrices are easy to invert (invert each diagonal entry) and exponentiate (raise each entry to the desired power).
  • DiagonalizableCh 13
    A square matrix A is diagonalizable if it can be written as A = PDP⁻¹ for some invertible matrix P and diagonal matrix D. Equivalent to having n linearly independent eigenvectors.
  • DimensionCh 9
    The number of vectors in a basis for a vector space or subspace. All bases for the same space have the same number of elements. The dimension of ℝⁿ is n.
  • Discrete dynamical systemCh 13
    A system of the form x_{k+1} = Ax_k that describes how a state vector evolves step by step. The state at step k is x_k = Aᵏx₀.
  • Dot productCh 2, 11
    For vectors u, v ∈ ℝⁿ, the scalar u · v = u₁v₁ + ··· + uₙvₙ = ‖u‖‖v‖cosθ, where θ is the angle between the vectors. Zero dot product means orthogonality.
  • EigenspaceCh 12
    The null space of (A − λI) for a given eigenvalue λ. The set of all eigenvectors for λ, together with the zero vector. Its dimension is the geometric multiplicity of λ.
  • EigenvalueCh 12
    A scalar λ such that Av = λv for some nonzero vector v. Measures the scaling factor along a stable direction of the transformation.
  • EigenvectorCh 12
    A nonzero vector v such that Av = λv for some scalar λ. Eigenvectors keep their direction (or exactly reverse it if λ < 0) under the transformation A.
  • Elementary row operationsCh 3
    Three operations on matrix rows that preserve the solution set: (1) swap two rows, (2) multiply a row by a nonzero scalar, (3) add a scalar multiple of one row to another.
  • Free variableCh 3
    A variable corresponding to a non-pivot column in the RREF of a system. Free variables may take any value, generating infinitely many solutions.
  • Geometric multiplicityCh 12
    The dimension of the eigenspace for a given eigenvalue λ. Equal to the nullity of (A − λI). Always less than or equal to the algebraic multiplicity.
  • Gram-Schmidt processCh 11
    An algorithm for converting a basis of a subspace into an orthogonal basis by successively removing projection components. Normalize each result to get an orthonormal basis.
  • Homogeneous systemCh 3, 10
    A linear system Ax = 0 with zero right-hand side. Always consistent (x = 0 is always a solution). The complete solution set is the null space of A.
  • Identity matrixCh 4
    The n × n matrix Iₙ with ones on the diagonal and zeros elsewhere. Satisfies AIₙ = A and IₘA = A for any compatible matrix A.
  • Inconsistent systemCh 3
    A system of linear equations with no solution. Identified by a row of the form [0 ··· 0 | c] with c ≠ 0 in the RREF of the augmented matrix.
  • Invertible (nonsingular)Ch 5
    A square matrix A is invertible if there exists A⁻¹ such that AA⁻¹ = A⁻¹A = I. Equivalently: det(A) ≠ 0, columns are linearly independent, rank = n, and Ax = b has a unique solution for every b.
  • Least squares solutionCh 11
    The vector x̂ that minimizes ‖Ax − b‖². Satisfies the normal equations AᵀAx̂ = Aᵀb. Makes Ax̂ the closest point in Col(A) to b.
  • Linear combinationCh 9
    An expression of the form c₁v₁ + c₂v₂ + ··· + cₖvₖ for scalars c₁, …, cₖ. The span of a set is the collection of all its linear combinations.
  • Linear independenceCh 9
    A set of vectors is linearly independent if the only linear combination equal to zero uses all zero scalars. Equivalently, no vector in the set is a linear combination of the others.
  • Linear transformationCh 7
    A function T: ℝⁿ → ℝᵐ that preserves vector addition T(u+v) = T(u)+T(v) and scalar multiplication T(cu) = cT(u). Represented by the standard matrix A via T(x) = Ax.
  • Magnitude (norm)Ch 2
    The magnitude of v = (v₁, …, vₙ) is ‖v‖ = √(v₁² + ··· + vₙ²). Measures the length of the vector. Comes from the Pythagorean theorem.
  • Markov chainCh 13
    A discrete dynamical system in which a state vector evolves by multiplication by a transition matrix whose columns each sum to 1. Long-term behavior is governed by the dominant eigenvalue.
  • Normal equationsCh 11
    The system AᵀAx̂ = Aᵀb whose solution gives the least squares solution to Ax = b. Always consistent; gives the unique least squares solution when AᵀA is invertible.
  • Null spaceCh 10
    The set of all solutions to Ax = 0, written Nul(A). A subspace of ℝⁿ for an m × n matrix. Its dimension is the nullity of A.
  • NullityCh 10
    The dimension of the null space of a matrix: nullity(A) = dim(Nul(A)). Equals the number of free variables. By the Rank-Nullity Theorem, rank(A) + nullity(A) = n.
  • Orthogonal complementCh 11
    For a subspace W, the orthogonal complement W⊥ consists of all vectors orthogonal to every vector in W. The row space and null space of A are orthogonal complements in ℝⁿ.
  • ProjectionCh 11
    The projection of u onto v is proj_v u = (u·v)/(v·v)·v. More generally, the projection of a vector onto a subspace gives the closest vector in that subspace — the "shadow" of the original vector.
  • RankCh 10
    The dimension of the column space (equivalently, the row space) of a matrix. Equals the number of pivot positions. Measures how much information the matrix preserves.
  • Rank-Nullity TheoremCh 10
    For any m × n matrix A: rank(A) + nullity(A) = n. The pivot columns (rank) and free-variable columns (nullity) together account for all n columns.
  • Row spaceCh 10
    The set of all linear combinations of the rows of matrix A, written Row(A). A subspace of ℝⁿ. The row space and null space of A are orthogonal complements.
  • Similar matricesCh 13
    Two n × n matrices A and B are similar if B = P⁻¹AP for some invertible P. They represent the same linear transformation in different bases and share eigenvalues, determinant, trace, and rank.
  • SingularCh 5
    A square matrix that is not invertible. Equivalently: det(A) = 0, columns are linearly dependent, or Ax = b has no unique solution.
  • SpanCh 9
    The set of all linear combinations of a given set of vectors. The span is always a subspace. To check if b is in span{v₁,…,vₖ}: determine whether the system [v₁ ··· vₖ | b] is consistent.
  • SubspaceCh 8
    A nonempty subset H of a vector space that contains the zero vector, is closed under addition, and is closed under scalar multiplication. Every linear combination of vectors in H is also in H.
  • Unit vectorCh 2
    A vector of magnitude 1. Any nonzero vector v gives the unit vector v̂ = v/‖v‖ pointing in the same direction.
  • VectorCh 2
    An ordered list of numbers, interpreted geometrically as an arrow with direction and magnitude, and algebraically as an element of ℝⁿ. Vectors can represent displacement, force, data, coordinates, or states.
  • Vector spaceCh 8
    A set with operations of addition and scalar multiplication satisfying ten axioms. Examples include ℝⁿ, spaces of matrices, and spaces of polynomials. Provides a common language for objects that behave like vectors.

Checking Your Own Work

Linear algebra offers built-in consistency checks. For system solutions: substitute back and verify every equation. For matrix inverses: confirm AA⁻¹ = I. For eigenvalue-eigenvector pairs: verify Av = λv directly. For projections: confirm the residual b − b̂ is orthogonal to the subspace. For rank and nullity: confirm they sum to the number of columns. These checks catch most errors without a complete reworking.
