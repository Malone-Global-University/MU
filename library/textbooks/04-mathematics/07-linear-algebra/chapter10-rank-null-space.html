Chapter 10 — Front Matter

Purpose and Overview

Every matrix carries hidden structure. When you write down a matrix — whether it represents a system of equations, a data table, a linear transformation, or a network — it contains information about dimension, dependence, consistency, and collapse. Chapter 10 is about learning to read that structure.

In previous chapters, you learned that vector spaces and subspaces are the natural setting for linear algebra, and that span, linear independence, and basis give you the tools to describe dimension precisely. Now you will connect those ideas directly to matrices. You will learn to ask: what output vectors can this matrix produce? What input vectors does this matrix send to zero? How much information does this matrix preserve, and how much does it collapse?

The answers are organized into four distinguished subspaces — the column space, the row space, the null space, and the left null space — collectively called the fundamental subspaces of a matrix. Two numbers, called rank and nullity, measure the dimensions of these spaces and are connected by a beautiful theorem called the Rank-Nullity Theorem. Together, these ideas form one of the most complete structural accounts in all of linear algebra.

Opening Question

Suppose a company tracks five products across three stores. An analyst organizes the sales data into a matrix A with 3 rows and 5 columns. She runs row reduction and finds that only two pivot columns appear. What does this mean? Can she reconstruct the full picture from two columns? Are some product patterns redundant? If she tries to fit a model Ax = b using this data, for which demand vectors b does a solution exist? How many solutions are there? These are questions about rank, column space, and null space. By the end of this chapter, you will be able to answer all of them.

Why This Chapter Matters

The ideas here appear throughout applied mathematics. In data science and machine learning, the rank of a data matrix tells you how many truly independent features exist — low-rank matrices indicate redundancy or correlation. In systems and engineering, the column space tells you which outputs a system can produce, and the null space describes which inputs have no effect on output. In statistics, the null space of the design matrix corresponds to combinations of predictors that cannot be estimated from the data. In computer graphics, transformations that collapse dimension produce degenerate projections. In differential equations, the null space of a coefficient matrix determines the structure of homogeneous solution sets. Understanding rank and null space means understanding how information flows through a linear system.

Section 10.1

Matrices as Maps Between Spaces

A matrix is not just a rectangular array of numbers. When you multiply a matrix A by a vector x, you produce a new vector Ax. This operation assigns to each input vector an output vector. In the language of functions, the matrix A defines a function from one vector space to another.

If A is an m × n matrix, then x must be a vector in ℝⁿ (with n components) and the product Ax is a vector in ℝᵐ (with m components). The matrix A maps vectors from ℝⁿ to vectors in ℝᵐ. This is a natural extension of the function idea from Precalculus, where a function maps a single real number to another. Here, inputs and outputs are vectors.

The domain of the map is ℝⁿ: all possible input vectors x. The codomain is ℝᵐ: the space where all outputs Ax live. But the matrix does not necessarily reach every vector in ℝᵐ. The collection of all vectors that A can actually produce — the set of all Ax as x ranges over all inputs — is called the image or range of the map. This is exactly the column space of A.

Similarly, some nonzero input vectors x may be sent to 0 by the matrix. The collection of all such vectors forms the null space of A. The rank and nullity measure how much of the output space is reached and how much of the input space is collapsed.

Section 10.2

Column Space

Definition — Column Space

The column space of a matrix A, written Col(A), is the set of all vectors that can be written as linear combinations of the columns of A. If A has columns a₁, a₂, …, aₙ, then:

Col(A) = span{a₁, a₂, …, aₙ}

This is a subspace of ℝᵐ. Because Ax = x₁a₁ + x₂a₂ + ··· + xₙaₙ, the column space is exactly the set of all vectors A can produce as outputs.

Geometric meaning: The column space is the reachable output region of the matrix. If b is in Col(A), the equation Ax = b has at least one solution. If b is not in Col(A), the equation has no solution at all.

Finding a Basis for the Column Space

The columns of A may not all be linearly independent. To find a basis for the column space, use row reduction to identify the pivot columns — the columns containing the leading entries — then select those columns from the original matrix A (not from the reduced form).

Critical Distinction

Use the pivot columns from the original matrix A, not from the row-reduced form. Row operations change column relationships, so the reduced columns do not lie in the original column space.

Example 10.2.1
Finding a basis for the column space
  • ProblemFind a basis for the column space of:
    1 2 3 A = 2 4 7 1 2 4
  • ThinkRow-reduce A to identify pivot columns, then select those columns from the original matrix.
  • Calculate Apply R₂ ← R₂ − 2R₁, then R₃ ← R₃ − R₁, then R₃ ← R₃ − R₂: 1 2 3 1 2 3 1 2 3 2 4 7 → 0 0 1 → 0 0 1 1 2 4 0 0 1 0 0 0 Pivots appear in columns 1 and 3. Column 2 is a free column.
  • Select A basis for Col(A) uses columns 1 and 3 of the original matrix: a₁ = (1, 2, 1)ᵀ a₃ = (3, 7, 4)ᵀ
  • CheckThese two vectors are not scalar multiples of each other — they are linearly independent. Together they span a two-dimensional subspace of ℝ³.
  • InterpretThe column space of A is a plane through the origin in ℝ³. The equation Ax = b is consistent only if b lies on this plane — that is, only if b is a linear combination of a₁ and a₃.
Answer: Basis for Col(A) = {(1,2,1)ᵀ, (3,7,4)ᵀ}. Dimension = 2.
Section 10.3

Row Space

Definition — Row Space

The row space of a matrix A, written Row(A), is the subspace of ℝⁿ spanned by the rows of A. The row space lives in the input space ℝⁿ, while the column space lives in the output space ℝᵐ.

The row space represents the directions in the input space that the matrix can "detect." Input vectors perpendicular to the row space are sent to zero — they lie in the null space.

Finding a Basis for the Row Space

Unlike the column space, the row space is preserved by row operations. Taking linear combinations of rows does not change what those rows span. Therefore, use the nonzero rows of the echelon form as a basis for Row(A) — you may use the reduced rows directly, which is not permitted for the column space.

Example 10.3.1
Finding a basis for the row space
  • ProblemFind a basis for the row space of the same matrix A from Example 10.2.1.
  • ThinkThe echelon form is already known. The nonzero rows of the echelon form are a basis for Row(A).
  • Calculate From the echelon form, the nonzero rows are: r₁ = (1, 2, 3) r₂ = (0, 0, 1)
  • CheckThese two rows are linearly independent: neither is a multiple of the other. They span a two-dimensional subspace of ℝ³.
  • InterpretThe row space of A is a plane in ℝ³ — the same dimension as the column space. This is not a coincidence. It is one of the central facts of this chapter: the column space and the row space always have the same dimension. That common dimension is the rank of A.
Answer: Basis for Row(A) = {(1,2,3), (0,0,1)}. Dimension = 2.
Key Asymmetry to Remember

For column space: take pivot columns from the original matrix. For row space: take nonzero rows from the echelon form. These rules are different because row operations change columns but preserve row span.

Section 10.4

Null Space

Definition — Null Space

The null space of a matrix A, written Null(A), is the set of all solutions to the homogeneous system Ax = 0:

Null(A) = {x ∈ ℝⁿ : Ax = 0}

The null space always contains the zero vector and is a subspace of ℝⁿ (the input space).

Why the Null Space Is a Subspace

Suppose u and v are both in Null(A), meaning Au = 0 and Av = 0. Then A(u + v) = Au + Av = 0 + 0 = 0, so u + v is also in Null(A). Similarly, A(cu) = cAu = c0 = 0, so scalar multiples stay in the null space. Closed under addition and scalar multiplication — the null space is a subspace.

Geometric meaning: The null space contains all input vectors that the matrix collapses to zero — vectors "invisible" to the matrix. If the null space contains only the zero vector, the matrix is one-to-one: different inputs produce different outputs. If the null space contains nonzero vectors, information is lost. The null space lives in the input space ℝⁿ, just like the row space.

Section 10.5

Solving Ax = 0

Finding the null space means solving the homogeneous system Ax = 0. Because the right-hand side is zero and row operations preserve this, you only need to row-reduce A itself, then identify pivot and free variables and write the solution in parametric vector form.

Example 10.5.1
Finding the null space — one free variable
  • ProblemFind the null space of A from Examples 10.2.1 and 10.3.1.
  • ThinkContinue to RREF from the echelon form to make back-substitution explicit.
  • Calculate From the echelon form, apply R₁ ← R₁ − 3R₂: 1 2 0 0 0 1 0 0 0 Pivot columns: 1 and 3. Free column: 2. Let x₂ = t (free parameter). From row 2: x₃ = 0 From row 1: x₁ + 2x₂ = 0 → x₁ = −2x₂ = −2t x = t · (−2, 1, 0)ᵀ
  • Check Verify: A · (−2, 1, 0)ᵀ = −2(1,2,1)ᵀ + 1(2,4,2)ᵀ + 0(3,7,4)ᵀ = (0,0,0)ᵀ ✓
  • InterpretThe null space is Null(A) = span{(−2, 1, 0)ᵀ} — a line through the origin in ℝ³. Every vector on this line is sent to 0 by A. The matrix collapses this entire line onto the origin.
Answer: Null(A) = span{(−2, 1, 0)ᵀ}. Nullity = 1.
Example 10.5.2
Finding the null space — two free variables
  • Problem Find the null space of: 1 2 0 3 B = 2 4 0 6
  • Calculate Apply R₂ ← R₂ − 2R₁: 1 2 0 3 0 0 0 0 Pivot column: 1. Free columns: 2, 3, 4. Let x₂ = s, x₃ = t, x₄ = u. From row 1: x₁ = −2x₂ − 3x₄ = −2s − 3u x = s·(−2,1,0,0)ᵀ + t·(0,0,1,0)ᵀ + u·(−3,0,0,1)ᵀ
  • InterpretThe null space is a three-dimensional subspace of ℝ⁴, spanned by three independent vectors. The matrix B collapses a three-dimensional subspace to the zero vector — it loses a great deal of information.
Answer: Null(B) = span{(−2,1,0,0)ᵀ, (0,0,1,0)ᵀ, (−3,0,0,1)ᵀ}. Nullity = 3.
Section 10.6

Rank

Definition — Rank

The rank of a matrix A, written rank(A), is the dimension of the column space of A. Equivalently, the rank equals:

· The number of pivot columns in the row echelon form of A

· The number of nonzero rows in the echelon form

· The dimension of the row space of A

All three descriptions give the same number. That the column space and row space always have the same dimension is one of the most important theorems in linear algebra.

Rank and Information Preservation

The rank measures how much independent information the matrix contains. A matrix with high rank preserves more information about the input; a matrix with low rank collapses more information. If A is m × n, then rank(A) ≤ min(m, n).

ConditionNameMeaning
rank(A) = nFull column rankColumns are independent; Ax = 0 has only x = 0; map is one-to-one
rank(A) = mFull row rankEvery b ∈ ℝᵐ is reachable; Ax = b always has at least one solution
rank(A) = n = mFull rank (square)Matrix is invertible; unique solution to Ax = b for every b
rank(A) < min(m,n)Rank-deficientSome output directions missing; some inputs collapsed to zero
Example 10.6.1
Computing rank from row reduction
  • Calculate Matrix A (3×3 from earlier): echelon form has 2 nonzero rows and 2 pivot columns. rank(A) = 2 Matrix B (2×4 from earlier): echelon form has 1 nonzero row and 1 pivot column. rank(B) = 1
  • InterpretMatrix A is 3×3 with rank 2 — not full rank. The column space is a plane, not all of ℝ³. Matrix B is 2×4 with rank 1 — most of the four input dimensions are collapsed to zero.
rank(A) = 2  |  rank(B) = 1
Section 10.7

Nullity

Definition — Nullity

The nullity of a matrix A, written nullity(A), is the dimension of the null space of A. Since the null space is spanned by one vector for each free variable in Ax = 0, the nullity equals the number of free variables — the number of non-pivot columns in the row echelon form.

Nullity and information loss: If the nullity is 0, the matrix is one-to-one — no information is lost. If the nullity is large, many different inputs map to the same output: the matrix collapses a high-dimensional subspace of inputs to zero.

Example 10.7.1
Computing nullity
  • Calculate Matrix A (3×3): 2 pivot columns → 1 free column. nullity(A) = 1 (confirmed: null space is a line — dimension 1) Matrix B (2×4): 1 pivot column → 3 free columns. nullity(B) = 3 (confirmed: null space is three-dimensional)
nullity(A) = 1  |  nullity(B) = 3

Section 10.8

The Rank-Nullity Theorem

Theorem — Rank-Nullity Theorem

Let A be an m × n matrix. Then: rank(A) + nullity(A) = n where n is the number of columns of A.

This says that rank and nullity partition the total number of input dimensions. Every column is either a pivot column (contributing to rank) or a free column (contributing to nullity). Since there are exactly n columns, the two counts must sum to n.

Interpretation — Conservation of Dimension

Think of the Rank-Nullity Theorem as a conservation law for dimensions:

The Conservation Law

dimensions preserved + dimensions collapsed = total input dimensions

Rank counts independent directions mapped to nonzero outputs. Nullity counts independent directions collapsed to zero. Together they account for all of ℝⁿ.

Example 10.8.1
Applying the Rank-Nullity Theorem — from rank to nullity
  • ProblemA 4 × 7 matrix A has rank 3. What is its nullity? What does this mean?
  • Calculate rank(A) + nullity(A) = n 3 + nullity(A) = 7 nullity(A) = 4
  • CheckThe rank cannot exceed 4 (the number of rows), and it is 3, so this is consistent.
  • InterpretThe matrix produces outputs in ℝ⁴, but its column space is only 3-dimensional — it cannot reach every vector in ℝ⁴. In the input space ℝ⁷, four independent directions are sent to zero. Only three independent directions are "detected" and mapped to nonzero outputs.
nullity(A) = 4. The matrix collapses a 4-dimensional subspace to zero.
Example 10.8.2
Using the theorem to draw structural conclusions
  • ProblemA 5 × 5 matrix A has nullity 0. What can you conclude?
  • Calculaterank(A) + 0 = 5 → rank(A) = 5
  • InterpretThe matrix has full rank. All 5 columns are independent. Every vector in ℝ⁵ is reachable as an output. The matrix is invertible, and the equation Ax = b has exactly one solution for every b ∈ ℝ⁵. All structural properties of invertibility follow from nullity 0.
rank = 5. The matrix is invertible. Unique solution for every right-hand side.
Section 10.9

Consistency and Column Space

Theorem — Consistency Criterion

The system Ax = b is consistent (has at least one solution) if and only if b ∈ Col(A) — that is, if and only if b is a linear combination of the columns of A.

To check: augment [A | b] and row-reduce. If rank([A|b]) = rank(A), the system is consistent. If rank([A|b]) = rank(A) + 1, the system is inconsistent — a new pivot appeared in the last column.

Example 10.9.1
Testing consistency — consistent and inconsistent cases
  • Problem For matrix A from before, test b = (1, 3, 2)ᵀ and b' = (1, 3, 3)ᵀ.
  • Calculate — b Row-reduce [A | b]: 1 2 3 | 1 1 2 3 | 1 1 2 3 | 1 2 4 7 | 3 → 0 0 1 | 1 → 0 0 1 | 1 1 2 4 | 2 0 0 1 | 1 0 0 0 | 0 No new pivot in the last column. rank([A|b]) = 2 = rank(A). Consistent.
  • Calculate — b' Row-reduce [A | b']: 1 2 3 | 1 1 2 3 | 1 1 2 3 | 1 2 4 7 | 3 → 0 0 1 | 1 → 0 0 1 | 1 1 2 4 | 3 0 0 1 | 2 0 0 0 | 1 New pivot in the last column (the equation 0 = 1). rank([A|b']) = 3 > 2 = rank(A). Inconsistent.
  • Interpretb = (1,3,2)ᵀ lies in the column space of A — the system has solutions. b' = (1,3,3)ᵀ does not — no combination of columns of A can produce it.
b = (1,3,2)ᵀ: consistent.  |  b' = (1,3,3)ᵀ: inconsistent.

Section 10.11

The Four Fundamental Subspaces

Every m × n matrix A has four fundamental subspaces, first identified and named by Gilbert Strang. These four subspaces describe everything about the structure of the linear map defined by A.

SubspaceSymbolLives inDimensionGeometric meaning
Column spaceCol(A)ℝᵐ (output)rAll reachable outputs
Left null spaceNull(Aᵀ)ℝᵐ (output)m − rOutput directions unreachable
Row spaceRow(A)ℝⁿ (input)rInput directions detectable
Null spaceNull(A)ℝⁿ (input)n − rInput directions collapsed to zero

where r = rank(A). Notice: in ℝᵐ, the column space and left null space have dimensions r and m − r that sum to m. In ℝⁿ, the row space and null space have dimensions r and n − r that sum to n. This is a second expression of the Rank-Nullity idea — both ambient spaces are completely partitioned.

The Orthogonality Relationship (Preview)

The row space and null space are orthogonal complements in ℝⁿ: every vector in the row space is perpendicular to every vector in the null space. Similarly, the column space and left null space are orthogonal complements in ℝᵐ. You can see a preview now: a vector v in the null space satisfies Av = 0. If r is any row of A, the dot product r · v equals the corresponding component of Av, which is 0. So null space vectors are perpendicular to all rows — perpendicular to the entire row space. Chapter 11 will develop this relationship fully and precisely.

Example 10.11.1
Dimensions of all four fundamental subspaces
  • ProblemFor matrix A (3×3, rank 2 from earlier), find the dimensions of all four fundamental subspaces.
  • Calculate r = rank(A) = 2, m = 3, n = 3. Col(A) ⊂ ℝ³ dim = r = 2 (a plane) Null(Aᵀ) ⊂ ℝ³ dim = m − r = 1 (a line) Row(A) ⊂ ℝ³ dim = r = 2 (a plane) Null(A) ⊂ ℝ³ dim = n − r = 1 (a line)
  • InterpretThe column space is a plane in ℝ³; the left null space is a line perpendicular to that plane. The row space is a plane in ℝ³; the null space is a line perpendicular to the row space. Each of the two ambient spaces ℝ³ is split into a plane and its perpendicular line.
Four dimensions: 2, 1, 2, 1. Each ambient ℝ³ splits into plane + perpendicular line.
Section 10.12

Rank in Data and Modeling

Rank and Redundancy in Data

In data science and statistics, a data matrix A might have m observations (rows) and n features (columns). The rank tells you how many truly independent patterns exist in the data. If the rank is much smaller than n, many features carry no information that cannot be recovered from the others. This is the foundation of principal component analysis (PCA), encountered in Data Science Foundations and Machine Learning Mathematics.

Underdetermined and Overdetermined Systems

When n > m (more unknowns than equations), a consistent system has infinitely many solutions — the null space is nontrivial. The solution set has dimension equal to the nullity: a whole family parameterized by the free variables. When m > n (more equations than unknowns), the system generally has no exact solution unless b happens to lie in the column space of A. This is the context for least squares approximation (Chapter 11): when Ax = b has no solution, find the x that makes Ax as close as possible to b.

Example 10.12.1
Rank in a data context — complete redundancy
  • Problem A data matrix records five students' scores on four tests: 80 40 160 80 D = 70 35 140 70 90 45 180 90 60 30 120 60 Determine the rank and interpret the result.
  • ThinkInspect the columns: column 2 appears to be ½ of column 1, column 3 appears to be 2× column 1, and column 4 equals column 1.
  • Calculate Every column is a scalar multiple of column 1. After row operations, only one nonzero row survives. rank(D) = 1
  • InterpretThe four tests are completely redundant — a student's score on any one test perfectly predicts their scores on all others. All information lives on a one-dimensional subspace. Only one parameter is needed to describe this dataset, even though four measurements were taken. In a real setting, this would raise concerns about test design or data entry errors.
rank(D) = 1. All four tests are scalar multiples of each other — complete redundancy.

Section 10.13

Common Rank and Null Space Mistakes

Mistake 1 — Taking pivot columns from the reduced form for Col(A)
Row-reduce A → RREF, then use pivot columns of the RREF as a basis for Col(A)

Row operations change columns. The RREF pivot columns (containing standard basis vectors) are not in the column space of the original A. Correct approach: use the pivot positions to identify which columns to take, but take those columns from the original matrix.

Mistake 2 — Confusing Row Space and Column Space
"The row space is spanned by the pivot columns" or "the column space is spanned by the nonzero rows"

Rows live in ℝⁿ (input space). Columns live in ℝᵐ (output space). They are not the same when m ≠ n, and even when m = n they are distinct subspaces with distinct meanings. Column space uses original pivot columns; row space uses reduced nonzero rows.

Mistake 3 — Forgetting free variables in the null space
Solving Ax = 0 and reporting "the null space is the vector v = (1, −2, 1)"

The null space is a subspace, not a single vector. When there is one free variable, the null space is span{v} — a whole line. When there are two free variables, the null space is a plane. Always express Null(A) as a span, and verify the nullity matches the number of free variables.

Mistake 4 — Confusing rank with number of rows or columns
Assuming a 4 × 7 matrix automatically has rank 4 (or 7)

The rank is at most min(m, n), but it may be less. A 4 × 7 matrix can have rank anywhere from 0 to 4. You must row-reduce the matrix to determine the actual rank. Many matrices are rank-deficient.

Mistake 5 — Assuming the null space is trivial for "tall" matrices
Assuming a 5 × 3 matrix (more rows than columns) must have only the trivial null space

A 5 × 3 matrix can still have a nontrivial null space if its rank is less than 3. Whether the null space is trivial depends entirely on the rank, not on whether m > n. Always check rank before drawing conclusions about the null space.

Mistake 6 — Applying the Rank-Nullity Theorem with the wrong n
For an m × n matrix, writing rank(A) + nullity(A) = m instead of n

The Rank-Nullity Theorem uses the number of columns n, not rows m. The null space lives in the input space ℝⁿ, and nullity counts free variables — which depend on the number of columns. Correct statement: rank(A) + nullity(A) = n.

Mistake 7 — Confusing consistency with invertibility
Assuming that a consistent system Ax = b implies A is invertible

A system can be consistent (have solutions) without A being invertible. If A has a nontrivial null space, the system may have infinitely many solutions. Invertibility requires both consistency (at least one solution) and uniqueness (exactly one solution), which requires nullity 0.

Section 10.14

Preparing for Orthogonality

Chapter 10 has revealed the deep structure inside every matrix: four subspaces, two dimensions (rank and nullity), and a beautiful theorem connecting them. But there is a geometric relationship you have only glimpsed: the row space and the null space are perpendicular, and the column space and the left null space are perpendicular.

To understand perpendicularity precisely in ℝⁿ, you need the concept of the dot product and orthogonality — the subjects of Chapter 11.

It will develop projections as a way to find the "closest" vector in a subspace, and introduce least squares as an application of projection when a system has no exact solution. The fundamental subspaces you identified in Chapter 10 will become the geometric stage for Chapter 11's orthogonality theory.

Chapter Summary

  • Sec 10.1An m × n matrix defines a linear map from ℝⁿ to ℝᵐ. The column space is the set of reachable outputs; the null space is the set of inputs collapsed to zero. Rank and nullity measure these two aspects of the map.
  • Sec 10.2The column space Col(A) is the span of the columns of A, a subspace of ℝᵐ. A basis is found by identifying pivot columns in the echelon form, then selecting those columns from the original matrix.
  • Sec 10.3The row space Row(A) is the span of the rows of A, a subspace of ℝⁿ. Row operations preserve the row space, so the nonzero rows of the echelon form are a valid basis.
  • Sec 10.4The null space Null(A) is the solution set of Ax = 0, a subspace of ℝⁿ. It contains all inputs that A collapses to zero. A nontrivial null space means information is lost.
  • Sec 10.5To find the null space, row-reduce A to RREF, assign free parameters to each free variable, express pivot variables in terms of free ones, and write the parametric vector form. One parametric vector per free variable.
  • Sec 10.6The rank of A is the dimension of the column space (equivalently, the row space). It equals the number of pivot columns and measures how much independent information the matrix preserves.
  • Sec 10.7The nullity of A is the dimension of the null space. It equals the number of free variables and measures how much information the matrix loses.
  • Sec 10.8The Rank-Nullity Theorem: rank(A) + nullity(A) = n (number of columns). It is a conservation law for dimensions: dimensions preserved plus dimensions collapsed equals total input dimensions.
  • Sec 10.9The system Ax = b is consistent if and only if b ∈ Col(A). This is detected by comparing rank(A) and rank([A|b]). A new pivot in the last column signals inconsistency.
  • Sec 10.10Row reduction provides a unified procedure for all three bases: original pivot columns for Col(A), echelon nonzero rows for Row(A), and parametric null space vectors for Null(A).
  • Sec 10.11The four fundamental subspaces are Col(A), Null(Aᵀ), Row(A), and Null(A), with dimensions r, m−r, r, and n−r. The row space and null space are orthogonal complements in ℝⁿ; the column space and left null space are orthogonal complements in ℝᵐ.
  • Sec 10.12In data science, rank measures independent patterns; low rank indicates redundancy. Underdetermined systems have nontrivial null spaces; overdetermined systems may require least squares. Rank also measures independent constraints in network and economic models.
  • Sec 10.13Seven common errors: using RREF pivot columns for Col(A); confusing row and column space; reporting a single vector instead of a spanning set for the null space; assuming rank equals m or n; assuming tall matrices have trivial null spaces; misapplying Rank-Nullity with m; confusing consistency with invertibility.
  • Sec 10.14Chapter 11 will prove the orthogonal complement relationship between the four subspaces, develop projections, and introduce least squares. The fundamental subspaces of Chapter 10 become the geometric stage for all of Chapter 11.

Key Terms Review

column space
Col(A): the span of the columns of A; a subspace of ℝᵐ; the set of all reachable outputs of the matrix.
row space
Row(A): the span of the rows of A; a subspace of ℝⁿ; the input directions detectable by the matrix.
null space
Null(A): the solution set of Ax = 0; a subspace of ℝⁿ; all inputs collapsed to zero by the matrix.
left null space
Null(Aᵀ): the solution set of Aᵀy = 0; a subspace of ℝᵐ; the output directions the matrix cannot reach.
rank
rank(A): the dimension of the column space (or row space); the number of pivot columns; how much information is preserved.
nullity
nullity(A): the dimension of the null space; the number of free variables; how much information is lost.
Rank-Nullity Theorem
For any m × n matrix A: rank(A) + nullity(A) = n. A conservation law for dimensions.
pivot column
A column in the row echelon form that contains a leading nonzero entry. Pivot columns determine the rank.
free column
A column with no pivot in the echelon form. Each free column corresponds to a free variable and a null space basis vector.
homogeneous system
The system Ax = 0 with zero right-hand side. Its solution set is always a subspace — the null space of A.
consistency
A system Ax = b is consistent if it has at least one solution — equivalently, if b ∈ Col(A).
full rank
A matrix has full rank when rank(A) = min(m, n). A square matrix with full rank is invertible.
rank-deficient
A matrix whose rank is strictly less than min(m, n) — it collapses more than the minimum possible.
fundamental subspaces
The four distinguished subspaces Col(A), Row(A), Null(A), and Null(Aᵀ) that describe the complete structure of the linear map defined by A.
particular solution
Any single vector x₀ satisfying Ax₀ = b. The general solution is x₀ plus any vector in Null(A).
general solution
The complete solution set of Ax = b: one particular solution plus the entire null space of A.
