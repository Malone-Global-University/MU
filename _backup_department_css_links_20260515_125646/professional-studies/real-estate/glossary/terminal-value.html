Terminal Value

Real Estate Investing Glossary – Malone Global University

Home Department Professional Studies Real Estate Track Glossary Terminal Value

Definition

Terminal value is the projected value of a property at the end of an investor’s holding period. It represents the expected sale price (or residual value) used in discounted cash flow (DCF) analysis.

How It’s Calculated

In real estate, terminal value is most commonly calculated using an exit cap rate.

  • Terminal Value = Projected NOI in Exit Year ÷ Exit Cap Rate

Example

An investor projects Year 5 NOI to be $1,500,000. They assume an exit cap rate of 7.5%.

  • Terminal Value = 1,500,000 ÷ 0.075 = $20,000,000

Selling costs (brokerage fees, legal, transfer taxes) are then deducted to calculate net sale proceeds.

Why It Matters

  • Often represents a large portion of projected total return
  • Strongly influences IRR calculations
  • Reflects market expectations at exit
  • Connects income growth assumptions with future pricing

Exit Cap Rate Assumptions

Investors typically use a conservative exit cap rate that is equal to or slightly higher than the going-in cap rate to account for market risk and uncertainty.

DCF Integration

In a DCF model, terminal value is discounted back to present value along with projected annual cash flows.

  • Present Value = Terminal Value ÷ (1 + Discount Rate)n

Where n equals the number of years in the holding period.

Investor Perspective

Because terminal value can heavily impact projected returns, disciplined investors stress-test exit cap rates and NOI projections. Overly optimistic assumptions can significantly inflate projected IRR.

Common Pitfalls

  • Using unrealistically low exit cap rates
  • Overestimating NOI growth
  • Ignoring selling costs
  • Relying too heavily on appreciation assumptions
