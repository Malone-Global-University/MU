Return on Cost

Real Estate Investing Glossary – Malone Global University

Home Department Professional Studies Real Estate Track Glossary Return on Cost

Definition

Return on cost measures how much stabilized income a property produces relative to the total cost basis to acquire and deliver the asset. It is commonly used in development and value-add underwriting to answer a simple question: “If I build or reposition this property, what yield do I earn on what I actually spent?”

Core Formula

  • Return on Cost = Stabilized NOI ÷ Total Project Cost

What counts as Total Project Cost?

“Total cost” should reflect the full cost basis required to deliver the stabilized asset. Depending on deal type, it can include:

  • Purchase price (or land basis in development)
  • Hard costs (construction, materials, labor)
  • Soft costs (architecture, engineering, permits, legal)
  • Financing costs (loan fees, interest during construction/reno)
  • CapEx / renovation budget
  • Lease-up / marketing costs and concessions (where applicable)
  • Reserves required at stabilization (sometimes included depending on convention)

How It’s Used

Return on cost is most useful when compared to the market’s prevailing cap rate for similar stabilized assets. The difference between your return on cost and the market cap rate is sometimes called a yield spread.

  • If Return on Cost > Market Cap Rate: you may be “creating value” (building below market pricing).
  • If Return on Cost ≈ Market Cap Rate: value creation is thin; execution risk may not be compensated.
  • If Return on Cost < Market Cap Rate: you are likely overpaying or under-delivering NOI.

Return on Cost vs. Cap Rate

These sound similar but answer different questions:

  • Cap rate: NOI ÷ Market value (what the market is paying today for stabilized NOI).
  • Return on cost: NOI ÷ Total cost (what you paid to create that NOI).

Investors use return on cost to underwrite the business plan; they use cap rate to estimate value at stabilization or exit.

Example

A sponsor buys an older apartment building for $12,000,000 and invests $2,000,000 in renovations. Total project cost is $14,000,000. After improvements and lease-up, stabilized NOI is $980,000.

  • Return on Cost = $980,000 ÷ $14,000,000 = 7.0%

If comparable stabilized assets trade at a 5.75% cap rate, implied stabilized value would be:

  • Implied Value = $980,000 ÷ 0.0575 ≈ $17,043,478

This spread suggests the plan may create value—assuming NOI is durable and the market cap rate holds.

Why It Matters

  • Reality check: ties returns to actual costs, not optimistic valuations.
  • Value-add discipline: forces clarity on whether CapEx truly produces enough NOI.
  • Development feasibility: helps determine if new construction is worth the risk vs. buying stabilized.
  • Capital allocation: helps compare different projects on a consistent basis.

Common Pitfalls

  • Using “pro forma” NOI too early: return on cost should use stabilized NOI, not a peak-year fantasy.
  • Understating total cost: ignoring interest carry, lease-up costs, or contingencies inflates returns.
  • Confusing NOI with cash flow: NOI excludes debt service; return on cost is not a levered return metric.
  • Ignoring cap rate risk: value creation can disappear if cap rates expand before stabilization.
  • Not stress-testing: small misses in rent, occupancy, or expenses can materially reduce ROI on cost.
