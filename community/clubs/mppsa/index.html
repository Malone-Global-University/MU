Welcome

The MPPSA (Malone Political Party Student Alliance) is a free assembly club organized around political engagement and learning party politics leadership. Members of the study and discuss the political platform of the Malone Party with the goal of spreading its agenda.

Links

  • Political Stances

    This is an internal link for a club hosted summary of the Malone Party's 32 leading platform issues in an accessible html format.

  • The Malone Platform

    This is an external link to the Official Malone Party Platform Library. The Malone Platform Library contains 96 pdfs. The 32 Manifesto pdfs define each of the 32 leading platform issues. The 32 Essay pdfs explain how the problem is structured and provides background on the issue for context. The 32 Polemic pdfs make arguments against opponents claims in defense of the Malone Party's position on these issues. This three layer system first defines, then isolates, then solves the problem.

  • Discussions

    This link is for an index of group discussions that evaluate different strategies for implenting solutions to the issues pertinent to the Malone Party belief system.

  • Malone Platform Mirror

    This link is for a index repository of Malone Party pdfs. All the PDFs from the offical site have been downloaded, copied, and re-hosted here. These PDFs are current as of 4-26-2026
