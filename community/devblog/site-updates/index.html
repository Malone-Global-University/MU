Site Update Archive

The MGU Devblog records visible progress on Malone Global University. It is designed as a lightweight update archive for new pages, department changes, library additions, design improvements, technical upgrades, and major content-production milestones.

Earlier versions of this page used screenshot cards. This revised version avoids heavy image loading and organizes updates by year so the archive can keep growing without becoming slow or visually crowded.

Posting Rule

Each update should be short, dated, and specific. A good entry should name the area changed, summarize the improvement, and link to the dated update page. Screenshots can still be used inside individual posts when helpful, but the main archive should remain text-first for speed and organization.
