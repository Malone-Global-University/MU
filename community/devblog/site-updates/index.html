Site Updates

Release notes and visible progress reports for the MGU website.

Purpose

The Site Updates page records visible changes to Malone Global University. This includes new pages, revised navigation, updated department structures, corrected links, improved templates, and major content additions.

Use this section when the change affects what visitors can see, use, browse, or understand on the public website.

What to Record

A strong site update should explain what changed, where it changed, and why the change matters. The goal is not only to announce progress, but to preserve a practical history of the site's growth.

Useful entries may include new index pages, department reorganizations, library expansions, homepage improvements, community additions, navigation changes, and fixes that improve the visitor experience.

Suggested Entry Types

New page announcements, revised section summaries, corrected navigation paths, updated landing pages, expanded reference areas, and deployment notes all belong here.

When possible, entries should include the affected path, the date of the change, a short description of the problem or improvement, and any follow-up work that remains.

Connection to the Devblog

Site Updates is the public-facing side of the Devblog. It records the finished changes that users can see, while other Devblog areas may focus more deeply on technical causes, design reasoning, or workflow lessons.
