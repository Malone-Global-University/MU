WORKFLOW NOTE

Title
MGU Prompt Bundle Production for Advanced Mathematics and AI Textbooks

Date
2026-05-17

Area
MGU Mathematics Series / Library Textbook Edition / Textbook Production Prompts

Problem
Several advanced MGU textbook outlines existed as planning material, but they had not yet been converted into reusable production bundles. The missing piece was a consistent set of master instruction sheets and chapter-by-chapter prompts that could be copied into Claude or another drafting model to generate full Markdown textbook chapters.

Intended Goal
Create downloadable ZIP bundles for each textbook area, with one master instruction sheet and one chapter prompt file per chapter. Each bundle should preserve the MGU Library Textbook Edition structure: full textbook identity, purpose, audience, chapter sequence, instructional philosophy, writing rules, section lists, back matter, and clear instructions to produce full Markdown chapters without answer keys.

Root Cause
The outlines were useful for planning, but they were not yet packaged into a repeatable production workflow. Without separate master instructions and chapter prompt files, each future chapter request would require manually reconstructing the textbook rules, sequence, and chapter-specific purpose.

Steps Taken

1. Built a Differential Equations chapter prompt ZIP using the previously established MGU master-instruction pattern and the 14-chapter Differential Equations outline.

2. Created an Artificial Intelligence Foundations bundle containing a master instruction sheet, README, and 14 chapter prompt files.

3. Created a calculus-based Probability and Statistics bundle with master instructions and 14 chapter prompts covering probability foundations, inference, regression, stochastic processes, and capstone modeling.

4. Created a calculus-based Data Science Foundations bundle with master instructions and 14 chapter prompts covering data workflow, cleaning, visualization, modeling, machine learning foundations, responsible practice, and capstone work.

5. Created a Machine Learning Mathematics bundle with master instructions and 14 chapter prompts covering vector representation, probability, loss functions, optimization, generalization, regression, classification, kernels, neural networks, representation learning, learning theory, and capstone modeling.

6. Used consistent file naming, folder naming, Markdown formatting, and ZIP packaging across all delivered bundles.

Key Change
Advanced textbook outlines were converted into reusable production-ready prompt bundles with standardized master instructions and chapter-level Markdown prompts.

Outcome
The MGU textbook production pipeline now has ready-to-use prompt packages for Differential Equations, Artificial Intelligence Foundations, Probability and Statistics, Data Science Foundations, and Machine Learning Mathematics. Each bundle can be reused to generate full textbook chapters without reconstructing the instructional rules or chapter structures from scratch.

Validation
Each ZIP bundle was generated and delivered as a downloadable file. The contents followed the same repeated structure: README, master instruction sheet, and 14 chapter prompt files. The chapter prompts preserved the requested chapter titles, purposes, sections, back matter, and “write as a full Markdown textbook chapter” instruction.

Rules Going Forward
Keep using the same bundle pattern for future MGU textbooks: one master instruction sheet, one README, and one Markdown prompt per chapter. Preserve the exact chapter sequence from the approved outline. Do not include answer keys unless explicitly requested. Keep prompts focused on full textbook chapter generation, not short lessons or worksheet packets. Maintain MGU Library Textbook Edition identity across all bundles.

Notes
This workflow is now efficient enough to apply to additional textbook outlines such as Scientific Computing, Numerical Methods, Combinatorics, Discrete Mathematics, Deep Learning, Natural Language Processing, Robotics, or AI Safety and Governance. Future bundles should continue using consistent folder names, stable prompt filenames, and chapter-specific back matter so the files remain easy to sort, copy, and archive.
