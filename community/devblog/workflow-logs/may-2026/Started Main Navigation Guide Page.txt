WORKFLOW NOTE

Title
Started Main Navigation Guide Page

Date
2026-05-17

Area
`orientation/navigation.html` — MGU Orientation / Site Navigation

Problem
The navigation guide page existed only as a blank structural scaffold. It had the basic HTML document setup, header injection, breadcrumb placeholder, main content area, bottom navigation placeholder, and footer injection, but it did not yet explain how visitors should move through Malone Global University. The meta description also still used placeholder maintenance-test wording, which did not match the purpose of the page.

Intended Goal
Create a clear first draft of the main MGU navigation guide that explains the major site areas, helps visitors understand the logic of the website, and provides a practical orientation layer for new and returning users.

Root Cause
The page structure was already in place, but the content layer had not yet been written. MGU has multiple major navigation areas, including Main, Orientation, Community, Department, Library, Tutorials, and Malone Panther, so the page needed explanatory content before it could function as a useful guide.

Steps Taken

1. Replaced the placeholder meta description with a navigation-focused description for Malone Global University.

2. Added a hero section with the title “Navigation Guide” and a short explanation of the page’s purpose.

3. Added an introductory section explaining that MGU is a large educational website organized around different kinds of use: study, reference, exploration, tutorials, community, and orientation.

4. Added an “On This Page” section with internal anchor links for the major guide sections.

5. Created explanatory sections for the main MGU navigation areas: Main, Orientation, Community, Department, Library, Tutorials, and Malone Panther.

6. Expanded the Library section with a nested “Major Library Areas” subsection covering Dictionary, Encyclopedia, Fiction, Guides, Malonicus Rex Educare, and Reference.

7. Added a “Recommended Starting Points” section for different visitor types, including new visitors, students, researchers, practical-task learners, and returning users.

8. Added a “Navigation Tips” section explaining breadcrumbs, section indexes, and bottom navigation.

9. Added a closing section titled “A Living Campus Map” to explain that the page should grow as MGU grows.

Key Change
The page was changed from a blank HTML scaffold into a usable first-draft orientation guide for navigating the major areas of Malone Global University.

Outcome
The navigation guide now gives visitors a clear explanation of the site’s top-level structure and helps them decide where to go based on what they are trying to do. It also establishes `orientation/navigation.html` as the central explanation page for MGU’s navigation logic.

Validation
The draft was checked against the existing MGU page structure: header placeholder, breadcrumb placeholder, main content area, bottom navigation placeholder, footer placeholder, and uploaded/updated date logic were preserved. The added content follows the intended site organization and uses semantic section headings with matching anchor IDs.

Rules Going Forward
Navigation guide pages should explain the logic of the site rather than trying to list every page. Top-level navigation should be described by function, not just by folder name. Orientation pages should help visitors understand how to use MGU, while department pages should focus on formal study and library pages should focus on reference, lookup, and organized knowledge.

Notes
This page can later be expanded with visual navigation cards, a site map, quick-start pathways, or links to more specific orientation guides such as a Library Guide, Department Guide, Tutorial Guide, or New Student Guide.
