WORKFLOW NOTE

Title
Jazz Musician Directory Expansion

Date
2026-05-16

Area
MGU Music Department / Jazz Musician Directory Pages

Problem
The Music Department had a starter HTML page for a “List of African American Jazz Musicians,” but the page was mostly empty. It needed organized musician lists, biography links, and a clearer directory role inside the larger MGU music structure. The page also raised a follow-up need for complementary jazz-style directory pages covering blues, gospel influence, ragtime, swing, bebop, hard bop, soul jazz, free jazz, fusion, and vocal jazz.

Intended Goal
Create a usable jazz musician directory system where the main page organizes African American jazz musicians by era, while separate complementary pages organize musicians by jazz style. Each musician name should function as a link to a future biography page, and each style page should be ready to drop into the MGU website as an `index.html` file.

Root Cause
The original page was only a structural shell. It had the MGU header, footer, breadcrumb, and bottom-navigation placeholders, but it lacked content architecture, musician groupings, internal page navigation, and a repeatable pattern for related jazz directory pages.

Steps Taken

1. Populated the main African American jazz musician page with era-based sections, including early jazz, blues/vocal jazz, swing, bebop, hard bop/soul jazz, avant-garde/free jazz, fusion/post-bop, and contemporary jazz.

2. Alphabetized musician names within each section and formatted each name as a biography link using clean folder-style URLs.

3. Corrected the page title spelling from “Musicans” to “Musicians.”

4. Recommended treating jazz-style musician lists as complementary standalone pages rather than trying to overload the main era-based page.

5. Defined a style-directory structure under a shared jazz musicians area, with folders for blues-jazz, gospel-influenced-jazz, ragtime, swing, bebop, hard-bop, soul-jazz, free-jazz, fusion, and vocal-jazz.

6. Created a zipped bundle named `mgu_jazz_style_musician_pages.zip`.

7. Generated 10 folder-based `index.html` pages, one for each jazz style directory.

8. Used consistent MGU page structure across the bundle: header injection, hero section, breadcrumb placeholder, overview section, “On This Page” links, grouped musician lists, editorial note, bottom navigation placeholder, and footer injection.

9. Used `/component/dept.css` for the generated department pages.

10. Added a `README.txt` explaining the bundle contents, suggested placement, and biography-link assumption.

Key Change
The jazz musician content was expanded from one empty shell page into a reusable directory system with both era-based and style-based navigation.

Outcome
The MGU Music Department now has a clearer jazz biography-index framework. The main page can serve as the broad African American jazz musician directory, while the bundled complementary pages allow visitors to browse by jazz style. The resulting structure supports future biography pages, cross-indexing, and expansion into instrument, region, and jazz-history pages.

Validation
The generated bundle contains the expected 10 style folders, each with an `index.html` file, plus a README. The pages use consistent MGU HTML scaffolding, working relative biography-link patterns such as `../charlie-parker/`, and alphabetized musician lists inside grouped sections.

Rules Going Forward
Style-based musician pages should remain separate from era-based musician pages. Musicians may appear on multiple pages when historically appropriate because cross-indexing is useful in a library system. Folder-based pages should continue using `index.html` inside descriptive directories. Biography links should remain clean, lowercase, hyphenated folder links unless the site’s biography path changes.

Notes
The next logical expansion would be instrument-based pages such as jazz trumpeters, jazz saxophonists, jazz pianists, jazz bassists, jazz drummers, jazz guitarists, and jazz vocalists. Regional pages such as New Orleans jazz musicians, Kansas City jazz musicians, Chicago jazz musicians, Harlem/New York jazz musicians, and West Coast jazz musicians would also complement the current directory system.
