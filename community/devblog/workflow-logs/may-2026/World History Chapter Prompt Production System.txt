WORKFLOW NOTE

Title
World History Chapter Prompt Production System

Date
2026-05-17

Area
MGU History Series / World History Textbook Production / Claude Chapter Prompt Workflow

Problem
The World History textbook had a strong chapter outline and overview schema, but it did not yet have a reusable full-depth chapter production system. The project needed the same kind of structured chapter template and Claude-ready prompt workflow previously created for the Government/Civics and U.S. History textbooks.

Intended Goal
Create a repeatable production method for generating full-depth World History textbook chapters as Markdown files. Each chapter prompt needed to preserve the book’s global, chronological, regional, comparative, and reference-oriented design while guiding Claude to produce substantial textbook chapters around 70,000 characters.

Root Cause
The outline defined the textbook’s 60-chapter structure, but the writing workflow still needed a standardized prompt architecture. Without that, chapters could become uneven, too brief, too Europe-centered, too focused on rulers and wars, or insufficiently connected to maps, timelines, regions, evidence, trade networks, environment, technology, and historical comparison.

Steps Taken

1. Reviewed the uploaded World History chapter outline and overview schema, which define **World History: Civilizations, Empires, and Global Change** as an MGU History Series Library Textbook Edition with 60 chapters grouped into 11 parts. 

2. Defined a full textbook chapter template for World History that includes chapter opening, learning objectives, guiding questions, historical context, chronology preview, regional orientation, vocabulary, main sections, regional lens, global connection, civilization/empire/network study, timeline study, map study, primary source window, evidence window, historical debate, comparison across societies, multiple perspectives, cause-and-effect analysis, continuity and change, everyday life, environment and technology, summary, review materials, writing prompts, research extension, and chapter reflection.

3. Adapted the template to the specific needs of world history by emphasizing regional balance, global connections, comparison across societies, trade and migration networks, empires, civilizations, religions, environment, technology, archaeology, material evidence, and non-European historical development.

4. Wrote a Claude-ready master prompt instructing Claude to produce full-depth World History chapters as serious Markdown textbook chapters of approximately 70,000 characters.

5. Used the uploaded 60-chapter outline to generate one prompt file per chapter, following the naming pattern `world_history_chapter_01_prompt.md`, `world_history_chapter_02_prompt.md`, and so on.

6. Formatted each prompt so Claude is instructed to write the matching output file, such as `world_history_chapter_01.md`, as a full Markdown chapter.

7. Created a zipped downloadable bundle containing all 60 individual chapter prompt files.

8. Created a combined Markdown file containing all 60 prompts in one document for easier review, copying, or archiving.

Key Change
The World History textbook now has a standardized full-depth chapter prompt system that can generate consistent, globally balanced, reference-grade Markdown chapters across all 60 chapters.

Outcome
The project moved from outline/schema planning into a usable production workflow. The user now has a downloadable ZIP bundle of individual Claude-ready chapter prompts and a single combined Markdown version. The prompt structure should help prevent shallow summaries, Europe-centered framing, and inconsistent chapter formats.

Validation
Confirmed that the generation process created **60 prompt files**, matching the 60 chapters in the uploaded World History outline. Also confirmed that both downloadable artifacts were created: `world_history_chapter_prompts.zip` and `world_history_all_chapter_prompts.md`.

Rules Going Forward
World History chapters should be generated from the standardized prompt files rather than improvised one at a time. Each chapter should remain chronological, regional, comparative, and globally connected. Chapters should not be reduced to rulers, generals, wars, Europe, or “rise and fall” narratives. Each chapter should include evidence, maps, timelines, regional comparison, global connections, ordinary life, environment, technology, and historical reasoning where relevant.

Notes
This workflow now matches the broader MGU textbook production pattern: define overview, define chapter outline, create full-depth chapter template, generate chapter-specific Claude prompts, then use those prompts to produce Markdown chapters. The World History textbook is positioned to become a major MGU History Library anchor supporting future timelines, maps, biography pages, primary source collections, regional guides, encyclopedia pages, and specialized textbooks.
