WORKFLOW NOTE

Title
MGU Orientation Folder Icon Creation

Date
2026-05-17

Area
MGU Orientation / Folder Icons / Visual Asset Organization

Problem
The MGU Orientation folder needed a distinctive square icon that could be used as a Windows `.ico` folder icon. There was no dedicated visual marker for the orientation area, making the folder harder to identify quickly in a larger local MGU file structure.

Intended Goal
Create a square MGU-themed orientation icon and provide a simple PowerShell/ImageMagick command to convert it into a 256×256 `.ico` file suitable for Windows folder customization.

Root Cause
The orientation folder did not yet have a dedicated icon asset. The user needed both the graphic concept and the correct conversion command, especially because Windows folder icons work best when exported as `.ico` files rather than ordinary PNG images.

Steps Taken

1. Designed a square MGU Orientation icon using a navy, gold, and white academic color scheme.

2. Used an orientation/navigation concept with a compass motif, open book, graduation cap, and the letters “MGU” to visually communicate guidance, onboarding, and academic direction.

3. Generated the PNG asset named `academic_compass_with_graduation_motif.png`.

4. Provided a PowerShell script using ImageMagick to convert the PNG into a Windows `.ico` file.

5. First provided a multi-size `.ico` conversion command using `icon:auto-resize=256,128,64,48,32,24,16`.

6. Revised the command after clarification so the output targets a 256×256 icon specifically.

7. Finalized the preferred command:

```powershell
magick "C:\MGU\academic_compass_with_graduation_motif.png" `
  -define icon:auto-resize=256 `
  "C:\MGU\mgu_orientation.ico"
```

Key Change
A dedicated MGU Orientation icon was created and paired with a 256×256 ImageMagick conversion command for Windows folder use.

Outcome
The MGU Orientation folder now has a custom visual identity that matches the broader MGU academic brand. The user also has a direct PowerShell command to convert the generated PNG into a usable `.ico` file without needing additional design or export steps.

Validation
The design was generated as a square image suitable for icon conversion, and the conversion command was adjusted to match the user’s stated requirement of a 256-pixel icon. The final command uses ImageMagick’s `.ico` export behavior with `icon:auto-resize=256`, which is appropriate for producing a 256×256 Windows icon file.

Rules Going Forward
MGU folder icons should use simple, high-contrast designs that remain readable at small sizes.

Orientation-related icons should visually suggest navigation, onboarding, direction, or getting started.

When converting generated PNG icons to `.ico`, use ImageMagick and specify the intended icon size clearly.

For single-size folder icons requested by the user, use `-define icon:auto-resize=256` instead of a multi-size icon list.

Keep icon filenames short, descriptive, and lowercase where practical, such as `mgu_orientation.ico`.

Notes
The generated design used a compass and academic motif, which makes it appropriate for the Orientation section. A future icon set could use the same visual language for related folders such as navigation, welcome, student guide, library orientation, course orientation, and site orientation.
