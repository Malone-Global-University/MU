WORKFLOW NOTE

Title
MGU Mathematics Textbook Prompt Frameworks

Date
2026-05-17

Area
MGU Mathematics Series / Library Textbook Edition / Chapter Prompt Production

Problem
The Geometry, Trigonometry and Precalculus, and High School Statistics textbook projects needed standardized master instructions and chapter prompt structures that matched the existing MGU Algebra I/Arithmetic-style textbook workflow. Without these, each textbook could drift in tone, chapter structure, assessment pattern, back matter, and instructional philosophy.

Intended Goal
Create reusable master instruction frameworks and chapter prompt files that preserve the MGU Library textbook format: meaning before procedure, full student-facing explanations, worked examples, applications, common mistakes, review material, error analysis, checkpoints, and bridge notes to later courses.

Root Cause
The outlines existed as chapter sequences and purpose notes, but they had not yet been converted into production-ready instruction sheets and chapter-level prompt files. Each textbook needed explicit rules for tone, formatting, examples, assessment sections, notation, diagrams, technology use, and course placement.

Steps Taken

1. Converted the Geometry outline into a full master instruction document modeled on the Algebra I master instructions.
2. Created a zipped folder of 14 Geometry chapter prompt Markdown files plus a README.
3. Converted the Trigonometry and Precalculus outline into a full master instruction document with guidance for functions, graphs, trigonometry, modeling, technology, and calculus readiness.
4. Created a zipped folder of 14 Trigonometry and Precalculus chapter prompt Markdown files plus a README.
5. Converted the High School Statistics outline into a full master instruction sheet with guidance for data collection, displays, summary statistics, probability, inference, modeling, ethics, technology, and capstone work.
6. Preserved the same MGU textbook identity across all three books: MGU Mathematics Series, Library Textbook Edition.
7. Maintained 14-chapter structures, 5-part organization, consistent chapter back matter, and robust full-textbook back matter for each textbook.

Key Change
The textbook outlines were transformed from planning drafts into reusable MGU production instructions and chapter prompt systems.

Outcome
Geometry, Trigonometry and Precalculus, and High School Statistics now have consistent writing rules, chapter expectations, instructional philosophy, back matter patterns, and production prompts. The Geometry and Trigonometry/Precalculus prompt bundles were delivered as zipped downloads, and the Statistics master instruction sheet is ready to support future chapter prompt generation.

Validation
The produced documents were checked against the provided outlines to ensure chapter titles, part divisions, section lists, textbook identity, instructional tone, back matter, and bridge-course placement were preserved. The zipped prompt folders included 14 chapter prompt files plus README files and followed the same prompt model requested by the user.

Rules Going Forward
Keep MGU Mathematics textbooks in the same Library Textbook Edition pattern unless explicitly changed. Do not rename books, reorder chapters, remove review material, omit worked examples, or turn chapters into worksheet packets. Use Markdown as the default chapter output format. Each chapter prompt should include chapter purpose, section list, back matter, and the instruction to write a full Markdown textbook chapter without the answer key.

Notes
The next logical production step is to create a zipped folder of High School Statistics chapter prompts using the same model already used for Geometry and Trigonometry and Precalculus.
