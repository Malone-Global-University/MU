WORKFLOW NOTE

Title
Calculus Chapter Prompt Bundle Creation

Date
2026-05-17

Area
MGU Mathematics Series / Calculus Textbook Production / Prompt Packaging

Problem
The Calculus textbook outline existed as a full chapter-outline draft, but it had not yet been converted into a reusable master instruction sheet or a packaged set of chapter-specific writing prompts. Without this, each chapter would require manually re-entering the same textbook identity, writing philosophy, structure, and chapter sequence instructions.

Intended Goal
Create a reusable Calculus master instruction sheet and package it with individual chapter prompts so the textbook can be generated chapter by chapter in a consistent MGU Library Textbook Edition format.

Root Cause
The outline described the textbook structure, chapter purposes, section lists, and back matter, but it was still in planning-outline form. It needed to be converted into production-ready prompt files that preserve the MGU instructional philosophy and prevent chapter drift during generation.

Steps Taken

1. Reviewed the uploaded Calculus outline and identified the intended book identity: Calculus, MGU Mathematics Series, Library Textbook Edition.
2. Converted the outline into a full master instruction sheet covering purpose, audience, writing style, instructional philosophy, chapter requirements, worked example formats, notation rules, graph/diagram guidance, technology expectations, modeling expectations, common mistake expectations, MGU connections, and series placement.
3. Preserved the 14-chapter, 5-part structure from the outline without renaming or reordering chapters.
4. Created individual chapter prompt files for Chapters 1–14, each tied to its chapter purpose and section sequence.
5. Added a full textbook back matter prompt so appendices, glossary, answer key, index, formula sheets, and readiness references can be generated later.
6. Packaged the master instruction sheet, chapter prompts, and back matter prompt into a zipped download folder named `calculus_chapter_prompts.zip`.

Key Change
The Calculus outline was transformed from a planning document into a reusable prompt-production bundle for generating the full textbook consistently.

Outcome
The Calculus textbook project now has a structured prompt kit that can be used to generate each chapter as a full MGU Library textbook chapter. This reduces repeated setup work, protects the chapter sequence, and keeps the tone, notation, examples, applications, review structure, and assessment format consistent across the book.

Validation
The generated master instruction sheet was checked against the uploaded outline for chapter order, part grouping, textbook title, series identity, chapter purposes, back matter, and pathway placement. The final ZIP bundle was delivered as a downloadable file and included the master instructions, all 14 chapter prompts, and the back matter prompt.

Rules Going Forward
Future MGU mathematics textbook prompt bundles should include a master instruction sheet, one prompt per chapter, and a back matter prompt when the outline includes appendices or full-book reference material. Chapter order, textbook title, series identity, and part structure should be preserved exactly unless explicitly revised. Prompts should maintain the MGU textbook philosophy: meaning before procedure, patient explanation, worked examples, applications, common mistakes, review, error analysis, checkpoints, and bridges to later study.

Notes
This Calculus book is positioned after Trigonometry and Precalculus and before Calculus II, Calculus III, Differential Equations, Linear Algebra, Discrete Mathematics, Probability and Statistics, and Data Science Foundations. It should function as the bridge from precalculus function modeling into higher mathematics, physics, engineering, economics, optimization, scientific computing, and advanced quantitative reasoning.
