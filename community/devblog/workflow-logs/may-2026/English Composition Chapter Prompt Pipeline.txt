WORKFLOW NOTE

Title
English Composition Chapter Prompt Pipeline

Date
2026-05-17

Area
MGU Humanities Series / English Composition and Academic Writing / Textbook Production Prompts

Problem
The English Composition textbook planning materials did not match. The uploaded chapter outline used a revised 8-part, 48-chapter structure, while the overview file included an older broader plan that suggested additional chapters and a possible 60-chapter version. This created uncertainty about which chapter list should be used for production prompts.

Intended Goal
Establish the 48-chapter outline as the working master structure and create a Claude-ready prompt system for producing full Markdown textbook chapters around 70,000 characters each.

Root Cause
The overview file was an earlier conceptual planning document that explored a larger possible structure, while the chapter outline file was the later, more settled production outline. Several topics in the overview were not missing from the outline; they had been consolidated into broader chapters.

Steps Taken

1. Compared the uploaded chapter outline against the uploaded overview document.
2. Identified the chapters or standalone topics that appeared in the overview but not in the 48-chapter outline.
3. Explained that the 48-chapter outline should be treated as the stronger master version.
4. Wrote a reusable master prompt for Claude to generate full textbook chapters at approximately 70,000 characters.
5. Required Claude output to be Markdown only, with no DOCX, HTML, JSON, XML, code, commentary, or front matter unless specifically requested.
6. Added chapter-quality rules covering structure, length, tone, examples, back matter, citation honesty, and MGU textbook identity.
7. Created a filled sample prompt for Chapter 1, “What Academic Writing Is.”
8. Generated a complete set of chapter prompts for all 48 chapters in the revised English Composition outline.
9. Delivered the prompts as a combined Markdown file and as a ZIP bundle containing the combined file plus individual chapter prompt files.

Key Change
The production workflow was standardized around the revised 48-chapter outline and Markdown-only Claude prompts for long-form textbook chapter generation.

Outcome
The English Composition and Academic Writing textbook now has a clear chapter-production prompt system. The mismatch between the old overview and the revised outline was resolved, and all 48 chapters now have prompt material ready for Claude-based Markdown chapter drafting.

Validation
The chapter mismatch was checked by comparing the chapter titles and structure from both uploaded files. The final prompt bundle was generated from the 48-chapter outline rather than the older 60-chapter overview. Download links were provided for both the combined Markdown prompt file and the ZIP bundle.

Rules Going Forward
Use the 48-chapter outline as the master production structure for this textbook.
Treat older overview documents as planning references, not final chapter lists.
Generate textbook chapters as Markdown files only unless a different format is explicitly requested.
Keep Claude prompts focused on complete, full-depth chapters around 70,000 characters.
Do not use DOCX production for this textbook pipeline unless specifically requested.
Do not split overview-only topics into extra chapters unless the master outline is intentionally revised.

Notes
Several overview topics remain useful as embedded material inside the 48-chapter structure. For example, timed essays, peer review, citation tools, research-paper writing, and final submission checks can be included inside relevant existing chapters rather than added as separate chapters.
