WORKFLOW NOTE

Title
Calculus Prompt Kit Production

Date
2026-05-17

Area
MGU Mathematics Series / Calculus Textbook / Chapter Prompt Workflow

Problem
The Calculus textbook had a complete outline, but it was still in draft planning form. It described the textbook’s chapter sequence, part structure, chapter purposes, section lists, and back matter, but it was not yet organized into reusable production prompts for generating the full textbook chapter by chapter.

Intended Goal
Create a reusable Calculus prompt kit that includes a master instruction sheet, individual chapter prompts, and a back matter prompt so the textbook can be generated consistently in the MGU Library Textbook Edition format.

Root Cause
The outline contained the correct structure, but without a separate master instruction sheet and chapter-level prompts, future chapter generation would require manually repeating instructions and would increase the risk of inconsistent tone, missing sections, renamed chapters, compressed explanations, or drift away from the MGU textbook model.

Steps Taken

1. Reviewed the Calculus outline and preserved its intended identity: **Calculus**, **MGU Mathematics Series**, **Library Textbook Edition**.
2. Converted the outline into a master instruction sheet covering purpose, audience, writing style, instructional philosophy, chapter structure, worked example formats, notation rules, graph guidance, technology expectations, modeling expectations, applications, common mistakes, MGU connections, and series placement.
3. Preserved the 5-part, 14-chapter sequence exactly as drafted.
4. Created individual chapter prompt files for Chapters 1–14, each tied to its assigned chapter purpose, section list, and instructional expectations.
5. Added a full textbook back matter prompt for appendices, formula references, readiness checks, glossary, answer key, index, and MGU Library connections.
6. Packaged the master instruction sheet, chapter prompts, and back matter prompt into a downloadable ZIP folder named `calculus_chapter_prompts.zip`.

Key Change
The Calculus outline was converted from a planning document into a reusable chapter-generation prompt kit.

Outcome
The Calculus textbook project now has a ready-to-use prompt bundle for producing full textbook chapters in a consistent format. This should reduce repeated setup work, keep chapter generation aligned with the MGU textbook philosophy, and make the Calculus book easier to produce systematically.

Validation
The completed prompt kit was checked against the original outline for title, series identity, chapter order, part grouping, chapter purposes, section structures, back matter, and pathway placement. The ZIP folder was created and delivered as a downloadable file.

Rules Going Forward
Future MGU textbook prompt bundles should include a master instruction sheet, one prompt file per chapter, and a back matter prompt when applicable. Chapter titles, sequence, part divisions, textbook identity, and series placement should remain stable unless intentionally revised. Prompts should preserve the MGU model: meaning before procedure, patient explanation, worked examples, applications, common mistakes, review, error analysis, checkpoints, and bridges to later study.

Notes
The Calculus book is positioned after Trigonometry and Precalculus and before Calculus II, Calculus III, Differential Equations, Linear Algebra, Discrete Mathematics, Probability and Statistics, and Data Science Foundations. It should function as the bridge from precalculus function modeling into higher mathematics, physics, engineering, economics, optimization, scientific computing, and advanced quantitative reasoning.
