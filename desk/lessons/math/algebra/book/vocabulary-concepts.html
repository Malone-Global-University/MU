Math Vocabulary and Concepts

Pre-calculus (also known as trigonometry which is algebra/geometry/trigonometry combined):

1. Arithmetic- The study of adding, subtracting, multiplying, dividing, powering, and radicalizing.

2. Algebra- The study of balancing equations using variables to solve for an unknown quantity using variables, graphing and functions. It is arithmetic with variables (formulas). Calculus without derivatives.

3. Geometry- The study of shapes and their dimensions, number of sides, and measure of their angles using proofs.

4. Trigonometry- The study of right angles and their measure. Using trig functions to find unknown angles or distances.

5. Statistics- The study of odds and probability. This leads into further number theory.

Advanced math:

1. Calculus- The study of rates of change using formulas for slope such as (y2 -y1) √∑ (x2 -x1). The two parts are derivatives and integration finding slope of a curved function and taking the area under that curve.

2. Linear algebra- Studying higher level linear equations using matrices, determinants, and Gaussian elimination. Focuses on moving through vector spaces using multi dimensional plane geometry. Also called vector algebra or tensor calculus to use matrices and arrays as tables to list numbers.

3. Discrete mathematics- Studying logic in the form of formal written proofs, Getting into set theory and set notation to list things in groups of numbers called sets. Logic means symbolism.

4. Lambda calculus- Formalizing mathematical concepts in a pure form for later use in emerging fields such as science and computer technology.

5. Differential equations ‚Äì Higher levels of calculus concepts dealing with vector spaces.

6. Number theory ‚Äì Statistics using tools like Bayesian theorems and computer modeling. It helps to minor in programming using R language and libraries.

7. Combinatorics ‚Äì Set theory with application for computer algorithms.

8. Topology- Algorithms using ordered data in a set theory environment.
