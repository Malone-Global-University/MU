Lesson 4: Opposite of a Number

Understand opposite numbers (additive inverses) and how they are used in addition and subtraction.

Home Pre-Algebra Unit 1: Numbers & Operations Lesson 4: Opposite of a Number

Lesson Overview

An opposite number (also called an additive inverse) is a number that, when added to the original number, results in zero. Understanding opposites helps simplify calculations and solve equations.

Topics Covered

  • Definition of opposite numbers
  • Examples of opposites: 5 and -5, -8 and 8
  • Addition of opposite numbers: x + (-x) = 0
  • Using opposites to solve simple equations

Lesson Examples

Example 1: 7 + (-7) = 0

Example 2: -4 + 4 = 0

Example 3: Solve x + 5 = 0 ‚Üí x = -5

Practice Problems

  1. Find the opposite of: 9, -12, 0, 7.
  2. Simplify: -6 + 6, 8 + (-8), -3 + 3.
  3. Solve for x: x + 7 = 0; x + (-9) = 0.
  4. Explain why every number has exactly one opposite number.
