Lesson 8: Absolute Value

Understand absolute value as the distance from zero, and learn how to solve problems involving absolute values.

Home Pre-Algebra Unit 1: Numbers & Operations Lesson 8: Absolute Value

Lesson Overview

The absolute value of a number is its distance from zero on the number line, regardless of direction. Absolute values are always non-negative.

Topics Covered

  • Definition of absolute value
  • Absolute value of positive and negative numbers
  • Using absolute value in expressions and equations
  • Practical examples and real-life applications

Lesson Examples

Example 1: |5| = 5

Example 2: |-7| = 7

Example 3: |0| = 0

Example 4: Simplify: |3 - 8| = |-5| = 5

Practice Problems

  1. Find the absolute value: |12|, |-15|, |0|
  2. Simplify: |7 - 10|, |-9 + 4|, |5 - 5|
  3. Explain why absolute value is always non-negative in your own words.
  4. Give a real-world example where absolute value is used (distance, temperature, etc.).
