Lesson Description

This lesson explains how to add and subtract positive and negative numbers using rules, examples, and number line visualization.

Learning Objectives

  • Apply the rules for adding signed numbers.
  • Apply the rules for subtracting signed numbers.
  • Use a number line to visualize signed number operations.

Topics Covered

  • Adding positive and negative numbers
  • Subtracting positive and negative numbers
  • Number line visualization
  • Common mistakes with signed numbers

Key Terms

  • Signed Number: A number with a positive or negative sign.
  • Addition Rule: Same signs add normally, different signs subtract.
  • Subtraction Rule: Change subtraction to addition of the opposite number.

Lesson Examples

Example 1: Adding Signed Numbers

  1. Calculate 5 + (-3).
  2. Subtract smaller absolute value from larger: 5 - 3 = 2.
  3. Use the sign of the number with larger absolute value: Result = 2.

Example 2: Subtracting Signed Numbers

  1. Calculate -4 - 6.
  2. Change subtraction to addition of opposite: -4 + (-6).
  3. Add the numbers: -4 + (-6) = -10.

Practice Problems

  1. 7 + (-5) = ?
  2. -3 + 8 = ?
  3. 6 - (-2) = ?
  4. -5 - 4 = ?

Tips & Common Mistakes

  • When signs are different, subtract the smaller absolute value from the larger.
  • When subtracting, always change to addition of the opposite number.
  • Use number lines to help visualize operations with signed numbers.

Summary

Adding and subtracting signed numbers follows simple rules. Visualizing numbers on a number line and applying these rules ensures accuracy in calculations.

Challenge Problems

  1. Calculate -7 + (-12) using a number line.
  2. Compute 15 - (-9) and explain each step.
  3. Create three of your own problems adding and subtracting signed numbers.
