Lesson Description

This lesson focuses on adding and subtracting radicals. Students will learn to combine like radicals, simplify radical expressions, and understand the rules for operations involving radicals.

Learning Objectives

  • Identify like radicals and unlike radicals.
  • Learn to add and subtract radicals with the same radicand.
  • Simplify radical expressions after addition or subtraction.

Topics Covered

  • Like radicals
  • Adding radicals
  • Subtracting radicals

Key Terms

  • Like Radicals: Radicals with the same radicand and index.
  • Unlike Radicals: Radicals with different radicands or indexes that cannot be directly combined.
  • Simplify: To rewrite a radical expression in its simplest form.

Tips & Common Mistakes

  • Only combine like radicals; different radicands cannot be added directly.
  • Always simplify radicals before adding or subtracting.
  • Remember to combine coefficients, not radicands.

Summary

Radicals can be added or subtracted only when they are like radicals. Simplifying expressions first and combining coefficients ensures correct results. Unlike radicals must remain separate unless simplified to like radicals.
