Lesson Description

This lesson introduces exponents as a shorthand method for repeated multiplication. Students will learn the meaning of base and exponent, how to evaluate expressions, and how exponential notation simplifies large repeated products.

Learning Objectives

  • Identify the base and exponent in exponential expressions.
  • Rewrite repeated multiplication using exponent notation.
  • Evaluate simple exponential expressions.

Topics Covered

  • Repeated multiplication
  • Base and exponent
  • Evaluating powers

Key Terms

  • Exponent: The number that tells how many times the base is multiplied by itself.
  • Base: The number being multiplied repeatedly.
  • Power: The result of raising a base to an exponent.

Tips & Common Mistakes

  • The exponent tells how many times to multiply, not what to multiply by.
  • Do not multiply base and exponent together.
  • Always multiply step-by-step to avoid arithmetic errors.

Summary

Exponents provide a compact way to represent repeated multiplication. Understanding base and exponent is essential before learning exponent rules and advanced applications.
