Lesson Description

Radical expressions include square roots (and sometimes higher roots) that contain numbers, variables, or both. In this lesson, students learn how to simplify radical expressions, combine like radicals, and evaluate them when variable values are provided.

Learning Objectives

  • Simplify radical expressions containing numbers and variables.
  • Combine like radical terms correctly.
  • Evaluate radical expressions by substituting given values.

Topics Covered

  • Simplifying radicals with variables
  • Combining like radicals
  • Evaluating radical expressions

Key Terms

  • Radical Expression: An expression that contains a radical symbol.
  • Like Radicals: Radicals that have the same index and radicand.
  • Evaluate: To substitute a value for a variable and simplify.

Tips & Common Mistakes

  • Only combine radicals that have the same radicand.
  • Always factor out perfect squares when simplifying.
  • Remember to substitute values before taking the square root when evaluating.

Summary

Radical expressions can include numbers and variables. To simplify, factor out perfect squares and reduce to simplest form. Like radicals can be combined by adding or subtracting coefficients. When evaluating, substitute first, then simplify.
