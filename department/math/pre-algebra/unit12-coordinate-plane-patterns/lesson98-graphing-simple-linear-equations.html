Plot lines using slope and y-intercept and connect equations to their graphs.

Lesson Description

In this lesson, students learn how to graph simple linear equations written in the form y = mx + b. By identifying the slope (m) and y-intercept (b), students will accurately plot points and draw straight lines that represent linear relationships.

Learning Objectives

  • Identify slope and y-intercept in an equation.
  • Graph linear equations using slope-intercept form.
  • Create and verify tables of values for graphing.

Topics Covered

  • Slope-intercept form (y = mx + b)
  • Plotting points on the coordinate plane
  • Drawing and interpreting straight lines

Key Terms

  • Slope: The rate of change; how steep a line is.
  • y-intercept: The point where a line crosses the y-axis.
  • Slope-Intercept Form: An equation written as y = mx + b.

Lesson Examples

Example 1: Graphing Using Slope and Intercept

  1. Given equation: y = 2x + 1. Identify slope m = 2 and intercept b = 1.
  2. Plot the y-intercept at (0,1).
  3. Use slope (rise 2, run 1) to plot another point and draw the line.

Example 2: Graphing by Creating a Table

  1. Given equation: y = -x + 3.
  2. Choose x-values (0,1,2) and compute y-values (3,2,1).
  3. Plot the points and draw a straight line through them.

Practice Problems

  1. Graph the equation y = 3x + 2.
  2. Graph the equation y = -2x + 4.
  3. Find the slope and y-intercept of y = 5x - 1.
  4. Create a table of values for y = x - 3 and graph it.

Tips & Common Mistakes

  • Always plot the y-intercept first.
  • Pay attention to negative slopes (lines go downward left to right).
  • Ensure your points line up in a straight line.

Summary

Graphing simple linear equations involves identifying the slope and y-intercept, plotting points accurately, and drawing a straight line. Mastering this skill connects equations, tables, and graphs into a unified understanding of linear relationships.

Challenge Problems

  1. Graph y = 4x - 5 and describe the slope.
  2. Write an equation for a line with slope -3 and y-intercept 2, then graph it.
  3. Determine whether the line through (0,2) and (2,6) matches y = 2x + 2.
