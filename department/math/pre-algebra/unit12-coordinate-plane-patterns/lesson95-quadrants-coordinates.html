Lesson Description

This lesson explains how the coordinate plane is divided into four quadrants and how positive and negative values determine a point‚Äôs location. Students will learn to identify which quadrant a point lies in and recognize when a point is on an axis.

Learning Objectives

  • Identify and label Quadrants I, II, III, and IV.
  • Determine the quadrant of a given ordered pair.
  • Recognize points that lie on the x-axis or y-axis.

Topics Covered

  • Signs of x- and y-coordinates
  • Quadrant numbering (counterclockwise pattern)
  • Points on axes vs. points in quadrants

Key Terms

  • Quadrant: One of four sections of the coordinate plane.
  • x-coordinate: The horizontal value in an ordered pair.
  • y-coordinate: The vertical value in an ordered pair.

Lesson Examples

Example 1: Identify the Quadrant of (4, -3)

  1. The x-coordinate (4) is positive.
  2. The y-coordinate (-3) is negative.
  3. Positive x and negative y places the point in Quadrant IV.

Example 2: Identify the Location of (-5, 0)

  1. The x-coordinate is negative.
  2. The y-coordinate is 0.
  3. When y = 0, the point lies on the x-axis (not in a quadrant).

Practice Problems

  1. Which quadrant contains (3, 6)?
  2. Which quadrant contains (-4, 2)?
  3. Which quadrant contains (-7, -1)?
  4. Where is the point (0, 9) located?

Tips & Common Mistakes

  • Quadrants are numbered counterclockwise starting in the top-right.
  • If either coordinate is zero, the point is on an axis.
  • Do not confuse Quadrant II and Quadrant IV‚Äîthey differ by sign positions.

Summary

The signs of the x- and y-coordinates determine a point‚Äôs location on the coordinate plane. Understanding quadrants builds the foundation for graphing equations and analyzing linear relationships.

Challenge Problems

  1. Create an ordered pair located in Quadrant II.
  2. Is it possible for a point to be in two quadrants? Explain.
  3. Describe the sign pattern for each quadrant.
