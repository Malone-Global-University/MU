Lesson 17 — Division of Zero

🎯 Goal

Understand when division by zero is undefined and how to divide zero.

📖 Lesson Description

  • 0 ÷ any nonzero number = 0
  • Any number ÷ 0 = undefined

🔎 Examples

  • 0 ÷ 5 = 0
  • 7 ÷ 0 = undefined

✏️ Practice

  1. 0 ÷ 3 = ?
  2. 10 ÷ 0 = ?
  3. 0 ÷ (-7) = ?
