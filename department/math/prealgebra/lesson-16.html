Lesson 16 — Equivalent Fractions and Reducing

🎯 Goal

Identify equivalent fractions and simplify fractions.

📖 Lesson Description

Fractions that represent the same value are equivalent fractions. Example: 2/3 = 4/6 = 8/12.

🔎 Examples

  • 1/2 = 2/4 = 4/8
  • 3/5 = 6/10 = 9/15

✏️ Practice

  1. Find two fractions equivalent to 3/4
  2. Simplify 12/20
  3. Simplify 18/24
