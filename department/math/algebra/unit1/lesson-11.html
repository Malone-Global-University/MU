Lesson 11 — Combining Functions

Combining functions allows us to generate new functions through operations on two or more existing functions. Key operations: - Addition: (f+g)(x) = f(x) + g(x) - Subtraction: (f−g)(x) = f(x) − g(x) - Multiplication: (f⋅g)(x) = f(x) ⋅ g(x) - Division: (f/g)(x) = f(x)/g(x), g(x) ≠ 0 Each operation produces a new graph, and the properties of the original functions influence the resulting shape.

Overview

Why it matters

- Models real-world scenarios: total cost, profit, motion - Prepares students for composite functions and function inverses - Strengthens understanding of function operations and their graphical interpretation

Worked examples

Addition: f(x) = x², g(x) = 2x → (f+g)(x) = x² + 2x Graph: parabola shifted by linear term, vertex at x = −1

Subtraction: f(x) = x², g(x) = 3x−1 → (f−g)(x) = x² − 3x + 1 Graph: parabola shifted and stretched relative to f(x)

Multiplication: f(x) = x, g(x) = x−2 → (f⋅g)(x) = x(x−2) = x² − 2x Graph: parabola with zeros at x = 0 and x = 2

Division: f(x) = x²−1, g(x) = x−1 → (f/g)(x) = (x²−1)/(x−1) = x + 1, x ≠ 1 Graph: line with a hole at x = 1

Teaching tips

  • Remind students of domain restrictions, especially for division
  • Use graphing calculators or software for visualization
  • Emphasize step-by-step combination: evaluate function values, then perform operation
  • Clarify the difference between combining functions and composing functions (next lesson)
