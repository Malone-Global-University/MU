Lesson 19: Polynomial Equations – Solving by Factoring and Substitution

Polynomial equations set a polynomial equal to zero. Solving involves factoring, applying the Zero-Product Property, and using substitution to reduce higher-degree polynomials to quadratic form.

Overview

A polynomial equation is an equation in which a polynomial is set equal to zero: \(P(x) = 0\) Key strategies: - Factor the polynomial - Apply the Zero-Product Property: if \(A \cdot B = 0\), then \(A=0\) or \(B=0\) - Use substitution to simplify higher-degree polynomials

  • Factor out GCF first
  • Set each factor equal to zero
  • Use substitution \(y=x^2\) or similar to reduce degree
  • Back-substitute to solve for original variable

Why it matters

- Finding solutions (roots/zeros) is essential for graphing and function analysis - Roots are used in calculus, physics, and engineering - Builds foundation for rational expressions, inequalities, and advanced algebra

Worked examples

**Quadratic Polynomial:** \(x^2 + 5x + 6 = 0\) Factor: \((x+2)(x+3) = 0\) Solutions: \(x = -2, -3\)

**Cubic Polynomial with GCF:** \(x^3 - 3x^2 - 4x = 0\) Factor GCF: \(x(x^2 - 3x - 4) = 0\) Factor quadratic: \(x(x-4)(x+1)=0\) Solutions: \(x=0,4,-1\)

**Polynomial using Substitution:** \(x^4 - 5x^2 +4 = 0\) Substitute \(y=x^2\): \(y^2 -5y +4=0\) Factor: \((y-4)(y-1)=0\) ⇒ \(y=4,1\) Back-substitute \(x^2=y\): \(x^2=4 ⇒ x=±2\), \(x^2=1 ⇒ x=±1\)

Teaching tips

  • Always factor out GCF first
  • Teach the Zero-Product Property clearly
  • Use substitution for higher-degree polynomials
  • Remind students to include both positive and negative roots for squares
