Lesson 36: Curve Sketching & Critical Points

Students will learn to: - Identify critical points using \(f'(x) = 0\) or \(f'(x)\) undefined - Determine intervals of increase/decrease - Analyze concavity and inflection points with \(f''(x)\) - Apply the First and Second Derivative Tests for maxima and minima - Sketch functions systematically using derivative information

Overview

Why it matters

- Provides a structured method for graphing functions without plotting every point - Essential in optimization problems in physics, economics, and engineering - Bridges algebra, calculus, and real-world problem solving

Worked examples

**Step 1: Find Critical Points** \[ f(x) = x^3 - 6x^2 + 9x + 1 \] First derivative: \[ f'(x) = 3x^2 - 12x + 9 \] Set \(f'(x) = 0\): \[ 3x^2 - 12x + 9 = 0 \Rightarrow x^2 - 4x + 3 = 0 \Rightarrow (x-1)(x-3) = 0 \] Critical points: \(x = 1, 3\)

**Step 2: Determine Intervals of Increase/Decrease** - Test intervals: \((-\infty,1), (1,3), (3,\infty)\) - Pick test points in \(f'(x)\): \[ f'(0) = 3(0)^2 -12(0)+9 = 9 > 0 \Rightarrow f \text{ increasing on }(-\infty,1) \] \[ f'(2) = 3(4) -12(2) + 9 = 3 > 0 \Rightarrow f \text{ increasing on }(1,3) \] \[ f'(4) = 3(16)-12(4)+9 = 3 > 0 \Rightarrow f \text{ increasing on }(3,\infty) \] *(Notice: In this example, function is actually always increasing — teaching point to check carefully.)*

**Step 3: Concavity and Inflection Points** \[ f''(x) = 6x - 12 \] Set \(f''(x) = 0 \Rightarrow 6x - 12 = 0 \Rightarrow x = 2\) - Concave up: \(f''(x) > 0 \Rightarrow x > 2\) - Concave down: \(f''(x) < 0 \Rightarrow x < 2\) - Inflection point at \(x = 2\), \(f(2) = 8 - 24 + 18 +1 = 3\)

**Step 4: Sketch Function** - Mark critical points (max/min), inflection points - Use intervals of increase/decrease and concavity to guide shape - Plot key points for accuracy
