Lesson 37: Optimization Problems

Students will learn to: - Identify optimization problems from word problems - Express the quantity to be optimized as a function of a single variable - Use the first derivative to find critical points - Apply the second derivative test or endpoint analysis to classify maxima and minima - Interpret solutions in context

Overview

Why it matters

- Optimization is essential in physics, economics, engineering, and business - Connects algebra, calculus, and problem-solving skills - Reinforces the connection between derivatives, critical points, and function behavior

Worked examples

**Example 1: Max Area of a Rectangle** A farmer has 100 meters of fencing and wants to create a rectangular pen. Find dimensions for maximum area.

1. Let length = \(x\), width = \(y\). Constraint: \(2x + 2y = 100 \Rightarrow y = 50 - x\) 2. Area: \(A(x) = x \cdot y = x(50-x) = 50x - x^2\) 3. Derivative: \(A'(x) = 50 - 2x\) 4. Set \(A'(x) = 0 \Rightarrow 50 - 2x = 0 \Rightarrow x = 25\) 5. Second derivative: \(A''(x) = -2 < 0 \Rightarrow \text{maximum}\) 6. Dimensions: \(x = 25\), \(y = 50 - 25 = 25\)

**Example 2: Minimum Cost** A company wants to minimize the cost \(C(x) = 5x^2 - 40x + 150\).

1. Derivative: \(C'(x) = 10x - 40\) 2. Set \(C'(x) = 0 \Rightarrow 10x - 40 = 0 \Rightarrow x = 4\) 3. Second derivative: \(C''(x) = 10 > 0 \Rightarrow \text{minimum cost}\) 4. Minimum cost: \(C(4) = 5(16) - 40(4) + 150 = 30\)

**Example 3: Open-Top Box** A box with square base and open top must have volume \(V = 32 \text{ m}^3\). Find dimensions minimizing surface area.

1. Let base side = \(x\), height = \(h\). Volume: \(x^2h = 32 \Rightarrow h = \frac{32}{x^2}\) 2. Surface area: \(S(x) = x^2 + 4xh = x^2 + 4x \cdot \frac{32}{x^2} = x^2 + \frac{128}{x}\) 3. Derivative: \(S'(x) = 2x - \frac{128}{x^2}\) 4. Set \(S'(x) = 0 \Rightarrow 2x - \frac{128}{x^2} = 0 \Rightarrow 2x^3 - 128 = 0 \Rightarrow x^3 = 64 \Rightarrow x = 4\) 5. Height: \(h = 32 / 16 = 2\)
