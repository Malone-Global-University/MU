Lesson 23: Graphs of Exponentials

Graphing an exponential function \[ f(x) = a \cdot b^x \] shows how the function grows or decays over the real numbers. Key features: - Y-intercept: \(f(0) = a\) - Horizontal asymptote: usually \(y = 0\), unless vertically shifted - Increasing vs. decreasing: - \(b > 1 \Rightarrow\) increasing (growth) - \(0 < b < 1 \Rightarrow\) decreasing (decay) - Domain: \((-\infty, \infty)\) - Range: \((0, \infty)\) for standard form Graphing involves plotting key points and drawing a smooth curve approaching the asymptote.

Overview

Why it matters

- Visualizes exponential trends for intuition and prediction - Essential for finance, population, physics, and other applications - Reinforces link between algebraic formula and geometric representation

Worked examples

**Basic Growth:** \[ f(x) = 2^x \] Points: \(f(-2) = 1/4\), \(f(-1) = 1/2\), \(f(0) = 1\), \(f(1) = 2\), \(f(2) = 4\) Horizontal asymptote: \(y=0\) Curve rises as \(x \to \infty\), approaches 0 as \(x \to -\infty\)

**Decay Example:** \[ g(x) = (1/2)^x \] Points: \(g(-2) = 4\), \(g(-1) = 2\), \(g(0) = 1\), \(g(1) = 1/2\), \(g(2) = 1/4\) Horizontal asymptote: \(y=0\) Curve falls as \(x \to \infty\), rises as \(x \to -\infty\)

**Shifted Function:** \[ h(x) = 3 \cdot 2^x + 1 \] - Vertical shift moves asymptote to \(y=1\) - Y-intercept: \(h(0) = 3 \cdot 1 + 1 = 4\) - Curve grows rapidly above \(y=1\)

Practice problems

  1. Graph \(f(x) = 5^x\) for \(x = -2,-1,0,1,2\)
  2. Identify horizontal asymptote for \(g(x) = 4 \cdot (1/3)^x + 2\)
  3. Plot \(h(x) = 2 \cdot 3^x - 1\) and label y-intercept
  4. Determine if \(f(x) = (1/4)^x\) is increasing or decreasing
  5. Graph \(k(x) = 0.5^x + 3\) and describe end behavior

Teaching tips

  • Identify y-intercept and asymptote first
  • Use visual steepness to illustrate growth/decay rate
  • Students often confuse x and y roles in tables
  • Remind negative exponents: \(b^{-x} = 1/b^x\)
