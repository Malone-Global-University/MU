Lesson 26: Change of Base

The change of base formula allows computation of logarithms in any base using a more convenient base: \[ \log_b(M) = \frac{\log_k(M)}{\log_k(b)} \] Where: - \(b\) = original base - \(M\) = argument - \(k\) = new base (commonly 10 or \(e\)) Useful when: - Calculators provide only \(\log_{10}\) or \(\ln\) - Solving equations with unusual bases

Overview

Why it matters

- Enables practical computation of logarithms without specialized tables - Essential in science, finance, and computer science for arbitrary-base logarithms - Connects all logarithms to a single universal base, reinforcing algebraic understanding of exponential relationships

Worked examples

**Using base 10:** Compute \(\log_2(50)\) \[ \log_2(50) = \frac{\log_{10}(50)}{\log_{10}(2)} = \frac{1.69897}{0.30103} \approx 5.644 \]

**Using natural logarithm (\( \ln \))**: Compute \(\log_5(100)\) \[ \log_5(100) = \frac{\ln(100)}{\ln(5)} = \frac{4.60517}{1.60944} \approx 2.861 \]

**Solving an equation:** Solve \(2^x = 15\) \[ x = \log_2(15) = \frac{\ln(15)}{\ln(2)} \approx \frac{2.70805}{0.69315} \approx 3.906 \]

Practice problems

  2. Compute \(\log_7(50)\) using \(\ln\)
  3. Solve \(5^x = 200\) using change of base
  4. Solve \(10^{2x} = 500\)
  5. Express \(\log_8(32)\) using base 2

Teaching tips

  • Ensure numerator and denominator are not confused in the formula
  • Any base can be used, but 10 or \(e\) is preferred for calculators
  • Emphasize the connection to solving exponential equations
  • Show both decimal approximations and exact fraction forms
