Lesson 7 — Systems of Linear Equations

A system of linear equations consists of two or more linear equations with the same variables. Solving the system means finding the values of the variables that satisfy all equations simultaneously.

Overview

Systems of equations model real-world relationships: supply and demand, motion problems, chemistry mixtures, and financial planning. Mastering them builds a foundation for linear algebra, optimization, and multivariable calculus.

Why it matters

Solving systems allows students to analyze multiple constraints simultaneously, preparing for applications in economics, physics, engineering, and advanced mathematics.

Worked examples

Substitution method: { x + y = 7 x − y = 3 } Solve for x in the second equation: x = y + 3 Substitute into the first: (y+3)+y = 7 ⇒ 2y+3=7 ⇒ 2y=4 ⇒ y=2 Then x = 2 + 3 = 5 Solution: x=5, y=2

Elimination method: { 3x + 2y = 16 5x − 2y = 14 } Add equations to eliminate y: 3x+2y + 5x−2y = 16+14 ⇒ 8x=30 ⇒ x=3.75 Substitute back: 3(3.75)+2y=16 ⇒ 11.25+2y=16 ⇒ 2y=4.75 ⇒ y=2.375 Solution: x=3.75, y=2.375

Matrix method (2×2 example using inverse): A = [[2, 1], [3, -4]], x = [x, y], b = [8, -2] Equation: Ax = b Compute A⁻¹ and multiply: x = A⁻¹b Solution: x=2, y=3/4

Practice problems

Answer key (brief): 1 → x=5, y=−1 2 → x=2, y=−1 3 → x=1, y=2, z=0 4 → 100 ml of 10% solution, 100 ml of 30% solution

Teaching tips

  • Always check solutions by substituting back into all original equations
  • Substitution is best for simple systems or when one variable is easily isolated
  • Elimination works well when coefficients can easily cancel
  • Matrices are more systematic for larger systems
  • Watch for no solution (parallel lines) and infinite solutions (coincident lines)
