Lesson 61: Exponential Functions and Graphs

An exponential function is a function in which the variable appears in the exponent: f(x) = a¬∑b^x, where a ‚â† 0 and b > 0, b ‚â† 1. Exponential functions model growth and decay and have distinctive graph shapes.

Key Points About Exponential Graphs

  • If b > 1 ‚Üí Exponential growth, graph increases as x increases.
  • If 0 < b < 1 ‚Üí Exponential decay, graph decreases as x increases.
  • The y-intercept is always at (0, a).
  • The horizontal asymptote is y = 0 unless a transformation shifts it.
  • Graphs never cross the x-axis; y is always positive for standard exponential functions.
