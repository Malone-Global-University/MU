MGU Jazz Style Musician Pages
==============================

This bundle contains complementary style-based jazz musician directory pages.

Folder structure:
- blues-jazz/index.html
- gospel-influenced-jazz/index.html
- ragtime/index.html
- swing/index.html
- bebop/index.html
- hard-bop/index.html
- soul-jazz/index.html
- free-jazz/index.html
- fusion/index.html
- vocal-jazz/index.html

Placement suggestion:
music/jazz/musicians/[style-folder]/index.html

Biography link assumption:
Each musician link points one level up, such as ../charlie-parker/.
That assumes biography folders sit beside the style folders under the same musicians directory.
