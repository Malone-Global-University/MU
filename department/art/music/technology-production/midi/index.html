MIDI

Learn how MIDI represents musical performance data for sequencing, controllers, virtual instruments, drum programming, and electronic music production.

Overview

MIDI is not audio. It is a system for communicating musical performance information such as note, pitch, velocity, duration, controller movement, tempo, and program changes.

MIDI Data

MIDI data can describe notes, timing, velocity, sustain, modulation, pitch bend, controller movements, and instrument changes.

Controllers

MIDI controllers include keyboards, pads, drum controllers, wind controllers, control surfaces, and other devices that send performance data.

Sequencing

Sequencing arranges MIDI events across time, allowing producers to program drums, bass lines, chords, melodies, automation, and virtual instrument parts.
