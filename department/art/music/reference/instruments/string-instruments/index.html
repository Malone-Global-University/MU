String Instruments

Study string instruments through vibration, bowing, plucking, fingering, resonance, tone color, range, and ensemble roles.

Overview

String instruments produce sound through vibrating strings. They may be bowed, plucked, struck, amplified, fretted, fretless, acoustic, or electronic.

This page belongs to the Instrument Guide area of the Malone Global University Music Department. Instrument pages should connect sound production, performance technique, musical role, ensemble use, history, technology, and listening practice.

Sound Production

String sound depends on string vibration, instrument body, bow or plucking technique, finger placement, resonance, and amplification when used.

Common String Instruments

Common string instruments include violin, viola, cello, double bass, harp, guitar, bass guitar, mandolin, banjo, ukulele, and many global string instruments.

Ensemble Roles

Strings appear in orchestras, chamber music, folk traditions, jazz, blues, country, rock, pop, film music, worship, and studio recording.
