Branch Overview

The Digital Literacy branch provides foundational practical computing instruction for learners who need confidence using computers, operating systems, files, browsers, accounts, software, and everyday digital tools. This branch is designed for beginners, hobbyists, office workers, and students preparing for more technical study by building the baseline skills required to navigate, organize, troubleshoot, and use modern computing environments effectively.

Digital Literacy Courses

  • PC Fundamentals

    An introductory course covering the basic parts, functions, and everyday use of personal computers. Students learn about hardware, operating systems, user accounts, storage, desktop environments, settings, and safe routine computer operation.

  • File Management

    Practical instruction in organizing, naming, moving, copying, deleting, searching, and backing up files and folders. Students learn how directory structures, file extensions, storage locations, and organization habits affect everyday computer use.

  • Internet and Web Literacy

    Foundational training in using browsers, search engines, websites, online accounts, downloads, links, tabs, bookmarks, and basic web safety. Students learn how to navigate the internet effectively while recognizing common risks and reliability issues.

  • Troubleshooting

    Beginner-friendly problem-solving instruction for common computer issues. Students learn how to identify symptoms, check settings, restart services or devices, research errors, test simple fixes, and know when a problem requires higher-level support.

  • Digital Productivity

    Practical instruction in using digital tools to complete work efficiently. Topics include basic typing workflow, window management, screenshots, file sharing, cloud storage, calendars, notes, task lists, and personal organization systems.
