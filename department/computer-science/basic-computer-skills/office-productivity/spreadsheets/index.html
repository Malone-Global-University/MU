Course Overview

Spreadsheets teaches students how to organize, calculate, analyze, and present structured information using spreadsheet software. Learners develop practical competency in workbook setup, cell formatting, formulas, functions, tables, charts, sorting, filtering, and spreadsheet-based decision support for personal, academic, administrative, and professional work.

Spreadsheets Topics

  • Spreadsheet Interface Basics

    Introduction to rows, columns, cells, worksheets, workbooks, ribbons, menus, toolbars, and the basic navigation patterns used in spreadsheet applications.

  • Data Entry and Cell Formatting

    Practical instruction in entering, editing, formatting, aligning, and organizing spreadsheet data so information is clear, consistent, and readable.

  • Formulas and Basic Calculations

    Training in writing formulas, using operators, referencing cells, copying calculations, and performing basic arithmetic within spreadsheet environments.

  • Functions and Common Workflows

    Instruction in common spreadsheet functions such as SUM, AVERAGE, MIN, MAX, COUNT, IF, and basic lookup or summary workflows.

  • Tables, Sorting, and Filtering

    Practical methods for converting data into tables, sorting records, filtering lists, organizing categories, and managing structured datasets.

  • Charts and Data Visualization

    Instruction in creating basic charts and visual summaries that communicate patterns, comparisons, trends, and categories clearly.

  • Spreadsheet Quality Control

    Best practices for checking formulas, avoiding common errors, labeling data, protecting important cells, documenting assumptions, and maintaining reliable workbooks.
