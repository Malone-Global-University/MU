Overview

The Network Administration Department at Malone University focuses on the design, deployment, and management of modern digital infrastructure. Students gain hands-on experience with routers, switches, firewalls, servers, and enterprise network environments.

Degree Programs

📘 Associate of Science — Networking & Infrastructure (ASNI)

A two-year degree covering wired and wireless networks, basic routing and switching, server administration, and IT support fundamentals.

📗 Bachelor of Science — Network Administration (BSNA)

A complete four-year program that includes enterprise architecture, virtualization, cloud systems, network security, automation, and infrastructure engineering.

Course Catalog

  • NET 101 — Introduction to Networking: OSI model, TCP/IP, network types, cabling.
  • NET 120 — Routing & Switching I: VLANs, subnets, static routing, switch configuration.
  • NET 150 — Windows Server Administration: AD, DNS, DHCP, Group Policy.
  • NET 160 — Linux Server Fundamentals: CLI, services, permissions, SSH.
  • NET 201 — Routing & Switching II: OSPF, EIGRP, trunking, spanning tree, DHCP relay.
  • NET 230 — Wireless Network Design: RF theory, access points, enterprise Wi-Fi, security modes.
  • NET 250 — Virtualization & Cloud Systems: Hypervisors, VM management, AWS/Azure basics.
  • NET 310 — Network Automation: Scripting, Python, Ansible, automated configuration.
  • NET 401 — Enterprise Infrastructure Management: Large-scale networks, monitoring, SLAs.

Career Paths

Graduates are prepared for roles such as:

  • IT Infrastructure Engineer
