Overview

The Tools page provides an overview of the technical toolchain used across the IT department at Malone Global University. These tools equip students with practical experience in modern development, collaboration, infrastructure, and analytics workflows.

<

MGU Built Utilities

  • MGU Code Editor

    A lightweight browser-based writing and development utility designed for drafting, editing, testing, and organizing code directly inside the MGU environment. Useful for quick HTML, CSS, JavaScript, Markdown, and text editing workflows without relying on external software.

  • HTML Color Reference Table

    An interactive visual reference utility for exploring HTML and CSS colors through swatches, hexadecimal values, RGB values, and live preview tools. Designed to help developers quickly compare palettes, test combinations, and identify exact color codes during interface and theme design.

Development & Programming Tools

  • Integrated Development Environments (IDEs): Visual Studio Code, IntelliJ IDEA, PyCharm
  • Text Editors: Sublime Text, Atom, Vim
  • Version Control Systems: Git, GitHub, GitLab
  • Code Collaboration: GitHub Pull Requests, Merge Reviews, Branching Strategies

Cloud & Deployment Tools

  • Cloud Platforms: AWS, Azure, Google Cloud Platform
  • Containerization & Orchestration: Docker, Kubernetes
  • Continuous Integration / Continuous Deployment (CI/CD): Jenkins, GitHub Actions, GitLab CI
  • Server Management: SSH, Ansible, Terraform

Analytics & Data Tools

  • Data Analysis: Python (Pandas, NumPy), R
  • Data Visualization: Tableau, Power BI, Matplotlib, Seaborn
  • Databases: MySQL, PostgreSQL, MongoDB

Collaboration & Productivity Tools
