Course Overview

Documents and Formatting teaches students how to create, structure, format, and refine professional written documents using modern word processing software. Learners develop practical competency in document layout, typography, spacing, structure, visual hierarchy, and formatting standards used in academic, administrative, and business communication.

Documents and Formatting Topics

  • Word Processing Interface Basics

    Introduction to ribbons, menus, formatting panels, document views, editing tools, and the core interface components of modern word processors.

  • Text Formatting and Typography

    Practical instruction in font selection, sizing, emphasis, alignment, spacing, indentation, and typographic best practices for readable documents.

  • Document Structure and Layout

    Instruction in organizing content using headings, sections, margins, page breaks, columns, and layout structures for professional presentation.

  • Lists, Tables, and Embedded Content

    Practical methods for inserting and formatting lists, tables, images, charts, and other embedded visual elements within structured documents.

  • Templates and Style Systems

    Training in reusable formatting systems, style sets, templates, themes, and document standardization workflows for consistent output.

  • Proofreading and Document Review

    Instruction in reviewing, editing, proofreading, commenting, track changes, and collaborative document revision practices.

  • Professional Document Standards

    Best practices for creating polished reports, letters, proposals, memoranda, and formal documentation suitable for academic and workplace use.
