Program Overview

The Software Engineering & Fullstack Development program prepares students for professional application development across web, mobile, and enterprise systems. The curriculum emphasizes both front-end and back-end development, system architecture, API integration, and DevOps practices for scalable and maintainable software solutions.

Students learn modern frameworks, databases, continuous integration/deployment pipelines, and agile project management methodologies. Emphasis is placed on real-world problem-solving, team collaboration, and production-quality code delivery.

Degree Pathways

  • Bachelor of Science in Software Engineering & Fullstack Development (BSSE)
    A comprehensive four-year program covering front-end frameworks, back-end languages, cloud services, API design, and full lifecycle software engineering.
  • Professional Certificate in Fullstack Development
    Focused training on front-end and back-end skills, modern frameworks, and deployment practices.
