PC Literacy for Smartphone Users

Home Directory üìò Course Title:
  • Duration: 4 Weeks
  • Format: 2 classes per week (90 minutes each)
  • Goal: Transition learners from app-users to system-users

Course Objective

By the end of 4 weeks, students will be able to:

  • Navigate Windows or macOS confidently
  • Manage files and folders
  • Use Microsoft Word or Google Docs
  • Send professional emails with attachments
  • Download, install, and manage basic software
  • Understand basic troubleshooting
  • Apply for jobs online using a PC

Class 1: Introduction to the Desktop Environment

Objectives:

  • Understand desktop vs mobile interface

Topics:

  • What is an operating system?
  • Desktop, taskbar, start menu
