Program Overview

The Data Science & Artificial Intelligence program is engineered for learners pursuing careers in quantitative analytics, applied machine learning, neural networks, and large-scale data systems. Students develop fluency across the entire data lifecycle—from acquisition and cleaning to modeling, deployment, and automation. This track emphasizes rigorous mathematical foundations paired with real-world engineering practices.

What You Will Learn

  • Python for Data Science — NumPy, pandas, Jupyter, workflow design
  • Statistical Modeling — regression, distributions, inference, experimental design
  • Machine Learning — supervised, unsupervised, ensemble learning
  • Deep Learning — neural networks, CNNs, RNNs, transformers
  • Data Engineering — ETL pipelines, databases, large-scale batch & streaming systems
  • AI Deployment — model ops, containers, API integration, monitoring
  • Visualization & BI — dashboards, exploratory analysis, communication of insights
  • Ethics of AI — fairness, bias, interpretability, responsible development

Course Catalog

  • DSA 110 — Introduction to Data Science

    Covers data literacy, Python foundations, exploratory analysis, and data storytelling.

  • DSA 205 — Applied Statistics

    Probability models, regression, distributions, hypothesis testing, and real-world inference.

  • DSA 250 — Machine Learning I

    Supervised and unsupervised algorithms, model selection, tuning, and evaluation metrics.

  • DSA 315 — Data Engineering Fundamentals

    Covers SQL, ETL pipeline design, warehousing, and scalable storage architectures.

  • DSA 330 — Deep Learning

    Neural network architectures, image models, sequence models, and optimization techniques.

  • DSA 360 — AI Systems & Deployment

    Building and deploying production-grade ML systems, APIs, monitoring, and MLOps.

  • DSA 401 — Big Data Analytics

    Distributed computing, Spark, and large-scale data processing workflows.

  • DSA 499 — Capstone: Intelligent Systems Engineering

    Students build and deploy a complete AI solution tackling an industry-scale problem.
