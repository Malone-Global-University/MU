Course Overview

Active Directory teaches students how enterprise Windows domains manage identity, authentication, policy, and centralized administrative control across organizational networks. Learners develop practical competency in directory services architecture, domain structures, user and group management, Group Policy, organizational units, and domain administration workflows.

Active Directory Topics

  • What Is Active Directory?

    Introduction to Active Directory as Microsoft’s centralized directory and identity management platform.

  • Domains, Forests, and Organizational Units

    Instruction in the structural hierarchy and architecture of Active Directory environments.

  • Users, Groups, and Computer Objects

    Practical instruction in managing directory objects, identities, and logical administrative groupings.

  • Group Policy Management

    Training in creating, applying, and troubleshooting Group Policy Objects for centralized configuration management.

  • Authentication and Domain Services

    Guidance on how Active Directory handles authentication, trust relationships, and domain service operations.

  • Domain Controller Operations

    Instruction in the role, configuration, replication, and maintenance of domain controllers.

  • Active Directory Best Practices

    Best practices for secure, scalable, and maintainable Active Directory administration.
