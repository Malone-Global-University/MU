Linux Administration

Course Overview

Linux Administration teaches students how to manage, maintain, secure, and troubleshoot Linux-based systems in workstation and server environments. Learners develop practical competency in Linux system management, package administration, user permissions, services, networking, security, and operational maintenance workflows.

Linux Administration Topics

  • Linux Administration Overview

    Introduction to Linux systems administration responsibilities, environments, and core operational tooling.

  • Users, Groups, and Permissions

    Instruction in Linux user management, permission systems, ownership models, and access control structures.

  • Package Management and Software Installation

    Practical guidance on installing, updating, removing, and maintaining software through Linux package managers.

  • Services and Daemons

    Training in managing background services, startup processes, and daemon lifecycle administration.

  • Linux Networking Basics

    Instruction in interface configuration, hostname management, network diagnostics, and Linux network operations.

  • Linux Security and Hardening

    Guidance on securing Linux systems through permissions, patching, firewalling, and hardening measures.

  • Linux Administration Best Practices

    Best practices for reliable, scalable, and maintainable Linux systems administration.
