Branch Overview

The Systems Administration branch provides operational training in managing computing environments, operating systems, user accounts, permissions, services, infrastructure components, and enterprise support systems. Students learn how to configure, maintain, secure, troubleshoot, and administer the platforms and environments that underpin modern organizational computing operations.

Systems Administration Courses

  • Windows Administration

    Instruction in administering Microsoft Windows environments, including operating system management, user accounts, permissions, services, updates, troubleshooting, and administrative tooling.

  • Linux Administration

    Practical instruction in Linux system management, including shell administration, package management, permissions, services, networking, and server operations.

  • Active Directory

    Training in Microsoft Active Directory environments, including domain services, user/group management, organizational units, group policy, authentication, and enterprise identity administration.

  • Virtualization

    Instruction in virtual machine environments, hypervisors, resource allocation, virtual networking, snapshots, and infrastructure virtualization strategy.

  • Backup and Recovery

    Practical instruction in backup strategy, redundancy, restoration, disaster recovery planning, continuity operations, and data protection practices.
