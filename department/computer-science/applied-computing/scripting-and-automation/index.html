Branch Overview

The Scripting and Automation branch teaches students how to reduce manual work, automate repetitive tasks, manage systems programmatically, and build repeatable digital workflows through scripting languages and command-line tooling. Students learn how modern technical professionals leverage automation to improve speed, reliability, scalability, and operational efficiency across computing environments.

Scripting and Automation Courses

  • Command Line Fundamentals

    Foundational instruction in terminal and shell environments, including command syntax, navigation, file operations, pipes, redirection, and environment variables.

  • PowerShell

    Practical automation and administration using Microsoft PowerShell for Windows systems, scripting, pipeline processing, object manipulation, and task automation.

  • Bash

    Instruction in Bash shell scripting and Linux/Unix command-line automation, including shell environments, script logic, administration tasks, and process automation.

  • Python Automation

    Applied automation development using Python for scripting, file manipulation, data processing, web interaction, and automated workflow construction.

  • Task Automation

    Higher-level workflow and process automation focused on designing repeatable automation systems, scheduled jobs, integrated workflows, and scalable automation strategies.
