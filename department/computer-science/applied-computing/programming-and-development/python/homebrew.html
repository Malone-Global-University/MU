Python Homebrew Overview

This section collects small Python scripts, utilities, and experimental tools created for practical Malone Global University workflows. These scripts are shared as learning examples, working drafts, and reusable helpers for static-site organization, file management, and applied computing practice.

Link Maker

Link Maker is a small Python utility that scans a selected folder and creates an HTML file containing organized links to the files it finds. It can include subfolders and group output by folder, making it useful for static-site indexes, image collections, file inventories, and resource pages.
