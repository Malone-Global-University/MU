Programming and Development

Branch Overview

The Programming and Development branch provides applied instruction in writing code, building software, and developing technical solutions through modern programming practices. Students learn how to design, implement, debug, organize, and maintain software systems while developing practical fluency in languages, development workflows, and real-world engineering habits.

Programming and Development Courses

  • HTML

    Instruction in the structure and semantics of web documents using HyperText Markup Language. Students learn how to create properly structured pages, organize content, and build foundational website layouts.

  • CSS

    Practical instruction in styling and layout design for web interfaces. Topics include selectors, properties, responsive design, positioning, spacing, visual hierarchy, and interface presentation.

  • JavaScript

    Applied programming instruction in client-side and general-purpose scripting using JavaScript. Students learn variables, logic, DOM manipulation, events, functions, and interactive application behavior.

  • Python

    General-purpose programming instruction using Python for scripting, automation, data handling, and software development. Students learn syntax, control flow, functions, data structures, and practical project implementation.

  • SQL

    Instruction in querying, manipulating, and managing relational databases through Structured Query Language. Students learn how to retrieve, filter, join, update, and organize structured data.

  • Fullstack Development

    Integrated application development spanning front-end interfaces, back-end logic, databases, APIs, and deployment workflows. Students learn how complete modern web applications are architected and built.

  • Software Projects

    Applied project-based software development emphasizing implementation, debugging, documentation, architecture, and portfolio-quality deliverables through practical coding assignments and build work.
