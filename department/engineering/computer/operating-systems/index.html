Operating Systems

Operating Systems studies the system software layer that manages hardware resources and provides foundational services for applications. Students learn how operating systems schedule processes, manage memory, control devices, and coordinate system execution.

  • Process Management

    Study how operating systems create, schedule, synchronize, and terminate running processes and threads.

  • Memory Management

    Examine allocation, paging, segmentation, virtual memory, and system memory control mechanisms.

  • File Systems

    Learn how operating systems organize, store, retrieve, and manage persistent data on storage devices.

  • Device and I/O Management

    Explore drivers, interrupts, buffering, hardware communication, and peripheral resource control.
