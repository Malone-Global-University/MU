Computer Architecture

Computer Architecture studies the internal structure and design of computational hardware. Students examine processors, memory hierarchies, instruction execution, hardware pipelines, and the physical organization of modern computing systems.

  • Processor Design

    Study CPU structure, execution units, arithmetic logic, control units, and processor organization.

  • Memory Systems

    Examine cache, RAM, storage hierarchy, memory access patterns, and data retrieval architecture.

  • Instruction Set Architecture

    Learn how processor instructions are structured, decoded, and executed within hardware systems.

  • Parallel Processing

    Explore multicore systems, parallel execution, concurrency hardware, and performance scaling strategies.
