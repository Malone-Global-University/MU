Computer & Software Engineering

Computer and Software Engineering studies the design of computational systems across software, hardware, embedded architecture, and networked infrastructure. Students learn how digital systems are engineered from physical hardware through abstract software layers.

  • Software Engineering

    Study software architecture, development processes, testing, and scalable application design.

  • Computer Architecture

    Examine processor design, memory systems, instruction sets, and computational hardware.

  • Operating Systems

    Learn how system software manages hardware resources, processes, memory, and scheduling.

  • Networking Engineering

    Study network infrastructure, protocols, routing, switching, and distributed connectivity.

  • Cybersecurity Engineering

    Explore defensive engineering methods used to secure digital systems and infrastructure.
