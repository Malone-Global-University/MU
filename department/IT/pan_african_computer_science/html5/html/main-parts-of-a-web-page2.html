Intro to HTML Pages

There are two main parts to a web page: The Head and The Body.

The Head

According to w3schools.com, "The HTML <head> element is a container for the following elements: <title>, <style>, <meta>, <link>, <script>, and <base>."
"The <head> tag contains metadata (document title, character set, styles, links, scripts), specific information about the web page that is not displayed to the user. Metadata provides browsers and search engines with technical information about the web page." – w3docs.com

The <head> can include the following elements:

  • <title> – defines the title of the web page (required).
  • <style> – contains CSS code.
  • <base> – defines a base URL for all relative URLs.
  • <link> – links to external resources (like stylesheets).
  • <meta> – provides metadata (charset, description, etc.).
  • <script> – includes or references JavaScript code.
  • <noscript> – defines fallback content if scripts are disabled.

The Body

The <body> tag defines the main content of a web page. Everything visible to the user (text, images, videos, links, etc.) goes inside <body>.

Syntax:

<body>
    <!-- Content goes here -->
</body>

Example:

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Sample Page</title>
</head>
<body>
  <h1>Welcome to My Website</h1>
  <p>This is a paragraph of text.</p>
  <img src="example.jpg" alt="Example Image">
  <a href="https://example.com">Visit Example</a>
</body>
</html>

Explanation:

  • <h1>: Displays a heading.
  • <p>: Displays a paragraph.
  • <img>: Embeds an image.
  • <a>: Creates a hyperlink.

Attributes of <body>:

  • Global attributes: id, class, style.
  • Event attributes:
    • onload: runs JavaScript when the page loads.
    • onunload: runs JavaScript when the page is closed.

Example with Attributes:

<body style="background-color: lightblue;" onload="alert('Page Loaded!')">
  <h1>Hello, World!</h1>
</body>

Notes:

  • There can only be one <body> tag in an HTML document.
  • The <body> is essential for rendering visible content.
