Lesson 2 — Your First Webpage

⟵ Prev • Index • Next ⟶

Learning Objectives

  • Create a valid HTML document.
  • Understand the purpose of <head> and <body>.
  • Open an HTML file locally in a browser.

Starter Template

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8" />
  <meta name="viewport" content="width=device-width, initial-scale=1.0" />
  <title>My First Page</title>
</head>
<body>
  <h1>Hello, web!</h1>
</body>
</html>

Exercise

Create hello.html and open it in your browser.
