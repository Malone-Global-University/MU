Lesson 3: Lists

Unordered Lists

Unordered lists use <ul> (unordered list) and <li> (list item) tags. Items appear with bullet points.

<ul>
  <li>Apples</li>
  <li>Bananas</li>
  <li>Cherries</li>
</ul>

Ordered Lists

Ordered lists use <ol> (ordered list) and <li>. Items appear with numbers.

<ol>
  <li>Wake up</li>
  <li>Eat breakfast</li>
  <li>Go to school</li>
</ol>

Description Lists

Description lists use <dl> (description list), <dt> (term), and <dd> (definition).

<dl>
  <dt>HTML</dt>
  <dd>The standard markup language for creating web pages.</dd>
  <dt>CSS</dt>
  <dd>A style sheet language used for describing the presentation of a document.</dd>
</dl>

Exercise

Create a webpage that contains:

  1. An unordered list of your favorite foods.
  2. An ordered list of your top 5 movies.
  3. A description list of 3 school subjects with short definitions.
