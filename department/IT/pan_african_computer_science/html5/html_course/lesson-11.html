Final Project: Build a Multi-Page Website

In this final project, you will create a complete, functional multi-page website from scratch using all the HTML skills you have learned.

Project Requirements

  1. Create at least 3 HTML pages (e.g., Home, About, Contact).
  2. Include a consistent navigation menu across all pages.
  3. Use semantic HTML structure for each page.
  4. Add images, links, and at least one table or form.
  5. Include embedded multimedia (video or audio).

Bonus Challenges

  • Make the site mobile-responsive with CSS.
  • Validate your HTML with the W3C Validator.

When finished, submit your project for review. Congratulations on completing the HTML course!
