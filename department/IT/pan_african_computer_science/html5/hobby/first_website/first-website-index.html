Building a Website – Hobbyist Series

Lesson 1: Introduction

Welcome to the Series

This course will guide you through building your first website from scratch, step by step. Whether you’re doing this as a hobby or aiming for a professional project, you’ll learn the basics of HTML, CSS, and other tools that make the web work.

Lesson 1 introduces what a website is, the tools you need, and the structure of a basic web page.

Lessons in This Series

  • Lesson 1 – Introduction to Building Websites
  • Lesson 2 – Understanding HTML

  • Lesson 4 – Adding Images and Links
  • Lesson 5 – Structuring a Multi-Page Site

How to Use This Course

Each lesson builds on the previous one. Follow the links above in order, and try the exercises at the end of each lesson to practice what you’ve learned.
