Operators

Operators perform actions on variables and values. Common types include:

  • Arithmetic (+, -, *, /)
  • Assignment (=, +=)
  • Comparison (==, ===, !=, <)
  • Logical (&&, ||, !)

let x = 10;
let y = 5;
let sum = x + y; // 15
