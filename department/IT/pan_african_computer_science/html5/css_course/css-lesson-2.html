Lesson 2: CSS Selectors

What are Selectors?

Selectors tell CSS which HTML elements to style.

Common Selectors

  • p — selects all paragraph elements.
  • .className — selects all elements with that class.
  • #idName — selects a single element with that ID.

Example

p { color: blue; }
.myClass { background: yellow; }
#main { font-size: 20px; }
