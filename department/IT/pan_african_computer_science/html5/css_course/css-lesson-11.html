Lesson 11: Final Project - Build a Responsive Website

Project Overview

Use everything you’ve learned in this course to create a multi-page, mobile-friendly website.

Requirements

  • At least 3 pages linked with navigation.
  • Consistent color scheme and typography.
  • Responsive layout using Flexbox or Grid.
  • At least one animation or transition.

Tips

Start with HTML structure, then apply external CSS for styling. Test on desktop and mobile views.
