Lesson 8: CSS Grid

What is CSS Grid?

CSS Grid is a 2D layout system for creating rows and columns in web design.

Basic Properties

  • display: grid;
  • grid-template-columns — define column sizes.
  • grid-template-rows — define row sizes.

Example

.grid-container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr;
  gap: 10px;
}
