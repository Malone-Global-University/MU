Lesson 6: Positioning and Layout

Position Property

  • static — default, follows normal flow.
  • relative — positioned relative to itself.
  • absolute — positioned relative to nearest positioned ancestor.
  • fixed — stays in place when scrolling.

Example

.box {
  position: absolute;
  top: 50px;
  left: 100px;
}
