Programming Fundamentals – Core JavaScript Concepts 1. Variables – Storage for Data Variables hold values the program needs to track. Modern Standard let – reassignable const – fixed after assignment Examples let score = 10; const appName = "Dashboard"; 2. Data Types – What Goes Inside Variables JavaScript supports several key types: String (text): "Hello" Number (integer/float): 42 Boolean (true/false): true Array (lists): [1, 2, 3] Object (structured data): { name: "Joseph", age: 30 } Null (empty) Undefined (not assigned) Example const user = { id: 101, name: "Joseph", active: true }; 3. Operators – Basic Logic and Math Arithmetic +, -, *, /, % Comparison ==, === (strict), !=, <, >, <=, >= Logical && (and), || (or), ! (not) 4. Functions – Reusable Blocks of Logic Functions encapsulate behavior. Standard Function function greet(name) { console.log("Welcome, " + name); } Arrow Function (modern style) const greet = (name) => { console.log(`Welcome, ${name}`); }; 5. Conditionals – Decision Logic Conditional execution routes based on rules. If / Else let balance = 500; if (balance > 0) { console.log("Account active"); } else { console.log("Account empty"); } Else If if (score >= 90) { grade = "A"; } else if (score >= 80) { grade = "B"; } else { grade = "C"; } 6. Loops – Repeating Tasks For Loop for (let i = 0; i < 5; i++) { console.log("Iteration", i); } While Loop let x = 0; while (x < 3) { console.log("Value:", x); x++; } Looping Arrays const fruits = ["apple", "pear", "kiwi"]; for (let item of fruits) { console.log(item); } 7.
