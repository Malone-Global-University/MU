CSS Course Overview

Excerpts from CSS 101

Lesson 1: Introduction to CSS

What is CSS?

CSS (Cascading Style Sheets) is used to style and format HTML content. It lets you control layout, colors, fonts, spacing, and even animations.

Why Use CSS?

  • Separate content from design.
  • Keep a consistent style across pages.
  • Improve maintainability of websites.

Ways to Apply CSS

  1. Inline: Style applied directly in HTML.
  2. Internal: Using a <style> block in the HTML head.
  3. External: Linking to a .css file (recommended).

Lesson 2: CSS Selectors

What are Selectors?

Selectors tell CSS which HTML elements to style.

Common Selectors

  • p ‚Äî selects all paragraph elements.
  • .className ‚Äî selects all elements with that class.
  • #idName ‚Äî selects a single element with that ID.

Example

p { color: blue; }
.myClass { background: yellow; }
#main { font-size: 20px; }

Lesson 3: Colors and Backgrounds

Text Colors

Use the color property to set text colors using names, hex codes, or RGB.

Backgrounds

Use background-color or background-image to set backgrounds.

Example

body { background-color: lightblue; }
h1 { color: navy; }

Lesson 4: Text Styling

Font Properties

Change text appearance with font-family, font-size, font-weight, and font-style.

Text Alignment

Use text-align to align text: left, right, center, or justify.

Example

h1 { font-family: Arial; text-align: center; }
p { font-size: 18px; line-height: 1.5; }

Lesson 5: The Box Model

Parts of the Box Model

  • Content: The actual text or image.
  • Padding: Space between content and border.
  • Border: Line around the element.
  • Margin: Space between elements.

Example

div {
  margin: 20px;
  padding: 10px;
  border: 2px solid black;
}

Lesson 6: Positioning and Layout

Position Property

  • static ‚Äî default, follows normal flow.
  • relative ‚Äî positioned relative to itself.
  • absolute ‚Äî positioned relative to nearest positioned ancestor.
  • fixed ‚Äî stays in place when scrolling.

Example

.box {
  position: absolute;
  top: 50px;
  left: 100px;
}

Lesson 7: Flexbox

What is Flexbox?

Flexbox is a CSS layout module designed to arrange elements in rows or columns, making alignment and spacing easier.

Common Properties

  • display: flex;
  • justify-content ‚Äî aligns items horizontally.
  • align-items ‚Äî aligns items vertically.

Example

.container {
  display: flex;
  justify-content: space-between;
  align-items: center;
}

Lesson 8: CSS Grid

What is CSS Grid?

CSS Grid is a 2D layout system for creating rows and columns in web design.

Basic Properties

  • display: grid;
  • grid-template-columns ‚Äî define column sizes.
  • grid-template-rows ‚Äî define row sizes.

Example

.grid-container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr;
  gap: 10px;
}

Lesson 9: Transitions and Animations

Transitions

Transitions allow smooth changes between property values.

Animations

Animations use keyframes to define multiple steps of change.

Example

button {
  transition: background-color 0.3s ease;
}
button:hover {
  background-color: blue;
}

Lesson 10: Responsive Design

What is Responsive Design?

Responsive design ensures web pages look good on all devices using flexible layouts and media queries.

Example

@media (max-width: 600px) {
  body { background-color: lightgray; }
}
